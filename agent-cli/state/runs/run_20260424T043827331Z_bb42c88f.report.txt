我做了这些事：

1. 用 `mycli skill-library search aliyun` 找到了服务器 skill：
   - `aliyun-server-operation`
   - 服务器 IP：`39.106.125.149`

2. 读了 skill，确认连接方式和服务器目录：
   - 项目目录：`/opt/chat_soft`
   - 后端 PM2 服务：`chat-soft-server`
   - 后端端口：`3000`
   - 前端 mobile 目录：`/opt/chat_soft/apps/mobile`

3. 在服务器上构建了 mobile 网页端：

```bash
cd /opt/chat_soft
/usr/local/bin/pnpm --filter @chat-soft/mobile build
```

4. 把构建产物部署到：

```text
/www/wwwroot/chat-soft
```

5. 因为服务器现在没有 nginx/httpd 在监听 80，我先用 PM2 启了一个轻量静态服务：

```text
chat-soft-web
```

监听：

```text
0.0.0.0:80
```

6. 当前 PM2 里有两个服务：

```text
chat-soft-server  后端 API，端口 3000
chat-soft-web     前端网页，端口 80
```
