# Chat Soft

面向个人私有使用的跨网络双端通信系统。

当前目标：

- 安卓客户端
- Windows 客户端
- 香港小服务器
- V1 先跑通文本消息与语音消息
- 预留电脑端 AI 调用接口

## 当前阶段

当前处于正式需求与总体架构设计阶段。

已完成：

- 需求访谈与范围收敛
- V1 / V2 / V3 版本划分
- 项目文档骨架

下一步：

- 锁定 V1 技术栈
- 输出服务端与客户端模块拆分
- 开始搭建可运行原型
