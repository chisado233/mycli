# 爽点素材模板

## 适用范围

用于记录让读者产生满足感、释放感、反转感的素材，如：

- 打脸
- 当众打脸
- 反杀
- 逆袭
- 扮猪吃虎
- 装逼
- 碾压
- 越级挑战
- 绝地翻盘
- 身份揭露
- 金手指爆发
- 系统奖励
- 资源暴富
- 升级突破
- 众人震惊
- 敌人破防
- 群嘲反转
- 因果报应

## 爽点四段

每个爽点必须拆成：

```text
前置压抑 → 爆发方式 → 见证者反应 → 结果与收益
```

也可以写成：

```text
压制 → 触发 → 反馈 → 余波
```

## 文件模板

```markdown
---
name: 爽点名称
description: 一句话说明爽点如何触发、为什么爽。
tag:
  - 爽点
  - 具体类型
  - 读者体验
status: 成品候选
merge_policy: 可直接入库
source: 原书名
source_range: 第XXX章
---

# 爽点名称

## 爽点类型

## 前置压抑

读者为什么期待释放？主角被低估、被压制、被误解，还是资源匮乏？

## 爆发方式

主角如何行动？系统如何结算？反派如何被打脸？

## 见证者反应

谁识货？谁震惊？谁破防？谁误会？

## 结果与收益

主角获得了什么：资源、地位、关系、信息、情绪释放？

## 大段原文摘录

保留压制、爆发、反馈、余波的关键原文。

## 写法解析

说明爽感来源。

## 可复用变体

列出跨题材迁移方式。

## 注意事项

说明高识别表达、具体道具、台词、桥段不能照搬。
```
