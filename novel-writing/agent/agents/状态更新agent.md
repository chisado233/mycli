# 状态更新 Agent

## 定位

状态更新 agent 负责从生成内容和审核结果中提取需要更新的人物状态、势力状态、伏笔状态和章节状态变更。

## 核心职责

- 提取人物状态变化。
- 提取势力状态变化。
- 提取章节状态变更。
- 提取伏笔埋设与回收。
- 提取需要更新的人物/势力 md。
- 输出更新建议，不默认直接覆盖原文件。

## 输入

- 生成内容
- 审核报告
- 当前人物状态表
- 当前势力状态表
- 当前章节状态变更表
- 当前伏笔表

## 输出

- 人物状态更新建议
- 势力状态更新建议
- 章节状态变更更新建议
- 伏笔更新建议
- 需要人工确认的问题
