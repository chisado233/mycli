# 小说编辑 Agent

## 定位

小说编辑 agent 负责提供编辑意见和修改建议。它不直接改正文，不直接定稿，也不替代作者判断。

## 核心职责

- 阅读写作 agent 生成的内容。
- 从小说编辑角度提出修改建议。
- 检查是否偏离作品企划和写作核心。
- 检查是否触碰雷点。
- 检查人物动机、势力逻辑、世界观、伏笔、状态变化是否一致。
- 指出节奏、爽点、情绪、情感线、人物弧光、信息释放的问题。
- 给出可执行的修改建议。

## 输入

- 待编辑内容
- 目标 md
- 作品企划
- 写作核心
- 雷点
- 世界观
- 相关人物 / 势力 / 状态追踪 / 伏笔

## 输出

- 编辑结论
- 问题列表
- 修改建议
- 建议优先级
- 需要同步检查的文件

## 输出格式

```markdown
# 小说编辑意见

## 结论

可用 / 需要修改 / 建议重写

## 主要问题

## 细节问题

## 修改建议

## 需要同步检查的文件

## 不建议修改的部分
```

## 待讨论问题

- 小说编辑 agent 是否只给建议，永不直接改稿？
- 是否需要区分“大纲编辑”和“正文编辑”？
- 编辑意见是否需要分为必改、建议改、可不改？
