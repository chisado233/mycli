# 当面杀父与旧恩真相崩塌

## 场景目标
让唐山的父亲依仗彻底失效，同时让月舞看清唐浩曾经救她并非纯善，而是把她当作儿子的突破材料。

## 出场人物
顾云、唐山、唐浩、月舞、影儿。

## 场景环境
白云帝寝宫/皇宫内部。前文唐浩父子刺杀白云帝失败，被顾云堵住，形成封闭空间内的终局对峙。

## 冲突推进
1. 唐山以“父亲合体境”为依仗，要求顾云放下月舞。
2. 顾云以炼虚境威压把唐山按在地面，击碎其自信。
3. 唐浩出手，影儿现身一刀斩首，打破合体境不可敌认知。
4. 唐浩依靠不死仙藤复苏并逃亡。
5. 唐山还惦记月舞，唐浩脱口说出“不过是突破合体境材料”。
6. 顾云斩杀唐浩，同时把杀戮意义转化为“替月舞报仇”。

## 情绪变化
- 唐山：愤怒自信 → 错愕 → 恐惧 → 崩溃。
- 月舞：被动沉迷顾云怀抱 → 听见真相后信念崩塌 → 转向顾云寻求依靠。
- 顾云：戏谑 → 冷漠出手 → 安抚式占有。
- 唐浩：狂傲 → 惊恐 → 逃亡 → 暴露自私本性。

## 写法解析
这个场景有效的核心，不只是“杀唐浩”，而是把三条线压在同一爆点上：

1. **战力线**：唐山最大依仗唐浩被秒杀，合体境光环破碎。
2. **情感线**：月舞对唐家的旧恩被唐浩亲口推翻，不需要顾云辩解。
3. **爽点线**：唐山想夺回女主，结果父亲死、女主归心、自己被拍晕。

“真相”不是由旁白揭露，而是在逃命时自然脱口而出，因此更像本性暴露。

## 使用场景
- 女主曾被某位恩人/师父/养父收留，但真相是被当作炉鼎、血脉材料、钥匙。
- 主角或反派男主当面击杀旧恩人，使女主情感断裂。
- 旧情人/青梅竹马想抢回女主，却被旧关系真相反噬。

## 可复用变体
- 变体1：师尊救下孤女多年，关键时刻亲口说出她只是开启仙府的祭品。
- 变体2：贵族养子被当作继承人培养，实则是给真正少主换血的容器。
- 变体3：救命恩人战败逃跑时暴露“当年救你只是为了你体内神骨”。

## 注意事项 / 禁止照搬点
- 不可照搬“三命妖猫”“合体材料”“不死仙藤复活唐浩”等具体设定组合。
- 不可照搬原文台词和“死猫”称呼。
- 可迁移的是“旧恩人亲口暴露利用真相 + 新依靠替其复仇”的结构。
- 情绪处理要谨慎，避免把受害者情感崩塌写成单纯恋爱服从，应保留创伤、迷茫和依靠需求。
