# 传承守护者送钥匙上门

## 场景目标
让顾云不需要自行寻找海神传承入口，而是通过“守护者救传人”逻辑，让钥匙自动送到手上。

## 出场人物
顾云、影儿、海澜、唐山、月舞。

## 场景环境
飞舟外，唐山被挂在飞舟船头作为诱饵。海澜感应海神传人危险后赶来。

## 冲突推进
1. 顾云判断唐山还剩海泽国度机缘，故意保留其性命。
2. 海澜带着信仰怒火登场，救下唐山。
3. 海澜质问顾云亵渎海神传人。
4. 海澜化作人鱼，持三叉戟攻击。
5. 顾云空手挡下，捏住她脖子。
6. 顾云夺取钥匙，海澜神魂被人皇幡吞噬。

## 情绪变化
- 海澜：愤怒神圣 → 自信审判 → 惊恐无力 → 香消玉殒。
- 顾云：等待猎物 → 确认钥匙 → 冷淡掠夺。
- 唐山：被救希望出现，但并无实际自主权。
- 影儿：想杀 → 被顾云阻止 → 理解“钥匙来了”。

## 写法解析
这一场的爽感来自“敌人制度自动服务反派”。海澜本该是唐山的守护者、献祭者、引路人，但顾云把唐山挂在飞舟前，使海澜必须来救。她越忠诚，越会把钥匙送到反派手中。

“钥匙来了”一句把神圣传承降格成物资快递，形成反派式幽默。

## 使用场景
- 秘境守护者因感应传人危机自动现身。
- 天命主角的护道人、信使、系统NPC被反派截胡。
- 反派不破解谜题，而是抓住掌握钥匙的人。

## 可复用变体
- 变体1：圣女来救神子，身上带着开启圣墓的血玉，被反派夺走。
- 变体2：器灵感应主人濒死，主动现身护主，被反派反向契约。
- 变体3：宗门长老来接回真传弟子，却被敌人通过搜魂拿到宗门禁地路线。

## 注意事项 / 禁止照搬点
- 不可照搬“海神传人、海澜、人鱼三叉戟、掏心窝子”的组合与表达。
- 可迁移机制是“守护者救援 = 钥匙暴露 = 反派截胡”。
- 若写正派主角，需弱化残酷掠夺，改成智取或道德困境。
