# chapter-017：老宗主来了，老宗主跪了

- 拆解等级：A
- 本章一句话：顾云在屏风后压住风云韵的宗主姿态，正当两人关系反差升级时，老宗主风霄携怒闯入，却在顾云现身时被恐怖威压吓到当场跪下。
- 表层事件：
  1. 屏风后，顾云调侃风云韵刚才在众人面前装出的宗主魄力。
  2. 老宗主风霄气势汹汹登场，质问宗门惨状和凶徒所在。
  3. 中年长老借风霄撑腰叫嚣，却发现风霄已经恐惧下跪。
- 故事逻辑：顾云目标是让风云韵保住外部威严又确认私下主从关系 → 阻碍是老宗主风霄因宗门清洗闯入问罪 → 顾云搂风云韵从屏风后现身并释放压迫 → 风霄瞬间跪倒 → 新问题是风霄如何处理前来叫嚣的长老与宗门残局。
- 本章功能：人物关系反差深化、老宗主登场、实力威慑预热、结尾反转爽点。
- 关键人物/势力：顾云、风云韵、风霄、中年长老、紫云宗。
- 人物/势力/背景/世界设定：风霄是紫云宗上一代宗主、合体境大能、风云韵师尊；在苍玄界合体境已是顶级强者，但面对顾云仍不敢反抗。
- 人物设定素材：风云韵——公开端庄、私下羞怯臣服；顾云——懂得配合伴侣公开身份，又在私密空间掌握主动；风霄——外表暴烈护宗，实则极识时务。
- 力量体系素材：合体境威压可震慑炼魂/炼虚；但顾云的真实威压远超下界合体，形成层级碾压。
- 新增设定/伏笔：风霄认出顾云非普通强者，或至少感知到不可敌。状态：强化。后续风霄将成为紫云宗代理人。
- 常见场景类型：私密关系反差 / 强者登场反被压制 / 扮猪吃虎 / 当场跪服。
- 情绪/爽点/搞笑点/抽象点：前半章暧昧调教式反差，后半章从老宗主“问罪”瞬间反转为“下跪”，中年长老的叫嚣形成打脸前置。
- 原文摘录候选：
  > “风霄冷声道：‘云韵，做了几年宗主你是心高气傲了？连我这个师尊的话都敢不听了！’  
  > ‘宗门出了这么大的事情，全都是你这个宗主的责任，还不赶紧滚出来见我？’  
  > 说话间，合体境的磅礴气势轰然张开，吓得在场的所有修士瑟瑟发抖。”
  >
  > “眼见一名男子搂着自己的徒儿从屏风后走出，每踏一步。  
  > 他就感觉自己的心头恐惧增加三分，身上的压力更是暴涨。  
  > ……  
  > 扑通！  
  > 一声巨大的下跪声在自己身后响起，中年长老神色一怔，他颤抖着转身，就看见那被自己视作最后希望的老宗主竟然跪在了地上。”
- 结尾钩子：合体境老宗主当场跪下，下一章必然解释其恐惧与处置方式。
- 可借鉴技法：反转式权威登场——先用旁人对“合体境”的敬畏抬高权威，再让权威在更高层级前瞬间崩塌。
- 索引更新价值：人物：风霄新增；场景：老宗主问罪反跪；爽点：权威跪服；情感线：顾云-风云韵公开/私下双面关系。
- 判断与证据锚点：
  1. “合体境的大能”抬高风霄地位。
  2. “老宗主竟然跪在了地上”完成反转。
- 写作还原输入包：
  - 场景顺序：屏风后私密调侃 → 老宗主暴怒入堂 → 长老报告宗主在屏风后 → 风霄释放合体威压 → 顾云出声嘲讽 → 中年长老借势叫嚣 → 顾云搂风云韵现身 → 风霄下跪结尾。
  - 人物目标：顾云维持支配与趣味；风云韵想保住宗主脸面；风霄想追责护宗；中年长老想借靠山复仇。
  - 信息释放：风霄为合体境，紫云宗仍有顶级老祖级战力；顾云威压可让合体境失态。
  - 爽点触发链：压制：老宗主合体威压、长老叫嚣 → 触发：顾云现身 → 反馈：风霄跪倒 → 余波：老宗主将转向求饶。
  - 结尾落点：权威跪服的强钩子。
