# chapter-020：天命动向

- 拆解等级：A
- 本章一句话：影儿分魂伪装赤炎尊者引导林萧逃往无尽火山，顾云则带风云韵离开紫云宗前往白云州天骄战，正式切换到唐山线。
- 表层事件：
  1. 林萧在“赤炎尊者”引导下逃向无尽火山，并立誓复仇紫云宗、顾云、风云韵。
  2. 三日后，顾云给风霄续命宝药，风霄接手紫云宗并放风云韵随顾云离开。
  3. 顾云安抚风云韵失去宗主位置的不安，并透露未来可让她在上界继续做宗主。
  4. 飞舟前往白云州白云城，天骄战与唐山线即将开启。
- 故事逻辑：顾云目标是放养并监控林萧，同时转向另一天命之子唐山 → 阻碍是林萧逃亡路线与风云韵离宗安排 → 影儿分魂诱导林萧去无尽火山，顾云用宝药收买风霄 → 林萧进入预设机缘猎场，风云韵成为顾云身边侍女 → 新问题是白云帝国天骄战与唐山如何被顾云针对。
- 本章功能：双线转场、林萧仇恨强化、风云韵身份转换、白云州地图开启、唐山线预告。
- 关键人物/势力：顾云、林萧、影儿分魂/赤炎尊者、风云韵、风霄、唐山、白云帝国。
- 人物/势力/背景/世界设定：荒云州为妖蟒一族地盘、人类禁区；白云州霸主为白云帝国，老祖合体境；白云城举行天骄战；唐山在其中。
- 人物设定素材：林萧——被仇恨驱动、把风云韵从“仙女”贬为“烧鸡”，占有欲与羞辱欲加深；风云韵——担忧失去宗主身份会降低顾云兴趣；顾云——用未来资源与身份安抚女人，同时精确切换猎物。
- 力量体系素材：紫幽神冥火继续作为林萧机缘目标；百花圣体、百花圣地后续路线被预告；白云帝国合体境老祖作为下界霸主战力锚点。
- 新增设定/伏笔：
  - 林萧受影儿分魂误导前往无尽火山：状态：强化。
  - 风云韵未来百花圣地/百花圣体路线：状态：埋设。
  - 唐山将在天骄战出现：状态：埋设。
- 常见场景类型：逃亡追杀 / 反派操盘 / 情感安抚 / 飞舟转场 / 新地图开启。
- 情绪/爽点/搞笑点/抽象点：林萧对“老师”声音异常的疑惑形成悬念；风霄因一份续命宝药“卖徒弟”有黑色幽默；风云韵担心“失去宗主趣味”体现被调教后的自我认知。
