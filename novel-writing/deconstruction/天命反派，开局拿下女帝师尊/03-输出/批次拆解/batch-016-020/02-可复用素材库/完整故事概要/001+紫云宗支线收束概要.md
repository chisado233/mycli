# 紫云宗支线收束概要

## 一句话卖点
高位反派帝子在下界宗门中清洗天命之子助力、收服地方势力与气运女主，并放线钓鱼引导天命之子奔向下一场被截胡的机缘。

## 主角与目标
顾云的目标是拆解林萧天命剧本，收割风云韵与紫云宗价值，同时布局无尽火山；完成后转向白云州，寻找新天命之子唐山。

## 世界背景
苍玄界是下界修真世界，紫云宗为紫云州霸主但正在衰落。合体境风霄是紫云宗老宗主，但面对上界帝子顾云不堪一压。无尽火山、白云州天骄战是后续主线地图。

## 核心矛盾
林萧原剧本应获得风云韵和紫云宗助力，但顾云提前截胡并栽赃，紫云宗内部仍有海殇人等旧助力护林萧。顾云必须清除这些助力，避免林萧获得缓冲空间。

## 主要阶段
1. 清洗反对派：顾云杀海殇人等七人，林萧失去紫云宗助力。
2. 稳定风云韵关系：公开给面子、私下确认主从反差。
3. 老宗主跪服：风霄从潜在敌人转为执行者。
4. 林萧路线放线：影儿分魂伪装赤炎尊者，诱导林萧去无尽火山。
5. 阵营扩充：影儿摘面纱收心。
6. 地图转场：顾云带风云韵前往白云州天骄战。

## 关键反转
- 合体境老宗主本应是紫云宗最后底牌，却当场跪服。
- “赤炎尊者”指引林萧逃亡，实则是顾云阵营影儿分魂操控。
- 风云韵从一宗之主转为顾云身边侍女，但顾云承诺未来给她更大宗门舞台。

## 情感线
- 顾云-风云韵：公开宗主/私下臣服的反差进一步固化。
- 顾云-影儿：忠诚影卫越界并被接纳。
- 林萧-风云韵：从林萧单方面“仙女/我的女人”幻想，转为仇恨羞辱欲。

## 结局方向 / 阶段落点
紫云宗支线结束，林萧被送入无尽火山猎场；顾云前往白云州天骄战，唐山线开启。

## 可复用结构
“截胡原主角女主 → 栽赃主角 → 清除主角地方助力 → 收服地方老祖 → 放主角逃去下一机缘 → 主角转场猎杀第二主角”。
