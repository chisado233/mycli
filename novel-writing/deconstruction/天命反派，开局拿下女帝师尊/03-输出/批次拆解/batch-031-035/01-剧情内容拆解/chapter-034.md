# chapter-034：覆灭赤霄剑阁

- 拆解等级：A
- 本章一句话：顾云带四大合体势力降临赤霄剑阁，借月舞复仇请求斩断其与唐山情缘，并由影儿秒杀两大合体境，最终将赤霄剑阁化为人皇幡养料。
- 表层事件：
  1. 赤霄剑阁两位合体境发现魂灯全灭，外界四大强者降临。
  2. 顾云抱着月舞出现，赤霄修士认出三命妖猫并羞辱其母。
  3. 月舞向顾云下跪请求复仇，承诺“无所不从”。
  4. 系统判定唐山与月舞情缘断绝，唐山天命值大减。
  5. 影儿秒杀赤云老鬼并挖出赤羽心脏，四大合体破阵灭阁，人皇幡吞魂蜕变。
- 故事逻辑：月舞目标是为母复仇 → 阻碍是赤霄剑阁护宗大阵与两位合体境 → 行动是向顾云求助并献上从属承诺 → 结果是顾云出手灭阁，母亲心脏回收 → 新问题是顾云对苍玄界格局的屠灭影响扩散。
- 本章功能：完成月舞从唐山线向顾云线的核心转折；爆发大爽点“替弱者复仇+顶级刺客秒合体+势力覆灭”；推动人皇幡成长。
- 关键人物/势力：顾云、月舞、影儿、风霄、白炽、林擎天、青云和尚、赤云老鬼、赤羽、赤霄剑阁。
- 人物/势力/背景/世界设定：赤霄剑阁曾杀害月舞母亲，用三命妖猫心脏突破合体；护宗大阵在影儿面前无效。
- 人物设定素材：月舞从奶凶猫转为主动请求复仇的受害者；顾云表面调笑，实则精准满足她最核心需求；影儿对外冰冷高效，对顾云“幸不辱命”。
- 力量体系素材：三命妖猫心脏可助突破合体；影儿速度与隐匿能无视大阵秒杀合体；人皇幡吸收大量魂魄后蜕变。
- 新增设定/伏笔：
  - 月舞母亲心脏被赤羽炼化/保留残息，后续可复苏：埋设。
  - 人皇幡吞魂成长：强化。
  - 唐山与月舞情缘断绝：回收/阶段节点。
- 常见场景类型：复仇请愿、灭门、强者压境、刺客秒杀、势力覆灭、情感线断裂。
- 情绪/爽点/搞笑点/抽象点：月舞认真说“就算你让我用猫的……”被顾云打断，黑色幽默中完成情感臣服；四位合体抢着响应，构成反派团建式爽点。
