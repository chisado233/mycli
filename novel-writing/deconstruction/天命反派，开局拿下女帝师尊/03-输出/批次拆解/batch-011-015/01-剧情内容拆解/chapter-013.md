# chapter-013：林萧毁了紫云矿脉？

- 拆解等级：A
- 本章一句话：顾云借风云韵宗主权威将紫云矿脉被毁之事栽赃给林萧，并让林萧在公开审判中陷入众怒。
- 表层事件：
  1. 顾云将紫雷竹交给风云韵，并决定随她前往紫云宗当客卿。
  2. 林萧醒来误以为紫云山脉大破坏是自己/老师所为，又发现风云韵被带走，愤怒发誓报仇。
  3. 紫云宗内部议论矿脉被毁，风云韵在大殿陈述“黑袍人”袭击说法，将林萧押上来审问。
- 故事逻辑：目标：顾云要为自己毁矿找替罪羊并切断林萧紫云宗助力 → 阻碍：林萧只是外门弟子，按常识不可能毁矿 → 行动：风云韵捏造“气息忽强忽弱、疑似合体”的黑袍人，并抓住林萧自称杀虎的口供 → 结果：林萧被押上大殿，海殇人登场 → 新问题：海殇人会替林萧开脱。
- 本章功能：完成从山脉战斗到宗门审判的转场；把顾云破坏行为转移到林萧身上；引出林萧紫云宗内的关键助力海殇人。
- 关键人物/势力：顾云、风云韵、林萧、海殇人、紫云宗高层。
- 人物/势力/背景/世界设定：紫云矿脉是紫云宗修炼资源命脉；紫云宗因老祖衰弱而处于八大王州弱势；紫云州为苍玄界八大王州之一。
- 人物设定素材：林萧自我中心，将风云韵视作“自己的女人”；风云韵虽心有不忍但服从顾云命令；紫云宗弟子因资源断绝而失智愤怒。
- 力量体系素材：炼虚大战通常难以摧毁矿脉，疑似合体力量才合理；赤炎尊者沉睡导致林萧无解释能力。
- 新增设定/伏笔：
  - 紫云矿脉被毁会导致紫云宗修炼道途受损。
  - 海殇人认识林萧，是其前期投资人。
- 常见场景类型：宗门审判 / 栽赃陷害 / 舆论围剿 / 资源危机。
- 情绪/爽点/搞笑点/抽象点：林萧把顾云造成的大坑误认为自己战绩，随即被当罪魁祸首抓走，形成反讽。
- 原文摘录候选：
  > “这些……难道都是我做的吗？  
  > 他的记忆还停留在发现顾云亵渎风云韵，自己完全将身体交给老师的时候，后续的事情一概不知。”
  
  > “我本来与那狂雷虎交手正酣，忽然之间钻出来了一个黑袍人，黑袍人身上的气息波动非常的剧烈，忽而能达到合体，忽而又只有炼血境，极为诡异。”
  
  > “风云韵语气坚定开口。实则内心很是无奈，紫云矿脉其实是因为顾云那一掌才毁灭的，如今却要栽赃他人，她心中微微有些不忍。”
- 结尾钩子：海老认出林萧，林萧抓住救命稻草求助。
- 可借鉴技法：栽赃局先制造民怨，再用半真半假的战力描述让替罪羊符合逻辑。
- 索引更新价值：紫云宗世界设定、海殇人人物、林萧被栽赃爽点、紫云矿脉资源设定。
- 判断与证据锚点：
  - “忽而合体忽而炼血”：为林萧异常战力做证据链。
  - “价值远超紫云矿脉”：解释风云韵为何愿意演戏。
- 写作还原输入包：
  - 场景顺序：赠紫雷竹 → 林萧废墟醒来 → 紫云宗弟子议论 → 大殿高层审问 → 风云韵捏造黑袍人 → 林萧被押上来 → 海老震惊。
  - 人物目标：顾云要背后操控；风云韵要完成栽赃；林萧要自证清白并找风云韵；海老要保护林萧。
  - 信息释放：紫云矿脉重要性、紫云宗衰落、海老与林萧关系。
  - 爽点触发链：压制：顾云毁矿有暴露风险 → 触发：风云韵掌控宗门叙事 → 反馈：林萧被抓被骂 → 余波：海老入局。
  - 结尾落点：审判矛盾升级。
