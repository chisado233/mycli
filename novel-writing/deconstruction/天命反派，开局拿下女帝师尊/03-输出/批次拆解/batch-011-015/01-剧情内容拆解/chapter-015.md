# chapter-015：好戏落幕，演员们，该退场了

- 拆解等级：A
- 本章一句话：林萧因自相矛盾被判死后，在影儿伪装赤炎尊者引导下逃亡，海殇人及老派长老公然站队阻拦，顾云以“好戏落幕”准备清场。
- 表层事件：
  1. 顾云抓住林萧承认炼虚战力的口供，风云韵追问另外两位妖王之死，林萧错漏百出。
  2. “赤炎尊者”突然苏醒催促林萧逃跑，林萧放狠话遁走。
  3. 海殇人出手阻拦追兵，多名老派长老站队，矛头指向顾云；顾云祭出大印压场。
- 故事逻辑：目标：顾云要逼出林萧逃亡并顺势清洗紫云宗内反对派 → 阻碍：海殇人在宗内威望高，老派长老同情林萧 → 行动：先让林萧逃，再观察海殇人派系全部跳出来 → 结果：反对者集中暴露 → 新问题：第016章武力清场将引发宗门震荡。
- 本章功能：完成审判局高潮；将林萧从宗门线赶向逃亡线；把海殇人派系集中到顾云掌下，形成下一章杀伐爽点。
- 关键人物/势力：顾云、林萧、风云韵、海殇人、紫云宗老派长老、影儿分魂/伪装赤炎尊者。
- 人物/势力/背景/世界设定：紫云宗内部长老并非铁板一块，海殇人威望可对抗宗主；宗门对客卿外人有天然排斥。
- 人物设定素材：海殇人正直但迂腐，宁愿对抗宗主也要保林萧；风云韵恐惧不能完成顾云任务；顾云喜欢看戏，等待敌人全部跳出再一次清场。
- 力量体系素材：海殇人火系法术火墙、火龙啸天；风云韵风系术法风之缚、风之极-山河动；顾云大印类宝物即将压场。
- 新增设定/伏笔：
  - “赤炎尊者”声音异常，后文第018章确认是影儿分魂伪装。
  - 海殇人派系被集中暴露，下一章可清算。
- 常见场景类型：公开审判 / 逃亡 / 派系站队 / 大佬压场 / 名场面。
- 情绪/爽点/搞笑点/抽象点：海殇人热泪盈眶以为老伙计义气，实则全员进入顾云清算范围；风云韵怕“主人交代不了”的内心反差。
