# 紫云山脉截胡支线概要

## 一句话卖点

反派少主看完天命剧本后，不等男主成长，直接闯进其英雄救美节点，抢走女主、打飞男主、顺手灭掉区域三大妖王。

## 主角与目标

顾云目标是削弱林萧，抢走其原本会获得的风云韵与紫云宗助力，同时为一年后的苍玄界本源争夺提前拆除威胁。

## 世界背景

苍玄界是顾云下界试炼地图，紫云州紫云山脉中有妖王狂雷虎和紫云宗资源冲突。风云韵为救师尊/老宗主，需要紫雷竹。

## 核心矛盾

原剧本中林萧会在风云韵被狂雷虎重创时救下她，从而收获女主与宗门助力；顾云则利用剧本信息差提前介入，夺取这个因果。

## 主要阶段

1. 006：顾云确认天命剧本可改，决定先处理林萧艳遇。
2. 007：顾云发现风云韵、狂雷虎与暗处林萧，确认女主价值。
3. 008：林萧面板展开，风云韵与狂雷虎交易破裂。
4. 009：风云韵重伤，林萧借力欲救，顾云天降截胡，影儿拍飞林萧。
5. 010：顾云灭三妖、收魂、带风云韵入山洞疗伤，建立后续关系钩子。

## 关键反转

- “林萧英雄救美”反转成“顾云抢先救美”。
- “林萧借老爷爷爆发”反转成“还未出手就被影儿拍地”。
- “狂雷虎地利威胁”反转成“三妖一掌清场”。
- “风云韵原属林萧阵营”反转成“顾云救命因果建立”。

## 情感线

- 顾云-风云韵：陌生目标 → 救命恩人 → 疗伤羞恼，后续将快速收服。
- 顾云-影儿：主仆保护 → 影儿吃味 → 战斗配合 → 情感越界铺垫。
- 林萧-风云韵：原未来命定关系被顾云截断，林萧只剩单方面占有欲与误会。

## 结局方向 / 阶段落点

风云韵在山洞中醒来质问顾云身份。支线未闭合，后续011-012将通过亲密推进、身份揭露与资源承诺完成收服，并在013-016转入紫云宗栽赃林萧的大型打脸。

## 可复用结构

“读剧本锁机缘 → 到场观察原男主与女主 → 等待原男主高光前一秒 → 反派抢走救命因果 → 暗卫压制男主 → 清场收资源 → 转入女主关系推进”。
