# chapter-005：降临苍玄界，锁定天命之子

- 拆解等级：A
- 本章一句话：顾云与影儿降临苍玄界，确认地图三大机缘点后，没有盲目前往，而是消耗反派值锁定林萧、唐山两位天命之子并获得天命剧本。
- 表层事件：
  1. 顾云与影儿从暧昧姿势中起身，影儿解释陛下命她贴身保护。
  2. 影儿展示化作顾云腰间黑纹的隐匿保护能力。
  3. 顾云查看苍玄界地图，发现无尽火山、海泽国度、苍玄灵眼三处标记。
  4. 顾云消耗6400反派值获得林萧、唐山天命剧本，并在山洞中读完两本八百万字命运线。
- 故事逻辑：目标→顾云想找到小世界之灵与天命之子；阻碍→机缘地疑似天命安排，贸然前往可能空手；行动→借系统探测并购买天命剧本；结果→锁定两位天命之子和未来事件；新问题→林萧、唐山未来都会参与覆灭顾家大战，必须提前扼杀。
- 本章功能：从地图探索转向剧本猎杀；明确第一阶段敌人；强化顾云不是莽夫而是利用信息差的反派；推进影儿主仆关系。
- 关键人物/势力：顾云、影儿、系统、林萧、唐山、苍玄界。
- 人物/势力/背景/世界设定：苍玄界分八州；三处疑似核心机缘点；天命之子未来会飞升三千道域参与覆灭顾家。
- 人物设定素材：顾云谨慎、懂天命套路、愿意花资源买信息差；影儿怯懦顺从但眼神深邃，疑似另有秘密；系统提供剧本侦查功能。
- 力量体系素材：反派值可锁定天命之子或购买天命剧本；天命剧本记录主角重要事件；影儿可化作道纹/黑纹贴身护卫。
- 新增设定/伏笔：
  - 新增项：林萧《神火帝尊》天命之子，状态：埋设。
  - 新增项：唐山《娇妻生死录》天命之子，状态：埋设。
  - 新增项：影儿眼眸深邃异常，状态：待验证。
  - 新增项：三处地图机缘，无尽火山/海泽国度/苍玄灵眼，状态：埋设。
- 常见场景类型：降临新地图 / 主仆暧昧 / 地图分析 / 系统侦查 / 山洞读剧本。
- 情绪/爽点/搞笑点/抽象点：主角不按天命路线抢机缘，而是先买剧本锁主角，形成“反派开全图挂”爽点；影儿化纹被摸发痒，暧昧搞笑。
