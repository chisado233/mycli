# chapter-022：经典桥段，天命之子被打脸

- 拆解等级：A
- 本章一句话：金惜狂以皇子身份挑衅唐山并炫富，却因天字包厢被顾云预订而装逼失败，唐山趁机反嘲，双方冲突升级，顾云暗中确认唐山天命身份。
- 表层事件：
  1. 金惜狂调戏月舞，唐山杀意升起却因境界差距隐忍。
  2. 金惜狂拍出白云钱庄金卡炫耀身份，掌柜却告知最好包厢已被年轻炼虚强者订走，疑似龙城林家。
  3. 唐山嘲笑金惜狂装逼失败，金惜狂暴怒出手；顾云在楼上观看并获取唐山系统信息。
- 故事逻辑：目标=金惜狂想用身份财力抢月舞、唐山想维护占有权 → 阻碍=顾云预订包厢制造信息差 → 行动=金惜狂炫卡，唐山反嘲 → 结果=两人开打 → 新问题=顾云盯上唐山与月舞。
- 本章功能：完成新天命之子信息揭示；用“被第三方抢先预订”让反派配角装逼失败；让顾云从上帝视角介入唐山线。
- 关键人物/势力：唐山、月舞、金惜狂、顾云、风云韵、影儿、白云帝国、龙城林家。
- 人物/势力/背景/世界设定：
  - 金惜狂是白云帝国二殿下，在白云城拥有高地位。
  - 龙城林家是苍玄界顶级势力，连白云皇子都忌惮。
  - 顾云提前订下天字包厢，成为暗中搅局者。
- 人物设定素材：
  - 唐山：将月舞视为“禁脔”，外部挑衅会触发极端杀意。
  - 金惜狂：纨绔皇子，喜欢当众炫富、调戏女子、仗势欺人。
  - 影儿/风云韵：围绕顾云争宠，形成反派后宫日常。
- 力量体系素材：唐山炼骨境初期；金惜狂炼骨境后期；顾云系统读取唐山：海王体、天命之子白、天命值1200、关联女主月舞。
- 新增设定/伏笔：
  - 新增项：唐山为《娇妻生死录》男主，体质海王体，伏笔状态：埋设。
  - 新增项：月舞为关联女主，后续可通过截胡女主削弱唐山，伏笔状态：埋设。
  - 新增项：龙城林家威慑力，伏笔状态：强化。
- 常见场景类型：纨绔挑衅 / 炫富失败 / 楼上观战 / 系统识别天命之子 / 后宫争宠。
- 情绪/爽点/搞笑点/抽象点：
  - 爽点：金惜狂装逼失败，气焰被“龙城林家”压灭。
  - 搞笑点：唐山“此人已有取死之道”反复内心判刑。
  - 反派爽点：顾云没出面，只凭预订包厢就让两方狗咬狗。
