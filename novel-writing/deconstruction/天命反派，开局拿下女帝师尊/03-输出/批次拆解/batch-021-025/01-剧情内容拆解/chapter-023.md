# chapter-023：原形毕露卡显威，掳走气运之女

- 拆解等级：A
- 本章一句话：顾云趁唐山与金惜狂激战，用原形毕露卡将月舞变回三命妖猫并掳走，成功截断唐山最关键的气运女主关系。
- 表层事件：
  1. 唐山用藤蔓与步法缠斗金惜狂，金惜狂以虎系灵力强攻。
  2. 顾云在无人注意时对月舞使用原形毕露卡，月舞变成三花小猫。
  3. 顾云抱走月舞，系统奖励反派值；月舞被风云韵揉搓，又因顾云容貌心动。
  4. 系统揭示月舞身份、身世和未来牺牲线。
- 故事逻辑：目标=顾云想削弱唐山天命 → 阻碍=月舞在人前且与唐山关系紧密 → 行动=借战斗混乱用卡牌变猫并抱走 → 结果=月舞到手，唐山失去女主而暂不自知 → 新问题=月舞身份与杀母仇线引出赤霄剑阁、龙城林家等后续冲突。
- 本章功能：核心气运女主截胡；月舞妖猫身份揭露；开启月舞与顾云之间“被掳—羞愤—心动—被保护”的反向情感线。
- 关键人物/势力：顾云、月舞、唐山、金惜狂、风云韵、三命妖猫、龙城林家。
- 人物/势力/背景/世界设定：
  - 月舞是三命妖猫，可幻化成人；曾折损两条命。
  - 妖猫之心是合体境材料，母亲因此而死。
  - 月舞未来在原书中会为唐山牺牲，后被复活为九命猫妖、达准帝境。
- 人物设定素材：
  - 月舞：活泼、羞愤、爱炸毛、颜控；猫形时“奶凶”反差强。
  - 顾云：精准打击天命结构，不杀男主先夺其女主。
  - 风云韵：对可爱小动物有强烈揉搓欲，形成御姐反差萌。
- 力量体系素材：原形毕露卡可使化形妖兽恢复妖兽形态七日；唐山藤蔓术、虎系灵力、妖猫血脉材料价值。
- 新增设定/伏笔：
  - 新增项：三命妖猫身份和妖猫之心，伏笔状态：埋设。
  - 新增项：龙城林家未来会为妖猫心脏出手，伏笔状态：埋设。
  - 新增项：月舞未来牺牲复活线，伏笔状态：待验证。
- 常见场景类型：战斗 / 趁乱掳人 / 道具显威 / 萌宠化女主 / 女主身世揭露。
- 情绪/爽点/搞笑点/抽象点：
  - 爽点：顾云不费吹灰之力夺走唐山关联女主。
  - 搞笑点：月舞焦急喊“山哥”却只发出“喵”。
  - 反差萌：少女女主被迫变猫，愤怒却被当成撒娇。
