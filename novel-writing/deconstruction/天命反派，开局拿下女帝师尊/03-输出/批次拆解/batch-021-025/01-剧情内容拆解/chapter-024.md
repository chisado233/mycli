# chapter-024：金惜狂之死

- 拆解等级：A
- 本章一句话：顾云暗中杀死金惜狂并嫁祸唐山，使唐山从客栈冲突者变成白云帝国通缉犯，彻底破坏原著中二人结交的后续路线。
- 表层事件：
  1. 金惜狂猛攻唐山，唐山以金针麻痹其眉心后趁机逃走。
  2. 顾云用灵力虚影替代月舞，让唐山误以为自己带走了月舞。
  3. 金惜狂头颅炸开，众人认定唐山杀害皇子。
  4. 顾云解释原著中金惜狂本会成为唐山助力，如今被其提前清除。
- 故事逻辑：目标=顾云要让唐山失去白云帝国助力并背锅 → 阻碍=原本唐山只是麻痹金惜狂，不会杀人 → 行动=顾云暗中补刀并制造唐山逃跑假象 → 结果=金惜狂死亡，唐山被通缉，天命值减少 → 新问题=唐山必须靠天骄大战求生。
- 本章功能：完成嫁祸爽点；切断唐山未来盟友金惜狂；解释顾云对原著敌对联军的预防性清算。
- 关键人物/势力：顾云、唐山、金惜狂、月舞、白云帝国。
- 人物/势力/背景/世界设定：
  - 原著金惜狂会因战斗敬佩唐山，与其结交并夺得皇位。
  - 白云帝国皇子在皇城被杀，将引发全城通缉。
  - 天骄大战是唐山唯一可能获得外部庇护的破局机会。
- 人物设定素材：
  - 顾云：记仇且有未来视角，能因族姐未来死亡提前斩断因果。
  - 月舞：即使被掳，第一反应仍担忧唐山遭白云帝国追杀。
  - 唐山：逃跑时还未意识到月舞已经被掉包。
- 力量体系素材：唐山金针暗器只有麻痹效果；顾云灵力虚影可短暂骗过唐山；系统记录唐山天命值减少100、大罗道塔进度700/2000。
- 新增设定/伏笔：
  - 新增项：金惜狂原著后期加入讨伐顾家联军，伏笔状态：回收/改写。
  - 新增项：天骄大战可为通缉犯提供庇护路径，伏笔状态：强化。
- 常见场景类型：暗中补刀 / 嫁祸 / 皇子死亡 / 通缉危机 / 原著剧情改写。
- 情绪/爽点/搞笑点/抽象点：
  - 爽点：顾云一举两得，既报未来仇又压低唐山生存空间。
  - 震惊点：金惜狂头颅在众人吹捧中炸开。
  - 搞笑点：两名妖艳女子还在夸殿下“今晚奴婢想要举高高”时被血腥现实打断。

- 可借鉴技法：**把原著助力改造成通缉大锅**——反派利用未来视角，将本该结交的配角提前变成主角的死局。
