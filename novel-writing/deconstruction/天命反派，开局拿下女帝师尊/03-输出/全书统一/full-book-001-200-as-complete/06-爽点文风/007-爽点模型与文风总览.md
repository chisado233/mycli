# 爽点模型与文风总览

## 爽点模型

### 1. 截胡模型

最核心爽点。顾云提前读取剧本，等天命之子接近机缘时截走。

常见链条：

```text
天命之子即将得机缘 → 顾云已知剧本 → 让对方带路/触发机关 → 顾云登场截胡 → 系统扣天命并发奖励
```

代表：风云韵救美、炎姬密室、位面之灵、倪阳宝藏、藏剑山庄、仙藤祖地名额。

### 2. 女主夺缘模型

不是单纯抢女人，而是破坏其与天命之子的叙事功能。

```text
旧男主误判女主心意 → 顾云制造救命/复仇/资源/名分 → 女主公开站队顾云 → 旧男主破防 → 天命值下降
```

代表：风云韵、月舞、林紫衣、方轻烟、武灵、姬瑶珺、洛寒烟、百花圣地众女。

### 3. 反派压制模型

顾云常用身份、护道者、资源与规则，而非单纯拳头。

```text
对方以规则/境界/辈分压人 → 顾云搬出帝子身份或顾家资源 → 对方权衡利益退让 → 顾云顺势重定价
```

代表：五大势力跪拜、王家退婚、摇光宴会、百花圣地万花潭、洛寒烟妥协。

### 4. 系统奖励模型

系统播报是节奏器，常接在破防或关系不可逆节点后。

奖励类型：宝箱、反派值、天命值、体质本源、法宝、抽奖、剧本、机缘掠夺。  
作用：让爽点具备“可量化收益”，避免只是情绪满足。

### 5. 养肥再宰模型

顾云不急杀高价值天命，而是保留其寻宝/成长/破防功能。

代表：

- 林萧带路位面之灵。
- 王岩掌控藏剑山庄后再剥夺。
- 叶不凡被藤华带走，祖地机缘被预定。
- 苏昊自宫后仍被洛寒烟投喂，等道身再收割。

### 6. 反套路升级模型

传统主角爽点被倒置：

- 临阵突破被打断。
- 绝地惩罚被顾云阻止。
- 女主救场反成站队顾云。
- 金手指剑灵改投顾云。
- 师尊赠法变自毁陷阱。

### 7. 公共叙事权模型

顾云完成坏事后，抢占现场解释权，把夺宝/嫁祸包装为救人或除害。

代表：藏剑山庄自伤嫁祸王岩；顾云救武灵等天骄并取得龙泉；百花圣地中把苏昊冒犯转成“师尊买单”。

## 典型爽点四段模板

### 截胡机缘

```text
压制：主角/天命之子即将获得传承。
触发：反派提前入局或让其带路。
反馈：机缘转投反派，天命之子不可置信。
余波：系统扣天命，进入下一机缘。
```

### 女主公开站队

```text
压制：天命之子自认关系深。
触发：顾云以身份、资源、情绪救赎介入。
反馈：女主当众切割旧关系。
余波：天命之子破防，顾云收宝箱。
```

### 高位强者被拿捏

```text
压制：女帝/圣主/准帝自恃境界和身份。
触发：顾云用系统道具、护道者、信息差破防。
反馈：对方失去主动权。
余波：关系从敌对转为依赖或内应。
```

## 文风总览

### 节奏

快节奏、强钩子、单章标题直接暴露爆点。多数章节以“系统播报/人物破防/新女主登场/新副本开启”结尾。

### 叙述口吻

反派视角强，常用现代吐槽拆传统爽文套路。叙述会直接点破“天命之子”“女主”“剧本”“韭菜”“寻宝鼠”等概念，形成类型自觉。

### 吐槽机制

- 把传统主角台词转成笑点，如“莫欺少年穷”“取死之道”“龙王不可辱”。
- 用黑色幽默消解残酷，如“死全家了”“临阵突破没毛用”。
- 顾云常在内心吐槽天命剧本双标，增强读者同谋感。

### 对话风格

- 顾云：轻佻、慢条斯理、反问压迫、常用温柔话术包裹算计。
- 天命之子：怒吼、质问、道德绑架、自我感动。
- 女主：高位女主常嘴硬，弱势女主常自卑，后期被顾云一句承诺击穿。
- 顾家长辈：强者日常化，调侃、护短、老六式恶趣味。

### 章节钩子

常见类型：

- 新女主钩子：玄青宝鉴识别。
- 战斗钩子：某人出手/突破/大招。
- 系统钩子：宝箱、剧本、天命值。
- 关系钩子：女主主动、误会、同床/同池前夜。
- 新地图钩子：风华书院、藏剑山庄、百花圣地、仙藤祖地。

## 可迁移机制

- 标题直接调用类型梗，再正文反向兑现。
- 让敌人完成开图，主角收尾。
- 用女主作为“天命雷达”引出新男主。
- 资源赠礼先埋后炸，如龙魂道茶。
- 强势家族写成有互动的“老六生态”，而非纯背景。

## 禁止照搬表达

不可照搬具体人物名、系统名、玄青宝鉴、大罗道塔、天地熔炉、万化圣体、天命值播报格式、原章节标题与高识别桥段组合。复用时应抽象为机制：预知剧本、气运索引、资源熔炉、时空塔、体质兼容等。
