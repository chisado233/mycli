# range-191-200 情感线推进粗拆

## 目标范围

第191-200章。

## 上下文参考范围

第186-190章用于理解洛寒烟被迫赴约、苏昊被禁闭、顾云将洛寒烟带入大罗道塔的前因；第201-205章仅用于确认顾云-顾瑶紫吃醋线延续，不作为正式素材来源。

## 本批次情感主线

本批次核心是顾云对洛寒烟的关系攻陷：先用封帝丹打碎强者安全感，再借洛河虚影被灭斩断父权后手，随后以极阴反噬苦肉计制造“他为我赴死”的误读，使洛寒烟由羞怒敌对转为高依附。后半段转为分离后的通讯调情与对苏昊的处置安排，显示洛寒烟已把顾云置于苏昊之上。

## 不可逆节点

1. **第192章洛河虚影被灭**：洛寒烟最后的外援与尊严支点失效。
2. **第193章洛寒烟倾心提示**：系统明确提示“洛寒烟对宿主彻底倾心”，情感线由敌对转为占有依附。
3. **第199章苏昊自毁与情缘断绝**：苏昊作为后宫男主与洛寒烟等女主的情感可能性被斩断。
4. **第200章传讯石被藏回衣襟**：洛寒烟离开顾云后仍保留私密联系物，关系进入持续追踪状态。

## 高价值证据候选

- 第193章：洛寒烟从“你不要碰我”到“相信，我相信。你停下好不好？”体现极端情绪翻转。
- 第194章：洛寒烟救顾云三十日后，醒来仍嘴硬否认，却无法真正疏离。
- 第197章：冷华云通过“寒气泄露”发现端倪，推动保护欲变成对顾云的敌意。
- 第200章：洛寒烟想捏碎传讯石，最终又藏回衣襟，动作比台词更诚实。

## 可复用关系模型

- **强者高冷女主攻陷模型**：剥夺安全感 → 斩断外援 → 制造误读式救赎 → 亲密后分离 → 私密通讯维持依附。
- **护道者吃醋模型**：长辈式保护 → 被当工具人 → 旁观攻略别人 → 意识到占有欲 → 表面拒绝、继续靠近。
- **旧情感线拆除模型**：旧男主保有信任 → 女主受新关系影响转交“善意礼物” → 信任被反派利用 → 旧线断裂。
