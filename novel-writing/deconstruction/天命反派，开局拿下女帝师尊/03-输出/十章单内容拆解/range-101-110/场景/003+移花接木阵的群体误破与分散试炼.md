# 移花接木阵的群体误破与分散试炼

## 场景目标

进入藏剑山庄后，众天骄试图破解木人阵；王岩则按夭语指引潜入控制中心；顾云暗中观察，让王岩继续替自己推进机缘。

## 出场人物

武灵、武惊天、李乘风、赵乾、王岩、夭语、顾云、其他修士。

## 场景环境

深幽山谷中的巨大木人场，无数木人交错，似有诡异丝线连接。阵法名为“移花接木”，可转移承受伤害并反向激活木人战力。

## 冲突推进

1. 武灵率先冲入，为抢龙泉神剑骗过兄长，制造轻喜剧式入场。
2. 众人抵达木人场，李乘风识别“阵眼”，号召合力攻击。
3. 王岩已深入阵中，担忧外部破坏阵法；夭语催促他寻找控制中心。
4. 顾云暗中旁观，不阻止王岩，给出“敌人打工”的幕后视角。
5. 众人攻击所谓核心后，灵力反被汇聚到木人体内，使其短暂攀升至化龙境，一击震飞众人。
6. 迷雾自深处蔓延，除顾云外所有人被传送分散，各自面对木人；王岩因祸得福来到控制石碑所在小院。

## 情绪变化

抢先进场的急切 → 发现阵法的谨慎 → 群体破阵的自信 → 木人睁眼的惊悚 → 被反震吐血的狼狈 → 迷雾分割的恐慌 → 王岩误入核心地的狂喜。

## 高光片段

证据概述：李乘风指认阵法核心，众天骄剑气齐出，却见四面八方灵力向核心木人涌来。木人睁眼，眼中带着亘古苍茫，气息节节攀升至化龙境，反手一击将修士震飞。随后迷雾覆盖，众人被切割到不同区域，面前各有一尊木人虎视眈眈，弱者甚至刚入门就被轰成血雾。

关键短引：
- “移花接木之阵。”
- “齐齐攻击，第一时间破除阵眼！”
- “那木人刹那间睁开了双眼。”
- “所有人都被传送走，分散在大阵各处。”

## 写法解析

这个场景的可复用价值在于“群体自信误破阵”。不是靠旁白说阵法厉害，而是让一个看似懂行的角色给出方案，众人执行后反遭利用。阵法名“移花接木”与功能一致，攻击核心反而给木人充能，使错误有逻辑感。

空间调度清晰：外层木人场负责群像危机，深层小院负责王岩主线机缘，顾云处于第三层“暗中俯瞰”。三线并行，让探索场景既有即时危险，也有幕后算计。

## 使用场景

适合秘境、古墓、宗门试炼、机关城。尤其适合把团队拆散，使每个角色进入单独危机或单独机缘线。

## 可复用变体

- 变体1：众人合击祭坛核心，实际是在替沉睡傀儡充能。
- 变体2：破解幻境的“出口”其实是分割阵触发点，队伍被投放到各自心魔房间。
- 变体3：盗墓团队炸开主墓门，反而激活守墓机关，主角故意不提醒，等待反派拿钥匙。

## 注意事项 / 禁止照搬点

不要照搬“移花接木之阵”“木人场”“第三第七木人路线”等细节。可迁移的是：懂行者误判 → 集体执行 → 阵法吸收攻击反噬 → 空间分割 → 真机缘入口因混乱开启。
