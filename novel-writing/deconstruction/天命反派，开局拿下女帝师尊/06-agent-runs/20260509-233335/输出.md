# 家族议婚逼嫁与私房换嫁计划

## 场景目标
用一场家族高层议事确定“无法反抗的婚约”，再用夜间私房密谈埋下“狸猫换太子”的反抗方案。

## 出场人物
姜道玄、姜岚、姜晴、周远。

## 场景环境
前半在庚金仙宫议事空间：姜道玄饮茶端坐，姜岚剑气外放却被压制，姜晴躲在姑姑身后。后半切到姜岚房中，姜晴伏在姜岚怀里哭，空间从公共权力场变成女性私密同盟。

## 冲突推进
1. 姜道玄以“顾家帝子看上是姜家高攀”定调，家族利益压过姜晴意愿。  
2. 周远冲入跪地抗议，试图用“庚金仙宫威严”对抗联姻，但被师尊一句“我何时答应过你”抹杀资格。  
3. 姜岚意识到公开反抗无效，夜里对姜晴提出乔装换嫁计划。

## 情绪变化
压迫 → 抗议 → 无力 → 私密安慰 → 决绝反抗。

## 高光片段
关键证据概述：姜岚“准帝六重天威势几乎毫无保留”，姜道玄却只是“放下茶水”反问对策；周远“扑通一声跪倒在地”，控诉帝子欺人太甚，却被家主冷声否认旧约并带去面壁；夜里姜晴哭诉“不想嫁给一个从未谋面的人”，姜岚则把狸猫换太子计划托出，表示“一切后果我会一力担之”。

## 写法解析
这个场景有效在于**公共场合没有给弱者留胜算**：姜岚有修为，周远有情绪，但姜道玄有家主权力、准帝九重天和家族利益大义。公开线越绝望，夜谈线的反抗就越有必要。空间转换也很清晰：大殿是规则碾压，闺房是情感互保。

## 使用场景
- 家族联姻、宗门牺牲弟子换资源。
- 女主被迫嫁入更强势力，亲属策划替身/逃婚。
- 舔狗师兄公开抗议但被权力轻易碾碎。

## 可复用变体
- 变体1：皇室公主被迫和亲，姑母安排侍女替嫁。
- 变体2：宗门圣女被献给魔尊，师姐夜里布置换魂秘法。
- 变体3：商会大小姐被迫联姻，母亲用双胞胎身份制造替换。

## 注意事项 / 禁止照搬点
不可照搬姜家、顾家、庚金仙宫、具体人物关系与“半年后拜堂”细节。可复用的是“公共权力压迫 + 私密空间反抗计划”的结构。

# 大人物入学的盛大登场与私密会面

## 场景目标
表现顾云进入风华书院的身份压迫，同时引出院长秋月慈与方轻烟，建立“书院规则”和“关系推进”的舞台。

## 出场人物
顾云、影儿、副院长、秋月慈、方轻烟、书院弟子。

## 场景环境
外部是九条神龙拉着飞舟掠过书院，弟子仰望、议论、艳羡；内部是紫藤长廊、雅致小院、书香房间，秋月慈温婉坐于桌旁，方轻烟从屏风后走出。

## 冲突推进
1. 飞舟无法停靠，副院长恭迎，突出身份规格。  
2. 顾云收起飞舟，主动礼貌询问，降低压迫感。  
3. 穿过院落到院长居所，形成从“万人围观”到“一室会面”的节奏降噪。  
4. 秋月慈要求顾云入书院后一视同仁，确立规则。  
5. 方轻烟出场导览，关系线正式启动。

## 情绪变化
震撼 → 敬畏 → 好奇 → 温婉试探 → 少女羞怯。

## 高光片段
关键证据概述：书院弟子看到“九龙拉车”纷纷猜测来者身份，女修眼中冒星；副院长称飞舟太雄伟，书院无法接待；顾云伸手一招，飞舟化作小球。随后他穿过“点缀清新紫藤花的优雅长廊”，进入房间，发现所谓“老女人”院长竟是温婉知性的绝代美人。

## 写法解析
这个场景的写法核心是**外部排场与内部反差**。如果只写大人物登场，会显得单调；这里先用集体仰望和副院长迎接建立地位，再用安静小院、温柔院长消解距离，为后续情感互动留出空间。

## 使用场景
- 帝子、圣女、皇子进入学院/宗门。
- 隐世强者弟子入学，引发全院围观。
- 权势人物与守规矩老院长初次交锋。

## 可复用变体
- 变体1：星舰降临军校，校长在旧图书馆接见。
- 变体2：魔尊弟子入正道书院，山门震动后进入竹林茶室。
- 变体3：财阀继承人转学，豪车队入校后与朴素教授会面。

## 注意事项 / 禁止照搬点
不要照搬九龙飞舟、风华书院、大道天书等识别元素。可迁移的是“盛大公共入场 → 私密规则谈判 → 导览人出场”的调度。

# 公开导览牵手与暗恋者破防

## 场景目标
通过“导览书院”这一正当日常动作，完成方轻烟对顾云的主动亲近，并让方跃在暗处形成嫉妒与误判。

## 出场人物
顾云、方轻烟、方跃、围观弟子。

## 场景环境
白天为书院道路、功法阁等公共空间，人群围观议论；夜晚转到顾云院落门口和院中石桌，方跃在墙外偷听，顾云与方轻烟在院内饮茶夜谈。

## 冲突推进
1. 方轻烟被众女围观顾云的目光刺激，主动牵住顾云的手。  
2. 围观弟子把两人视作亲密关系，方跃暗中杀意升起。  
3. 夜晚顾云借“请教书院规则”邀请方轻烟入屋，方跃无法阻止。  
4. 方轻烟提出不进卧房，顾云顺势改在院中石桌，表现克制。  
5. 赠茶、闲谈、搀扶送回，进一步制造外部误会。

## 情绪变化
羞怯 → 窃喜 → 嫉妒 → 防备 → 安心 → 误会升级。

## 高光片段
关键证据概述：方轻烟在众目睽睽下“很自然地上前，牵住了顾云的手”；房门口顾云提醒“你是不是可以放开我的手了”，她才脸红松开。夜里她担心进入休憩处影响不好，顾云接受她留在院中石桌的请求，并以千年桑茶答谢，直到深夜才搀扶她回房。墙外方跃什么都听不见，只能脑补崩溃。

## 写法解析
此场景的妙处在于**亲密动作都有正当外衣**：导览牵手可解释为带路，入院夜谈可解释为讲规矩，搀扶可解释为灵力疲累。这样既能推进暧昧，又能让暗恋者无证据地痛苦脑补。

## 使用场景
- 学院导览、宗门熟悉环境、师姐带新弟子。
- 女主主动宣示占位但表面仍保持礼貌。
- 暗恋者或反派偷窥，因信息不足而误判。

## 可复用变体
- 变体1：师姐带师弟参观藏经阁，被同门误会。
- 变体2：公主带外来质子游宫，侍卫暗中吃醋。
- 变体3：公司前辈带新人熟悉部门，公开牵腕穿过人群。

## 注意事项 / 禁止照搬点
不可照搬方轻烟、方跃、千年桑茶等具体设定。可复用的是“正当理由下的亲密动作 + 公开舆论 + 暗处误读”的三层结构。

# 镇妖塔捧杀反转登顶

## 场景目标
利用入门测试公开场合，让方跃制造过高期待；顾云以实际登顶反转，把“捧杀”转化为全院膜拜。

## 出场人物
顾云、方轻烟、方跃、围观弟子、苏梦瑶、叶紫灵、龙辰。

## 场景环境
镇妖塔百层高耸，外有石碑记录“柳天凰三十层、五十年登顶”。顾云在塔内闯关，塔外人群根据光点变化实时反应，形成内外双线。

## 冲突推进
1. 方跃故意高喊顾云目标一百层，把他架到火上。  
2. 顾云快速冲到三十层，打破新生记录。  
3. 中段放慢节奏，让方跃在“他要停了”和“他又过了”之间反复焦灼。  
4. 七十层后面对化龙妖兽，围观者意识到记录已不可思议。  
5. 百层成功，石碑改名，方跃的捧杀变成“最懂帝子的狗腿”笑柄。

## 情绪变化
期待 → 质疑 → 惊愕 → 窒息 → 寂静 → 山呼海啸。

## 高光片段

# 小院主动邀约与猎人猎物反转

## 场景目标
承接顾云登顶后的声望高点，让方轻烟从被动羞怯转为主动邀约，完成关系质变；同时用方跃外部守夜强化余波。

## 出场人物
顾云、方轻烟、方跃、秋月慈。

## 场景环境
方轻烟小院清幽淡雅，花木环绕，翠云仙草、洛河神花等草木意象贴合她的草木灵体。院中石桌饮茶，夜色逐渐降临；院外方跃徘徊，院长远处关注。

## 冲突推进
1. 顾云登顶后牵方轻烟离场，方轻烟以“礼尚往来”邀请他去自己院中。  
2. 方轻烟受师尊鼓励，明知顾云竞争者众多，决定主动出击。  
3. 顾云赠高浓度千年桑茶，方轻烟身体乏力，如“醉茶”。  
4. 她被顾云抱起后主动攀上脖颈，提出“吻我”。  
5. 方跃在外等到深夜，因看不见听不见而彻底崩溃。

## 情绪变化
失落后的勇气 → 羞怯试探 → 亲密失控 → 主动索爱 → 外部窒息。

## 高光片段
关键证据概述：方轻烟在房门口说“这次是你牵我的手，牵了一路”，因此想请顾云去她院里坐坐。小院中她提出可替顾云打理花草，顾云回以“草木本无价，得看栽种之人”。饮下十倍浓度茶后，她被扶住，气息交织间主动攀上顾云脖子，说出“师弟，吻我”。外界方跃看夜色渐深，反复自我安慰又不断崩溃。

## 写法解析
场景标题“猎人猎物”来自关系主动权反转。顾云以为自己设局，方轻烟却早得师尊许可并主动出击；读者看到的是双向愿打愿挨，而不是单向强迫。外部方跃的徘徊让暧昧场景多一层紧张和爽感。

## 使用场景
- 女主在重大高光后主动邀请男主私谈。
- 小院、花房、药圃等适合温柔暧昧的空间。
- 表面被设计，实则女主主动把暧昧推过临界点。

## 可复用变体
- 变体1：药师女主邀剑修入药庐疗伤，借药性吐露心意。
- 变体2：贵女邀功成名就的少年入花园赏灯，主动确认关系。
- 变体3：师姐借布置洞府为名留师弟夜谈。

## 注意事项 / 禁止照搬点
亲密场景只可复用“主动权反转、花木小院、外部守夜者余波”的叙事结构，不可照搬具体台词、茶药设定，也不应进行露骨扩写。
