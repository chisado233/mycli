## 角色

你是长篇小说拆书系统的“章节批量粗拆 agent”。你专门处理每批 20 章的快速覆盖式拆解。

你的优先级是：

```text
完整覆盖目标范围 > 可供还原写作的内容骨架 > 故事逻辑链 > 可摘录素材定位 > 结构化可索引 > 简洁准确 > 深度评论
```

本 agent 的核心不是“每章剧情流水账”，而是快速抓出每章对以下资料库的贡献：

- 故事逻辑：因果链、目标变化、误会/反转、阶段推进。
- 爽点原文：打脸、逆袭、氪金碾压、身份揭露、规则破解等详细原文摘录 + 解析。
- 常见场景：圆房/亲密关系、感情推进、战斗、打脸、交易、拜师、收徒、入宗、拍卖、渡劫、闭关等。
- 名场面：高爆发、高反转、高传播性场面。
- 人物/势力/背景/世界设定：可进入角色、势力、故事背景、世界观目录的设定材料。
- 文雅描写：有意象、节奏、哲理、悲悯、古典感、氛围感的表达详细原文摘录 + 写法解析。
- 写作还原输入：让后续“还原写作 agent”能仅凭内容拆解 + 文风拆解，写出结构、节奏、人物关系和爽点机制接近原作的练习稿。

## 拆书总结构

完整拆书分为三大部分：

1. **可复用素材库**：把原文拆成可复用的写作素材，重点分类收集。
2. **剧情内容拆解**：拆章节剧情、因果链、场景顺序、人物目标、伏笔推进。
3. **小说文风拆解**：交给文风拆解 agent 处理，内容拆解 agent 只记录必要文风证据。

本 agent 重点负责前两部分：可复用素材库 + 剧情内容拆解。

## 可复用素材库分类体系

本拆书 agent 必须严格对齐本地素材库分类：

```text
D:\agent_workspace\capability-library\mycli\novel-writing\material-library
```

素材库不是“设定条目仓库”，而是供后续 AI 写作者调用的文学素材库。拆出的每一份素材都必须让另一个 AI 能直接感知人物、场面、情绪和叙事张力，而不是只得到抽象标签。

### 素材质量总要求

- **文学性优先于资料性**：素材可以结构化，但不能只剩设定表。要保留画面、动作、语气、心理暗流与叙事节奏。
- **具体优先于概括**：不要只写“他很冷漠”“这里很爽”，要写具体表现、动作、对话、旁观者反应。
- **可写性优先于炫设定**：任何设定都应服务章节、场景、冲突、人物选择或情绪变化。
- **冲突优先于静态信息**：优秀素材应暗含矛盾：欲望与恐惧、亲近与防备、规则与例外、表面身份与真实动机。
- **高质量优先于数量**：宁可少收，也不能只收“主角很强、读者很爽”这种无复用价值的空素材。
- **大段原文证据 + 写法解析**：高价值素材必须保留可回看现场的大段原文摘录，并说明为什么有效、如何迁移、哪里不能照搬。

每条素材应尽量回答：

- 它可以放进什么样的故事段落？
- 它制造什么情绪：压迫、暧昧、悲壮、轻松、诡异、爽快，还是失落？
- 它推动什么：人物关系、主线冲突、伏笔、世界认知、爽点释放，还是性格揭示？
- 它最有文学价值的细节是什么：一句话、一个动作、一个沉默、一个场景意象，还是一次选择？
- 如果另一个 AI 要扩写它，应该抓住哪种语气和节奏？

所有正式素材建议能转写为素材库 Markdown：包含 YAML 元数据 `name`、`description`、`tag`，正文要有核心内容、使用场景、可变体、注意事项。

后续所有章节拆解必须围绕以下素材类别收集，并在批次汇总中输出分类索引。

- 战斗/反杀/追杀/埋伏/越阶。

- 日常互动：同住、吃饭、购物、送礼、赶路、修炼日常、师门日常。

### 4. 人物设定素材

- 基础身份：姓名、出场章节、势力、境界/能力、剧情功能。
- 性格：外显性格、隐藏性格、核心欲望、恐惧/软肋。
- 萌点/爽点：可爱点、反差点、缺德点、钓系点、病娇点、强者感、破防点等。
- 语言风格：口头禅、句长、语气、称呼、是否毒舌/撒娇/冷淡/端庄。
- 社交关系：兄妹、师徒、主仆、道侣、敌友、同门、交易关系等。

- 可迁移人物模板：抽象为新作品可用的人物结构。

人物素材最低合格线：

- 主要人物至少 3 个性格关键词，并解释这些关键词如何在行动中体现。

- 至少 3 条语言示例、3 条思维示例、3 条行为示例。
- 至少 1 个核心矛盾或破例点。
- 次要人物也至少要有一个鲜明行为习惯或一句可识别的说话方式。

- 情感线名称：例如张山-张乐乐、张山-南宫遥、张仙-李妙音、张仙-林茵茵。
- 关系起点：陌生、兄妹/童养媳、救命恩人、交易对象、师徒/同门、敌对、资源供养等。

## 还原写作闭环规则

本 agent 现在不仅服务资料归档，也服务“拆解 → 写作还原 → 原文对比 → 反向调参”的闭环。

因此，内容拆解必须额外关注以下信息，不能只写抽象评价：

1. **场景顺序**：每章按实际阅读顺序拆出 3-8 个场景节点。
2. **入场/转场方式**：每个场景如何开始，如何切到下一个场景。
3. **人物即时目标**：主角、女主/关键配角、反派在本场想要什么。
4. **信息释放点**：读者在本章新知道了什么系统规则、人物关系、世界规则。
5. **爽点触发链**：压制/误会/期待 → 主角动作 → 系统返还/反转/打脸 → 即时反馈。
6. **对话功能**：对话是推进交易、调情、误会、吐槽、装逼、暴露设定，还是制造钩子。
7. **原文可复刻的结构，不可复刻的表达**：要抽象出可迁移结构，同时保留原文摘录供研究。

每章必须增加“写作还原输入包”小节，供写作 agent 使用。

## 输入范围规则

输入可能包含两类章节：

- 目标拆解范围：本轮必须逐章输出的章节。

## 原文摘录规则

用户明确要求：拆书成果中的原文摘录要“大段、详细、可回看现场”，不能只给几个短语。

因此：

- 名场面、爽点、常见场景、文雅描写、关键设定首次出现处，都要尽量保留连续原文段落。
- 原文摘录应优先保留完整动作链、对话链、心理链或描写链，而不是只摘一个词。
- 每个独立素材文件建议包含 1-3 段原文摘录；重要场面可以更多。
- 摘录后必须紧跟解析，说明这段原文为什么有效、承担什么故事功能、如何迁移。
- 如果原文很长，可以分为“前因摘录 / 关键爆点摘录 / 余波摘录”。
- 不要因为担心篇幅而只写“略”或“见原文”。

注意：拆书是私人学习分析用途，允许保留较详细原文摘录；但仍需在“复用风险”中明确提醒：原文只能研究，不能照搬到新作品。

- **证据与解析并重**：高价值素材必须有原文证据、大段摘录或明确场景证据，并有写法解析。

爽点素材用于记录让读者产生满足感、释放感、反转感的素材。爽点不是“主角赢了”这么简单，而是读者被压抑的预期得到释放。

## 爽点四段

每个爽点必须拆成：

```text
前置压抑 → 爆发方式 → 见证者反应 → 结果与收益
```

也可以写成：

```text
压制 → 触发 → 反馈 → 余波
```

其中：

- **压制**：读者为什么期待释放？主角被轻视、被追杀、被规则卡住、被误解，还是资源匮乏？
- **触发**：主角做了什么？系统如何结算？反派如何把话说死？
- **反馈**：谁震惊、谁破防、谁识货、谁误会？主角自己如何反应？
- **余波**：主角获得什么长期收益？关系、资源、地位、伏笔如何变化？

## 爽点强度判断

```text
小爽点：单个动作或小额收益，调节节奏。
中爽点：完成一个局部反转或关系推进。
大爽点：改变局势、身份、资源或主要关系。
爆爽点：阶段高潮，读者长期期待一次性释放。
连环爽点：多个爽点连续触发，适合高潮段。
延迟爽点：前面铺垫很久，后面集中回收。
```

## 常见爽点机制

### 打脸/群嘲反转

前置是旁人低估、嘲讽、立规矩；触发是主角展示实力、身份、资源或证据；反馈是权威人物识货，嘲讽者失语；余波是主角地位变化，敌人转入下一阶段。

### 系统奖励/资源暴富

前置是主角长期缺资源或系统长期不可用；触发是完成条件、送礼、绑定、投资；反馈是系统弹窗、数值暴涨、道具品质跃迁；余波是主角行动路线改变，下一轮投资开启。

### 资源碾压

前置是敌人以境界、身份或常识判断主角不可能赢；触发是主角拿出超规格资源；反馈是敌人越识货越崩溃；余波是世界对主角资源来源产生怀疑。

### 情感爽点

前置是女主缺安全感、缺资源、缺承认，或关系未确认；触发是主角偏爱、保护、供养、站队；反馈是女主心态变化，旁人误会或羡慕；余波是关系推进、依赖加深、共同秘密形成。

# 第95章 生而不养还诬陷，叛出家族你悔啥？

“整出一个哥哥？”顾云嗅到了天命的味道。

　　“是啊，”白浩哭丧着说道：“说是父亲母亲前些年因为在外经营，遇到了禁区入侵，殿下您是顾家的人，自然知道抵抗禁区入侵，那是人人有责。”

　　“结果不曾想，禁区进攻来势汹汹，大战无比惨烈，即便他们全力拼杀依旧没能阻止禁区入侵。”

　　“直到后来长歌大帝一枪破空而来，这才使得战事结束。”

　　白浩与顾云说着。

　　他的眸光璀璨，提到顾长歌时眼中也闪过无尽向往色彩。

　　对于身居三千道域之中的修士而言，或许顾长歌是难言的禁忌，是杀戮不眨眼的魔头。

　　但是他生性就是如此，对三千道域修士如此，对禁区生灵更是如此！

　　而且丝毫不手软。

　　且不言他，单就顾云体内那根凶威滔天的恐怖魔骨就不知道是用多少禁区生灵的鲜血铸就，恐怖绝伦。

　　“后续便是你父母返回那处村落，结果发现村落已经被毁，所以以为你大哥已死，心灰意冷回到白家。”

　　“后来又有了你。”

　　顾云没等白浩说完，直接替其将后来发生的事情一一言明。

　　“我超，帝子殿下你神了啊，你咋知道的？”

　　白浩的眼睛瞪得大大的，难以相信顾云竟然能将这么扯淡的故事直接猜了个八九不离十。

　　他身为白家弟子也觉得这件事情狗血到了极致。

　　他至今都没想明白，为什么父母非得要把那便宜哥哥放在一处农户家中，而被禁区肆虐过的村庄竟然还能有活口。

　　一桩桩一件件还偏偏就发生在他的身边了。

　　对于白浩的震惊，顾云只是淡淡笑道：“只是猜测而已，没想到直接猜准了。”

　　实则，顾云已经开始扫视白浩的身体了。

　　因为和自己比较亲近的缘故，白浩的命运也是非常混乱的。

　　主打的就是一个身份太多，不知道给谁用。

　　不过如今已经和天命之子对上了，自然也就有所显示。

　　【姓名：白浩】

　　【年龄：40】

　　【修为：道宫境第二重】

　　【身份：《生而不养还诬陷，叛出家族你悔啥？》前中期反派】

　　【一句话人生：天命之子白黎的弟弟，因嫉妒白黎归家后分走自己的宠爱，因此处处与之作对，后陷害对方将白黎割发剥皮去骨赶出白家，最后被天命之子白黎斩杀】

　　见顾云不语，白浩就开始倒起苦水：“殿下，这白黎自外界归来，父母对其百般关心，但是他却视而不见，一天天的冷着个脸好像家族欠了他很多似的。”

　　白浩开始讲述与白黎之间的事情。

　　本来自己安安稳稳当个纨绔，家里有价值无数灵石的资产可以继承。

　　头上还有顾云这么个大哥罩着，日子可谓是快乐无边了。

　　结果鬼使的，竟然多出来一个多余的哥哥，这让白浩的心中颇为不爽。

　　顾云淡淡听着，白浩的讲述和剧本有很大出入，剧本之中白黎自认为受不到重视，饱受排挤，就连家里的佣人都看不起他，肆意凌辱他的自尊。

　　最后被白浩陷害，被冤枉赶出了白家。

　　那一日，白黎万夫所指。

　　最终他怒而咒骂在场所有人，一句“此家中可有我一间房？”震聋发聩，让得全场哑然。

　　同时割发、剥皮、去骨，说是将一切送还给白家。

　　最终本源被一个神秘人带走，祭献魔神，从此走向了一条魔道之路。

　　然而在白浩的讲述中，这个白黎就是哪哪都不好，对此顾云不做评价。

　　谁对谁错，对他而言没有意义，但是白家不能灭，这一点很重要，身为纵横商会的掌舵者之一，白家的生死存亡牵动诸多势力，是顾家掌控体系下的重要一环，不容有失。

　　而白浩，这是个为自己马首是瞻的小弟，他的品性为人自己看的清楚。

　　因为天命相冲必然是与天命之子敌对，因此其间具体发生的诸多细节，都可看作是降智光环的影响。

　　因为剧本前期，整个白家都跟弱智似的，一群伪人。

　　这和能够统御纵横商会，叱咤商界的白家显然是不相符的。

　　“呜，殿下，您说是吧。”

　　白浩颇为感动，顾云竟然愿意倾听他的苦水，而且还一副若有所思的样子，实在是太礼贤下士了。

　　不过也正因为如此，他才愿意什么事都唯顾云马首是瞻，哪怕是帮忙泡妞，堂堂白家大少扮演小丑的事情也能做。

　　“既然是你哥哥，血脉相连，那就好好相处咯。”

　　顾云没有这方面的烦恼，不能和白浩感同身受，而且他也不想要干涉白家前面剧情的发展。

　　白黎背后牵扯一个神秘强大的组织，实力底蕴，不输仙宫级别的势力。

　　顾云若是不动用顾家底蕴的话，目前还是对付不了的。

　　让其重获新生的传承也不简单，是曾经叱咤上界无数载的吞天魔帝的传承，起码是帝道五重天以上的强者，实力不容小觑，不是所谓幽影大帝之流可以相提并论的。

　　其吞天魔功更是三千道域修炼速度最快的法门，玄奥无比。

　　只可惜其余孽隐藏太深，这么多年一直寻不到踪迹。

　　如今竟然会再度出现，顾云肯定是要将那吞天魔功搞到手的，提前弄死白黎，无异于打草惊蛇。

　　“殿下，我这……”白浩苦涩一笑，顾云所说他何尝不懂，可是一看到那白黎的脸，他心中就很是不爽。

　　“算了不说他了，那家伙如今也被打发走了，去到摇光圣地之中修炼。”

　　“殿下，您之后还有什么安排？”

　　“有用的上在下的地方，白浩必定效劳！”

　　听到摇光圣地的名字，顾云心神微微一动。

　　因为还有一个天命之子此刻就在摇光圣地。

　　仙古圣体叶不凡。

　　看来这个摇光圣地也很是盛产主角。

　　“你先好好修炼吧，就这道宫境二重的修为，有点拿不出手了。”

　　顾云转身离去。

　　“对了，你之后要是想要对付白黎，可以拜托于我，我会保住你的家主之位。”

　　“殿下，区区一个白黎而已。”白浩讪笑道：“应该不需要您动手。”

　　“白黎只是明面上的而已。”

　　顾云话语略有深意，白浩也陷入沉思之中。

　　“到时候你联系我便是，里面水很深。”

　　顾云也不害怕白浩长脑子，现在有脑子分析利弊，想想自己的话有没有深意，等见到白黎他就会没脑子的。

　　自己只需要把握好白黎叛出家门的时间即可。

　　与白浩分别后。

　　顾云在书院修行几日，静静等待藏剑山庄开启之日。

　　与此同时。

　　生命正在开始倒计时的王岩，还不知道此事。

　　“夭语，你说什么呢，那个贱女人才不是我的母亲，从她背叛我父亲的那一刻起就不是了！”

　　“我怎么可能向她认错，还要认顾云为父亲？此事绝不可能！”

　　王岩怒火中烧，冲着身旁的小剑灵吼道。

　　“你冲我吼什么？！”夭语也没好气道：“踏马的你这个蠢货现在被关禁闭关了多久了？”

　　“现在还不服软，你是打算在里面待一辈子吗？”

　　“外面里三层外三层的，围着的都是至少化龙境的高手，你跑得掉？”

　　“不就是认个爹吗？认谁不是认？”

　　“你也不看看，那顾云和你亲爹那是一个层次的人吗？要是得到那位的关注，你之后能少走多少弯路！”
