# 人物设定素材分类索引

## 收集字段

| 字段 | 说明 |
|---|---|
| 基础身份 | 姓名、出场章节、势力、境界/能力、剧情功能 |
| 性格 | 外显性格、隐藏性格、核心欲望、恐惧/软肋 |
| 萌点/吸引力 | 可爱点、反差点、钓系点、强者感、破防点、缺德点 |
| 语言风格 | 口头禅、句长、语气、称呼、毒舌/撒娇/冷淡/端庄 |
| 社交关系 | 兄妹、师徒、主仆、道侣、敌友、同门、交易关系 |
| 关系型张力 | 兄妹感、师徒禁忌感、救命恩人、资源供养、误会、身份差、年龄差、强弱差 |
| 互动模式 | 谁主动、谁试探、谁误会、谁破防、谁获益 |
| 可迁移模板 | 新作品可复用的人物结构 |
