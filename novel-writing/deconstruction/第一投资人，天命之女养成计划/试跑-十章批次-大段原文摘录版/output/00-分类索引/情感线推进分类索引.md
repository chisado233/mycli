# 情感线推进分类索引

情感线推进专门记录“人物关系如何一步步变近、变复杂、变暧昧、变敌对或完成确认”。它和单个场景不同，重点是跨章节的关系阶段变化。

## 收集字段

| 字段 | 说明 |
|---|---|
| 情感线名称 | 例如张山-张乐乐、张山-南宫遥、张仙-李妙音、张仙-林茵茵 |
| 关系起点 | 陌生、兄妹/童养媳、救命恩人、交易对象、师徒/同门、敌对、资源供养等 |
| 推进节点 | 相遇、救助、送礼、误会、同住、共同战斗、坦白、背叛感、保护、关系确认 |
| 情绪变化 | 警惕、好奇、依赖、感激、暧昧、吃醋、羞恼、信任、占有、破防 |
| 张力来源 | 身份差、年龄差、强弱差、师徒/兄妹伦理、资源供养、误会、救命恩情、共同秘密 |
| 关键场景 | 每个节点对应的场景文件或章节 |
| 语言/称呼变化 | 称呼、语气、对话距离如何变化 |
| 主动权变化 | 谁主动靠近，谁后退，谁试探，谁误会，谁破防 |
| 可迁移模板 | 可复用的关系推进模型 |

## 可迁移情感线模板

| 模板 | 结构 |
|---|---|
| 贫苦依赖型 | 童年/同住/弱者依赖 → 主角保护 → 天命伏笔 → 后续觉醒 |
| 强者战损型 | 高位强者跌落 → 主角救助 → 系统绑定/交易 → 强弱关系反复 |
| 资源供养型 | 女方现实需求 → 主角高规格投喂 → 女方实力提升 → 情感与利益绑定 |
| 误会喜剧型 | 主角真实动机功利 → 配角理解成风流/阴谋 → 误会加深 → 笑点和阻力并存 |
| 共同战斗型 | 女方被迫站队 → 主角给底牌 → 配合反杀 → 信任升级 |
