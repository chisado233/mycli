# 拆解说明

## 拆解对象

《第一投资人，天命之女养成计划》

## 拆解目标

验证并完善小说拆书 agent 流程：先用前 3 章进行试拆，检查输出是否满足“结构同构、可追溯、可迁移、含高光素材解析”的预期，再根据审核结果精修提示词。

## 拆解范围

首轮：第 1-3 章。

后续：根据试拆效果扩展到前 10 章、第一阶段、第一卷。

## 拆解方式

- 使用 `mycli novel-writing agent run <request.json>`。
- 所有请求 JSON 存入 `11-调度与拆解/requests/`。
- 所有模型运行记录存入 `11-调度与拆解/agent-runs/`。
- 所有人工判断和提示词修订存入 `10-拆解记录/`。

## 使用的 agent

- 章节粗拆 agent
- 审核评估 agent
- 后续可能使用：高光素材摘录 agent、阶段汇总 agent、作品企划反推 agent。

## 产物说明

本项目目录是从 `deconstruction/templates/拆书成果模板` 复制而来，结构尽量对应小说项目模板。
