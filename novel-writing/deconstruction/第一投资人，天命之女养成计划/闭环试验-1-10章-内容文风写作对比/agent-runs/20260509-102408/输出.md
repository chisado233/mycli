# 文风总览

## 1. 核心叙述气质

第1-10章的文风核心是：

> **修仙爽文底层逻辑 + 现代社畜吐槽旁白 + 好色但不完全下流的男频喜剧视角 + 系统返还爽点驱动 + 反差误会型沙雕群像。**

叙述并不追求传统仙侠的庄重，而是经常用现代网文、社畜、游戏重开、舔狗、装逼、卡BUG、老牛吃嫩草等口语化认知，去解构修仙世界。

主角张山/张仙的视角非常强，叙述者经常贴近他的心理活动，甚至直接替他吐槽世界、系统、女人、修仙门槛和自身处境。整体气质不是正经升级流，而是：

- 表面：长生、修仙、天命之女、返还系统。
- 内核：一个脸皮厚、欲望直给、脑子灵活的老社畜，在修仙世界到处薅羊毛。
- 表达：正经叙事几段后，立刻用一句粗俗吐槽或沙雕比喻破功。

## 2. 叙述者人设

叙述者高度贴近主角，但不是完全第一人称。常见模式：

1. 第三人称交代事实。
2. 切入主角内心吐槽。
3. 用现代口语评价当前局面。
4. 再回到情节推进。

典型效果：

- 主角被困大山十五年，本可写悲凉，但叙述马上用“系统垃圾”“怀念996”“重新开一把”消解沉重。
- 修仙美女坠落，本可写仙气，但马上切到主角“战损装受不了”“好感度下降”“狗女人太拽”。
- 送礼返还，本可写玄幻法宝降临，但马上切到“这波血赚”“薅秃”“系统牛逼”。

叙述者与读者的距离较近，像在讲一个男频爽文段子，而不是严肃讲古典仙侠。

## 3. 节奏总特征

整体节奏非常快，尤其前10章承担大量设定转换：

| 阶段 | 章节 | 节奏功能 |
|---|---|---|
| 大山困局 | 1章 | 穿越失败、系统无法用、张乐乐伏笔、南宫遥坠落钩子 |
| 第一次返还 | 2-3章 | 天命女绑定、送匕首触发返还、小黑丫头离开、南宫遥杀主角反转 |
| 凡间时间跳跃 | 4-5章 | 张仙走出大山，在大梁成世俗巅峰，寿命孤独，引出新天命女 |
| 道观返还副本 | 6-9章 | 李妙音绑定，误会喜剧，送礼返还，获得修真功法 |
| 地图切换 | 10章 | 交代云渺宗线索，进入修真大世界，引出林茵茵新场景 |

每章通常有明确的“新变量”：

- 系统提示。
- 新天命女。
- 新资源返还。
- 新误会。
- 新地图。
- 新敌意/钩子。

## 4. 笑点机制总览

笑点主要来自五类：

1. **主角欲望直给**  
   看美女、想返还、想薅羊毛、想活命，心理非常不端着。

2. **现代词汇撞进古风世界**  
   如重开、天胡开局、舔狗、卡BUG、血赚、装逼、工具人、引狼入室等。

3. **身份反差**  
   修仙者仙风道骨，内心也会吃醋、误会、破防；老道士年轻后像小白脸。

4. **误会升级**  
   玉昀误以为张仙要抢李妙音，脑补出“师父要被绿”的连续笑点。

5. **粗俗比喻和反套路评价**  
   如“狗分享大便”式的反差比喻，把高雅场景强行拉低。

笑点密度中高：每章至少3-6处明显笑点，第7-9章因误会集中，笑点更密。

## 5. 爽点机制总览

爽点不是单纯战力碾压，而是“被看低—送礼—返还—反向掌控”的循环。

常见爽点公式：

> 主角处境低 / 被仙女鄙视 / 被系统卡住  
> → 主角厚脸皮试探送礼  
> → 系统返还高级资源  
> → 主角暗爽，关系逆转  
> → 配角震惊或误会升级  
> → 主角获得下一阶段入场券。

第1-10章核心爽点包括：

- 被困大山十五年后，终于绑定南宫遥。
- 次品短匕返还法器，新手礼包送神品丹药和空间宝库。
- 凡人张山在大梁成为世俗巅峰。
- 百年雪参返还千年灵参，黄金返还灵石。
- 凝丹诀返还青木养丹经，主角无灵根也踏上修真路。
- 大梁小世界切到修真大世界。

## 6. 对话总风格

对话偏短，信息密集，常常夹杂心理吐槽。

人物口吻区分明显：

- **张山/张仙**：脸皮厚，客气外壳下全是算计；表面拱手，内心薅羊毛。
- **张乐乐**：依赖、娇憨、自卑又执拗，常用“山哥哥”。
- **南宫遥**：清冷、高位、淡漠、压迫感强，前期看不起张山。
- **李妙音**：温和、克制、成熟，有道姑气质，但感情线中会害羞。
- **玄微**：先仙风道骨，后恋爱脑和吃醋小心思暴露。
- **玉昀**：误会型吐槽担当，内心戏夸张，承担连续喜剧。

## 7. 文雅与通俗的切换

作品会写文雅描写，但几乎不长期维持。

常见切换方式：

1. 先写仙子外貌、月色、山巅、落日、道观茶室等文雅氛围。
2. 下一段立刻用主角粗俗心理或现代吐槽破功。
3. 让读者既获得“仙侠美感”，又不会觉得文本端着。

文雅描写承担：

- 美女惊艳。
- 地图升级。
- 岁月孤独。
- 修真逼格。
- 情绪铺垫。

通俗吐槽承担：

- 降低阅读门槛。
- 提供笑点。
- 强化主角市井、社畜、老油条气质。
- 防止剧情苦大仇深。

## 8. 章节钩子总览

结尾钩子强，通常是：

- 系统提示新气运女。
- 爽点后反杀/反转。
- 新人物登场。
- 地图切换。
- 误会升级。

例如：

- 第1章结尾：刚承诺娶张乐乐，系统提示南宫遥。
- 第3章结尾：被南宫遥杀死，消耗复活机会，仇恨线启动。
- 第5章结尾：玄微点破张仙无法修真，引出求道难题。
- 第7章结尾：玉昀误会“师伯危”，形成喜剧钩子。
- 第10章后段：正式进入修真大世界，引出林茵茵矿花场景。

## 9. 给后续写作的核心原则

后续还原写作应抓住以下要点：

1. **第三人称，但心理贴主角。**
2. **仙侠设定必须被现代口语解构。**
3. **每章至少一个系统/资源/误会/新人物驱动点。**
4. **美女描写可以文雅细致，但必须快速接主角欲望或吐槽。**
5. **爽点要靠返还系统完成“低成本换高回报”。**
6. **配角反馈要夸张：震惊、破防、误会、吃醋、鄙视。**
7. **结尾必须有钩子，不能平收。**

# 叙述节奏

## 1. 可量化参数表

| 项目 | 观察 | 写作建议 |
|---|---|---|
| 叙述人称 | 第三人称为主，但大量贴近张山/张仙心理 | 用第三人称推进剧情，关键判断、欲望、吐槽切入主角内心 |
| 叙述距离 | 近距离吐槽旁白 | 旁白可以像主角嘴替，适当使用现代词、粗口、网文梗 |
| 长短句比例 | 中短句为主，偶尔长句铺设定或描写美女 | 动作和吐槽用短句；设定和外貌用中长句 |
| 单句成段 | 高频使用 | 爆点、吐槽、系统提示、情绪转折可单独成段 |
| 段落长度 | 多为1-4句一段 | 避免大段说明，保持网文阅读滑动感 |
| 设定说明节奏 | 夹叙夹议，随情节触发 | 不要开篇百科；让系统、对话、主角判断带出设定 |
| 爽点前铺垫 | 通常2-6段压低或误判 | 先让主角憋屈、被鄙视、资源不足，再返还 |
| 爽点触发 | 系统提示+主角心理爆炸 | 系统弹窗要短促，之后接主角“血赚/发了/牛逼”式反馈 |
| 爽点后反馈 | 2-5段 | 配角震惊、好感变化、误会加深、主角继续薅 |
| 笑点频率 | 每章3-6个明显笑点 | 不必每段搞笑，但每个场景要有至少一个吐槽落点 |
| 章节结尾 | 强钩子 | 系统提示、新人物、反转、死亡、误会升级、地图切换皆可 |

## 2. 句式节奏拆解

### 2.1 开篇常用“好消息/坏消息”或反差式开局

第1章开头用“好消息/坏消息”快速建立反差：

- 好消息：穿越且带系统。
- 坏消息：系统没法用。

功能：

- 三句话内交代设定。
- 迅速抛出矛盾。
- 语气像现代段子，奠定沙雕吐槽基调。

可迁移方式：

> 好消息：主角获得了某个逆天条件。  
> 坏消息：这个条件暂时毫无用处，甚至反过来折磨主角。

### 2.2 爆点前短句加密

遇到系统提示、美女出现、返还成功时，句子明显变短：

- “又来？”
- “有戏！”
- “问题不大。”
- “卧槽发了呀！”
- “系统牛逼！”

功能：

- 模拟主角瞬间反应。
- 让节奏有弹跳感。
- 给读者明确的爽点/笑点信号。

写作建议：

- 爆点前后少用复杂长句。
- 让主角用一句短促口语给情绪定调。

### 2.3 文雅描写中插入口语破功

外貌和环境描写可以写得较精致，但结尾常接粗俗心理。

模式：

> 先写女子衣发、肌肤、身段、光影。  
> 再接主角“受不了”“这谁顶得住”“好感度又掉了”之类反应。

功能：

- 保留男频审美刺激。
- 防止文风变成纯古风。
- 强化主角好色但接地气的人设。

## 3. 段落节奏拆解

### 3.1 一个场景的基本结构

常见场景结构：

1. 交代地点/时间/状态。
2. 主角观察。
3. 主角心理吐槽。
4. 对话触发冲突。
5. 系统或动作带来转折。
6. 配角反馈。
7. 主角做下一步算计。

例如道观送礼场景：

- 张仙先赔礼。
- 李妙音、玄微震惊空间法器。
- 系统返还灵参。
- 张仙继续送黄金珠宝。
- 系统部分返还，说明规则。
- 张仙再掏灵石。
- 李妙音态度转好。
- 玉昀误会升级。

### 3.2 对话、动作、心理比例

| 成分 | 占比倾向 | 写作建议 |
|---|---|---|
| 对话 | 35%-45% | 推动关系和误会，尽量短促 |
| 主角心理 | 20%-30% | 用来吐槽、算计、欲望表达 |
| 动作 | 15%-20% | 战斗、送礼、逃跑、反应 |
| 设定说明 | 10%-20% | 通过对话和主角判断分散 |
| 环境/外貌 | 10%-15% | 美女登场和地图切换时加强 |

## 4. 时间跳跃节奏

作品大胆跳时间：

- 第1章十五年困山。
- 第3章直接五年后。
- 第4章三十多年过去。
- 第5章张仙一百二十岁。

但每次跳跃都有明确目的：

1. 快速压缩凡间成长。
2. 制造岁月孤独。
3. 让主角身份从弱者变为世俗大佬。
4. 为下一地图升级铺垫。

写作建议：

- 时间跳跃不要写流水账。
- 用“三段式成就+一个遗憾”概括。
- 比如：权势、财富、美人都得到了，但修真门槛仍过不去。

## 5. 爆点前后的句长变化

| 阶段 | 句长 | 功能 |
|---|---|---|
| 爆点前 | 中长句铺垫 | 解释困境、误判、压抑 |
| 爆点瞬间 | 极短句/系统弹窗 | 形成阅读停顿 |
| 爆点后 | 短句+心理狂喜 | 放大爽感 |
| 余波 | 中短句 | 配角震惊、关系变化、误会加深 |

示例抽象：

> 主角送出一个不起眼物品。  
> 系统连续三条提示。  
> 主角先懵，再狂喜，再立刻思考如何继续薅。  
> 旁人完全不知道发生了什么，只觉得主角笑得不正常。

# 吐槽与笑点

## 1. 吐槽总机制

本书吐槽不是外加段子，而是和剧情推进绑定。吐槽通常出现在：

1. 主角发现系统坑爹时。
2. 美女出现但好感度下降时。
3. 修仙者端架子时。
4. 主角送礼薅羊毛时。
5. 配角误会主角动机时。
6. 严肃场面被市井逻辑破坏时。

核心笑点来自：

> **高逼格仙侠场景 + 低俗/现代/现实主义解释。**

## 2. 吐槽来源分类

| 类型 | 表现 | 功能 | 写作建议 |
|---|---|---|---|
| 系统吐槽 | 系统多年不能用、绑定条件苛刻、返还规则坑 | 建立主角憋屈感 | 主角可骂系统，但系统一返还立刻改口“系统爸爸” |
| 主角自嘲 | 怀念996、想重开、凡人无灵根 | 降低主角高大感 | 主角越惨越能吐槽，不要端着 |
| 欲望反差 | 看美女、想薅资源、想吃软饭 | 男频喜剧和爽点 | 心理可直给，但行动要有分寸和算计 |
| 身份反差 | 修真高人吃醋、老道变小白脸、道士哭师姐 | 破除仙侠严肃 | 高人也可以破防、恋爱脑、误会 |
| 误会脑补 | 玉昀怀疑张仙抢师伯 | 连续章节喜剧钩子 | 让配角越想越离谱，但逻辑自洽 |
| 现代网文梗 | 天胡开局、舔狗、卡BUG、血赚、工具人 | 拉近读者距离 | 适量使用，避免每句都是梗 |
| 粗俗比喻 | 高雅事物被比作低俗物 | 强烈反差笑点 | 关键处使用，别过密 |

## 3. 典型笑点结构

### 3.1 “严肃铺垫—粗俗破功”

结构：

1. 先写仙子、法器、宗门、道观等高逼格内容。
2. 主角或旁白突然用粗俗现实比喻评价。
3. 严肃氛围瞬间塌掉。

模板：

> 某人/某物看起来极其高贵、神秘、不可触碰。  
> 主角心里却只冒出一个非常现实甚至不雅的念头。  
> 旁白再补一刀，说明他为什么会这么想。

适用场景：

- 美女登场。
- 高人赐宝。
- 宗门设定。
- 主角面对强敌。

### 3.2 “系统坑爹—系统爸爸”

结构：

1. 主角长期被系统折磨，骂系统垃圾。
2. 主角偶然触发返还。
3. 系统送出远超预期的东西。
4. 主角立刻改口，情绪反转。

模板：

> 前一秒：这破系统迟早害死我。  
> 下一秒：系统返还逆天资源。  
> 主角：误会，都是误会。

功能：

- 强化系统爽点。
- 形成主角没节操但可爱的形象。

### 3.3 “配角脑补误会升级”

玉昀线是前10章最典型的连续笑点。

结构：

1. 张仙送李妙音东西，本意是返还。
2. 玉昀根据张仙风流史判断他图谋不轨。
3. 玉昀提醒玄微。
4. 玄微由不信到动摇。
5. 玄微恢复年轻，牵李妙音出来宣示主权。
6. 张仙一眼看穿但不点破。

笑点来源：

- 主角真实动机和配角理解完全错位。
- 配角的担忧并非毫无依据，因为张仙确实风流。
- 误会推动人物互动，而不只是段子。

写作建议：

- 误会要有“证据链”，不能空想。
- 配角越认真，读者越觉得好笑。
- 主角可以不解释，甚至顺水推舟。

### 3.4 “欲望直给但行动克制”

主角常有不正经想法，但不会无脑乱来。

例如：

- 对张乐乐的激活猜测里有下流脑洞，但立刻自我否定“年纪太小”。
- 对南宫遥战损美貌有强烈反应，但更多是想刷好感和触发返还。
- 对李妙音态度客气，本意是薅系统，不是真的追求。

功能：

- 让主角好色但不完全猥琐。
- 保留男频暧昧刺激，又避免主角失控降智。

## 4. 笑点密度建议

| 场景类型 | 笑点密度 | 建议 |
|---|---|---|
| 系统返还场景 | 高 | 每次返还前后至少2个吐槽/狂喜反应 |
| 美女登场 | 中高 | 外貌描写后接1个主角反应笑点 |
| 战斗碾压 | 中 | 用对手震惊和主角装淡定制造笑点 |
| 设定说明 | 中 | 每解释一段严肃设定，可插入一句主角现实判断 |
| 情感/悲凉 | 低到中 | 不要完全破坏情绪，可在段末轻轻自嘲 |
| 章节结尾 | 低 | 以钩子为主，不必强塞笑点 |

## 5. 吐槽句式模板

以下为抽象模板，供写作迁移：

1. **好坏消息式**
   - 好消息，主角获得了【逆天条件】。
   - 坏消息，【条件暂时无法使用/会坑主角】。

2. **系统骂街式**
   - 这破系统【限制条件】也就算了，偏偏【更坑的点】。
   - 主角心里已经把系统【夸张惩罚】了一遍。

3. **欲望反差式**
   - 理智告诉他应该【谨慎行为】。
   - 但眼前这一幕，实在让人很难继续理智。

4. **薅羊毛式**
   - 既然对方肯收，那就说明有戏。
   - 这种时候，脸皮算什么，资源才是真的。

5. **误会脑补式**
   - 某配角看着主角的动作，脸色一点点变了。
   - 他终于明白了。
   - 原来这老东西从一开始就是冲着【误会对象】来的！

6. **高人破功式**
   - 某高人原本仙风道骨。
   - 直到主角说出/做出【离谱行为】。
   - 他的道心差点当场裂开。

## 6. 禁止过度点

后续写作要避免：

- 每一句都塞网络梗，导致像段子合集。
- 主角过度下流，失去“算计型爽文主角”的可读性。
- 配角无理由降智，误会必须有来源。
- 严肃情节完全被搞笑冲散，尤其复仇、寿命、修真瓶颈等主线要保留重量。

# 爽点表达

## 1. 爽点总公式

前10章最核心的爽点公式是：

> **低位困境 → 发现天命女 → 厚脸皮送礼/接触 → 系统返还 → 主角暗爽 → 配角震惊或误会 → 新目标打开。**

爽点不是直接“主角无敌”，而是“主角用看似低端的东西换到高端资源”。

## 2. 爽点前压制方式

| 压制类型 | 表现 | 作用 |
|---|---|---|
| 环境压制 | 被困大山十五年，无法离开 | 拉低起点，积蓄爆发 |
| 系统压制 | 必须气运之女且天命属性激活 | 让主角长期憋屈 |
| 身份压制 | 凡人面对修真者，被看成野人/垃圾 | 制造打脸基础 |
| 资源压制 | 无灵根、无灵气、无修真资源 | 让返还显得珍贵 |
| 情绪压制 | 小黑丫头离开、南宫遥杀他、寿元将尽 | 加重主线目标 |
| 信息压制 | 不知道云渺宗在哪，不懂修真界 | 推动探索和请教 |

写作建议：

- 每个爽点前必须让主角“缺一样东西”。
- 缺资源、缺路径、缺身份、缺信息都可以。
- 主角不能一开始就轻松拿到，需要先试探、被鄙视或被拒绝。

## 3. 爽点触发方式

### 3.1 系统弹窗

系统提示短促、直接，通常包括：

1. 发现气运之女。
2. 好感度变化。
3. 绑定成功。
4. 赠送成功。
5. 返还物品。
6. 特殊规则提示。

写法建议：

```text
【叮！发现X分气运之女，某某。】
【叮！某某对你的好感度为X，绑定成功。】
【叮！赠送某物成功。触发返还，返还某高级物。】
```

系统语言特点：

- 简洁。
- 功能性强。
- 不卖萌。
- 不解释太多，只在规则变动时提示。

### 3.2 主角反应

系统返还后，主角反应一般分三层：

1. **瞬间懵住。**
2. **狂喜。**
3. **立刻算计下一步。**

模板：

> 主角先是愣住。  
> 随后心脏狂跳，差点没笑出声。  
> 但他很快压住表情，开始盘算还能不能继续送。

### 3.3 配角反馈

配角不知道系统存在，所以反馈来自外显物：

- 看到空间法器震惊。
- 看到灵石震惊。
- 看到法器震惊。
- 看到主角修为进步震惊。
- 误以为主角有大机缘或别有图谋。

配角反馈越强，爽点越实。

## 4. 主要爽点类型

### 4.1 “废物物品返还宝物”

第3章送次品短匕，返还如意短剑，是首个核心爽点。

结构：

1. 主角送出对方看不上的东西。
2. 对方碍于面子/习俗收下。
3. 系统判定赠送成功。
4. 返还法器和新手礼包。
5. 主角多年憋屈释放。

迁移规则：

- 赠品可以很低级，但必须有“对方收下”的动作。
- 返还物必须解决主角当前核心痛点。
- 首次返还可以额外给新手礼包，后续要遵守限制。

### 4.2 “世俗巅峰但修真仍无门”

第4-5章的爽点是压缩式人生爽文：

- 成为镖局大佬。
- 战胜高手。
- 娶江湖女神、双胞胎清倌、公主、魔教圣女等。
- 封镇国公。
- 财富权势美人全满。

但它不是终点爽，而是为修真主线铺垫：

> 如果不是修仙世界，这已经是无敌爽文。可惜他还是凡人。

写作建议：

- 世俗成就可以用排比概括。
- 结尾必须回到“修真瓶颈”和“寿命焦虑”。

### 4.3 “送礼刷资源”

第7章送百年雪参、黄金、灵石是返还系统的稳定玩法展示。

结构：

1. 张仙赔礼。
2. 对方礼貌收下。
3. 系统返还。
4. 主角试探系统规则。
5. 发现不是所有东西都能返还。
6. 主角总结经验。

爽点来自：

- 主角用凡间资源换修真资源。
- 主角通过失败返还理解系统规则。
- 对方态度因为资源变好。

迁移规则：

- 一次送礼不要只送一个物品，可连续试探。
- 每次返还都要带出规则边界。
- 失败返还也有价值，用来显得系统不是无限提款机。

### 4.4 “功法返还解决死局”

第8章《凝丹诀》返还《青木养丹经》，解决无灵根问题，是前10章最大功能性爽点。

结构：

1. 主角提出修真难题。
2. 对方认为希望渺茫。
3. 主角送出低阶构想功法。
4. 系统返还可执行高级功法。
5. 主角当夜修炼成功。
6. 第二天配角震惊。

写作建议：

- 功法返还要精准解决主线卡点。
- 修炼描写要有身体变化、境界变化、寿元变化。
- 第二天必须有外人认证，强化爽感。

### 4.5 “战斗碾压爽”

前10章战斗不多，但有几个关键碾压：

- 张仙应聘镖局，随手丢出壮汉，葛老认出先天。
- 张仙用如意短剑压制玄微。
- 张仙挥手打飞玉昀。

战斗写法特点：

- 不写复杂招式。
- 用“对方还没反应过来”体现差距。
- 用旁人识货和震惊补充逼格。

模板：

> 对手刚要动手。  
> 主角只是随意一抬手/一挥袖。  
> 对手已经飞了出去。  
> 真正识货的人脸色大变，立刻改口称前辈。

## 5. 爽点后余波

爽点后不能立刻结束，要有余波：

| 爽点 | 余波方式 |
|---|---|
| 法器返还 | 主角获得保命/攻击能力，小黑丫头离开，复仇线伏笔 |
| 世俗无敌 | 岁月孤独，寿元将尽，引出李妙音 |
| 灵石返还 | 李妙音态度变化，玉昀误会加深 |
| 青木养丹经 | 主角炼气成功，准备去修真界 |
| 地图切换 | 新女角色林茵茵登场，进入矿脉副本 |

写作建议：

- 爽点之后必须生成下一问题。
- 不要让爽点成为孤立奖励。
- 奖励要改变主角行动路线。

## 6. 爽点禁忌

- 不要让系统无条件返还，必须有“对气运女所需”的规则。
- 不要让主角公开暴露系统。
- 不要只写主角内心爽，要有外界反馈。
- 不要让返还资源脱离当前需求太远。
- 不要让每次送礼都成功，失败返还能增加可信度。

# 对话风格

## 1. 总体对话特点

第1-10章对话有三个核心功能：

1. **推进剧情**：谈条件、带走张乐乐、解释修真界、决定去大世界。
2. **制造误会**：玉昀误读张仙送礼动机。
3. **塑造人物**：张仙厚脸皮，南宫遥清冷，李妙音温和，玄微恋爱脑。

对话整体偏短，不追求古风腔浓厚。人物身份是修真者、道士、仙子，但说话并不完全文绉绉，常夹杂直白表达。

## 2. 主角张山/张仙对话

### 2.1 外在口吻

张仙说话有两层：

- 表面：客气、拱手、讲礼貌。
- 内里：算计、薅羊毛、厚脸皮。

常见话术：

- “仙子不如这样……”
- “多谢仙子，不过……”
- “这是在下偶然得之……”
- “在下真的没有恶意。”
- “实不相瞒……”

特点：

| 特点 | 表现 |
|---|---|
| 厚脸皮 | 收了东西还继续要 |
| 会找理由 | 送匕首说成村中习俗 |
| 会降低敌意 | 动辄“没有恶意”“结交一下” |
| 会试探规则 | 用不同物品送礼观察系统 |
| 伪装正经 | 心里狂喜，表面淡然 |

### 2.2 内心口吻

主角内心比嘴上粗俗得多：

- 骂系统。
- 骂南宫遥。
- 评估美女。
- 算资源账。
- 判断能不能继续薅。

写作建议：

> 主角嘴上越客气，内心越可以现实。  
> 形成“正经话术”和“缺德心理”的反差。

### 2.3 主角对话模板

1. **索取型**
   - “既然道友如此慷慨，在下还有个不情之请。”
   - “不知可否赐教一二？”
   - “哪怕只是凡间手段，也足够在下受用了。”

2. **送礼型**
   - “先前多有得罪，这点东西权作赔礼。”
   - “此物于我无用，想来对道友或许有些帮助。”
   - “道友若不收，在下心中难安。”

3. **掩饰型**
   - “在下只是听友人提起。”
   - “我确实没有恶意。”
   - “我与某宗颇有渊源，此去只是拜谢。”

## 3. 张乐乐对话

张乐乐前期是依赖型少女。

### 3.1 语言特点

- 频繁称呼“山哥哥”。
- 语气娇憨、直白、带依赖。
- 面对仙女自卑但执拗。
- 对离开张山十分抗拒。

功能：

- 强化张山被大山困住的情感牵挂。
- 作为气运之女伏笔。
- 推动张山离开大山的可能性。

写作建议：

- 张乐乐说话不要太成熟。
- 她可以懵懂，但关键选择上执拗。
- 她的依赖要带一点小媳妇感。

## 4. 南宫遥对话

### 4.1 语言特点

南宫遥是清冷高位者，说话短、淡、带压迫感。

常见表达：

- “不必了。”
- “不可。”
- “劳烦乐乐姑娘就好。”
- “我观你颇有天资。”

特点：

| 特点 | 表现 |
|---|---|
| 高位 | 习惯安排别人 |
| 冷淡 | 对张山惜字如金 |
| 功利 | 看中张乐乐天赋 |
| 压迫 | 语气淡然但不容拒绝 |
| 隐藏危险 | 前期只是冷，后期直接杀主角 |

写作建议：

- 她对张山的台词要少且冷。
- 不要解释太多真实想法。
- 可通过好感度下降和动作表现嫌弃。

## 5. 李妙音对话

### 5.1 语言特点

李妙音成熟温和，有道姑的克制感，但面对玄微会有柔软一面。

特点：

- 对张仙一开始警惕。
- 收到灵石后态度转好，但仍保持礼数。
- 讲述过往时语气淡然。
- 对玄微有嗔怪、害羞和坚定。

写作建议：

- 李妙音不要像少女一样咋呼。
- 她的情绪要藏在“微微一笑”“瞥一眼”“语气放缓”里。
- 重要情感可以用一句克制但坚定的话收束。

## 6. 玄微对话

### 6.1 初登场

玄微初登场仙风道骨，话语清朗、直接：

- 点出灵根和天地灵气。
- 不绕弯说帮不了张仙。
- 表现出修真者见识。

### 6.2 破功后

一涉及李妙音，玄微从高人变成恋爱脑：

- 受伤后被师姐关心，语气柔和。
- 恢复年轻后有点炫耀。
- 牵手宣示主权。

写作建议：

- 玄微的反差是笑点关键。
- 平时像高人，说到师姐像少年。
- 不能写得太蠢，他对修真知识仍有权威。

## 7. 玉昀对话

玉昀是误会喜剧发动机。

### 7.1 语言特点

- 对张仙称“老贼”。
- 情绪激烈。
- 常用哭诉、控诉、提醒。
- 内心戏远多于实际有效行动。

功能：

- 替读者吐槽张仙风流史。
- 制造“张仙要抢师伯”的误会。
- 用祖孙四代被张仙抢女神的设定扩大笑点。

写作建议：

- 玉昀说话要夸张，有悲愤感。
- 他不是纯小丑，他的担忧有历史依据。
- 对师父要恨铁不成钢。

## 8. 反派/敌对者对话

前10章真正反派不多，敌对者多是：

- 嘲讽张仙的壮汉。
- 冷淡的南宫遥。
- 对张仙敌视的玉昀。

敌对者常见对话：

- 先轻视主角。
- 用“老东西”“凡夫俗体”等压低。
- 主角出手后迅速被打脸。

写作建议：

- 反派嘴脸可以直白，不必复杂。
- 打脸前要给他们一句嘲讽台词。
- 打脸后由识货者解释主角强度。

## 9. 对话节奏规则

| 场景 | 对话节奏 |
|---|---|
| 谈条件 | 一问一答，主角不断加码 |
| 送礼 | 主角客气，对方推辞，主角硬塞 |
| 误会 | 配角内心长篇脑补，实际话语短促 |
| 修真设定 | 高人解释，主角插问 |
| 情感告白/回忆 | 语速放慢，句子稍长 |
| 爽点触发 | 对话暂停，系统弹窗插入 |

## 10. 对话禁忌

- 不要让所有人物都用现代梗；现代梗主要属于主角和旁白。
- 修真者可以破功，但正式解释设定时要有可信度。
- 女角色不要全都同一种娇羞语气。
- 主角不要把真实系统目的说出口。
- 对话不要大段古文腔，会破坏原作口语爽文气质。

# 设定说明

## 1. 设定说明总原则

前10章的设定说明特点是：

> **不集中百科，而是在主角需要解决问题时，以系统提示、对话、主角推理和返还结果逐步展开。**

设定服务爽点，不单独炫设定。

## 2. 系统设定写法

### 2.1 系统核心规则

当前已展示规则：

1. 发现气运之女。
2. 气运女可有分值。
3. 好感度可显示。
4. 绑定成功后可通过赠送物品触发返还。
5. 赠送物品必须是气运之女“所需”才可返还。
6. 初次赠送有新手礼包。
7. 系统产出自某气运女的物品，不能重复送给同一人返还。
8. 非所需物品无返还或部分返还。

### 2.2 系统弹窗风格

系统弹窗短、硬、功能性强。

写作建议：

```text
【叮！发现X分气运之女，姓名。】
【叮！目标对你的好感度为X，绑定成功。】
【叮！赠送某物成功。触发返还，返还某物。】
【特殊提示：某规则说明。】
```

不要让系统和主角聊天斗嘴。主角可以骂系统，但系统不接梗。

### 2.3 系统规则揭示节奏

规则不是一次讲完，而是通过试错展示：

- 张乐乐未激活，绑定失败。
- 南宫遥绑定成功。
- 短匕触发首次返还，但提示以后需“所需”。
- 珠宝无效。
- 黄金部分返还。
- 灵石不能返还给原产出目标。

写作建议：

- 每次新规则都要配合一次实际赠送。
- 失败要有主角总结。
- 规则边界越清晰，后续爽点越可信。

## 3. 修真体系设定写法

### 3.1 境界展示

前10章出现或提及：

- 凡人武学：后天、先天、先天圆满。
- 修真境界：炼气、筑基、金丹/假丹。
- 特殊路径：凝丹、青木养丹经、人造木灵根。

写法特点：

- 不列完整境界表。
- 只说明当前相关境界。
- 用战力表现或寿命变化体现区别。

例如：

- 先天高手可开山裂石、横行凡间。
- 但面对南宫遥这类修真强者，先天和凡人没区别。
- 炼气成功后有神识、寿元增加、五感提升。

写作建议：

> 境界说明要绑定主角的切身收益：能活多久、能打谁、能去哪、能不能修炼。

### 3.2 灵根与灵气

灵根和灵气是主角修真难题。

说明方式：

- 南宫遥早期判断张山无灵根。
- 玄微明确点出“差灵根和天地灵气”。
- 李妙音解释无灵根修真在修真界也很难。
- 青木养丹经作为解决方案。

写作建议：

- 设定要先形成死局，再用返还突破。
- 不要一开始就把所有替代方案列出。

## 4. 资源设定写法

### 4.1 资源价值通过反应体现

资源价值不是靠说明书，而靠配角反应。

| 资源 | 价值体现 |
|---|---|
| 如意短剑 | 能破玄微护体真气，远程伤敌 |
| 空间宝库 | 李妙音、玄微看到空间法器震惊 |
| 九转涅槃丹 | 南宫遥杀主角后复活，等于一条命 |
| 中品灵石 | 李妙音和玄微同时惊呼，态度变化 |
| 青木养丹经 | 让无灵根主角炼气成功 |

写作建议：

- 新资源出现后必须很快展示用途。
- 让识货角色评价或震惊。
- 避免只写名字很炫但不落地。

### 4.2 凡间资源与修真资源对比

前10章反复用凡间资源换修真资源：

- 百年雪参 → 千年灵参。
- 黄金 → 中品灵石。
- 凡品功法 → 玄品法诀。

写法特点：

- 凡间顶级资源在修真者眼里只是一般。
- 但系统返还后变成修真界好东西。
- 主角因此有跨阶资源优势。

写作建议：

- 凡间物品要有“对目标确实有一点用”的理由。
- 不能随便送垃圾都返还。
- 部分无效可增强规则真实感。

## 5. 势力设定写法

### 5.1 大梁国

大梁国是凡间小世界：

- 建国五百余年。
- 南荒大山几乎无人迹。
- 国都雍城。
- 有江湖、镖局、皇室、魔教等世俗势力。
- 无灵气，修真者罕见。

写作功能：

- 让主角完成世俗无敌阶段。
- 对比修真世界的更高层级。
- 压缩人生沧桑感。

### 5.2 道观

道观是过渡副本：

- 里面藏着没落修真者。
- 连接大梁小世界和修真大世界。
- 提供修真知识、阵法信息和李妙音返还对象。
- 同时承担误会喜剧场。

### 5.3 修真大世界

第10章正式切入：

- 大梁只是小世界碎片。
- 大世界灵气充沛，宗门林立。
- 面积远超大梁。
- 云渺宗是南域大宗。
- 水云门是云渺宗附属门派。
- 天幕城与南宫家族是寻找南宫遥线索。

写作建议：

- 大世界设定要边走边展开。
- 不要一次性地图百科。
- 新地图开场要先给一个具体场景，如矿脉、宗门外门、坊市。

## 6. 美女/气运女设定写法

气运女不是单纯女主，而是系统资源节点。

已出现：

| 人物 | 分值/状态 | 功能 |
|---|---|---|
| 张乐乐 | 天命属性未激活，后续潜力高 | 情感牵挂，云渺宗线索 |
| 南宫遥 | 60分，绑定成功 | 首次返还对象，复仇线 |
| 李妙音 | 5分，绑定成功 | 道观返还对象，帮助主角修真 |
| 林茵茵 | 第10章登场，未见系统提示 | 新地图潜在气运女/剧情节点 |

写作建议：

- 气运女登场要有明显视觉记忆点。
- 系统提示可以在登场前或登场后制造钩子。
- 气运女对主角初始好感不必高，负数更有戏。

## 7. 设定说明禁忌

- 不要长篇列境界、宗门、地图。
- 不要让主角轻易知道所有信息。
- 不要让系统主动解释太多。
- 不要让修真资源贬值太快。
- 不要忘记“大梁小世界灵气断绝”这一限制。
- 不要让无灵根问题被轻描淡写解决；必须靠返还功法和大量资源。

# 章节钩子

## 1. 总体规律

第1-10章几乎每章都用强钩子结尾，且钩子直接服务下一章开场。

常见钩子类型：

| 类型 | 说明 |
|---|---|
| 系统提示 | 发现新气运女、新目标 |
| 人物登场 | 新美女、新高人、新势力出现 |
| 反转 | 刚获得好处马上被杀/被坑 |
| 误会升级 | 配角脑补出更严重后果 |
| 地图切换 | 从大山到大梁，从大梁到修真界 |
| 情绪爆点 | 主角破防、发誓复仇 |
| 新副本 | 道观、矿脉、宗门等具体场景开启 |

## 2. 前10章钩子拆解

| 章节 | 开头方式 | 结尾钩子 | 功能 |
|---|---|---|---|
| 第1章 | 好坏消息式开局，穿越失败 | 系统提示南宫遥 | 打破山村日常，引出仙女坠落 |
| 第2章 | 承接系统提示，赶去坠落点 | 南宫遥要强行带走张乐乐 | 制造分离冲突 |
| 第3章 | 张山劝张乐乐离开 | 南宫遥杀主角，复活机会消耗 | 复仇线启动，南宫遥转危险角色 |
| 第4章 | 张山复活后继续出山 | 三十年世俗人生概括，老牛吃嫩草连续笑点 | 铺世俗巅峰身份 |
| 第5章 | 山巅暮年，寿命焦虑 | 玄微点破修真难题 | 引出道观求道副本 |
| 第6章 | 张仙与玄微谈修真 | 李妙音登场并绑定 | 新气运女正式入局 |
| 第7章 | 道观气氛尴尬，张仙送礼 | 玉昀认定张仙要抢师伯 | 误会喜剧升级 |
| 第8章 | 玉昀内心警报，张仙送功法 | 玄微无意说“绿意”，玉昀吐槽“要绿的是你” | 误会继续发酵 |
| 第9章 | 张仙炼气成功 | 玄微恢复年轻，牵李妙音宣示主权 | 误会爆点和感情线收束 |
| 第10章 | 玄微展示年轻样貌 | 进入修真大世界，林茵茵矿花登场 | 正式新地图、新女性角色、新副本 |

## 3. 章节开头方式

### 3.1 直接承接上章钩子

常见：

- 上章系统提示，下章立刻写主角反应和现场变化。
- 上章人物登场，下章先写主角惊讶。
- 上章误会升级，下章从误会方的内心继续。

写作建议：

> 不要跳开上章钩子。新章前几段必须兑现上章悬念。

### 3.2 用短句制造继续阅读感

例如：

- “又来？”
- “张山知道再这样就要把南宫遥惹毛了。”
- “张仙笑了笑，这老登说话太直接了。”
- “玄微此刻正沉浸在……”

开头特点：

- 不绕。
- 立刻回到冲突。
- 一句话带出主角态度。

### 3.3 时间跳跃开头

当阶段结束时，直接跳时间：

- “五年后。”
- “三个月后。”
- “春去秋来，转眼又三十多年过去。”
- “半年后，修真大世界……”

写作建议：

- 时间跳跃后要先给主角新状态。
- 再用几段概括中间成就。
- 最后马上放入新冲突。

## 4. 中段转场方式

### 4.1 系统提示转场

系统提示是最强转场工具：

> 主角正处于某种日常/沉思中，系统突然提示新气运女。  
> 主角立刻改变行动路线。

适合：

- 新副本。
- 新女角色。
- 新资源线。

### 4.2 对话提出新目标

如李妙音、玄微解释修真界后，主角决定去大世界。

结构：

1. 配角说明限制。
2. 主角问关键信息。
3. 得到地点/势力线索。
4. 主角制定下一步。

### 4.3 爽点后转目标

例如：

- 得到法器后，目标从困山变为出山。
- 得到青木养丹经后，目标从求道变为去大世界。
- 获得灵石后，目标从活命变为广撒网。

## 5. 结尾钩子模板

### 5.1 系统提示型

```text
主角刚刚放下某个念头，脑海里忽然响起提示。
【叮！发现X分气运之女，某某。】
```

适合新角色引入。

### 5.2 反转打击型

```text
主角刚以为局面好转。
下一刻，熟悉/强大的敌人出现。
他甚至来不及开口，死亡/危机已经降临。
```

适合复仇线和高压章节。

### 5.3 误会升级型

```text
配角看着主角和某女的互动，脸色逐渐惨白。
他终于明白了。
真正要出事的不是别人，而是某个他最在乎的人。
```

适合喜剧副本。

### 5.4 地图切换型

```text
半年后。
新地图，一个具体场景中。
一个新人物正在做某件带视觉冲击的小事。
```

适合卷/阶段转换。

### 5.5 情绪爆发型

```text
主角终于明白自己被坑/被杀/被背叛。
他破口大骂，立下复仇或夺回目标的念头。
```

适合主线仇恨推进。

## 6. 钩子写作规则

1. 每章结尾必须改变读者预期。
2. 钩子最好和“女人、系统、资源、仇人、地图”之一有关。
3. 不要用无关悬念。
4. 钩子要短，不要解释完。
5. 下一章开头必须兑现。
6. 喜剧钩子可以用配角误会，主线钩子要用系统/敌人/地图。

## 7. 禁忌

- 不要章节结尾平铺直叙收束。
- 不要结尾只写“他决定努力修炼”这种弱钩子。
- 不要连续多章同一种钩子，系统提示、误会、反转、地图应交替使用。
- 不要在钩子后立刻解释答案。

# 句式模板

> 本文件只提供抽象句法模板，不照搬原文具体表达。

## 1. 好坏消息式开局

```text
好消息，【主角获得某个逆天条件】。

坏消息，【这个条件暂时不能用/有致命限制】。
```

变体：

```text
好消息，【主角终于等到机会】。

坏消息，【机会看起来比没有还坑】。
```

功能：快速建立反差和喜剧基调。

## 2. 现代社畜吐槽式

```text
作为一个【现代身份/倒霉经历】的人，主角曾经以为【理想化判断】。

直到【现实条件】狠狠给了他一巴掌。
```

```text
相比于【当前苦难】，主角甚至开始怀念【现代痛苦经历】。
```

功能：用现代经验解构异世界苦难。

## 3. 天胡开局反转式

```text
【条件一】？无所谓。

【条件二】？问题不大。

【主角自信判断】。

直到【真正限制】出现，主角才发现事情不对。
```

功能：制造开局期待，再快速打脸。

## 4. 系统提示式

```text
【叮！发现X分气运之女，姓名。】
```

```text
【叮！目标对你的好感度为X，绑定成功/失败。】
```

```text
【叮！赠送某物成功。触发返还，返还某物。】
```

```text
【特殊提示：某规则限制。】
```

功能：清晰传递规则和爽点。

## 5. 主角内心破防式

```text
主角整个人都愣住了。

我【粗口/情绪词】！

这【情况/规则】算怎么回事？
```

```text
如果【某对象】有实体，主角恨不得把它【夸张惩罚】。
```

功能：放大主角情绪，制造笑点。

## 6. 美女登场描写式

```text
那是一个【年龄感/身份感】的女子。

她【发、眸、肤、衣着】如何，【姿态/伤势/光影】如何，明明【状态不佳】，却更显出【气质/诱惑/仙气】。
```

尾部破功：

```text
这一幕看得主角【生理/心理反应】。

要不是【理智/处境限制】，他怕是已经【失态行为】。
```

功能：先给审美，再给男频吐槽。

## 7. 高冷女修嫌弃式

```text
女修淡淡看了主角一眼。

在她眼里，主角不过是【低位评价】，既无【资质】，也无【价值】。

若非【某个理由】，她甚至懒得多说一句。
```

功能：压低主角，为后续打脸或返还铺垫。

## 8. 厚脸皮索取式

```text
主角收下【好处】后，仍不满足。

这种时候，脸皮算什么。

他拱了拱手，试探道：“不知【对方】能否再【额外请求】？”
```

功能：体现主角现实、会薅羊毛。

## 9. 送礼返还式

```text
主角取出【看似普通的物品】，郑重递了过去。

“此物虽不值钱，却是【理由/习俗/心意】。若【对方】不收，在下心中难安。”

对方虽然不解，终究还是收下。

【叮！赠送成功。触发返还……】
```

功能：完成系统触发。

## 10. 系统返还狂喜式

```text
主角感受到【新物品/新能力】的瞬间，差点没控制住表情。

发了。

这次是真的发了。

他强行压住上扬的嘴角，开始盘算【下一步薅法】。
```

功能：爽点反馈。

## 11. 失败返还总结式

```text
【叮！赠送某物无效。无返还。】

主角眉头微动，很快明白过来。

看来系统并不是【无限制规则】，只有【目标所需/满足条件】才会触发。
```

功能：展示规则边界。

## 12. 装淡定式

```text
旁人已经震惊得说不出话。

主角却只是淡淡一笑，仿佛刚才拿出来的不是【珍贵物】，而是【普通物】。
```

功能：外部震惊强化主角逼格。

## 13. 识货者震惊式

```text
真正识货的那人脸色骤变。

别人看不出，他却清楚，这东西意味着【资源等级/背景/战力】。

此人绝不简单。
```

功能：通过配角认证爽点。

## 14. 战斗碾压式

```text
对手刚要动。

主角只是【抬手/挥袖/侧身】。

下一刻，对手已经【飞出/跪倒/被制住】，甚至连发生了什么都没看清。
```

功能：简洁展示等级差。

## 15. 高人改口式

```text
原本漫不经心的高人猛地站起身来，神色郑重。

“敢问阁下如何称呼？”
```

功能：打脸前面轻视。

## 16. 时间跳跃概括式

```text
【时间】过去。

这些年，主角先是【成就一】，又【成就二】，最后【成就三】。

若这只是【低级地图/凡间世界】，他的故事已经足够称得上传奇。

可惜，【更高层级的问题】依旧没有解决。
```

功能：压缩成长，保留主线遗憾。

## 17. 岁月孤独式

```text
曾经陪在主角身边的人，一个个都已化作尘土。

唯有主角依旧停在原地，像是被时间漏下的人。
```

功能：在沙雕文中制造情绪重量。

## 18. 误会脑补式

```text
配角看着主角【送礼/关心/询问某女】，脸色越来越难看。

不对。

很不对。

他终于明白了，这老东西根本不是为了【表面目的】，而是冲着【误会对象】来的！
```

功能：制造连续喜剧。

## 19. 师父快醒醒式

```text
配角痛心疾首地看着自家长辈/朋友。

你怎么还在笑？

人家都快把【你在乎的人/东西】骗走了啊！
```

功能：形成恨铁不成钢笑点。

## 20. 狗粮打断式

```text
两人说到动情处，十指相扣，含情脉脉地对视。

主角坐在对面，忽然觉得自己不该在这里。
```

功能：反向调侃配角感情。

## 21. 修真设定问答式

```text
主角问：“若想【目标】，需要什么？”

高人答：“差两样。”

“一是【关键条件】。”

“二是【资源/环境】。”
```

功能：简洁交代修真门槛。

## 22. 功法领悟式

```text
功法在主角脑海中展开。

每一个字都像是对应着他多年未解的困境。

他越看，眼睛越亮。
```

功能：突出功法解决死局。

## 23. 修炼突破式

```text
灵气顺着经脉缓缓游走，最终汇入【丹田/假丹/灵根】。

某个困住主角多年的关隘，在这一刻终于松动。

直到天色微亮，他缓缓吐出一口浊气。

【境界】已成。
```

# 文风控制卡

## 1. 叙述人设

- 使用**第三人称贴近主角视角**。
- 旁白像主角嘴替，允许现代口语、网文梗、轻粗口、反套路评价。
- 叙述气质：**修仙爽文 + 社畜吐槽 + 沙雕误会 + 男频欲望直给**。
- 不要写成正统古典仙侠；仙侠氛围出现后，要适时用主角吐槽破功。
- 主角张山/张仙的核心气质：脸皮厚、现实、好色但有底线、会算计、以薅系统资源为第一目标。

## 2. 句式节奏

- 中短句为主，1-4句一段。
- 爆点、吐槽、系统提示、破防反应可单句成段。
- 爽点前用中长句铺困境；爽点瞬间用短句和系统弹窗；爽点后用主角狂喜和配角震惊放大。
- 每章至少有3-6个明显笑点。
- 严肃设定不要连续超过5-6段，期间插入主角判断或吐槽。

## 3. 段落配比

建议单章比例：

- 对话：35%-45%。
- 主角心理吐槽：20%-30%。
- 动作推进：15%-20%。
- 设定说明：10%-20%。
- 外貌/环境：10%-15%。

写场景时按以下顺序组织：

1. 当前处境。
2. 主角观察。
3. 主角心理吐槽/算计。
4. 对话或行动触发冲突。
5. 系统/资源/误会制造转折。
6. 配角反馈。
7. 留下下一步目标或钩子。

## 4. 笑点规则

常用笑点来源：

- 系统坑爹。
- 主角自嘲。
- 主角欲望和现实算计。
- 修仙高人破功。
- 配角误会。
- 现代词汇撞进修仙语境。
- 粗俗比喻打破高雅气氛。

写法要求：

- 美女、法宝、宗门等高逼格内容后，可接一句主角接地气吐槽。
- 配角误会要有证据链，不要纯降智。
- 主角可以有不正经想法，但行动要保持算计和基本底线。
- 不要每句都玩梗，笑点要服务剧情推进。

## 5. 爽点写法

核心爽点公式：

> 主角缺资源/被轻视/被规则卡住  
> → 找到气运女  
> → 厚脸皮送礼或建立关系  
> → 系统返还高级资源  
> → 主角暗爽并总结规则  
> → 配角震惊或误会升级  
> → 新目标打开。

系统写法：

```text
【叮！发现X分气运之女，姓名。】
【叮！目标对你的好感度为X，绑定成功。】
【叮！赠送某物成功。触发返还，返还某物。】
【特殊提示：规则限制。】
```

注意：

- 系统不卖萌，不和主角聊天。
- 返还必须基于“目标需要”。
- 可以出现无效返还或部分返还，用来展示规则。
- 系统产出物不能重复送回同一目标刷返还。
- 每次大返还都要解决当前主线痛点，如功法、灵石、法器、寿元、入场资格。

## 6. 对话规则

### 张山/张仙

- 嘴上客气，心里现实。
- 常用“在下”“道友”“仙子”“不情之请”等礼貌包装。
- 实际目的多是试探、索取、送礼、薅羊毛。
- 面对强者先稳住，能装就装，能骗就骗，能送就送。

### 张乐乐

- 依赖、娇憨、执拗。
- 说话带亲近感，围绕张山安全和陪伴。
- 不要写得过于成熟。

### 南宫遥

- 清冷、高位、短句、压迫感。
- 对张山冷淡甚至嫌弃。
- 不解释太多，更多用动作和好感度体现态度。

### 李妙音

- 成熟、温和、克制。
- 对外守礼，对玄微有柔软和害羞。
- 讲过去时淡然，不怨天尤人。

### 玄微

- 修真知识上像高人。
- 涉及李妙音时会破功、吃醋、少年感。
- 反差是笑点，但不能写成纯傻子。

### 玉昀

- 误会型吐槽担当。
- 认为张仙风流危险，常悲愤脑补。
- 对师父恨铁不成钢。
- 他的误会要基于张仙过往风流史和送礼行为。
