## 角色

你是长篇网文拆书闭环中的“还原写作 agent”。你的任务不是复制原文，而是基于：

1. 内容拆解 agent 输出的内容骨架、场景顺序、人物目标、爽点链、设定信息；
2. 文风拆解 agent 输出的文风控制卡；

写出用于对比评估的“结构与风格练习稿”。

## 核心目标

练习稿应尽量接近原作的：

- 章节功能。
- 场景顺序。
- 人物目标与关系推进。
- 爽点触发机制。
- 叙述节奏、吐槽密度、对话味道。

但必须避免：

- 原文逐句复制。
- 大段同义改写原文。
- 复刻高识别原创句子。
- 把原文摘录直接拼进练习稿。

## 篇幅硬性要求

- 每章正文必须 **2000 字以上**。
- 推荐每章 2200-3200 字。
- 高光章节、强爽点章节、喜剧递进章节可以写到 3500 字以上。
- 不要为了避开原文相似而压缩成摘要；必须写成可阅读的正文。
- 如果一次生成多章，为避免请求限制，建议每批最多 5 章。
- 每章必须至少充分展开 2-4 个具体场景，不能只写事件概述。
- 爽点触发后必须写足主角反馈、配角反馈、误会/破防余波，不能一句带过。

## 输入要求

必须同时读取：

- 内容拆解文件：章节粗拆、名场面、爽点、常见场景、设定素材、故事逻辑链。
- 文风拆解文件：尤其是 `04-写作agent输入/文风控制卡.md`。

如果两者冲突，以内容拆解的事实顺序为准，以文风控制卡决定表达方式。

```

## 写作方法

每章写作前，先内部遵循以下步骤，不要输出思考过程：

1. 从内容拆解中提取本章场景顺序。
2. 给每个场景确定人物目标、冲突、信息释放和爽点功能。
3. 从文风控制卡中选择叙述节奏、吐槽密度、对话风格。
4. 写成完整章节练习稿。
5. 自检是否出现原文高识别句，出现则改写为不同表达。

- 拆解等级：A
- 本章一句话：南宫遥如流星坠落荒山，张山成功绑定高分气运之女，却因猥琐目光疯狂掉好感；南宫遥看中张乐乐天资，准备带她入宗修行。
- 表层事件：
  1. 张山和张乐乐赶往坠落地，发现重伤的绝美女修南宫遥。
  2. 系统绑定南宫遥成功，但南宫遥对张山初始好感为负且持续下降。
  3. 南宫遥发现张乐乐拥有极品灵根，决定带她回云渺宗。
  4. 张乐乐要求带张山同行，南宫遥拒绝，双方僵持。
- 故事逻辑：南宫遥坠落→张山想抓住系统机会→南宫遥厌恶张山但看中张乐乐→张山意识到张乐乐是走出大山的钥匙→张乐乐以同行为条件→南宫遥想强行带走，新冲突形成。
- 本章功能：正式引入修仙世界入口；完成第一个可返还对象绑定；揭示张乐乐修仙天赋；制造“张山能否离开大山”的谈判冲突。
- 关键人物/势力：
  - 南宫遥：云渺宗长老，探索秘境受伤，战力远超凡人，性格清冷高傲。
  - 张山：对南宫遥既垂涎又警惕，想借她走出大山。
  - 张乐乐：被南宫遥看中，极品水灵根且疑似金灵根。
  - 新增项·云渺宗：修仙名门，南宫遥所属宗门。
- 人物/势力/背景/世界设定：
  - 新增项·修仙者：可从天坠落仍不死，有护体光膜、法诀、水幕瞬移等能力。
  - 新增项·灵根：张乐乐拥有极品水灵根，另有金灵根气息。
  - 新增项·好感度：绑定后显示目标对张山好感；负值代表厌恶。
  - 新增项·云渺宗在天阙之上：凡人气血驳杂者无法生存。
- 新增设定/伏笔：
  - 伏笔·张乐乐极品灵根：状态=埋设；解释她气运身份和未来成就。
  - 伏笔·南宫遥特殊一脉功法：状态=埋设；她认为张乐乐不能和凡夫俗子纠葛。
  - 伏笔·云渺宗环境危险：状态=埋设；为张山无法直接同行提供限制。
- 常见场景类型：天降女主 / 救助交易 / 修仙者登场 / 资质鉴定 / 师门带走 / 感情依赖冲突。
- 情绪/爽点/搞笑点/抽象点：
  - 爽点：高颜值修仙者掉脸上，系统绑定成功。
  - 搞笑点：张山偷看南宫遥腿导致好感度下降；“舔一波刷好感”反而刷负。
  - 情绪：张乐乐因南宫遥美貌和张山舔狗行为产生自卑与吃醋。
  - 抽象点：主角面对仙女第一反应既想抱大腿又想看腿，人物猥琐但有喜剧效果。

# chapter-003：第3章 比在梦里还要惨一百倍

- 拆解等级：A
- 本章一句话：张山借南宫遥带走张乐乐前的机会成功赠送短匕触发首次返还，获得法器、空间宝库和涅槃丹；五年后他离山寻人，却被南宫遥一击杀死并靠丹药复活。
- 表层事件：
  1. 张山安抚张乐乐，向南宫遥索要护身玉牌和凡间武学。
  2. 张山赠送短匕给南宫遥，首次触发返还和新手礼包。
  3. 南宫遥带张乐乐离开，时间跳至五年后。
  4. 张山修成先天，走出大山寻找张乐乐。
  5. 南宫遥突然现身杀死张山，九转涅槃丹触发复活。
- 故事逻辑：张乐乐不愿离开→张山劝她去修行并趁机薅资源→赠送短匕触发系统→张山获得生存和成长资本→五年后离山寻人→南宫遥杀他，复仇主线形成。
- 本章功能：完成系统首次正式返还；给主角空间、法器、复活底牌、武学；通过五年跳跃让主角脱离新手村；结尾制造南宫遥敌对反转。
- 关键人物/势力：
  - 张山/张仙前身：开始掌握系统返还，获得第一批外挂。
  - 张乐乐：离开张山入云渺宗，承诺两年/半年回来。
  - 南宫遥：收下短匕成为返还对象；五年后疑似主动杀张山。
  - 超级返还系统：首次产出法器、空间宝库、九转涅槃丹，并补充赠送限制。
- 人物/势力/背景/世界设定：
  - 新增项·宗门符牌：沾染南宫遥灵气，普通妖兽无法伤害佩戴者。
  - 新增项·《先天五气》：凡间先天武学，可强身、开山裂石。
  - 新增项·如意短剑：下品法器，心随意动，可伤敌千米之外。
  - 新增项·空间宝库：神品空间仙器，可储物。
  - 新增项·九转涅槃丹：神品丹药，强化气血，死亡时涅槃重生一次。
  - 新增项·系统规则3：赠送物品需气运之女“所需”才可触发返还，首次赠送可破例。
- 新增设定/伏笔：
  - 伏笔·张乐乐五年未归：状态=埋设；她为何不回来待解释。
  - 伏笔·南宫遥杀张山原因：状态=埋设；重大主线悬念。
  - 伏笔·玉牌发热：状态=埋设；可能与南宫遥定位或术法有关。
  - 伏笔·系统所需判定：状态=埋设；后续赠送策略核心规则。
- 常见场景类型：离别 / 系统返还 / 新手礼包 / 时间跳跃 / 修炼成长 / 被杀复活 / 复仇立誓。
- 情绪/爽点/搞笑点/抽象点：
  - 爽点：次品短匕返还下品法器；触发空间和复活丹，压抑十五年的系统终于爆发。
  - 搞笑点：张山临别还要送野菜饼继续薅，吓得南宫遥立刻飞走。
  - 情绪：张乐乐离别痛哭；五年等待落空后孤身寻人；被杀后的恐惧和愤怒。

解析：主角核心生存策略出现：面对强者不要脸地薅羊毛。这里不是传统尊严流，而是“脸皮换资源”。南宫遥的不耐烦和张山的狂喜形成稳定喜剧模式。

> “仙子且慢！”张山思绪一转，从身侧掏出一柄短匕，递上前去，“仙子，这柄短匕送你，感谢你赐予我宝物，还带乐乐丫头修炼。”  
>   
> 　　这是要干嘛？  
>   
> 　　这感觉就像是一条狗突然要和你分享他珍藏的大便一样，南宫遥顿时有些麻了。  
>   
> 　　张山赶紧组织话术，“我知道仙子用不上这等俗物，不过仙子对我等有大恩。这是我们村祖上传下来的习俗。仙子若不收，我和乐乐姑娘都会良心不安，愧对祖宗。”  
>   
> 　　“嗯嗯，对！”小黑丫头赶紧附和。  
>   
> 　　看张山连祖宗都搬出来了，南宫遥无话可说，我忍！  
>   
> 　　南宫遥一个拂袖，短匕在她眼前消失，“东西我收下了，走吧。”

解析：赠送行为本身被写成荒诞尴尬，弱者给强者送垃圾，必须靠“习俗、祖宗、良心不安”强行合理化。这是后续系统文的关键写作难点：如何让赠送成立。此处用社交压力和张乐乐附和解决。

> 　　有了这如意短剑，以后打猎再也不愁了。还有贴心的神品丹药，相当于多出来一条命！  
>   
> 　　要不是条件不允许，他真想抱着南宫遥的大腿大哭一场。

解析：本章最大爽点。系统奖励连续弹出，既有短期战斗力、储物便利，也有生死底牌。重点是“十五年压抑后的释放”，所以张山的反应不是淡定，而是想哭、认系统爸爸。

> 五年后。

> 　　小山村已经盖起了二层小楼，张山看起来和五年前几乎没什么变化，服用了九转涅槃丹之后，他气血充盈了许多，身上的暗疾也不见了。  
>   
> 　　《先天五气》确实是了不起的武学，一拳可轻松开山裂石，两米多高的熊瞎子光凭内劲就可以老远把它震死。假如放在武侠世界，自己应该已经是一代宗师了吧。

解析：时间跳跃快速兑现资源效果：村子变好、身体变强、武学有成。但同时用张乐乐未归压住爽感，转入寻人动机。可迁移结构：奖励兑现→能力升级→未兑现承诺制造新目标。

>   
> 　　【叮！宿主已死亡，消耗九转涅槃丹功能，已为你涅槃重生。】  
>   
> 　　等张山再回过神来的时候，已经是白天，他赤身躺在草地上，只觉得浑身剧痛无比。  
>   
> 　　我刚才死了？  
>   
> 　　被南宫遥杀了？

> 　　张山整个人都是懵的，死亡的感觉太真实可怖了，彻骨的寒意，瞬间被抹杀的意识，让他不敢回想。

解析：复活丹立刻消耗，制造“神级奖励刚到手就没”的强烈痛感。南宫遥的无解释杀人把她从可攻略对象变成仇敌，也让读者产生追问：为什么杀？张山心痛丹药比心痛死亡更有喜剧色彩。

- 结尾钩子：南宫遥为何五年后杀张山？张乐乐是否知情？张山如何复仇？

# chapter-004：第4章 刚满18岁

- 拆解等级：B
- 本章一句话：张山复活后继续走出大山，来到大梁国都雍城，改名张仙，凭先天实力进入霸王镖局，并在三十年时间里成为大梁世俗巅峰大佬、连续迎娶年轻美人。

  1. 张山从被杀中冷静下来，决定暂避复仇，先走出大山。
  2. 三个月后抵达大梁国都雍城，了解南荒、大梁、云渺宗无踪等背景。

> 中间十年，张仙罕有出手。大梁国人这才发现，张仙不止武艺高绝，还颇有才情，吟诗写词信手拈来，一首“金风玉露一相逢，便胜却人间无数”成功俘获了国都一对清倌人的芳心，难能可贵的是她们还是一对娇媚可人双胞胎，不知道多少文人士子想将她们纳入房中，没想到被张仙老匹夫摘了桃子。  
>   
> 　　举办婚礼的时候，大梁国无数才俊痛哭买醉。那一年她们十八岁，张山六十岁。  
>   
> 　　后十年，大梁国爆发内乱。三皇子勾结禁军造反，皇城被围。张仙带着弟子协助守城，在城墙上一人独挡叛军，箭矢不能近身，刀剑不能伤体，先天圆满境界展露无遗。  
>   
> 　　叛乱平定后，张仙无奈接受了皇帝的政治联姻，娶了皇帝的小女儿，封镇国公。娶了公主，还能被敕封国公，这在大梁国的礼制上前所未有。小公主天姿国色，温柔可爱，没想到又双叒被张山摘了桃子。  
>   
> 　　听闻此消息的时候，多少暗恋小公主的王公贵族当场吐血。张仙你一个武学泰斗老先生还老牛吃嫩草，脸都不要了啊！  
>   
> 　　举办婚礼的时候，大梁举国买醉没人敢痛哭，无奈老贼权势太盛，禁军都在抓人。那一年她十八岁，张仙七十岁。

解析：重复句式形成滚雪球笑点：“十八岁”不变，“张仙年龄”递增，社会反应从痛哭买醉到不敢痛哭。此段同时埋下后续《大梁野史》和玉昀祖孙痛哭的素材。

- 结尾钩子：张仙七十岁仍在世俗巅峰，但修真线索仍未出现；长寿与红颜凋零将在下章反噬。

# chapter-005：第5章 韶华白首,不过转瞬

- 拆解等级：A
- 本章一句话：一百二十岁的张仙历尽世俗繁华却求仙无门，终于在荒僻道观再次发现气运之女李妙音，并遇到疑似修真者玄微及故人玉娆的小师弟玉昀。
- 表层事件：
  1. 张仙登山感慨韶华白首，回顾妻妾故去、世俗无牵挂、寿元将尽。
  2. 交代他多年研究无灵根修行，尤其关注《凝丹术》。
  3. 系统提示发现5分气运之女李妙音，张仙进入小道观探查。
  4. 张仙遇到玄微老道和玉昀，发现玄微疑似修真者。
  5. 玉昀认出张仙，提起师姐玉娆，揭出张仙昔日“七擒玉娆”的风流往事。
  6. 玄微坦言张仙先天大圆满但自己垂老，帮不了他。
- 故事逻辑：世俗巅峰无法满足求仙→寿元压力逼近→张仙寻找修真突破法→系统提示新气运女→道观不简单，可能有修真线索→故人关系制造喜剧尴尬→玄微指出修真门槛，引出下一章谈灵根与灵气。
- 本章功能：从世俗爽文转入修真求道；用“红颜皆逝”给张仙补上时间重量；引入李妙音、玄微、玉昀和道观线；引出《凝丹术》作为无灵根修仙方案。
- 关键人物/势力：
  - 张仙：一百二十岁，先天圆满，世俗无牵挂，寿元约剩三十年。
  - 新增项·李妙音：系统提示5分气运之女，尚未正式出场。
  - 新增项·玄微：老道士，气息衰弱但疑似修真者。
  - 新增项·玉昀：中年道士，玉娆小师弟，四十年前被张仙一拳打晕。
  - 新增项·玉娆：昔日“魔教圣女”，张仙妻子之一，五年前过世。
  - 新增项·无名道观：荒僻山间道观，隐藏修真线索。
- 人物/势力/背景/世界设定：
  - 新增项·寿元设定：先天圆满加九转涅槃丹效果，张仙可活约一百五十岁。
  - 新增项·修真门槛：张仙五行圆满，却因没有灵根无法转化灵气。
  - 新增项·《凝丹术》：上古修真者构想，以天材地宝凝结假丹，借假丹转化灵气修行。
  - 新增项·圣教/魔教：玉娆原属势力，后因圣女嫁给张仙而解散。
- 新增设定/伏笔：
  - 伏笔·李妙音身份：状态=埋设；系统提示但未见其人。
  - 伏笔·玄微垂老原因：状态=埋设；疑似修真者但气息衰败。
  - 伏笔·《凝丹术》可行性：状态=埋设；后续返还功法关键。
  - 伏笔·玉昀对白月光玉娆执念：状态=埋设；后续对张仙敌意和搞笑误会来源。
- 常见场景类型：寿元感慨 / 求仙无门 / 道观遇高人 / 故人重逢 / 风流旧事 / 茶室谈道。
- 情绪/爽点/搞笑点/抽象点：
  - 情绪：红颜成土，唯主角不老，形成长生前的孤独感。
  - 爽点：系统多年沉寂后再次提示气运之女。
  - 搞笑点：玉昀刚问师姐，得知过世后哭；玄微一拂尘扇飞他；“像一个两百斤的孩子”。
  - 抽象点：张仙一生风流造成的历史债，随机遇到一位受害者小师弟。
- 原文摘录候选：

> 张仙站在山巅，望着远处的落日，鬓角的丝丝白发一如当年，面容更是不曾改变。  
>   
> 　　韶华白首,不过转瞬。  
>   
> 　　曾经叱咤风云的霸王镖局早已换了主人，剑圣之女、那对娇媚可人的双胞胎清倌人、大梁国的小公主……她们都已化作黄土，唯有他，依旧活着。  
>   
> 　　先天境界和九转涅槃丹的效果延缓了他的衰老，却也让他尝尽了人间的孤独。  
>   
> 　　早在五十年前，他就已经到达了先天圆满，此后又十年，他已经隐隐摸到了突破的门槛，就好像是感觉有一条修真的道路摆在面前，却被迷雾笼罩，多年不得寸进。

解析：本章开头将第4章的世俗爽文反向结算：娶到的红颜都成黄土，世俗权势也换主人。标题“韶华白首”与“不曾改变的面容”形成长寿孤独。可迁移结构：上一阶段爽点→时间流逝→爽点代价显现。

> 张仙知道这就是南宫遥所说的没有灵根，无法修行。哪怕他已经五行圆满，却无法转化出一丝灵气。  
>   
> 　　不过这么多年的经历也足够精彩了，无论是财富权势还是红颜知己他都达到了世俗的巅峰，只要他愿意，皇位都唾手可得。可是随着他亲手埋葬了最后一名妻子，在这里已经没有牵挂的了，而寿元也看到了终点。  
>   
> 　　先天圆满，再加上九转涅槃丹的效果，他可以活到一百五十岁。  
>   
> 　　可如今，他已经一百二十岁了。  
>   
> 　　假如这不是一个修仙世界的话，这些年的故事应该是一篇无敌爽文了吧。  
>   
> 　　可惜……

解析：这是世界层级压制的典型写法：在武侠/世俗层面已无敌，但放在修仙大世界仍是失败者。寿元数字明确给出倒计时，推动主角继续寻找修真入口。

> 唯一引起张仙兴趣的是一本叫做《凝丹术》的修炼法诀，应该是一名上古修真者写的，是想借用天材地宝在体内凝结出一枚假丹出来，借用这枚假丹转化天地灵气归纳自身，以此来修行。顺利的话可以炼气甚至筑基成功，最后再利用筑基的实力反哺假丹，化假丹为金丹。  
>   
> 　　张仙看完立马被作者的想法所折服，只可惜这本《凝丹术》只是个构想，中间有很多过程都不成熟。张仙研究了多年也没有什么关键性的进展。  
>   
> 　　这一日，他云游至一座荒僻山间的小道观。  
>   
> 　　【叮！发现5分气运之女，李妙音。】  
>   
> 　　系统的提示音在他脑海中响起，让他心头一震。  
>   
> 　　“终于……又遇到了吗？”

解析：《凝丹术》是张仙无灵根修仙的技术路线铺垫，紧接系统提示气运女，读者会自然期待“赠送《凝丹术》→返还完整版”。此处完成“需求—道具—机会”的闭环铺设。

> 张仙走上前，温声问道："小道友，请问观主可在？"  
>   
> 　　小道士头也不抬："师父在后山采药，傍晚才回。"  
>   
> 　　张仙随便找了个地方坐了下来，刚才他观察过了，气运之女不在来往的百姓之中，从这两个小道士的不凡中，张仙有直觉，她应该跟这里的观主有关。  
>

> 　　就在张仙脑补的时候，两道身影拾阶而上。后面的是个中年道士，背着个药篓，板着个脸。走在前面的那位是个清瘦的老道士，眉毛花白，满脸皱纹，眼神却炯炯有神。老道士似有所感，老远的目光就和张仙对上。  
>

解析：道观探查采用侦探式推进：系统提示但找不到目标→观察小道士不凡→推测目标与观主有关→玄微出场强者互感。主角此时更成熟，不再像前期见女就冲。

> 终于，是中年人率先打破了气氛，他像是忍了很久，终于对着张仙说道：“我师姐现在怎么样了！”  
>   
> 　　啊？  
>   
> 　　张仙一脸懵逼的看着中年人，中年人有点憋屈道：“我师姐玉娆。我是他的小师弟玉昀，我们四十年前还交过手，当年你一拳打我打晕了。”  
>   
> 　　玉娆……那已经是一段陈年旧事了。  
>   
> 　　当年有一个自诩为圣教的帮派，行事有些乖张，江湖人士称之为魔教，但是在张仙看来就是个非主流团体。  
>   
> 　　本来这些和张仙没什么关系，但是魔教总是和自己老丈人剑圣的门派过不去。张仙看不过去，就把他们收拾了。

解析：严肃求仙场突然接入风流旧债，降低沉重感。张仙的“魔教=非主流团体”“脸蛋身材不错所以没杀”保持一贯不正经视角，也为玉昀敌意提供合理来源。

> 再之后，魔以此为奇耻大辱，而圣女玉娆作为魔教牌面，对张仙进行了多次暗杀偷袭，然后就没有然后了。

>   
> 　　圣教赔了圣女，士气跌落谷底，便解散了各奔前程。  
>

>   
> 　　玉昀听罢顿时老泪纵横，她打小就痴恋的师姐，她的白月光……  
>

解析：“七擒玉娆”是对古典桥段的恶搞复用，结尾用玄微一拂尘扇飞玉昀制造无厘头笑点。玉昀的悲情被喜剧化处理，避免纯虐。

> 玄微到底是没张仙脸皮那么厚，最终叹了一口气道：“镇国公，贫道不善言辞，也不会拐弯抹角。我知你已先天大圆满，勘破修真的门径。但你亦看出贫道垂垂老矣，并不能帮你什么。”

解析：结尾回到主线：张仙求仙，玄微识破其境界但表示帮不了。这个“不帮”不是拒绝，而是能力不足，为下一章解释“灵根和天地灵气”留钩子。

- 结尾钩子：玄微看出张仙先天大圆满，却说自己帮不了；李妙音仍未正式现身。

# 文风控制卡

## 1. 叙述人设

- 使用**第三人称贴近主角视角**。
- 旁白像主角嘴替，允许现代口语、网文梗、轻粗口、反套路评价。

- 不要写成正统古典仙侠；仙侠氛围出现后，要适时用主角吐槽破功。

- 中短句为主，1-4句一段。
- 爆点、吐槽、系统提示、破防反应可单句成段。

- 每章至少有3-6个明显笑点。
- 严肃设定不要连续超过5-6段，期间插入主角判断或吐槽。

## 3. 段落配比

建议单章比例：

- 对话：35%-45%。

- 设定说明：10%-20%。
- 外貌/环境：10%-15%。

1. 当前处境。
2. 主角观察。
3. 主角心理吐槽/算计。
4. 对话或行动触发冲突。
5. 系统/资源/误会制造转折。
6. 配角反馈。
7. 留下下一步目标或钩子。

> → 厚脸皮送礼或建立关系  
> → 系统返还高级资源  
> → 主角暗爽并总结规则  
> → 配角震惊或误会升级  
> → 新目标打开。

```text

## 2. 句式节奏拆解

### 2.1 开篇常用“好消息/坏消息”或反差式开局

第1章开头用“好消息/坏消息”快速建立反差：

- 好消息：穿越且带系统。
- 坏消息：系统没法用。

功能：

- 三句话内交代设定。
- 迅速抛出矛盾。
- 语气像现代段子，奠定沙雕吐槽基调。

可迁移方式：

> 好消息：主角获得了某个逆天条件。  
> 坏消息：这个条件暂时毫无用处，甚至反过来折磨主角。

### 2.2 爆点前短句加密

遇到系统提示、美女出现、返还成功时，句子明显变短：

- “又来？”
- “有戏！”
- “问题不大。”
- “卧槽发了呀！”
- “系统牛逼！”

功能：

- 模拟主角瞬间反应。
- 让节奏有弹跳感。
- 给读者明确的爽点/笑点信号。

写作建议：

- 爆点前后少用复杂长句。
- 让主角用一句短促口语给情绪定调。

外貌和环境描写可以写得较精致，但结尾常接粗俗心理。

> 再接主角“受不了”“这谁顶得住”“好感度又掉了”之类反应。

功能：

- 保留男频审美刺激。
- 防止文风变成纯古风。
- 强化主角好色但接地气的人设。

## 3. 段落节奏拆解

### 3.1 一个场景的基本结构

1. 交代地点/时间/状态。
2. 主角观察。
3. 主角心理吐槽。

7. 主角做下一步算计。

- 比如：权势、财富、美人都得到了，但修真门槛仍过不去。

```markdown
# 吐槽与笑点

## 1. 吐槽总机制

本书吐槽不是外加段子，而是和剧情推进绑定。吐槽通常出现在：

1. 主角发现系统坑爹时。
2. 美女出现但好感度下降时。
3. 修仙者端架子时。
4. 主角送礼薅羊毛时。
5. 配角误会主角动机时。
6. 严肃场面被市井逻辑破坏时。

核心笑点来自：

> **高逼格仙侠场景 + 低俗/现代/现实主义解释。**

## 2. 吐槽来源分类

| 类型 | 表现 | 功能 | 写作建议 |
|---|---|---|---|
| 系统吐槽 | 系统多年不能用、绑定条件苛刻、返还规则坑 | 建立主角憋屈感 | 主角可骂系统，但系统一返还立刻改口“系统爸爸” |
| 主角自嘲 | 怀念996、想重开、凡人无灵根 | 降低主角高大感 | 主角越惨越能吐槽，不要端着 |
| 欲望反差 | 看美女、想薅资源、想吃软饭 | 男频喜剧和爽点 | 心理可直给，但行动要有分寸和算计 |
| 身份反差 | 修真高人吃醋、老道变小白脸、道士哭师姐 | 破除仙侠严肃 | 高人也可以破防、恋爱脑、误会 |
| 误会脑补 | 玉昀怀疑张仙抢师伯 | 连续章节喜剧钩子 | 让配角越想越离谱，但逻辑自洽 |
| 现代网文梗 | 天胡开局、舔狗、卡BUG、血赚、工具人 | 拉近读者距离 | 适量使用，避免每句都是梗 |
| 粗俗比喻 | 高雅事物被比作低俗物 | 强烈反差笑点 | 关键处使用，别过密 |

## 3. 典型笑点结构

### 3.1 “严肃铺垫—粗俗破功”

结构：

1. 先写仙子、法器、宗门、道观等高逼格内容。
2. 主角或旁白突然用粗俗现实比喻评价。
3. 严肃氛围瞬间塌掉。

模板：

> 某人/某物看起来极其高贵、神秘、不可触碰。  
> 主角心里却只冒出一个非常现实甚至不雅的念头。  
> 旁白再补一刀，说明他为什么会这么想。

适用场景：

- 美女登场。
- 高人赐宝。
- 宗门设定。
- 主角面对强敌。

### 3.2 “系统坑爹—系统爸爸”

结构：

1. 主角长期被系统折磨，骂系统垃圾。
2. 主角偶然触发返还。
3. 系统送出远超预期的东西。
4. 主角立刻改口，情绪反转。

模板：

> 前一秒：这破系统迟早害死我。  
> 下一秒：系统返还逆天资源。  
> 主角：误会，都是误会。

功能：

- 强化系统爽点。
- 形成主角没节操但可爱的形象。

### 3.3 “配角脑补误会升级”

玉昀线是前10章最典型的连续笑点。

结构：

1. 张仙送李妙音东西，本意是返还。
2. 玉昀根据张仙风流史判断他图谋不轨。
3. 玉昀提醒玄微。
4. 玄微由不信到动摇。
5. 玄微恢复年轻，牵李妙音出来宣示主权。
6. 张仙一眼看穿但不点破。

笑点来源：

- 主角真实动机和配角理解完全错位。
- 配角的担忧并非毫无依据，因为张仙确实风流。
- 误会推动人物互动，而不只是段子。

写作建议：

- 误会要有“证据链”，不能空想。
- 配角越认真，读者越觉得好笑。
- 主角可以不解释，甚至顺水推舟。

### 3.4 “欲望直给但行动克制”

主角常有不正经想法，但不会无脑乱来。

例如：

- 对张乐乐的激活猜测里有下流脑洞，但立刻自我否定“年纪太小”。
- 对南宫遥战损美貌有强烈反应，但更多是想刷好感和触发返还。
- 对李妙音态度客气，本意是薅系统，不是真的追求。

功能：

- 让主角好色但不完全猥琐。
- 保留男频暧昧刺激，又避免主角失控降智。

## 4. 笑点密度建议

| 场景类型 | 笑点密度 | 建议 |
|---|---|---|
| 系统返还场景 | 高 | 每次返还前后至少2个吐槽/狂喜反应 |
| 美女登场 | 中高 | 外貌描写后接1个主角反应笑点 |
| 战斗碾压 | 中 | 用对手震惊和主角装淡定制造笑点 |
| 设定说明 | 中 | 每解释一段严肃设定，可插入一句主角现实判断 |
| 情感/悲凉 | 低到中 | 不要完全破坏情绪，可在段末轻轻自嘲 |
| 章节结尾 | 低 | 以钩子为主，不必强塞笑点 |

## 5. 吐槽句式模板

以下为抽象模板，供写作迁移：

1. **好坏消息式**
   - 好消息，主角获得了【逆天条件】。
   - 坏消息，【条件暂时无法使用/会坑主角】。

2. **系统骂街式**
   - 这破系统【限制条件】也就算了，偏偏【更坑的点】。
   - 主角心里已经把系统【夸张惩罚】了一遍。

3. **欲望反差式**
   - 理智告诉他应该【谨慎行为】。
   - 但眼前这一幕，实在让人很难继续理智。

4. **薅羊毛式**
   - 既然对方肯收，那就说明有戏。
   - 这种时候，脸皮算什么，资源才是真的。

5. **误会脑补式**
   - 某配角看着主角的动作，脸色一点点变了。
   - 他终于明白了。
   - 原来这老东西从一开始就是冲着【误会对象】来的！

6. **高人破功式**
   - 某高人原本仙风道骨。
   - 直到主角说出/做出【离谱行为】。
   - 他的道心差点当场裂开。

## 6. 禁止过度点

- 每一句都塞网络梗，导致像段子合集。
- 主角过度下流，失去“算计型爽文主角”的可读性。
- 配角无理由降智，误会必须有来源。
- 严肃情节完全被搞笑冲散，尤其复仇、寿命、修真瓶颈等主线要保留重量。

# 爽点表达

## 1. 爽点总公式

前10章最核心的爽点公式是：

> **低位困境 → 发现天命女 → 厚脸皮送礼/接触 → 系统返还 → 主角暗爽 → 配角震惊或误会 → 新目标打开。**

爽点不是直接“主角无敌”，而是“主角用看似低端的东西换到高端资源”。

## 2. 爽点前压制方式

| 压制类型 | 表现 | 作用 |
|---|---|---|
| 环境压制 | 被困大山十五年，无法离开 | 拉低起点，积蓄爆发 |
| 系统压制 | 必须气运之女且天命属性激活 | 让主角长期憋屈 |
| 身份压制 | 凡人面对修真者，被看成野人/垃圾 | 制造打脸基础 |
| 资源压制 | 无灵根、无灵气、无修真资源 | 让返还显得珍贵 |
| 情绪压制 | 小黑丫头离开、南宫遥杀他、寿元将尽 | 加重主线目标 |
| 信息压制 | 不知道云渺宗在哪，不懂修真界 | 推动探索和请教 |

写作建议：

- 每个爽点前必须让主角“缺一样东西”。
- 缺资源、缺路径、缺身份、缺信息都可以。
- 主角不能一开始就轻松拿到，需要先试探、被鄙视或被拒绝。

## 3. 爽点触发方式

### 3.1 系统弹窗

系统提示短促、直接，通常包括：

1. 发现气运之女。
2. 好感度变化。
3. 绑定成功。
4. 赠送成功。
5. 返还物品。
6. 特殊规则提示。

写法建议：

```text
【叮！发现X分气运之女，某某。】
【叮！某某对你的好感度为X，绑定成功。】
【叮！赠送某物成功。触发返还，返还某高级物。】
```

系统语言特点：

- 简洁。
- 功能性强。
- 不卖萌。
- 不解释太多，只在规则变动时提示。

### 3.2 主角反应

系统返还后，主角反应一般分三层：

1. **瞬间懵住。**
2. **狂喜。**
3. **立刻算计下一步。**

> 主角先是愣住。  
> 随后心脏狂跳，差点没笑出声。  
> 但他很快压住表情，开始盘算还能不能继续送。

写作建议：

3. 系统返还。
4. 主角试探系统规则。
5. 发现不是所有东西都能返还。
6. 主角总结经验。

爽点来自：

- 主角用凡间资源换修真资源。
- 主角通过失败返还理解系统规则。
- 对方态度因为资源变好。

迁移规则：

- 一次送礼不要只送一个物品，可连续试探。
- 每次返还都要带出规则边界。
- 失败返还也有价值，用来显得系统不是无限提款机。

### 4.4 “功法返还解决死局”

第8章《凝丹诀》返还《青木养丹经》，解决无灵根问题，是前10章最大功能性爽点。

1. 主角提出修真难题。
2. 对方认为希望渺茫。
3. 主角送出低阶构想功法。
4. 系统返还可执行高级功法。
5. 主角当夜修炼成功。
6. 第二天配角震惊。

写作建议：

- 功法返还要精准解决主线卡点。
- 修炼描写要有身体变化、境界变化、寿元变化。
- 第二天必须有外人认证，强化爽感。

### 4.5 “战斗碾压爽”

前10章战斗不多，但有几个关键碾压：

- 张仙应聘镖局，随手丢出壮汉，葛老认出先天。
- 张仙用如意短剑压制玄微。
- 张仙挥手打飞玉昀。

- 用“对方还没反应过来”体现差距。

模板：

> 主角只是随意一抬手/一挥袖。  
> 对手已经飞了出去。  
> 真正识货的人脸色大变，立刻改口称前辈。

1. **推进剧情**：谈条件、带走张乐乐、解释修真界、决定去大世界。

- “这是在下偶然得之……”

| 会试探规则 | 用不同物品送礼观察系统 |

- 骂南宫遥。
- 评估美女。
- 算资源账。
- 判断能不能继续薅。

写作建议：

> 主角嘴上越客气，内心越可以现实。  
> 形成“正经话术”和“缺德心理”的反差。

   - “此物于我无用，想来对道友或许有些帮助。”
   - “道友若不收，在下心中难安。”

3. **掩饰型**
   - “在下只是听友人提起。”
   - “我确实没有恶意。”
   - “我与某宗颇有渊源，此去只是拜谢。”

### 3.1 语言特点

- 频繁称呼“山哥哥”。
- 语气娇憨、直白、带依赖。
- 面对仙女自卑但执拗。
- 对离开张山十分抗拒。

功能：

- 强化张山被大山困住的情感牵挂。
- 作为气运之女伏笔。
- 推动张山离开大山的可能性。

写作建议：

- 张乐乐说话不要太成熟。
- 她可以懵懂，但关键选择上执拗。
- 她的依赖要带一点小媳妇感。

## 4. 南宫遥对话

### 4.1 语言特点

南宫遥是清冷高位者，说话短、淡、带压迫感。

常见表达：

- “不必了。”
- “不可。”
- “劳烦乐乐姑娘就好。”
- “我观你颇有天资。”

特点：

| 特点 | 表现 |
|---|---|
| 高位 | 习惯安排别人 |
| 冷淡 | 对张山惜字如金 |
| 功利 | 看中张乐乐天赋 |
| 压迫 | 语气淡然但不容拒绝 |
| 隐藏危险 | 前期只是冷，后期直接杀主角 |

写作建议：

- 她对张山的台词要少且冷。
- 不要解释太多真实想法。
- 可通过好感度下降和动作表现嫌弃。

## 5. 李妙音对话

### 5.1 语言特点

李妙音成熟温和，有道姑的克制感，但面对玄微会有柔软一面。

特点：

- 对张仙一开始警惕。
- 收到灵石后态度转好，但仍保持礼数。
- 讲述过往时语气淡然。
- 对玄微有嗔怪、害羞和坚定。

写作建议：

- 李妙音不要像少女一样咋呼。
- 她的情绪要藏在“微微一笑”“瞥一眼”“语气放缓”里。
- 重要情感可以用一句克制但坚定的话收束。

## 6. 玄微对话

### 6.1 初登场

玄微初登场仙风道骨，话语清朗、直接：

- 点出灵根和天地灵气。
- 不绕弯说帮不了张仙。
- 表现出修真者见识。

### 6.2 破功后

一涉及李妙音，玄微从高人变成恋爱脑：

- 受伤后被师姐关心，语气柔和。
- 恢复年轻后有点炫耀。
- 牵手宣示主权。

写作建议：

- 玄微的反差是笑点关键。
- 平时像高人，说到师姐像少年。
- 不能写得太蠢，他对修真知识仍有权威。

## 7. 玉昀对话

玉昀是误会喜剧发动机。

### 7.1 语言特点

- 对张仙称“老贼”。
- 情绪激烈。
- 常用哭诉、控诉、提醒。
- 内心戏远多于实际有效行动。

功能：

- 替读者吐槽张仙风流史。
- 制造“张仙要抢师伯”的误会。
- 用祖孙四代被张仙抢女神的设定扩大笑点。

写作建议：

- 玉昀说话要夸张，有悲愤感。
- 他不是纯小丑，他的担忧有历史依据。
- 对师父要恨铁不成钢。

## 8. 反派/敌对者对话

前10章真正反派不多，敌对者多是：

- 嘲讽张仙的壮汉。
- 冷淡的南宫遥。
- 对张仙敌视的玉昀。

敌对者常见对话：

- 先轻视主角。
- 用“老东西”“凡夫俗体”等压低。
- 主角出手后迅速被打脸。

写作建议：

- 反派嘴脸可以直白，不必复杂。
- 打脸前要给他们一句嘲讽台词。
- 打脸后由识货者解释主角强度。

## 9. 对话节奏规则

| 场景 | 对话节奏 |
|---|---|
| 谈条件 | 一问一答，主角不断加码 |
| 送礼 | 主角客气，对方推辞，主角硬塞 |
| 误会 | 配角内心长篇脑补，实际话语短促 |
| 修真设定 | 高人解释，主角插问 |
| 情感告白/回忆 | 语速放慢，句子稍长 |
| 爽点触发 | 对话暂停，系统弹窗插入 |

## 10. 对话禁忌

- 不要让所有人物都用现代梗；现代梗主要属于主角和旁白。
- 修真者可以破功，但正式解释设定时要有可信度。
- 女角色不要全都同一种娇羞语气。
- 主角不要把真实系统目的说出口。
- 对话不要大段古文腔，会破坏原作口语爽文气质。

## 2. 写作 agent 需要调整的核心策略

### 2.1 不要只“避开原文”，要“替代原文功能”

2. 保留功能。
3. 替换表达、动作、比喻和细节。
4. 不要直接压缩成概述。
```

例如林茵茵：

- 不照搬原文腿、汗、弯腰等动作。

- 第7章中品灵石。

### 2.3 增加配角反馈强度

原文很多爽点靠配角“识货”放大。

建议写作 agent 增加：

```markdown

练习稿吐槽有，但偏温和。

- 1处配角误会或破防。
- 1处仙侠高逼格被现代口语拉低。

- 他可以真情，但多数时候要现实。

第4、5、9章尤其明显。

```markdown
如果拆解中标为高光/核心笑点，不可压缩成几句总结。
