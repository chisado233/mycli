# 文风控制卡

## 1. 叙述人设

- 使用**第三人称贴近主角视角**。
- 旁白像主角嘴替，允许现代口语、网文梗、轻粗口、反套路评价。
- 叙述气质：**修仙爽文 + 社畜吐槽 + 沙雕误会 + 男频欲望直给**。
- 不要写成正统古典仙侠；仙侠氛围出现后，要适时用主角吐槽破功。
- 主角张山/张仙的核心气质：脸皮厚、现实、好色但有底线、会算计、以薅系统资源为第一目标。

## 2. 句式节奏

- 中短句为主，1-4句一段。
- 爆点、吐槽、系统提示、破防反应可单句成段。
- 爽点前用中长句铺困境；爽点瞬间用短句和系统弹窗；爽点后用主角狂喜和配角震惊放大。
- 每章至少有3-6个明显笑点。
- 严肃设定不要连续超过5-6段，期间插入主角判断或吐槽。

## 3. 段落配比

建议单章比例：

- 对话：35%-45%。
- 主角心理吐槽：20%-30%。
- 动作推进：15%-20%。
- 设定说明：10%-20%。
- 外貌/环境：10%-15%。

写场景时按以下顺序组织：

1. 当前处境。
2. 主角观察。
3. 主角心理吐槽/算计。
4. 对话或行动触发冲突。
5. 系统/资源/误会制造转折。
6. 配角反馈。
7. 留下下一步目标或钩子。

## 4. 笑点规则

常用笑点来源：

- 系统坑爹。
- 主角自嘲。
- 主角欲望和现实算计。
- 修仙高人破功。
- 配角误会。
- 现代词汇撞进修仙语境。
- 粗俗比喻打破高雅气氛。

写法要求：

- 美女、法宝、宗门等高逼格内容后，可接一句主角接地气吐槽。
- 配角误会要有证据链，不要纯降智。
- 主角可以有不正经想法，但行动要保持算计和基本底线。
- 不要每句都玩梗，笑点要服务剧情推进。

## 5. 爽点写法

核心爽点公式：

> 主角缺资源/被轻视/被规则卡住  
> → 找到气运女  
> → 厚脸皮送礼或建立关系  
> → 系统返还高级资源  
> → 主角暗爽并总结规则  
> → 配角震惊或误会升级  
> → 新目标打开。

系统写法：

```text
【叮！发现X分气运之女，姓名。】
【叮！目标对你的好感度为X，绑定成功。】
【叮！赠送某物成功。触发返还，返还某物。】
【特殊提示：规则限制。】
```

注意：

- 系统不卖萌，不和主角聊天。
- 返还必须基于“目标需要”。
- 可以出现无效返还或部分返还，用来展示规则。
- 系统产出物不能重复送回同一目标刷返还。
- 每次大返还都要解决当前主线痛点，如功法、灵石、法器、寿元、入场资格。

## 6. 对话规则

### 张山/张仙

- 嘴上客气，心里现实。
- 常用“在下”“道友”“仙子”“不情之请”等礼貌包装。
- 实际目的多是试探、索取、送礼、薅羊毛。
- 面对强者先稳住，能装就装，能骗就骗，能送就送。

### 张乐乐

- 依赖、娇憨、执拗。
- 说话带亲近感，围绕张山安全和陪伴。
- 不要写得过于成熟。

### 南宫遥

- 清冷、高位、短句、压迫感。
- 对张山冷淡甚至嫌弃。
- 不解释太多，更多用动作和好感度体现态度。

### 李妙音

- 成熟、温和、克制。
- 对外守礼，对玄微有柔软和害羞。
- 讲过去时淡然，不怨天尤人。

### 玄微

- 修真知识上像高人。
- 涉及李妙音时会破功、吃醋、少年感。
- 反差是笑点，但不能写成纯傻子。

### 玉昀

- 误会型吐槽担当。
- 认为张仙风流危险，常悲愤脑补。
- 对师父恨铁不成钢。
- 他的误会要基于张仙过往风流史和送礼行为。

## 7. 设定说明规则

- 不要集中百科。
- 设定通过系统提示、对话问答、主角试错、配角震惊展开。
- 境界说明要绑定实际收益：能打谁、能活多久、能去哪、消耗多少资源。
- 新资源出现后尽快展示用途。
- 修真资源价值通过识货者反应体现。
- 大地图设定要落在具体场景里，如矿脉、坊市、宗门、洞府。

当前关键设定：

- 大梁是灵气断绝的小世界。
- 修真大世界灵气充沛，宗门林立。
- 云渺宗是南域大宗，水云门是其附属门派。
- 主角无天然灵根，靠《青木养丹经》与灵石凝假丹，踏上修真路。
- 主角需要更多气运女和更多资源，才能继续升级。

## 8. 文雅与通俗切换

- 美女登场、山巅落日、修真大世界、法宝功法可以写文雅。
- 文雅描写后要适时接主角口语反应，形成反差。
- 悲凉场景可保留情绪，不要立刻用重梗砸碎；可以用轻自嘲收尾。
- 外貌描写重点：衣着、姿态、肌肤、动作、周围人反应；但避免照搬原文具体意象。

## 9. 章节钩子规则

每章结尾必须留钩子，优先选择：

1. 系统提示新气运女。
2. 新美女/新高人登场。
3. 返还出关键资源。
4. 主角身份暴露风险。
5. 配角误会升级。
6. 仇人线索出现。
7. 地图切换或新副本开启。

结尾不要平收。钩子要短，下一章开头必须兑现。

## 10. 禁用事项

不得照搬：

- 原文章节标题。
- 原文标志性粗俗比喻。
- 原文连续句子。
- 原文“好消息/坏消息”具体句。
- 原文“摘桃子”重复梗。
- 玉昀原文标志性哭诉句。
- 南宫遥杀主角后的原文骂句。
- 林茵茵挖矿外貌段落原句。

可迁移但需改写：

- 好坏消息式反差。
- 系统弹窗结构。
- 送礼返还循环。
- 配角误会喜剧。
- 美女文雅描写后接主角吐槽。
- 世俗巅峰后回到修真瓶颈的结构。

## 11. 单章执行检查表

写完每章后检查：

- [ ] 是否贴近主角视角？
- [ ] 是否有现代口语吐槽？
- [ ] 是否有明确目标推进？
- [ ] 是否有至少3个笑点？
- [ ] 是否有爽点或爽点铺垫？
- [ ] 是否有配角反馈或误会？
- [ ] 设定是否通过行动/对话带出，而非百科？
- [ ] 结尾是否有强钩子？
- [ ] 是否避免照搬原文标志性表达？
