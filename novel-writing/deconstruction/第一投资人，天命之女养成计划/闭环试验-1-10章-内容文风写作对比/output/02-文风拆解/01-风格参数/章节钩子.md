# 章节钩子

## 1. 总体规律

第1-10章几乎每章都用强钩子结尾，且钩子直接服务下一章开场。

常见钩子类型：

| 类型 | 说明 |
|---|---|
| 系统提示 | 发现新气运女、新目标 |
| 人物登场 | 新美女、新高人、新势力出现 |
| 反转 | 刚获得好处马上被杀/被坑 |
| 误会升级 | 配角脑补出更严重后果 |
| 地图切换 | 从大山到大梁，从大梁到修真界 |
| 情绪爆点 | 主角破防、发誓复仇 |
| 新副本 | 道观、矿脉、宗门等具体场景开启 |

## 3. 章节开头方式

### 3.1 直接承接上章钩子

常见：

- 上章系统提示，下章立刻写主角反应和现场变化。
- 上章人物登场，下章先写主角惊讶。
- 上章误会升级，下章从误会方的内心继续。

写作建议：

> 不要跳开上章钩子。新章前几段必须兑现上章悬念。

### 3.2 用短句制造继续阅读感

例如：

- “又来？”
- “张山知道再这样就要把南宫遥惹毛了。”
- “张仙笑了笑，这老登说话太直接了。”
- “玄微此刻正沉浸在……”

开头特点：

- 不绕。
- 立刻回到冲突。
- 一句话带出主角态度。

### 3.3 时间跳跃开头

当阶段结束时，直接跳时间：

- “五年后。”
- “三个月后。”
- “春去秋来，转眼又三十多年过去。”
- “半年后，修真大世界……”

写作建议：

- 时间跳跃后要先给主角新状态。
- 再用几段概括中间成就。
- 最后马上放入新冲突。

## 4. 中段转场方式

### 4.1 系统提示转场

系统提示是最强转场工具：

> 主角正处于某种日常/沉思中，系统突然提示新气运女。  
> 主角立刻改变行动路线。

适合：

- 新副本。
- 新女角色。
- 新资源线。

### 4.2 对话提出新目标

如李妙音、玄微解释修真界后，主角决定去大世界。

结构：

1. 配角说明限制。
2. 主角问关键信息。
3. 得到地点/势力线索。
4. 主角制定下一步。

### 4.3 爽点后转目标

例如：

- 得到法器后，目标从困山变为出山。
- 得到青木养丹经后，目标从求道变为去大世界。
- 获得灵石后，目标从活命变为广撒网。

## 5. 结尾钩子模板

### 5.1 系统提示型

```text
主角刚刚放下某个念头，脑海里忽然响起提示。
【叮！发现X分气运之女，某某。】
```

适合新角色引入。

### 5.2 反转打击型

```text
主角刚以为局面好转。
下一刻，熟悉/强大的敌人出现。
他甚至来不及开口，死亡/危机已经降临。
```

适合复仇线和高压章节。

### 5.3 误会升级型

```text
配角看着主角和某女的互动，脸色逐渐惨白。
他终于明白了。
真正要出事的不是别人，而是某个他最在乎的人。
```

适合喜剧副本。

### 5.4 地图切换型

```text
半年后。
新地图，一个具体场景中。
一个新人物正在做某件带视觉冲击的小事。
```

适合卷/阶段转换。

### 5.5 情绪爆发型

```text
主角终于明白自己被坑/被杀/被背叛。
他破口大骂，立下复仇或夺回目标的念头。
```

适合主线仇恨推进。

## 6. 钩子写作规则

1. 每章结尾必须改变读者预期。
2. 钩子最好和“女人、系统、资源、仇人、地图”之一有关。
3. 不要用无关悬念。
4. 钩子要短，不要解释完。
5. 下一章开头必须兑现。
6. 喜剧钩子可以用配角误会，主线钩子要用系统/敌人/地图。

## 7. 禁忌

- 不要章节结尾平铺直叙收束。
- 不要结尾只写“他决定努力修炼”这种弱钩子。
- 不要连续多章同一种钩子，系统提示、误会、反转、地图应交替使用。
- 不要在钩子后立刻解释答案。
