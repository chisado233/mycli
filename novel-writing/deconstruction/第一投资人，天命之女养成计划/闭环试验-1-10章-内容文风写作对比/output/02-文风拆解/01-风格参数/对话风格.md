# 对话风格

## 1. 总体对话特点

第1-10章对话有三个核心功能：

1. **推进剧情**：谈条件、带走张乐乐、解释修真界、决定去大世界。
2. **制造误会**：玉昀误读张仙送礼动机。
3. **塑造人物**：张仙厚脸皮，南宫遥清冷，李妙音温和，玄微恋爱脑。

对话整体偏短，不追求古风腔浓厚。人物身份是修真者、道士、仙子，但说话并不完全文绉绉，常夹杂直白表达。

## 2. 主角张山/张仙对话

### 2.1 外在口吻

张仙说话有两层：

- 表面：客气、拱手、讲礼貌。
- 内里：算计、薅羊毛、厚脸皮。

常见话术：

- “仙子不如这样……”
- “多谢仙子，不过……”
- “这是在下偶然得之……”
- “在下真的没有恶意。”
- “实不相瞒……”

特点：

| 特点 | 表现 |
|---|---|
| 厚脸皮 | 收了东西还继续要 |
| 会找理由 | 送匕首说成村中习俗 |
| 会降低敌意 | 动辄“没有恶意”“结交一下” |
| 会试探规则 | 用不同物品送礼观察系统 |
| 伪装正经 | 心里狂喜，表面淡然 |

### 2.2 内心口吻

主角内心比嘴上粗俗得多：

- 骂系统。
- 骂南宫遥。
- 评估美女。
- 算资源账。
- 判断能不能继续薅。

写作建议：

> 主角嘴上越客气，内心越可以现实。  
> 形成“正经话术”和“缺德心理”的反差。

### 2.3 主角对话模板

1. **索取型**
   - “既然道友如此慷慨，在下还有个不情之请。”
   - “不知可否赐教一二？”
   - “哪怕只是凡间手段，也足够在下受用了。”

2. **送礼型**
   - “先前多有得罪，这点东西权作赔礼。”
   - “此物于我无用，想来对道友或许有些帮助。”
   - “道友若不收，在下心中难安。”

3. **掩饰型**
   - “在下只是听友人提起。”
   - “我确实没有恶意。”
   - “我与某宗颇有渊源，此去只是拜谢。”

## 3. 张乐乐对话

张乐乐前期是依赖型少女。

### 3.1 语言特点

- 频繁称呼“山哥哥”。
- 语气娇憨、直白、带依赖。
- 面对仙女自卑但执拗。
- 对离开张山十分抗拒。

功能：

- 强化张山被大山困住的情感牵挂。
- 作为气运之女伏笔。
- 推动张山离开大山的可能性。

写作建议：

- 张乐乐说话不要太成熟。
- 她可以懵懂，但关键选择上执拗。
- 她的依赖要带一点小媳妇感。

## 4. 南宫遥对话

### 4.1 语言特点

南宫遥是清冷高位者，说话短、淡、带压迫感。

常见表达：

- “不必了。”
- “不可。”
- “劳烦乐乐姑娘就好。”
- “我观你颇有天资。”

特点：

| 特点 | 表现 |
|---|---|
| 高位 | 习惯安排别人 |
| 冷淡 | 对张山惜字如金 |
| 功利 | 看中张乐乐天赋 |
| 压迫 | 语气淡然但不容拒绝 |
| 隐藏危险 | 前期只是冷，后期直接杀主角 |

写作建议：

- 她对张山的台词要少且冷。
- 不要解释太多真实想法。
- 可通过好感度下降和动作表现嫌弃。

## 5. 李妙音对话

### 5.1 语言特点

李妙音成熟温和，有道姑的克制感，但面对玄微会有柔软一面。

特点：

- 对张仙一开始警惕。
- 收到灵石后态度转好，但仍保持礼数。
- 讲述过往时语气淡然。
- 对玄微有嗔怪、害羞和坚定。

写作建议：

- 李妙音不要像少女一样咋呼。
- 她的情绪要藏在“微微一笑”“瞥一眼”“语气放缓”里。
- 重要情感可以用一句克制但坚定的话收束。

## 6. 玄微对话

### 6.1 初登场

玄微初登场仙风道骨，话语清朗、直接：

- 点出灵根和天地灵气。
- 不绕弯说帮不了张仙。
- 表现出修真者见识。

### 6.2 破功后

一涉及李妙音，玄微从高人变成恋爱脑：

- 受伤后被师姐关心，语气柔和。
- 恢复年轻后有点炫耀。
- 牵手宣示主权。

写作建议：

- 玄微的反差是笑点关键。
- 平时像高人，说到师姐像少年。
- 不能写得太蠢，他对修真知识仍有权威。

## 7. 玉昀对话

玉昀是误会喜剧发动机。

### 7.1 语言特点

- 对张仙称“老贼”。
- 情绪激烈。
- 常用哭诉、控诉、提醒。
- 内心戏远多于实际有效行动。

功能：

- 替读者吐槽张仙风流史。
- 制造“张仙要抢师伯”的误会。
- 用祖孙四代被张仙抢女神的设定扩大笑点。

写作建议：

- 玉昀说话要夸张，有悲愤感。
- 他不是纯小丑，他的担忧有历史依据。
- 对师父要恨铁不成钢。

## 8. 反派/敌对者对话

前10章真正反派不多，敌对者多是：

- 嘲讽张仙的壮汉。
- 冷淡的南宫遥。
- 对张仙敌视的玉昀。

敌对者常见对话：

- 先轻视主角。
- 用“老东西”“凡夫俗体”等压低。
- 主角出手后迅速被打脸。

写作建议：

- 反派嘴脸可以直白，不必复杂。
- 打脸前要给他们一句嘲讽台词。
- 打脸后由识货者解释主角强度。

## 9. 对话节奏规则

| 场景 | 对话节奏 |
|---|---|
| 谈条件 | 一问一答，主角不断加码 |
| 送礼 | 主角客气，对方推辞，主角硬塞 |
| 误会 | 配角内心长篇脑补，实际话语短促 |
| 修真设定 | 高人解释，主角插问 |
| 情感告白/回忆 | 语速放慢，句子稍长 |
| 爽点触发 | 对话暂停，系统弹窗插入 |

## 10. 对话禁忌

- 不要让所有人物都用现代梗；现代梗主要属于主角和旁白。
- 修真者可以破功，但正式解释设定时要有可信度。
- 女角色不要全都同一种娇羞语气。
- 主角不要把真实系统目的说出口。
- 对话不要大段古文腔，会破坏原作口语爽文气质。
