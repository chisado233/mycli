# 句式模板

> 本文件只提供抽象句法模板，不照搬原文具体表达。

## 1. 好坏消息式开局

```text
好消息，【主角获得某个逆天条件】。

坏消息，【这个条件暂时不能用/有致命限制】。
```

变体：

```text
好消息，【主角终于等到机会】。

坏消息，【机会看起来比没有还坑】。
```

功能：快速建立反差和喜剧基调。

## 2. 现代社畜吐槽式

```text
作为一个【现代身份/倒霉经历】的人，主角曾经以为【理想化判断】。

直到【现实条件】狠狠给了他一巴掌。
```

```text
相比于【当前苦难】，主角甚至开始怀念【现代痛苦经历】。
```

功能：用现代经验解构异世界苦难。

## 3. 天胡开局反转式

```text
【条件一】？无所谓。

【条件二】？问题不大。

【主角自信判断】。

直到【真正限制】出现，主角才发现事情不对。
```

功能：制造开局期待，再快速打脸。

## 4. 系统提示式

```text
【叮！发现X分气运之女，姓名。】
```

```text
【叮！目标对你的好感度为X，绑定成功/失败。】
```

```text
【叮！赠送某物成功。触发返还，返还某物。】
```

```text
【特殊提示：某规则限制。】
```

功能：清晰传递规则和爽点。

## 5. 主角内心破防式

```text
主角整个人都愣住了。

我【粗口/情绪词】！

这【情况/规则】算怎么回事？
```

```text
如果【某对象】有实体，主角恨不得把它【夸张惩罚】。
```

功能：放大主角情绪，制造笑点。

## 6. 美女登场描写式

```text
那是一个【年龄感/身份感】的女子。

她【发、眸、肤、衣着】如何，【姿态/伤势/光影】如何，明明【状态不佳】，却更显出【气质/诱惑/仙气】。
```

尾部破功：

```text
这一幕看得主角【生理/心理反应】。

要不是【理智/处境限制】，他怕是已经【失态行为】。
```

功能：先给审美，再给男频吐槽。

## 7. 高冷女修嫌弃式

```text
女修淡淡看了主角一眼。

在她眼里，主角不过是【低位评价】，既无【资质】，也无【价值】。

若非【某个理由】，她甚至懒得多说一句。
```

功能：压低主角，为后续打脸或返还铺垫。

## 8. 厚脸皮索取式

```text
主角收下【好处】后，仍不满足。

这种时候，脸皮算什么。

他拱了拱手，试探道：“不知【对方】能否再【额外请求】？”
```

功能：体现主角现实、会薅羊毛。

## 9. 送礼返还式

```text
主角取出【看似普通的物品】，郑重递了过去。

“此物虽不值钱，却是【理由/习俗/心意】。若【对方】不收，在下心中难安。”

对方虽然不解，终究还是收下。

【叮！赠送成功。触发返还……】
```

功能：完成系统触发。

## 10. 系统返还狂喜式

```text
主角感受到【新物品/新能力】的瞬间，差点没控制住表情。

发了。

这次是真的发了。

他强行压住上扬的嘴角，开始盘算【下一步薅法】。
```

功能：爽点反馈。

## 11. 失败返还总结式

```text
【叮！赠送某物无效。无返还。】

主角眉头微动，很快明白过来。

看来系统并不是【无限制规则】，只有【目标所需/满足条件】才会触发。
```

功能：展示规则边界。

## 12. 装淡定式

```text
旁人已经震惊得说不出话。

主角却只是淡淡一笑，仿佛刚才拿出来的不是【珍贵物】，而是【普通物】。
```

功能：外部震惊强化主角逼格。

## 13. 识货者震惊式

```text
真正识货的那人脸色骤变。

别人看不出，他却清楚，这东西意味着【资源等级/背景/战力】。

此人绝不简单。
```

功能：通过配角认证爽点。

## 14. 战斗碾压式

```text
对手刚要动。

主角只是【抬手/挥袖/侧身】。

下一刻，对手已经【飞出/跪倒/被制住】，甚至连发生了什么都没看清。
```

功能：简洁展示等级差。

## 15. 高人改口式

```text
原本漫不经心的高人猛地站起身来，神色郑重。

“敢问阁下如何称呼？”
```

功能：打脸前面轻视。

## 16. 时间跳跃概括式

```text
【时间】过去。

这些年，主角先是【成就一】，又【成就二】，最后【成就三】。

若这只是【低级地图/凡间世界】，他的故事已经足够称得上传奇。

可惜，【更高层级的问题】依旧没有解决。
```

功能：压缩成长，保留主线遗憾。

## 17. 岁月孤独式

```text
曾经陪在主角身边的人，一个个都已化作尘土。

唯有主角依旧停在原地，像是被时间漏下的人。
```

功能：在沙雕文中制造情绪重量。

## 18. 误会脑补式

```text
配角看着主角【送礼/关心/询问某女】，脸色越来越难看。

不对。

很不对。

他终于明白了，这老东西根本不是为了【表面目的】，而是冲着【误会对象】来的！
```

功能：制造连续喜剧。

## 19. 师父快醒醒式

```text
配角痛心疾首地看着自家长辈/朋友。

你怎么还在笑？

人家都快把【你在乎的人/东西】骗走了啊！
```

功能：形成恨铁不成钢笑点。

## 20. 狗粮打断式

```text
两人说到动情处，十指相扣，含情脉脉地对视。

主角坐在对面，忽然觉得自己不该在这里。
```

功能：反向调侃配角感情。

## 21. 修真设定问答式

```text
主角问：“若想【目标】，需要什么？”

高人答：“差两样。”

“一是【关键条件】。”

“二是【资源/环境】。”
```

功能：简洁交代修真门槛。

## 22. 功法领悟式

```text
功法在主角脑海中展开。

每一个字都像是对应着他多年未解的困境。

他越看，眼睛越亮。
```

功能：突出功法解决死局。

## 23. 修炼突破式

```text
灵气顺着经脉缓缓游走，最终汇入【丹田/假丹/灵根】。

某个困住主角多年的关隘，在这一刻终于松动。

直到天色微亮，他缓缓吐出一口浊气。

【境界】已成。
```

功能：写突破爽感。

## 24. 地图切换式

```text
【时间】后，【新地图】。

这里不是【旧地图】那种【特点】，而是【新地图特点】。

主角站在其中，终于意识到，真正的故事才刚刚开始。
```

功能：开启新阶段。

## 25. 新女性角色场景式

```text
在【具体地点】，最引人注目的是一个【颜色/身份标识】的身影。

她正做着【普通劳动/日常动作】，却因为【身段、神态、汗水、衣摆】显得格外惹眼。

周围不少人都忍不住看了过去。
```

功能：用具体动作引出女角色魅力和社交地位。

## 26. 结尾系统钩子式

```text
主角刚要【做出决定/放松下来】。

脑海里忽然响起熟悉的声音。

【叮！发现X分气运之女，姓名。】
```

## 27. 结尾反杀式

```text
熟悉的身影出现在面前。

主角还没来得及开口，对方已经抬手。

下一刻，【致命危机】吞没了一切。
```

## 28. 结尾误会式

```text
配角看着眼前这一幕，脸色惨白。

完了。

这次真的完了。
```

## 29. 结尾新地图人物式

```text
【新地图】某处。

一个看似无关的人物，正在【日常动作】。

而主角还不知道，这个人会成为他进入新世界后的第一个麻烦/机会。
```
