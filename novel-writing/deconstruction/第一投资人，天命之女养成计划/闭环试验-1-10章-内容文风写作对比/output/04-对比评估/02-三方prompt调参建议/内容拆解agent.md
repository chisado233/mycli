# 内容拆解 agent 调参建议

## 1. 总体评价

内容拆解 agent 本轮表现较好。

优点：

- 章节事件顺序拆得完整。
- 主线因果链清晰。
- 关键设定、伏笔、系统规则提取充分。
- 每章“写作还原输入包”对写作 agent 帮助很大。
- 第7-8章这种机制型爽点拆解尤其有效。

主要不足：

- 对部分章节的“正文场景密度”提示不够。
- 对高识别桥段的“结构级降相似替代方案”不足。
- 对喜剧递进的细粒度拆解不够，导致写作 agent 容易写成概述。
- 对人物口吻中的“关键原文功能”拆了，但没有给足可替代表达策略。

## 2. 需要补充的字段

### 2.1 增加“场景展开权重”

当前拆解有“场景顺序”，但没有告诉写作 agent 哪些场景必须展开，哪些可以压缩。

建议每章新增字段：

```markdown
- 场景展开权重：
  1. 必须展开：
     - 场景A：原因=本章爽点/人物关系核心。
     - 场景B：原因=章节钩子。
  2. 可中等展开：
     - 场景C：原因=设定释放。
  3. 可压缩：
     - 场景D：原因=过渡。
```

例如第4章：

- 必须展开：镖局打脸、三段世俗传奇群体破防。
- 可压缩：入城找工作。
- 不应压缩：婚史递进笑点。

本轮第4章练习稿把世俗传奇写成履历，说明写作 agent 不知道这里应重点展开。

### 2.2 增加“爽点反馈链细分”

当前有“爽点触发链”，但有些只到“触发”，缺少“反馈”和“余波”拆细。

建议改为：

```markdown
- 爽点机制四段：
  - 压制：
  - 触发：
  - 即时反馈：
    - 主角心理：
    - 配角反应：
    - 系统提示：
  - 余波：
    - 新资源改变什么：
    - 新问题是什么：
```

例如第3章首次返还：

- 压制：系统十五年不能用。
- 触发：短匕被南宫遥收下。
- 即时反馈：
  - 系统连发奖励。
  - 张山狂喜、变脸认系统。
  - 南宫遥毫不知情。
- 余波：
  - 张山获得法器、空间、复活底牌。
  - 但复活底牌很快被南宫遥消耗。

练习稿触发写到了，但即时反馈不够长。

### 2.3 增加“喜剧递进层级”

第4章、第9章尤其需要。

建议新增字段：

```markdown
- 喜剧递进层级：
  1. 第一层笑点：
  2. 第二层加码：
  3. 第三层反转/爆点：
  4. 旁观者反馈：
  5. 结尾补刀：
```

例如第9章玉昀受害史：

1. 祖父被抢白月光。
2. 父亲被抢双胞胎。
3. 小叔被抢公主。
4. 自己被抢玉娆。
5. 祖孙四代雨夜抱头哭。

如果拆解中明确“每一代都要先立希望，再被张仙截胡，再给后果”，写作稿会更像正文。

### 2.4 增加“人物即时误解/真实动机对照”

道观线效果较好，但仍可更精确。

建议新增字段：

```markdown
- 误会对照表：
  - 张仙真实动机：
  - 玉昀理解：
  - 玄微理解：
  - 李妙音理解：
  - 读者知道的信息：
```

例如第7章：

- 张仙真实动机：触发返还、刷好感。
- 玉昀理解：张仙用资源腐蚀李妙音。
- 玄微理解：张仙有恩于师姐。
- 李妙音理解：张仙神秘但暂时无恶意。
- 读者知道：玉昀误会但有历史依据。

这样写作 agent 更容易写出错位喜剧。

### 2.5 增加“高相似桥段替代建议”

目前拆解有“复用风险”，但多是提醒不可照搬，没有给写作 agent 可执行替代方案。

建议新增：

```markdown
- 高相似风险与替代方向：
  - 原功能：
  - 不可复刻点：
  - 可替代写法1：
  - 可替代写法2：
```

例如林茵茵登场：

- 原功能：展示矿花魅力和钓系能力。
- 不可复刻点：原文具体衣着、腿、汗水、弯腰动作链。
- 可替代写法：
  1. 写她分配灵泉、让男修主动替她挖矿。
  2. 写她一句话让两个男修争着献殷勤。
  3. 写她表面娇憨，内心精准计算收益。

这样既保留功能，又降低相似。

## 3. 分章节重点改进建议

### 第4章

内容拆解应明确：

- 三十年世俗传奇不是背景摘要，而是本章核心喜剧段。
- 三段传奇每段都要有：
  - 主角装逼成就。
  - 一个“全民女神/稀缺美人”被主角拿下。
  - 对应群体破防。
  - 主角年龄递增带来的荒诞。
- 原文“摘桃子”不可照搬，但必须提供替代的重复笑点机制。

### 第5章

应补充：

- 本章情绪结构是“孤独求仙正剧”与“风流旧债喜剧”并置。
- 玉昀问师姐不是普通信息点，而是突然把求仙剧情拽回张仙风流黑历史。
- 玄微扇飞玉昀是高人破功笑点，应标为必须展开。

### 第9章

应补充：

- 祖孙四代受害史是核心高光，不应压缩成几句。
- 每代受害要有“期待—被截胡—荒诞后果”。
- 该段需要明确“主角过去爽点从受害者视角重讲”的写法。

### 第10章

应补充：

- 林茵茵登场不是单纯外貌描写，而是“外貌 + 社交操控 + 资源获取能力”的人物功能。
- 需要写出她“主动拿捏”，否则只是普通美女登场。

## 5. 优先级

| 优先级 | 调整项 |
|---|---|
| 高 | 增加场景展开权重 |
| 高 | 增加爽点反馈链细分 |
| 高 | 增加高相似桥段替代建议 |
| 中 | 增加喜剧递进层级 |
| 中 | 增加人物误会对照表 |
| 低 | 增加单章字数/篇幅建议 |
