# 文风拆解 agent 调参建议

## 1. 总体评价

文风拆解 agent 本轮方向判断准确。

优点：

- 抓住了“修仙爽文 + 现代社畜吐槽 + 男频欲望直给 + 沙雕误会”的核心气质。
- 对系统弹窗、送礼返还、误会脑补、对话口吻有较好总结。
- 输出的“文风控制卡”对写作 agent 有直接帮助。
- 明确列出了不可照搬表达，有助于降低句面复制风险。

主要不足：

1. **参数偏宏观，缺少章节类型专用强度表。**
2. **“可替代粗俗笑点”不足，导致写作 agent 只能弱化。**
3. **对原文低俗/缺德/破功程度的量化不够。**
4. **对高风险外貌描写的安全改写策略不足。**
5. **对重复梗递进的替代表达模板不足。**

## 2. 需要补充的文风参数

### 2.1 增加“吐槽强度等级”

当前文风拆解说“允许现代口语、轻粗口、反套路评价”，但写作 agent 不知道该用多狠。

建议新增：

```markdown
## 吐槽强度等级

- 1级：轻吐槽，用于情绪段，不破坏氛围。
- 2级：现代口语吐槽，用于设定说明后。
- 3级：缺德旁白，用于配角破防、主角薅羊毛。
- 4级：粗俗破功，用于系统坑爹、美女登场后主角失态。
- 5级：极端破防，慎用，用于绑定失败、复活丹消耗等爆点。
```

并给每章建议：

| 章节 | 建议吐槽强度 |
|---|---|
| 第1章绑定失败 | 4 |
| 第3章首次返还 | 4 |
| 第4章世俗传奇 | 3-4 |
| 第5章寿元孤独 | 1-2，后半玉昀旧债升到3 |
| 第7章送礼返还 | 3 |
| 第9章祖孙受害史 | 4 |
| 第10章林茵茵登场 | 2-3 |

本轮练习稿普遍停在2-3级，因此原文味道偏弱。

### 2.2 增加“文雅描写后的破功模板”

当前已有“美女描写后接主角反应”，但写作 agent 在南宫遥、林茵茵处都明显保守。

建议补充安全替代模板：

```markdown
- 文雅描写后破功方式：
  1. 生理反应弱化版：主角喉咙发干、呼吸顿一下。
  2. 现实判断版：这种颜值放前世能让全公司加班狗集体失智。
  3. 系统绑定版：主角还没来得及欣赏，系统先扣好感。
  4. 自我警告版：看归看，命还是要的。
  5. 资源脑版：漂亮是漂亮，关键是能不能返还。
```

这样可以替代原文高风险性化句子，又不丢“男频欲望直给”。

### 2.3 增加“重复笑点递进模板”

第4章和第9章最需要。

建议新增：

```markdown
## 重复梗递进写法

结构：
1. 第一次：建立梗。
2. 第二次：同类事件加码。
3. 第三次：后果升级或社会反应升级。
4. 最后一次：旁白补刀，形成章节记忆点。

注意：
- 不重复原句。
- 重复的是功能，不是措辞。
- 每次要换受害群体、反应强度或荒诞后果。
```

替代模板示例：

```markdown
第一次，江湖少年们只是喝闷酒。
第二次，文人士子开始写檄文。
第三次，王公贵族连骂都不敢骂，只能在家砸杯子。
```

### 2.4 增加“爽点反馈语言库”

当前文风拆解给了“发了”“血赚”等方向，但禁用清单又禁掉了部分高识别表达。写作 agent 缺少替代表达。

建议提供可迁移替代词：

| 原功能 | 可替代口语 |
|---|---|
| 发财狂喜 | 起飞了、这把活了、终于轮到我了、十五年没白熬 |
| 系统变脸 | 之前是我误会你了、你还是有点用的、刚才骂早了 |
| 薅羊毛 | 机会摆在眼前，不伸手对不起自己、礼貌归礼貌，资源不能少 |
| 规则试探 | 这系统不是提款机，得按需求投喂 |
| 配角震惊 | 识货的人已经坐不住了、他看张仙的眼神像看一座灵矿 |

### 2.5 增加“女性角色登场类型卡”

当前所有美女描写容易被写作 agent 一律降级。

建议区分：

```markdown
- 张乐乐：贫苦温情型，重点是眼睛、依赖、自卑。
- 南宫遥：高冷战损型，重点是危险、距离感、强者压迫。
- 李妙音：成熟余韵型，重点是岁月、温和、克制。
- 林茵茵：钓系经营型，重点不是身体，而是她如何让别人自愿付出。
```

尤其林茵茵应提示：

- 可以少写身体细节。
- 必须写主动操控男修。
- 必须写她表面娇憨、内心清醒。
- 必须写周围男修的竞争和被拿捏。

## 3. 对话口吻需加强的地方

### 3.1 南宫遥

文风拆解已写“短、冷、压迫”，但可以更具体：

```markdown
- 南宫遥对张山一句话最好不超过15字。
- 她不解释自己的判断。
- 她的嫌弃更多通过沉默、皱眉、避开、好感度下降表现。
```

练习稿中南宫遥有时解释较多，可再冷一点。

### 3.2 张仙

需要强调：

```markdown
- 张仙嘴上客气时，内心必须有一句现实算盘。
- 每次收礼/送礼/问话，都要有“真实目的”和“表面话术”的反差。
```

第7章做得好，第4-5章稍弱。

### 3.3 玉昀

需补充：

```markdown
- 玉昀的台词要像控诉，不像普通提醒。
- 他越认真，笑点越强。
- 他的脑补要建立证据链：张仙风流史、送礼、李妙音变年轻、玄微信任。
```

第9章需要更夸张的悲愤语气。
