# 还原写作 agent 调参建议

## 1. 总体评价

还原写作 agent 本轮完成度较高。

优点：

- 没写成纯大纲，基本是正文形态。
- 章节结构跟内容拆解高度吻合。
- 人物名、势力名、系统机制使用准确。
- 大部分章节都有吐槽、对话、系统提示、钩子。
- 主角张仙“嘴上客气，心里算计”的特征有体现。
- 原文句面复制风险控制较好。

主要问题：

1. **偏稳，缺少原文的离谱和缺德。**
2. **部分章节像压缩版正文，场景展开不足。**
3. **爽点触发后反馈段偏短。**
4. **对高风险桥段采取删减策略，而不是替代策略。**
5. **配角反馈不够夸张，导致爽点和笑点削弱。**

## 2. 写作 agent 需要调整的核心策略

### 2.1 不要只“避开原文”，要“替代原文功能”

本轮很多高风险桥段被弱化：

- 南宫遥战损登场。
- 第4章三段婚史。
- 第9章祖孙四代受害。
- 第10章林茵茵登场。

这些地方不能照搬原文，但也不能只删。

写作 agent 应遵守：

```markdown
高风险桥段处理规则：
1. 先识别原桥段功能。
2. 保留功能。
3. 替换表达、动作、比喻和细节。
4. 不要直接压缩成概述。
```

例如林茵茵：

- 不照搬原文腿、汗、弯腰等动作。
- 但要保留：
  - 她是矿花。
  - 男修围着她转。
  - 她会主动给予一点甜头。
  - 她内心知道自己在拿捏。
  - 她能从关系中获取资源。

### 2.2 爽点后必须写够反馈

当前写法常见问题：

```markdown
系统返还 → 主角高兴一句 → 进入下一事件
```

应改为：

```markdown
系统返还
→ 主角第一反应：愣住/懵
→ 第二反应：狂喜/差点失态
→ 第三反应：压住表情
→ 第四反应：立刻算计下一步
→ 配角外部反馈/误会
→ 新资源改变行动方向
```

重点章节：

- 第3章首次返还。
- 第7章中品灵石。
- 第8章《青木养丹经》。
- 第9章炼气成功。

### 2.3 增加配角反馈强度

原文很多爽点靠配角“识货”放大。

练习稿有时只写“玄微震惊”“李妙音不淡定”，但缺少更具体的价值标价。

建议写作 agent 增加：

```markdown
- 识货者脸色变化。
- 识货者解释价值。
- 旁人不懂但感觉气氛变了。
- 主角表面淡定，内心暗爽。
```

例如中品灵石：

- 不只写“是命”。
- 还要写它能让李妙音多恢复多少、在宗门相当于多少供奉、玄微为什么失态。

### 2.4 提高“缺德旁白”密度

练习稿吐槽有，但偏温和。

建议每章至少安排：

- 1处系统/规则吐槽。
- 1处主角现实算计。
- 1处配角误会或破防。
- 1处仙侠高逼格被现代口语拉低。

重点：

- 张仙不能太正经。
- 他可以真情，但多数时候要现实。
- 他嘴上越客气，心里越要“算账”。

### 2.5 场景不要过度压缩成履历

第4、5、9章尤其明显。

写作 agent 应判断：

```markdown
如果拆解中标为高光/核心笑点，不可压缩成几句总结。
```

第4章三十年可以仍然概括，但每个十年段应至少包含：

- 一个具体场面。
- 一个旁观者反应。
- 一个缺德旁白补刀。

第9章祖孙四代受害史应至少展开成四个递进小段，而不是流水账。

## 3. 分章节改写重点

### 第1章

- 增加绑定失败后的破防短句。
- 保持安全尺度的同时，加强张乐乐误会和张山自我刹车。
- 开篇反差再锋利些。

### 第2章

- 南宫遥登场需要更强视觉记忆点。
- 好感度下降要有更明确触发动作。
- 南宫遥台词更短、更冷。

### 第3章

- 首次返还后至少增加3-5段心理反馈。
- 南宫遥杀人后突出三重情绪：
  1. 死亡恐惧。
  2. 复活丹没了的心痛。
  3. 对南宫遥的仇恨。
- 送短匕场景要更尴尬、更荒诞。

### 第4章

- 不要把世俗传奇写成简历。
- 强化“不同群体被张仙气疯”的递进。
- 用替代表达取代原文“摘桃子”，但保留反复加码的笑点结构。

### 第5章

- 寿元孤独后加轻自嘲。
- 玉昀旧事加戏，让悲伤与荒诞并存。
- 玄微高人感要更强，后续破功才有效。

### 第6章

- 玉昀救场失败可以更具体。
- 李妙音登场后增加张仙对“气运女居然是成熟道姑”的反差心理。
- 玄微受伤后的尴尬可以更喜剧化。

### 第7章

- 本章保持较好。
- 可加强灵石价值标价和玄微激动。
- 玉昀尾钩可以更阴谋论。

### 第8章

- 《青木养丹经》出现后增加“多年死局被打开”的情绪。
- 李妙音年轻化要让玄微更失态，玉昀更崩溃。
- 绿帽双关要更自然从“木属青气”引出。

### 第9章

- 祖孙四代受害史必须展开。
- 每代都要“梦想—被张仙截断—后果”。
- 张仙炼气成功后增加玉昀负面吐槽，形成反差。

### 第10章

- 林茵茵登场不能只写普通漂亮。
- 要写她如何经营矿区男修关系。
- 她的钓系能力要通过行动展示，而不是只说“拿捏”。

## 4. 原文相似风险控制建议

当前写作 agent 的风险控制偏句面，下一轮应增加结构级控制。

### 4.1 句面风险

继续保持：

- 不照搬原文句子。
- 不照搬标志性比喻。
- 不照搬章节标题。
- 不照搬高识别粗口。

### 4.2 结构风险

如果是同作还原练习，结构接近是实验目标，可以接受。

但若目标是“仿写新作品”，则必须改：

- 首次返还物品。
- 复活道具消耗方式。
- 世俗婚史配置。
- 玉昀家族受害史。
- 林茵茵登场场景。

本轮是还原对比评估，因此结构高相似不算失败，但需标记风险。

## 5. 建议写作 agent 增加自检表

每章写完后自检：

```markdown
- [ ] 本章核心事件是否完整？
- [ ] 是否至少有一个具体场景充分展开？
- [ ] 爽点后是否写了主角反馈和配角反馈？
- [ ] 是否有3个以上吐槽/笑点？
- [ ] 是否有一句主角现实算计？
- [ ] 是否有配角误会/震惊/破防？
- [ ] 是否避免照搬原文标志句？
- [ ] 高风险桥段是否保留功能而非直接删弱？
- [ ] 结尾钩子是否能直接引出下一章？
```

## 6. 优先级

| 优先级 | 调整项 |
|---|---|
| 高 | 爽点后反馈写足 |
| 高 | 高风险桥段用替代，不用删减 |
| 高 | 第4、第9章喜剧递进展开 |
| 高 | 林茵茵登场加强人物功能 |
| 中 | 增加缺德旁白和配角破防 |
| 中 | 增强南宫遥、玉昀口吻 |
| 低 | 调整章节字数，使高光场景更充分 |
