# chapter-029：合法萝莉

- 拆解等级：A
- 本章一句话：张仙服用升仙草把自己从零灵根提升到下品木灵根，修成《青帝回春诀》，又结识云栖的“姐姐”云枕儿并通过赠礼绑定返还。
- 表层事件：
  1. 张仙夜里服用多株升仙草，自测灵根由0到下品。
  2. 他修炼《青帝回春诀》，获得恢复和治疗能力。
  3. 林茵茵成功炼化上品法器，云栖震惊于二人资质提升。
  4. 云枕儿登场，因先天魂魄有缺被时砂锁和玄冰阵封住肉身心智；张仙赠送上品防御玉坠并绑定。
- 故事逻辑：目标补足自身短板 → 阻碍资质太差、升仙草递减 → 张仙尝试服用并改修恢复法诀 → 获得下品木灵根与治疗续航 → 云枕儿登场提供新返还目标和魂魄缺失支线。
- 本章功能：主角个人力量体系补强；云家支线人物云枕儿登场；展示云栖对张仙背景判断进一步拔高。
- 关键人物/势力：张仙、林茵茵、云栖、云枕儿、云家/云裳阁。
- 人物/势力/背景/世界设定：
  - 新增项：云枕儿，12分气运之女，外形幼童，实际辈分高，金丹修为，先天魂魄有缺。
  - 新增项：灵宝【时砂锁】、玄冰阵可封住肉身生长和心智。
- 人物设定素材：
  - 张仙：务实修炼，知道自己不缺攻击，缺续航；对小孩型角色用玩具式赠礼。
  - 林茵茵：被包养幸福感强，对可爱云枕儿主动释放善意。
  - 云栖：商业精英但频繁被张仙财富震惊；对云枕儿宠溺。
  - 云枕儿：童稚、守礼、爱玩光点，称云栖“弟弟”形成反差萌。
- 力量体系素材：
  - 张仙：从0提升至下品木灵根；修成《青帝回春诀》，恢复灵气快，有治疗能力。
  - 上品法器炼化：正常需三个月到半年，林茵茵筑基五重可尝试炼化。
  - 系统返还：赠送云枕儿上品法器，返还上品法器×100。
- 新增设定/伏笔：
  - 伏笔：云枕儿魂魄有缺，需补魂宝物。状态：埋设。
  - 伏笔：云家可能因云枕儿与张仙深度绑定。状态：埋设。
- 常见场景类型：修炼 / 送礼 / 新角色登场 / 萝莉反差 / 资源震惊。
- 情绪/爽点/搞笑点/抽象点：
  - 搞笑点：“我绿了！！我进化到下品灵根了！”把灵根颜色测试写成自嘲。
  - 爽点：低资质主角终于突破0，哪怕只是下品也有阶段满足。
  - 萌点：云枕儿想要玉坠但记得“不能随便拿陌生人东西”。
- 原文摘录候选：
  > 张仙怀着激动的心情吞下第三株，这次他周身爆发出刺目的青光，随后内敛缓缓消散。  
  > 嗯？测灵珠的刻度只提升了一点。  
  > 第四株，刻度没动。  
  > 好了，到此为止吧。张仙从系统空间翻出一本《青帝回春诀》，之前因为灵根资质过低无法修炼，现在是下品灵根了，应该有戏。  
  > ……  
  > “哥哥，你这是……”  
  > “新学的一套法诀。”张仙弹指射出一道绿光，林茵茵前几日磕碰的膝盖瞬间愈合，“虽然打架不行，当个奶妈倒是合格了。”
- 结尾钩子：云枕儿收下上品防御法器，好感+5，云栖误以为张仙看他面子。
- 可借鉴技法：**小突破也能爽**——不必每次都跨大境界，零到下品的“终于不是0”也能制造读者满足。
- 索引更新价值：张仙个人力量体系；云枕儿人物设定；补魂伏笔；上品法器返还新目标。
- 判断与证据锚点：
  - “我再也不是0了”证明主角资质线阶段推进。
  - “先天魂魄有缺”确立云枕儿支线核心问题。
- 写作还原输入包：
  - 场景顺序：张仙夜服升仙草 → 灵根测试 → 修炼青帝回春诀 → 给林茵茵治疗 → 赠上品法器 → 云栖震惊 → 云枕儿登场 → 张仙赠玉坠。
  - 人物目标：张仙补短板和找返还对象；林茵茵炼化法器；云栖维护关系；云枕儿玩耍。
  - 信息释放：升仙草递减、青帝回春诀定位、时砂锁与魂魄缺陷。
  - 爽点触发链：资质低压抑 → 升仙草尝试 → 下品灵根达成 → 新法诀治疗 → 新目标赠礼返还。
  - 结尾落点：云枕儿成为新绑定对象。
