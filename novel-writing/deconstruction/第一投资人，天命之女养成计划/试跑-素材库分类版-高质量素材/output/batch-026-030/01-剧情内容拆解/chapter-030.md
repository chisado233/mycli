# chapter-030：你以为你是杨过啊

- 拆解等级：A
- 本章一句话：张仙在玉京城集市偶遇杨破霄与龙芷，用升仙草换升仙丹份额，既绑定龙芷又恶心了竞拍对手。
- 表层事件：
  1. 张仙因资源消耗与返还目标不足，外出逛集市。
  2. 杨破霄摆摊售丹并收天材地宝，因失去升仙草而烦躁。
  3. 张仙伪装身份购买延寿丹、气血丹，并拿出升仙草试探。
  4. 杨破霄暴露会炼升仙丹，张仙要求把升仙草交给龙芷保管，完成赠送绑定与返还。
- 故事逻辑：目标拓展资源与新目标 → 阻碍杨破霄不可信且有敌意 → 张仙伪装交易、诱导其说出升仙丹 → 改为赠龙芷并让杨破霄立誓分丹 → 获得龙芷绑定、升仙草返还、升仙丹未来收益。
- 本章功能：把拍卖会竞争转化为后续丹药合作/仇怨；正式绑定龙芷；强化杨破霄敌对心理。
- 关键人物/势力：张仙、杨破霄、龙芷、云裳阁。
- 人物/势力/背景/世界设定：
  - 强化项：玉京城集市可摆摊售丹、收购天材地宝。
  - 新增项：延寿丹：凡人延寿百年，炼气约十年，筑基约一年，更高境界无效。
  - 新增项：气血丹：提升凡人/武者气血。
  - 强化项：升仙丹：杨破霄可炼，一炉最多两三粒，实际可能更多。
- 人物设定素材：
  - 张仙：伪装交易、心理恶心对手、用“不信你只信你师父”精准戳杨破霄痛点。
  - 杨破霄：对龙芷有强烈占有式崇拜，被轻视时容易暴怒；炼丹能力强。
  - 龙芷：淡漠、少言、重实际利益；愿为徒弟代收升仙草。
- 力量体系素材：
  - 心魔大誓：交易保障机制。
  - 升仙丹炼制：需升仙草主材，成丹后可提升灵根，未来可用于主角资质线。
- 新增设定/伏笔：
  - 伏笔：两个月后升仙丹送至云裳阁。状态：埋设。
  - 伏笔：杨破霄将云裳阁视作线索，可能追查张仙。状态：强化。
  - 伏笔：龙芷绑定为0好感，后续可被投资。状态：埋设。
- 常见场景类型：集市交易 / 伪装身份 / 规则交易 / 情敌误会 / 反向恶心对手。
- 情绪/爽点/搞笑点/抽象点：
  - 爽点：张仙既触发返还，又白嫖未来升仙丹，还让杨破霄心态爆炸。
  - 搞笑点：张仙吐槽杨破霄“带银色面具，还背个大剑，你以为你是杨过啊？”
  - 反派破防点：杨破霄觉得张仙“染指宝贝师父”，敌意升级。
- 原文摘录候选：
  > “我有个小条件，我信不过你！升仙草你只给你身后的那位。”那人指了指一旁的龙芷。  
  > 【叮！龙芷对你的好感度为0，绑定成功。】  
  > 龙芷睁开双眸，看了那人一眼，随即又闭上眼睛继续打坐。  
  > 杨破霄勃然大怒，“道友这是什么意思？心魔大誓之下怎会毁诺！”  
  > “不行的话交易就取消。”那人话语之中毫无商量的余地。  
  > ……  
  > “不不，在下怎敢让仙子立誓。我只是将升仙草送你，心魔大誓你徒弟来就好了。”
- 结尾钩子：张仙成功返还升仙草×100，并决定用“恶心杨破霄”作为报复。
- 可借鉴技法：**交易里夹带羞辱**——主角表面谈合作，实际通过信任对象选择戳对手软肋，让爽点不只在收益，也在心理打击。
- 索引更新价值：龙芷人物线开启；杨破霄敌对强化；升仙丹伏笔；集市交易场景。
- 判断与证据锚点：
  - “升仙丹普天之下只有我一个可以炼制”证明杨破霄核心技能。
  - “赠送龙芷升仙草成功”证明龙芷正式进入投资系统。
- 写作还原输入包：
  - 场景顺序：张仙出门吃逛 → 杨破霄摆摊烦躁 → 买延寿丹气血丹 → 拿升仙草试探 → 谈升仙丹分成 → 指定龙芷收草 → 杨破霄立誓 → 张仙离开返还。
  - 人物目标：张仙绑定龙芷与白嫖丹药；杨破霄夺回丹材；龙芷帮徒弟促成交易。
  - 信息释放：延寿丹、气血丹、升仙丹、心魔大誓、龙芷绑定。
  - 爽点触发链：张仙伪装试探 → 杨破霄低价收购失败 → 暴露丹方 → 张仙指定龙芷 → 返还百株。
  - 结尾落点：杨破霄记住云裳阁，仇怨转入暗线。
