# 青璃卷级状态轨迹

> 状态对象：青璃  
> 状态来源：计划状态  
> 覆盖范围：卷0开篇前 至 第七卷结束  
> 人物定位：成年古剑剑灵 / 修仙剑冢线女主 / 苏念近战剑修化核心搭档 / 剑冢守灵传承核心  
> 关系原则：青璃与苏念的关系始终建立在成年、智慧人格、自愿认可、并肩作战基础上；“剑主”不是主人含义，而是她主动选择的同行者称呼。

### 4. 待解决问题

- 本体剑圣痕裂纹必须修复或压制。
- 剑魂长期镇压剑冢失控残意，存在沉眠或失控风险。
- 驭剑派可能趁乱夺取本体。
- 剑冢灵泉被污染，恢复速度不足。
- 万剑归墟图与青璃的共鸣被干扰。
- 她需要找到真正尊重剑灵意志、能并肩承担剑道因果的剑主。

### 5. 下卷承接

- 第一卷中，剑冢残玉将作为青璃呼唤的前置媒介出现。
- 青璃不会正式登场，但她的声音会在卷末成为第二卷剑冢线钩子。
- 她的状态必须保持“濒危但未失控”，以便第二卷苏念介入时仍能进行自主选择。
