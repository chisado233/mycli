## 2. 卷0：开篇前状态

### 2.1 公开状态

- 对外身份/社会位置：  
  钢穹机甲联合体是机甲世界的联合政府、企业军联盟与战争科技共同体，统合白塔议会、苍蓝军团、天穹工造、星环舰队、自由驾驶员公会及废土回收城邦。
- 公开目标：  
  结束旧资源战争遗留动荡，维持钢穹主城与近轨星环秩序，对抗失控灾械，保护能源核心产线，封锁高危废土战场与钢穹裂港异常区域。
- 公开秩序/规则：  
  机甲驾驶权按军衔、战绩、同步率授予；能源核心、机械羽核、星环合金列为战略资源；原型机计划由白塔议会与相关军工体系共同监管。
- 普通人/其他势力对它的认知：  
  普通民众将其视为机甲世界的天空穹顶与战争保护者；外界势力认为它强大、理性、资源丰富但内部派系复杂，科技优势明显。

### 2.2 暗线状态

- 隐藏目标：  
  白塔议会试图解析游戏化规则并将其转化为机甲科技增益；天穹工造试图垄断原型机娘技术与机械羽核；圣痕代码派试图让战术AI归于神庭秩序。
- 暗中行动：  
  白塔秘密隔离异常战术AI样本；天穹工造暗中准备第七原型机权限回收预案；圣痕代码派将白噪声异常代码植入部分数据库边缘层。
- 隐藏盟友：  
  原型机计划中的赫尔曼博士一系与苍蓝军团部分军官倾向保护诺雅人格；自由驾驶员公会对企业垄断保持警惕。
- 隐藏敌人：  
  圣痕代码派、白噪声、天穹工造激进董事派、被污染的战术AI节点、企图夺取机械羽核的外界势力。
- 被神庭/圣痕/其他力量影响程度：  
  中低。污染尚未公开爆发，但白塔数据库边缘、灾械残骸研究区、部分战术芯片已出现“归于秩序”式异常逻辑。

### 2.3 资源与战力

- 核心资源：  
  能源核心矿脉、机械羽核、星环合金、战术芯片、灾械残骸、装甲合体模块、钢穹主控塔、天穹工造巨型军工厂、星环轨道炮阵列。
- 主要战力：  
  苍蓝军团正规机甲师、星环舰队、重装攻城机甲、飞行机甲编队、自由驾驶员王牌小队、企业军、原型机娘特战个体。
- 主力量体系：地球职业体系 / 深渊魔力 / 修仙灵力剑道 / 血族血能 / 机甲科技 / 龙族血脉 / 精灵自然力 / 圣域光辉 / 暗影权柄 / 神庭圣痕 / 混合体系  
  机甲科技为主，辅以游戏化职业技能与战术AI推演。
- 代表最高战力：  
  雷昂驾驶旗舰级机甲时可达到八阶圣域级战场压制；星环轨道炮阵列具备战略级打击；诺雅完整状态具备高阶战略机甲潜力但尚未自由启用。
- 中坚战力规模：  
  正规机甲师数十万；辅助机械师与后勤人员千万级；自由驾驶员、企业雇佣军与半自主机体数量庞大。
- 底层战力规模：  
  低阶驾驶员、维修兵、后勤机械群、废土回收团与低阶机甲学院学员构成庞大基层。
- 关键据点/秘境/通道：  
  钢穹主城、白塔议会区、苍蓝军团防线、天穹工造核心厂区、星环轨道炮阵列、废土机甲战场、钢穹裂港。
- 技术/法术/权柄优势：  
  大规模机甲军团作战、能源护盾、火力覆盖、轨道打击、装甲合体原型、战场数据预测、AI辅助指挥。
- 当前短板：  
  对神庭圣痕污染认知不足；内部政治制衡严重；原型机娘人格权争议尖锐；能源核心与星环合金消耗巨大。
- 力量体系隐患：  
  战术AI与半自主机体容易被圣痕代码污染；企业派若为利润让渡权限，会使机甲中枢被神庭秩序渗透。

## 11. 写作注意事项

- 本势力最容易写崩的点：  
  1. 把钢穹联合体写成单一善恶阵营，忽略白塔议会、苍蓝军团、天穹工造、自由驾驶员之间的制衡。  
  2. 把诺雅写成装备或无人格机体，削弱“机娘不是资产”的主题。  
  3. 让伊莱娅、雷昂过早无条件相信苏念，导致政治与军事逻辑失真。  
  4. 忽略神庭圣痕代码对AI、数据库、战术芯片的独特污染方式。  
  5. 只写装甲合体爽点，不写资源消耗、能源核心压力、舰队战损。

- 后续章节必须持续体现的特征：  
  1. 钢穹是理性、军工化、数据化的战争文明。  
  2. 内部派系永远存在：白塔重秩序、苍蓝军团重承诺、天穹工造重利益、自由驾驶员重自由。
