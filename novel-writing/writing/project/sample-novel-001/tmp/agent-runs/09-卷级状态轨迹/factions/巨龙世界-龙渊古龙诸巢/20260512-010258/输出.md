## 2. 卷0：开篇前状态

### 2.1 公开状态

- 对外身份/社会位置：  
  龙渊古龙诸巢是巨龙世界最古老的统治性势力群，由赤金祖巢、霜翼巢、黑曜巢、翡翠巢、雷脊巢、幽蓝巢等龙巢组成。它们依靠古龙王、龙裔城邦、龙血骑士团和龙语祭司团维持对龙渊山脉与古龙天穹的统治。

- 公开目标：  
  守护祖巢、维护龙族血脉尊严、掌控龙鳞矿和龙血晶等核心资源、禁止外族染指祖龙禁地。

- 公开秩序/规则：  
  祖巢议会处理诸巢大事；古龙王拥有各巢最高权威；龙裔王管理眷族城邦但须服从古龙册封；古龙誓约以龙血、真名、鳞片为约束。

- 普通人/其他势力对它的认知：  
  外界普遍认为龙渊古龙诸巢傲慢、强大、资源丰厚且难以谈判。地球、机甲、修仙、深渊等势力都觊觎龙鳞矿、龙血晶、古龙符文，但也畏惧古龙王与龙兽潮。

### 2.2 暗线状态

- 隐藏目标：  
  祖巢议会真正担忧的是神庭秩序试图将古龙册封为战争兵器，并进一步染指祖龙残骨。

- 暗中行动：  
  幽蓝巢与龙语祭司团收集关于神庭、圣痕、游戏化规则的古老记忆；黑曜巢加强祖龙禁地守卫；霜翼巢暗中评估是否需要与外界建立有限联系。

- 隐藏盟友：  
  暂无正式隐藏盟友。潜在可接触对象包括华夏镇守体系开拓派、昆墟清醒派、钢穹反神庭派，但卷0阶段尚未建立关系。

- 隐藏敌人：  
  神庭秩序、神眷龙裔派、试图夺取祖龙残骨的外界掠夺者、深渊掠夺派、黑市偷猎者。

- 被神庭/圣痕/其他力量影响程度：  
  祖巢核心暂未被控制；龙裔城邦和部分野心龙裔王已被神庭诱惑。影响程度为“低到中”，但隐患极高。

### 2.3 资源与战力

- 核心资源：  
  龙鳞矿、龙血晶、古龙符文、龙息琥珀、霜翼羽鳞、祖龙残骨、龙渊誓约卷、龙语祭坛。

- 主要战力：  
  古龙王、成年巨龙、青年龙、龙裔战士、龙血骑士团、龙语祭司团、龙兽潮、巢穴守卫。

- 主力量体系：地球职业体系 / 深渊魔力 / 修仙灵力剑道 / 血族血能 / 机甲科技 / 龙族血脉 / 精灵自然力 / 圣域光辉 / 暗影权柄 / 神庭圣痕 / 混合体系  
  主体系为龙族血脉，辅以龙语符文、龙息、古龙誓约、巢穴地利；少量龙裔开始受到游戏化等级体系影响。

- 代表最高战力：  
  奥古斯都，赤金古龙王，九阶星冕级对应；格罗姆，黑曜沉眠者，九阶沉眠古龙；希薇娅，霜翼古龙女王，八至九阶。

- 中坚战力规模：  
  成年巨龙数量有限但单体强大；青年龙与龙血骑士团构成主要机动作战层；龙裔战士和龙兽牧群可形成区域级军团压力。

- 底层战力规模：  
  龙裔眷族、矿工城邦、低阶龙兽、巢穴守卫数量庞大，可维持山脉级防御和资源开采。

- 关键据点/秘境/通道：  
  龙渊山脉、赤金祖巢、霜翼巢、黑曜巢、幽蓝巢、龙语祭坛、祖龙禁地、龙鳞矿脉、龙渊关外侧裂谷。

- 技术/法术/权柄优势：  
  龙息破坏力、龙鳞防御、血脉压制、龙语符文结界、古龙誓约约束、飞行机动、巢穴地利。

- 当前短板：  
  情报体系弱于地球、狐族、神庭；龙族傲慢导致对外误判；内部龙裔王有摆脱古龙控制的野心。

- 力量体系隐患：  
  血脉等级森严使低阶龙裔容易被神庭“升格”诱惑；游戏化等级成长会冲击古老血脉秩序；古龙誓约若被圣痕污染，可能反向成为支配枷锁。

- 战力变化：  
  龙族中坚受损但未崩溃；神眷龙裔派被削弱；祖巢核心古龙王仍保有威慑。

- 最高战力变化：  
  奥古斯都未失势但态度从纯强硬转为“先审判神庭，再谈外界”；希薇娅话语权上升；格罗姆继续守禁地。

- 中坚战力变化：  
  龙裔王私军被清洗一批，龙血骑士团重新整编；雷加若被主角团击败，则从盲目好战转为尊重强者。

- 力量体系变化：  
  古龙誓约开始与反圣痕理念结合，形成未来龙巢誓约基础。

- 神庭/圣痕/污染影响变化：  
  由暗中扩张转为公开清洗；圣痕影响总体下降，但残余仍危险。

- 据点/秘境/通道变化：  
  龙渊关由纯冲突战场转为多界会盟重点防线之一；部分矿区暂时封锁，等待盟约分配。

- 伤亡/污染/封印：  
  部分龙裔战士、龙血骑士、青年龙伤亡；污染龙裔被封印或清洗；祖龙禁地进入最高戒严。

  初步理解苏念契约与神庭奴役、黑契会强制契约不同；仍需通过古龙誓约对照来接受。

- 对相关女主认知：  
  承认莉莉娅、克露露、青璃、白九璃、诺雅、希尔薇、伊芙琳、影弥等不是苏念的工具，而是各世界代表战力；但龙族仍以强者视角评估她们。

  奥古斯都仍可能误判多界会盟会稀释龙族自主权；格罗姆仍不信任任何外族接近祖龙残骨。

  敌意从对外界整体敌意下降为对资源掠夺者和神庭残余敌意；合作意向为签订有限反神明誓约。

- 敌人变化：

- 中立势力变化：  
  对精灵、圣域等清醒派保持谨慎礼貌；对深渊反神明派仍有本能排斥但接受会盟框架。

- 与神庭秩序关系：  
  从暗中对抗升级为公开敌对；龙族拒绝神庭驯化。

### 8.8 下卷行动计划与承接约束

- 下卷会主动做什么：  
  在神门降临时守护龙渊祖巢神门，准备以龙巢誓约加入反神明联军；清算神眷龙裔派残余。

- 下卷会被动承受什么：  
  神庭将直接攻击祖巢、祖龙残骨和古龙誓约，试图强制神权化龙族。

- 必须回收的问题：  
  龙族最终加入新秩序必须基于反神庭生存压力、苏念证明尊重自主、希薇娅与星眠长老推动，而非突然臣服。

- 不能突然改变的内容：  
  龙族保留骄傲与自治；即使结盟，也不会称苏念为主人或臣服者。

## 9. 第七卷结束状态

### 9.1 终局公开状态

- 对外结局：  
  龙渊古龙诸巢参与神门防守与诸界反神终局，在龙渊祖巢神门战中以古龙王、龙血骑士团、龙语祭司团、忠诚龙裔军团守住龙渊核心。神庭秩序崩塌或被重塑后，龙族正式加入多界盟约体系，但保留祖巢自治与祖龙禁地独立守护权。

- 世界评价：  
  多界承认龙族是反神明战争中的高阶支柱之一；龙族也不再被单纯视为傲慢资源怪物，而是有古老尊严、强大血脉和自主秩序的盟约强族。

- 合法性/统治权/话语权变化：  
  祖巢议会合法性增强，因其最终拒绝神庭册封并清算神眷龙裔派；希薇娅、星眠长老等清醒派话语权上升；维克托一系被剥夺龙名与统治权。龙族在多界新秩序中获得龙渊关防线与龙族资源谈判席位。

### 9.2 终局暗线状态

- 暗线是否暴露：  
  神眷龙裔派主干暴露；神庭利用龙裔自卑与权力欲污染龙族的路线公开。

- 暗线是否清算：  
  维克托及核心神眷龙裔被击败、封印、处决或驱逐；被轻度污染的龙裔接受伊芙琳净化、龙语誓约重塑或祖巢审判。

- 残余隐患：  
  少量圣痕龙裔逃入边境裂谷；祖龙残骨虽未被夺取，但曾被神庭标记，后续需长期封印净化；部分保守古龙仍不适应多界新秩序，可能在后日谈中反对资源开放。

### 9.3 终局资源与战力

- 保留资源：  
  祖龙残骨、主要祖巢、龙渊誓约卷、核心龙语祭坛、部分龙鳞矿脉、未污染龙血晶、赤金龙王冠、霜翼长誓碑、黑曜沉眠棺。

- 损失资源：  
  外围龙鳞矿脉受损；一部分龙血晶被污染后销毁；龙兽牧群大幅消耗；部分龙语祭坛在神门战中破碎。

- 战力残余：  
  古龙王核心尚存但多有伤势；成年巨龙伤亡有限但不可忽视；龙裔叛军被清算导致中坚数量下降；忠诚龙血骑士团重组后质量提升、规模下降。

- 据点/通道最终归属：  
  龙渊祖巢归属龙族自治；龙渊关纳入多界盟约防线，由龙族与华夏镇守体系等共同协调，但资源开采必须经龙巢誓约与多界盟约双重约束。祖龙禁地仍由黑曜巢与祖巢议会独立守护，禁止商业化开放。

## 11. 写作注意事项

- 本势力最容易写崩的点：  
  1. 把龙族写成单纯反派或单纯盟友，忽视“傲慢但不愿被神庭奴役”的复杂性。  
  2. 让古龙王过早被苏念说服，削弱T1势力尊严。  
  3. 忽视神眷龙裔派的内部阶层矛盾，使维克托背叛缺乏动机。  
  4. 把龙鳞矿、龙血晶、祖龙残骨写成随手可取资源，破坏龙族核心利益。

- 后续章节必须持续体现的特征：  
  1. 龙族重视血脉、真名、鳞片、誓约与尊严。  
  2. 祖巢议会不是统一声音，奥古斯都、希薇娅、格罗姆、星眠长老、雷加、维克托立场不同。  
  3. 龙族与苏念合作的前提是共同反神庭，而不是被个人魅力轻易收服。  
  4. 龙族对“契约”的理解必须经历从支配误判到理解自愿盟约的过程。

- 禁止突然消失的资源/战力：  
  龙鳞矿、龙血晶、古龙符文、祖龙残骨、龙语祭坛、古龙王、成年巨龙、龙血骑士团、龙裔城邦、龙兽潮、龙渊关防线。

- 需要在小篇章纲中重点追踪的状态：  
  1. 神眷龙裔派污染程度与维克托暴露节奏。  
  2. 希薇娅与星眠长老对苏念的观察与信任增长。  
  3. 奥古斯都从强硬试探到有限承认的因果证据。  
  4. 龙渊关矿区损失、封锁、谈判与盟约分配。  
  5. 祖龙残骨是否被神庭标记、封印是否稳定。  
  6. 龙巢誓约与多界盟印如何并存，避免龙族自治被写没。
