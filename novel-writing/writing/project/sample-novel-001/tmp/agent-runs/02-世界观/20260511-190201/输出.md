# 世界观总览

## 01. 世界类型

本书世界为 **现代都市全球觉醒世界 + 秘境降临 + 多异界召唤通道**。

表层是近现代社会：城市、学校、高考、学院、财阀、官方机构、公会、直播舆论、资源交易市场并存。

里层是“星潮”之后出现的超凡体系：全球人类觉醒异能，世界各地出现秘境裂隙、异界通道、禁区和召唤回响。人类以异能高考、学院训练营、官方探索队、公会佣兵、财阀实验室等形式开发超凡资源。

主角林烬的特殊性在于：他的召唤天赋并非随机召唤低级生物，而是能连接多条异界世界线中 **成年类人智慧女性强者**，并以“平等契约/羁绊契约”的方式建立战斗与情感双线。

## 02. 世界地图

世界地图分为四层推进：

1. **前期核心舞台：江海市及周边**
   - 林烬参加觉醒、高考、选拔、城市秘境。
   - 林清雪、莉莉娅、米娅/白九璃初步登场。
   - 适合快节奏打脸、考试、副本、初级资源争夺。

2. **第一阶段拓展：华夏学院体系与训练营**
   - 顶级学院、国家训练营、排行榜、秘境实战课。
   - 学院不长期困住主线，只承担资源、身份、队友、公开赛、前期保护伞作用。
   - 学院篇结束后，林烬转入都市秘境、官方任务、财阀争夺和跨城事件。

3. **中期核心舞台：全球都市秘境、禁区、异界裂隙**
   - 各国都有不同属性禁区。
   - 财阀、公会、官方机构争夺秘境资源。
   - 地下召唤组织利用非法契约、污染召唤、异界走私制造冲突。
   - 艾瑟琳、希尔薇、诺雅、伊芙琳等女主陆续通过不同异界回响加入。

4. **后期核心舞台：女主原世界与星门战争**
   - 魅魔王庭、狐族情报网、血族王朝、精灵圣域、机械天界、光辉圣国、影魔暗域等逐步揭开。
   - 主角团反向介入女主们的原世界危机。
   - 地球不再只是被动承受异界入侵，而是成为跨界秩序核心之一。

## 03. 世界规则

世界规则围绕“星潮”“觉醒”“秘境”“召唤契约”“异界通道”展开。

核心规则：

1. **星潮开启全球觉醒**
   - 十八年前，地球第一次遭遇星潮。
   - 星潮使人类灵魂频率被激活，部分人觉醒异能。
   - 异能高考成为新时代阶层上升通道。

2. **秘境是异界碎片与地球空间重叠**
   - 秘境内有怪物、材料、传承、矿脉、特殊规则。
   - 秘境分稳定、波动、失控、深渊化等状态。
   - 稳定秘境可用于考试和学院训练，失控秘境会吞噬城市区域。

3. **召唤不是奴役，而是契约连接**
   - 正规召唤契约必须满足双方意志响应。
   - 智慧召唤对象拥有独立人格、拒绝权、解除权和契约谈判空间。
   - 召唤师提供现实锚点、灵能供给和成长资源；召唤兽提供战斗能力、知识、异界情报和羁绊反馈。

4. **亲密羁绊可以增强契约稳定性，但不得违背自愿原则**
   - 情绪信任、并肩战斗、共同生活、互相选择都会提高契约亲和度。
   - 亲密关系只在成年、自愿、类人智慧对象之间成立。
   - 羁绊增强表现为召唤时长增加、技能同步、合击解锁、精神抗性提升等。
   - 不写露骨过程，只可通过次日氛围、称呼变化、契约波动、旁人调侃等隐晦呈现。

5. **异界通道会反向暴露地球坐标**
   - 越强大的召唤、秘境开发、星门开启，越容易被异界势力感知。
   - 主角越强，女主原世界危机越可能追到地球。
   - 这为中后期全球事件和异界危机提供主线升级。

## 04. 力量体系

人类觉醒者分为九阶，前期简单直观，中后期可扩展。

- 一阶：觉醒者  
- 二阶：实战者  
- 三阶：精英者  
- 四阶：统领级  
- 五阶：君主级  
- 六阶：灾厄级  
- 七阶：天灾级  
- 八阶：圣域级  
- 九阶：星冕级  

召唤兽可用对应战力等级，但保留原世界称谓，例如魅魔女王候选、血族公主、精灵圣射手、机械天使等。

主角林烬的升级爽点不只是单纯堆数值，而是：

1. 召唤位增加。
2. 女主契约稳定度提升。
3. 合击技能解锁。
4. 可召唤时长延长。
5. 召唤兽从“投影降临”到“完全降临”。
6. 主角本人获得契约反馈，补足体质、精神、空间感知和战术能力。
7. 女主原世界权限逐步开启。

## 05. 器物层级

器物分为现代装备、灵能武装、秘境遗物、异界圣器、星门核心五类。

常用等级：

- F级：制式训练装备。
- E级：低阶灵能装备。
- D级：初级秘境装备。
- C级：学院/公会主力装备。
- B级：精英觉醒者装备。
- A级：君主级装备，拥有特殊词条。
- S级：灾厄/天灾级装备，可改变战局。
- SS级：圣域级遗物，关联异界历史。
- SSS级：星冕级权柄器物，关联世界规则。

召唤师专属器物包括：契约戒、召唤媒介、灵魂锚、星纹卡、秘境钥匙、共鸣项链、召唤阵盘等。

## 06. 资源体系

世界资源要直接驱动争夺、考试、势力行动和后宫成长。

核心资源：

1. **星晶**
   - 秘境怪物、星潮矿脉、异界裂隙产出。
   - 通用超凡能源，可修炼、交易、驱动装备。
   - 异能高考、学院积分、公会悬赏都以星晶价值衡量。

2. **灵能积分**
   - 学院、官方、训练营内部货币。
   - 可兑换装备、技能训练、秘境名额、导师课程。

3. **秘境钥匙**
   - 进入特定秘境的凭证。
   - 高价值秘境钥匙会引发财阀、公会、学院、地下组织争夺。

4. **召唤媒介**
   - 用于提高特定异界回应概率。
   - 例如魅魔系梦魇晶、狐族幻灵铃、血族月血石、精灵圣木枝等。
   - 为女主登场、进阶、修复原世界联系提供剧情道具。

5. **契约亲和度**
   - 不作为露骨数值系统，而作为剧情内隐性指标。
   - 通过信任、并肩作战、日常相处、情感选择提升。
   - 提升后可解锁新技能、合击、完全降临时长。

6. **异界坐标**
   - 中后期关键资源。
   - 谁掌握坐标，谁就能进入女主原世界、开发资源或发动战争。
   - 地下召唤组织和异界敌对势力都会争夺。

## 07. 世界故事背景

十八年前，地球遭遇第一次“星潮”。

夜空出现持续七日的银色星河，全球通讯短暂中断，大量人类在昏迷中觉醒异能。随后，世界各地陆续出现秘境入口、异界裂隙、异常生物和灵能矿脉。

各国迅速建立觉醒者制度。普通高考之外，异能高考成为最重要的上升渠道。觉醒者学院、官方探索局、民间公会、财阀研究所、地下召唤组织在十几年内快速成型。

人类以为星潮只是地球获得超凡力量的开端，但少数顶层强者知道：星潮并不是恩赐，而是多世界壁垒松动的前兆。地球正在成为多个异界争夺的“新生锚点”。

林烬的特殊召唤天赋，正是这场大变局的核心变量之一。他召唤来的并不是普通兽宠，而是来自不同世界线、拥有身份、立场、欲望和危机的成年智慧女性强者。她们选择林烬，也意味着她们背后的世界逐步与地球建立联系。

后期读者感知：

## 风格

整体风格为 **快节奏、轻爽、暧昧、战斗爽、修罗场日常并行**。

不走沉重末世压抑路线，也不写过于复杂的政治设定。官方、学院、财阀、公会、地下组织、异界势力都以“制造事件和冲突”为主，不堆设定。

基调：

- 都市生活有烟火气。
- 学院/高考有竞争爽点。
- 秘境探索有战斗和宝物爽点。
- 后宫日常有轻喜剧、吃醋、拱火、主动撩拨。
- 异界危机有长期伏笔，但前期不压读者。

## 与题材卖点的关系

1. **服务魅魔吸量**
   - 星潮召唤体系允许莉莉娅作为第一位成年魅魔女王候选合理登场。
   - 精神控场、幻术、情绪破防在高考实战中有强烈反差爽点。
   - 魅魔身份天然适合主动暧昧、坏女人感、后宫控场。

2. **服务召唤兽后宫**
   - 契约规则明确召唤兽女主不是宠物，而是拥有独立人格的成年智慧伙伴。
   - 召唤契约可通过并肩战斗、信任、日常相处提升，天然绑定感情线和战斗线。

3. **服务学院/高考前期**
   - 异能高考是制度化选拔，方便开局打脸、排名、资源奖励、选拔副本。
   - 学院只是前期资源平台，不会限制后续展开。

4. **服务中后期扩展**
   - 秘境、禁区、异界通道、女主原世界线都从星潮规则自然延伸。
   - 每位女主背后都可以带出一条世界线和一个阶段事件。

# 世界地图

## 01. 地图层级总览

本书地图采用“四层递进结构”，便于分卷推进。

### 第一层：江海市——开局高考与城市秘境舞台

**新增项：江海市。**

定位：华夏东部大型沿海城市，超凡教育发达，拥有稳定的低阶秘境群。

主要功能：

- 林烬觉醒与异能高考开局。
- 林清雪现实侧羁绊建立。
- 莉莉娅首次登场并打脸考官/同学。
- 米娅或白九璃的媒介伏笔出现。
- 财阀、公会、地下召唤组织初次露头。

关键地点：

1. **江海一中异能班**
   - 高考前期舞台。
   - 负责觉醒检测、实战模拟、校内排名。
   - 可出现看不起召唤系的同学、偏心导师、强势天才等打脸角色。

2. **江海觉醒中心**
   - 官方觉醒登记和初步评级机构。
   - 可检测天赋类型，但无法准确识别林烬的特殊召唤本质。
   - 开局可给林烬“不稳定召唤系”的低评价。

3. **江海第一秘境：镜湖秘境**
   - 稳定低阶秘境，常用于高考模拟。
   - 地形：湖泊、雾林、废弃祭坛。
   - 怪物：幻雾水妖、镜影兽、低阶梦魇。
   - 适合莉莉娅精神控场高光。

4. **江海黑市：下城区星晶集市**
   - 地下资源交易点。
   - 白九璃可在这里留下情报伏笔。
   - 地下召唤组织可通过黑市售卖污染媒介。

5. **城市裂隙：旧港区临时秘境**
   - 高考/选拔期间突发失控事件。
   - 适合首卷中后段大型实战。
   - 可让林烬小队从考试竞争升级为城市危机处理。

### 第二层：华夏学院与训练营体系

**新增项：华夏三大顶级学院。**

1. **星海学院**
   - 主打召唤、精神、空间、秘境探索。
   - 适合林烬入学或获得训练营资格。
   - 校内有召唤师专属资源、契约研究室、模拟秘境。

2. **昆仑战院**
   - 主打近战、元素、军方体系。
   - 强硬派多，适合制造竞争与团队战。
   - 可有看不起后宫召唤组合的战斗狂角色。

3. **天枢研究院**
   - 主打科技、机械、符文、数据解析。
   - 为诺雅等机械天使相关剧情做铺垫。
   - 也可引出财阀实验与非法召唤研究。

学院相关地点：

- **新生秘境塔**
  - 学院入门资源塔，每层不同规则。
  - 适合排名战、女主轮流高光、召唤组合展示。

- **星潮训练营**
  - 国家级年轻天才集训。
  - 学院篇向都市/全球事件过渡的桥梁。
  - 各地天才、财阀子弟、公会继承人汇聚。

- **跨校联赛秘境**
  - 公开赛事，可配合直播舆论、名声暴涨、资源奖励。

### 第三层：华夏都市秘境与全球禁区

**新增项：华夏七大都市秘境圈。**

1. **江海镜湖圈**
   - 前期低阶秘境为主。
   - 精神、幻境、水系资源较多。
   - 服务莉莉娅初期成长。

2. **西南万兽山脉**
   - 生物型怪物、血脉材料丰富。
   - 可引出狐族、兽化污染、白九璃线索。
   - 注意白九璃本体设定为成年类人智慧狐女，不写动物化性内容。

3. **北境冰渊**
   - 冰系、亡灵、空间裂缝资源。
   - 可作为艾瑟琳血族夜战副本之一。

4. **南海星墟**
   - 海底遗迹、空间潮汐、异界沉船。
   - 适合米娅的空间/封印/秘宝鉴定能力发挥。

5. **中州龙脉裂谷**
   - 华夏核心战略资源区。
   - 官方严控。
   - 中期可围绕财阀、公会、官方利益冲突展开。

6. **东境机械遗迹**
   - 古代机械、数据生命、异常系统。
   - 为诺雅登场做铺垫。

7. **西北光蚀荒原**
   - 光暗混杂、圣遗物、污染教团。
   - 为伊芙琳、影弥线埋伏笔。

**新增项：全球十大禁区概念。**

不必前期全部展开，只作为中后期地图池。

- 欧洲血月古堡群：艾瑟琳原世界血族王朝接口。
- 北美机械天坑：诺雅机械天界接口。
- 南美翡翠神林：希尔薇精灵圣域接口。
- 非洲太阳裂谷：光辉圣女/古神污染接口。
- 极北永夜冰冠：深渊裂隙与星潮源头接口。
- 太平洋星门群岛：全球星门战争核心接口。

### 第四层：女主原世界线

各女主原世界不在前期大量展开，只通过召唤片段、梦境、秘境残片、敌对追兵、召唤媒介逐步揭示。

1. **莉莉娅原世界：夜冕魔界**
   - 魅魔王庭、梦魇贵族、深渊议会并存。
   - 莉莉娅是魅魔女王候选，因王庭继承斗争回应林烬召唤。
   - 长期冲突：魅魔王位、梦魇权柄、深渊议会对地球坐标的觊觎。

2. **米娅原世界：星锁迷宫**
   - 空间、封印、炼金、秘宝相关世界。
   - 米娅外表娇小但实际年龄与心智均为成年。
   - 可根据千束拍板选择能力方向。
   - 长期冲突：星锁核心失窃、迷宫主权争夺。

3. **白九璃原世界：青丘影市**
   - 狐族情报、幻术、交易、契约中介。
   - 白九璃有情报商和绿茶拱火属性。
   - 长期冲突：青丘影市被某个跨界势力渗透。

4. **艾瑟琳原世界：血月王朝**
   - 血族贵族、月血仪式、王位继承。
   - 艾瑟琳是成年吸血鬼公主。
   - 长期冲突：王朝内乱、血祖苏醒、地球血月禁区共鸣。

5. **希尔薇原世界：翡翠圣域**
   - 精灵、自然结界、古树、圣射手传承。
   - 长期冲突：圣树枯萎与秘境污染。

6. **诺雅原世界：机械天界**
   - 数据神殿、机械羽翼、战场预测系统。
   - 长期冲突：失控主脑、星潮算法、地球数据化风险。

7. **伊芙琳原世界：光辉圣国**
   - 圣女、净化、治疗、信仰体系。
   - 长期冲突：光辉被污染，圣国高层可能有问题。

8. **影弥原世界：无光暗域**
   - 影魔、刺客、反召唤、潜伏。
   - 长期冲突：暗域被用作入侵地球的跳板。

# 世界规则

## 显性规则

### 01. 星潮规则

**新增项：星潮。**

星潮是多世界壁垒周期性松动产生的宇宙级现象。地球在十八年前第一次被星潮覆盖，因此进入全球觉醒时代。

星潮带来的变化：

1. 人类觉醒异能。
2. 秘境与异界碎片降临。
3. 星晶矿脉生成。
4. 异界通道偶发开启。
5. 召唤师职业出现。
6. 部分人类灵魂频率可与异界生命共鸣。

前期只让读者知道“星潮导致全球觉醒”，中后期再揭示“星潮其实是地球被多世界盯上的信号”。

### 02. 觉醒规则

人类一般在十八岁前后进行正式觉醒检测，因此异能高考成为制度核心。

觉醒方向常见分类：

- 强化系：体质、速度、力量、防御。
- 元素系：火、水、雷、冰、风等。
- 精神系：幻术、念力、感知。
- 空间系：传送、储物、封锁。
- 召唤系：召唤战兽、元素灵、异界生命。
- 辅助系：治疗、增幅、净化、炼金。
- 特殊系：命运、时间、复制、规则类等稀有能力。

林烬表面检测结果：  
“不稳定召唤系，初始输出低，召唤对象未知，成长曲线不明确。”

林烬真实天赋：  
**新增项：万界羁绊召唤。**

特点：

1. 可连接多异界成年智慧强者。
2. 契约对象以类人女性强者为主，符合本书卖点。
3. 契约建立需要双方意志回应。
4. 契约越深，能力同步越强。
5. 不适合传统仪器检测，因此开局被低估。

### 03. 异能高考规则

**新增项：异能高考机制。**

异能高考分为四项：

1. **基础灵能值**
   - 测试灵能容量、恢复速度、肉体承载力。
   - 召唤系若初期无法稳定召唤，常在此项吃亏。

2. **天赋评级**
   - 官方仪器给出初始评级。
   - 评级从 F 到 SSS。
   - 仪器只能检测已表现出的能力，无法准确识别林烬的隐藏召唤潜力。

3. **实战模拟**
   - 在安全秘境或模拟战场内对抗怪物。
   - 莉莉娅可在这里用精神控场完成第一次惊艳反杀。

4. **团队副本**
   - 考验协作、资源分配、战场判断。
   - 林清雪与林烬可在此建立搭档关系。
   - 召唤兽女主的战术价值被正式体现。

高考奖励：

- 学院录取资格。
- 星晶奖金。
- 秘境进入名额。
- 召唤媒介兑换权。
- 官方临时保护身份。

### 04. 召唤契约规则

正规召唤契约分为三类：

1. **战斗契约**
   - 最常见。
   - 双方约定战斗协作、灵能供给和召唤时长。
   - 适合普通召唤师。

2. **羁绊契约**
   - 林烬核心契约类型。
   - 建立在信任、情绪共鸣、并肩作战和双向选择上。
   - 可解锁合击、感知共享、技能反馈。
   - 允许关系自然推进，但必须成年、自愿、平等。

3. **本命契约**
   - 深层契约，极少数召唤师可达成。
   - 可让召唤兽从投影降临逐步接近完全降临。
   - 风险是双方精神状态会互相影响。
   - 可作为女主重要关系节点，不滥用。

非法契约：

- 强制契约。
- 污染契约。
- 血祭契约。
- 洗脑契约。
- 奴役契约。

这些由地下召唤组织使用，是明确反派行为，主角团会与之冲突。

### 05. 召唤兽合法身份规则

**新增项：召唤伙伴临时身份制度。**

为了让召唤兽女主能合理进入都市生活，官方制定了“智慧召唤伙伴临时身份制度”。

规则：

1. 召唤对象若具备智慧、语言、类人社会适应能力，可申请“召唤伙伴临时身份”。
2. 召唤师为现实侧担保人，但不拥有召唤对象人身所有权。
3. 召唤伙伴可拥有独立行动权、财产收益权、任务分成权。
4. 召唤伙伴在城市内不得随意攻击普通人。
5. 高危异界身份需定期接受安全评估，但官方不得强制解剖、囚禁、奴役。
6. 学院和官方任务中，召唤伙伴可作为队伍成员登记。
7. 成年智慧召唤对象可自行决定私人关系，召唤师不得以契约强迫。

剧情功能：

- 让莉莉娅、白九璃、艾瑟琳等可合法进入学院、都市、据点日常。
- 制造“保守派质疑异界召唤兽风险”的冲突。
- 地下组织通过非法契约与主角形成鲜明对照。
- 保障后宫关系合规边界。

## 隐性规则

### 01. 地球是新生锚点

地球不是普通低级世界，而是星潮中的“新生锚点”。多个异界都能通过地球稳定坐标。

隐性影响：

- 地球秘境越来越多。
- 异界强者逐渐能定位地球。
- 林烬的召唤契约可能加速坐标暴露。
- 女主原世界危机最终会汇聚到地球。

### 02. 林烬的召唤并非随机

林烬召唤到的女主并不是纯随机，而是“世界线关键节点”之间的互相选择。

每位女主回应召唤时，通常都处在某种危机或转折：

- 莉莉娅：魅魔王庭继承危机。
- 米娅：星锁迷宫封印异常。
- 白九璃：青丘影市交易局。
- 艾瑟琳：血月王朝内乱。
- 希尔薇：圣树污染。
- 诺雅：机械天界失控算法。
- 伊芙琳：光辉圣国信仰腐化。
- 影弥：暗域反召唤计划。

### 03. 契约羁绊会改变双方命运

羁绊契约不是单方面增强，而是双向改写命运轨迹。

表现：

- 女主获得地球锚点，能暂时摆脱原世界追杀或束缚。
- 林烬获得女主能力反馈和异界知识。
- 双方共同面对契约带来的追踪、嫉妒、势力争夺和世界线修正。

## 限制与代价

### 01. 召唤时长限制

前期林烬灵能不足，召唤女主不能无限维持完全实体。

解决方式：

- 星晶补能。
- 契约亲和度提升。
- 主角等级提升。
- 召唤媒介强化。
- 学院训练召唤稳定性。

剧情作用：  
避免主角前期无脑碾压，同时给日常同居、契约加深、资源争夺提供动力。

### 02. 召唤位限制

林烬前期可稳定维持一到两位召唤伙伴，随着等级提升增加。

建议节奏：

- 一阶：临时召唤莉莉娅。
- 二阶：稳定莉莉娅，可短时召唤第二位。
- 三阶：两位稳定，一位临时。
- 四阶：三位稳定，合击体系成型。
- 五阶以上：后宫小队轮换作战，部分女主可常驻据点。

### 03. 灵魂负荷

召唤高位异界强者会给主角精神带来压力。

表现：

- 头痛。
- 灵能透支。
- 梦境被女主原世界片段影响。
- 高阶敌人可通过契约反向追踪。

莉莉娅等精神系女主可帮助主角稳定灵魂，也为暧昧陪伴、事后氛围、贴身照顾提供合理桥段。

### 04. 异界排斥

女主在地球完全降临会受到世界排斥，尤其是等级过高时。

解决方式：

- 契约锚点。
- 身份登记。
- 星晶供能。
- 秘境中战斗时排斥降低。
- 主角等级提升后可承载更强降临。

### 05. 合规边界

世界规则明确：

- 智慧召唤对象不是宠物。
- 契约不能替代自愿。
- 成年类人智慧对象才可推进亲密关系。
- 萝莉感角色必须实际成年且心智成年。
- 白九璃等狐族按成年类人智慧族群处理，不写动物化性内容。
- 禁止强迫、奴役、洗脑式亲密关系。
- 亲密描写只使用隐晦方式。

## 规则如何制造冲突

1. **高考评级误判**
   - 林烬天赋被判低潜力，方便开局打脸。

2. **召唤系偏见**
   - 召唤系前期不稳定、耗资源，被认为拖后腿。
   - 林烬用莉莉娅等女主组合证明召唤系上限。

3. **合法身份争议**
   - 莉莉娅等异界女主强大又美艳，引来官方保守派、同学嫉妒、财阀窥探。
   - 主角维护她们独立人格，强化情感羁绊。

4. **资源争夺**
   - 星晶、秘境钥匙、召唤媒介稀缺。
   - 财阀、公会、学院、地下组织围绕资源爆发冲突。

5. **非法召唤对照**
   - 地下召唤组织使用奴役契约、污染契约。
   - 主角的平等羁绊契约与反派形成价值对立。

6. **异界追踪**
   - 每次强力召唤都可能暴露坐标。
   - 女主原世界敌人逐步进入地球主线。

# 力量体系

## 等级结构

本书采用“九阶觉醒者体系”，简单直观，方便爽文升级。

### 人类觉醒者等级

| 等级 | 名称 | 大致定位 | 剧情阶段 |
|---|---|---|---|
| 一阶 | 觉醒者 | 刚觉醒，掌握基础能力 | 开局高考 |
| 二阶 | 实战者 | 可进入低阶秘境 | 高考/学院入门 |
| 三阶 | 精英者 | 城市年轻天才 | 首卷末/学院前期 |
| 四阶 | 统领级 | 可带队处理城市危机 | 学院后期/都市秘境 |
| 五阶 | 君主级 | 区域强者，公会会长级 | 中期 |

| 七阶 | 天灾级 | 可镇压大型禁区 | 后期 |
| 八阶 | 圣域级 | 接触世界规则 | 后期跨界 |
| 九阶 | 星冕级 | 星潮顶层战力 | 大后期 |

### 异界召唤兽等级

召唤兽女主可对应九阶，但使用原世界身份称谓增强特色。

示例：

- 莉莉娅：魅魔女王候选，初登场受世界排斥，表现约二阶上限到三阶局部爆发，后续逐步恢复。
- 白九璃：青丘情报商/狐族幻术高手，擅长潜入与资源牵线，正面战力未必最高。
- 艾瑟琳：血族公主，夜战爆发强，自愈能力强，受阳光/圣光类规则克制。
- 诺雅：机械天使，战场预测强，依赖数据和能量核心。
- 伊芙琳：光辉圣女，治疗净化强，但对污染类敌人有天然仇恨。
- 影弥：影魔刺客，暗杀强，正面对抗需配合队友。

### 召唤师专属成长维度

林烬除自身阶位外，还有以下成长线：

1. **召唤位**
   - 决定同时稳定召唤数量。

2. **契约深度**
   - 决定女主降临强度、技能同步和合击权限。

3. **锚点强度**
   - 决定女主在地球停留时间和日常实体稳定度。

4. **灵魂承载**
   - 决定能否承受高位女主完全降临。

5. **羁绊回馈**
   - 女主能力反哺主角，如精神抗性、魅惑免疫、空间感、夜视、自愈、战场预测等。

## 升级方式

### 01. 常规升级

- 吸收星晶。
- 秘境实战。
- 学院训练。
- 击败怪物获得灵能沉淀。
- 完成官方/学院任务获得资源。

### 02. 召唤师升级

- 契约稳定训练。
- 召唤阵优化。
- 媒介强化。
- 与召唤伙伴合击磨合。
- 战斗中完成高风险共鸣。

### 03. 羁绊升级

羁绊升级不写成机械好感度，但剧情中可通过以下节点体现：

- 女主主动选择留下。
- 主角在危机中保护女主人格与自由。
- 双方并肩击败追兵。
- 女主透露原世界秘密。
- 日常同居/据点相处加深信任。
- 隐晦亲密节点后契约波动变得稳定。
- 女主称呼变化、态度变化、吃醋争宠、合击解锁。

### 04. 女主恢复/进阶

女主初到地球时可因世界排斥、契约不稳、伤势、封印等原因实力受限。

恢复方式：

- 找到对应召唤媒介。
- 完成个人原世界任务。
- 击败追踪敌人。
- 林烬阶位提升。
- 契约深度增加。
- 获得专属器物。

## 升级资源

1. 星晶。
2. 学院积分。
3. 秘境核心。
4. 召唤媒介。
5. 异界残片。
6. 契约锚石。
7. 星门碎片。
8. 女主专属圣物。
9. 高阶导师指导。
10. 官方任务权限。

## 风险与代价

1. 过度吸收星晶会灵能紊乱。
2. 强行召唤会精神反噬。
3. 非法契约会污染灵魂。
4. 召唤高位女主会暴露地球坐标。
5. 合击失败可能导致双方受伤。
6. 秘境深处规则异常，可能改变战斗常识。
7. 女主原世界敌人可能顺着契约追踪林烬。

## 剧情功能

1. **制造升级爽感**
   - 从被判低潜力，到高考逆袭，再到学院顶尖，再到全球核心。

2. **制造女主高光**
   - 每位女主有不同战斗定位，避免重复。
   - 莉莉娅精神控场，米娅空间/封印，白九璃情报潜入，艾瑟琳夜战突袭等。

3. **制造团队组合**
   - 魅魔控场 + 主角指挥 + 林清雪现实侧输出/辅助。
   - 狐女潜入 + 吸血鬼突袭。
   - 机械天使预测 + 精灵狙击。
   - 圣女净化 + 影魔反召唤。

4. **制造感情推进**
   - 契约越深，战斗越强。
   - 但契约加深必须来自信任和自愿，不是强制绑定。

由现代科技结合星晶制造。

示例：

- 战术护甲。
- 秘境通讯器。
- 便携检测仪。

功能：前期常用，方便都市感和考试感。

### 02. 召唤师装备

召唤师专属器物，用于稳定契约和提升召唤效率。

- 星纹卡。
- 共鸣项链。

### 03. 秘境遗物

秘境中产出的古代器物，通常带特殊词条。

示例：

- 镜湖幻晶。
- 梦魇面具。
- 星锁钥匙。
- 血月匕首。
- 圣木长弓残片。
- 机械羽核。

功能：推动副本奖励和女主进阶。

### 04. 异界圣器

女主原世界相关高阶器物。

# 资源体系

## 01. 资源总览

本书资源体系强调“看得懂、能争、能爽”。

资源不宜过多，核心围绕以下七类：

1. 星晶。
2. 学院/官方积分。
3. 秘境钥匙。
4. 召唤媒介。
5. 异界材料。
6. 契约锚点。
7. 异界坐标。

## 02. 星晶

星晶是最通用的超凡能源。

来源：

- 秘境怪物核心。
- 星潮矿脉。
- 禁区采集。
- 异界通道沉积。
- 官方奖励。
- 公会任务。

用途：

- 修炼。
- 交易。
- 驱动装备。
- 稳定召唤。
- 补充秘境消耗。
- 制造灵能武装。

稀缺性：

- 低阶星晶可交易。
- 高阶星晶受官方和财阀控制。
- 特殊属性星晶会被特定女主需要，如梦魇晶、月血晶、幻灵晶、圣光晶。

剧情冲突：

- 高考奖励争夺。
- 秘境矿脉抢夺。
- 财阀垄断星晶市场。
- 地下组织走私污染星晶。

## 03. 学院/官方积分

学院积分用于前期推动主角成长，官方积分用于中后期任务体系。

来源：

- 考试排名。
- 秘境通关。
- 击杀怪物。
- 救援任务。
- 公开赛事。
- 官方悬赏。

用途：

- 兑换训练室。
- 兑换召唤媒介。
- 兑换秘境名额。
- 兑换装备。
- 请导师指导。
- 获得高阶资料权限。

剧情功能：

- 让林烬在学院短期内有明确目标。
- 让女主需要的道具可通过积分兑换或争夺。
- 避免学院篇只有排名，没有实际收益。

## 04. 秘境钥匙

秘境钥匙是进入特定秘境的凭证，分为一次性和永久权限。

类型：

1. 低阶考试钥匙。
2. 学院训练钥匙。
3. 公会探索钥匙。
4. 禁区外围钥匙。
5. 星门残钥。

争夺方：

- 学院。
- 官方。
- 财阀。
- 公会。
- 地下召唤组织。
- 异界追兵。

剧情功能：

- 每卷可以用一枚关键秘境钥匙引出副本。
- 女主专属媒介可藏在对应秘境中。
- 财阀和地下组织可因钥匙与主角冲突。

## 05. 召唤媒介

召唤媒介是本书最贴合后宫召唤卖点的资源。

媒介不是“抽卡道具”，而是能让特定异界频率更清晰的锚物。

示例：

- 莉莉娅：梦魇晶、夜冕残片、魅魔王庭纹章。
- 米娅：星锁钥匙、迷宫银砂、封印核心。
- 白九璃：幻灵铃、青丘狐火玉、影市契约牌。
- 艾瑟琳：月血石、血月蔷薇、王族血印。
- 希尔薇：圣木枝、翡翠箭羽、自然结界核。
- 诺雅：机械羽核、数据圣环、天界芯片。
- 伊芙琳：圣光晶、光辉书页、净化冠冕。
- 影弥：无光刃片、影核、暗域面具。

剧情功能：

- 为女主登场提供合理前置。
- 为女主进阶提供副本目标。
- 为修罗场提供“谁的媒介先兑换”之类轻喜剧争宠点。
- 为财阀/地下组织觊觎女主价值制造冲突。

## 06. 异界材料

异界材料用于装备、炼金、药剂、召唤阵升级。

分类：

- 怪物核心。
- 异界金属。
- 灵植。
- 血脉材料。
- 幻术材料。
- 空间材料。
- 圣光材料。
- 暗影材料。
- 机械残骸。

注意：

- 涉及智慧种族身体材料时谨慎处理，避免物化召唤兽女主。
- 反派可进行残酷实验，主角团明确反对。
- 主角获得女主相关材料时，多以圣物、遗物、矿石、结晶形式出现。

## 07. 契约锚点

契约锚点用于让召唤兽女主在地球稳定停留。

形式：

- 据点房间召唤阵。
- 契约戒。
- 灵魂锚石。
- 星纹居住证。
- 学院召唤宿舍。
- 主角精神海中的羁绊印记。

剧情功能：

- 解释女主为何可常驻主角身边。
- 支撑同居/据点日常。
- 锚点被攻击时可制造危机。
- 锚点升级后可解锁更多女主常驻。

## 08. 异界坐标

异界坐标是中后期核心战略资源。

掌握坐标意味着：

- 可定位女主原世界。
- 可追踪主角召唤链路。
- 可开启星门。
- 可发动跨界入侵。
- 可进行救援或反攻。

争夺方：

- 官方机构。
- 顶级学院。
- 财阀研究所。
- 地下召唤组织。
- 异界种族势力。
- 女主原世界敌对派系。

剧情功能：

- 将个人召唤后宫升级为全球事件。
- 女主个人线与地球主线交汇。
- 后期主角建立跨界召唤秩序的基础。

# 世界故事背景

## 01. 星潮前时代

星潮前，地球与现实世界近似。

人类社会有普通学校、大学、公司、财团、国家机构、网络舆论和现代科技。超凡只存在于神话、宗教、传说和都市怪谈中。

少数古老遗迹曾出现无法解释的符号，但长期被当作考古异常，直到星潮降临后，人类才发现这些遗迹可能是上一次多世界接触留下的痕迹。

## 02. 第一次星潮

十八年前，全球夜空出现银色星河。

七日内：

- 大量人类昏迷。
- 部分人醒来后觉醒异能。
- 野外出现怪物。
- 城市边缘出现秘境入口。
- 通讯卫星短暂失灵。
- 世界各国记录到无法解释的空间重叠现象。

这场事件被命名为：**第一次星潮**。

星潮后，全球进入觉醒时代。

## 03. 觉醒时代建立

星潮后的最初三年，人类经历混乱期。

怪物袭击、秘境失控、觉醒者犯罪、资源争夺频繁发生。随后，各国逐渐建立制度。

华夏建立：

- 觉醒者登记制度。
- 异能高考制度。
- 觉醒者学院体系。
- 秘境管理局。
- 星潮防卫署。
- 公会许可制度。
- 智慧召唤伙伴临时身份制度。

世界由此进入相对稳定的发展期。

## 04. 第二次星潮前兆

主线开局时，距离第一次星潮已经十八年。

表面上，人类已经适应觉醒时代，但实际上第二次星潮正在逼近。

前兆：

- 秘境波动变多。
- 异界通道出现频率上升。
- 召唤系觉醒者数量增加。
- 地下召唤组织活跃。
- 全球禁区出现异常潮汐。
- 林烬这样的特殊召唤天赋出现。

官方顶层知道部分真相，但不愿公开引发恐慌。

## 05. 林烬的特殊性

林烬并不是单纯“运气好召唤出魅魔”。

他的灵魂频率与地球新生锚点高度契合，因此能被多个异界关键女性强者感应。

这意味着：

1. 他能召唤高潜力女主。
2. 他能通过羁绊契约稳定多世界连接。
3. 他会成为官方、学院、财阀、地下组织、异界势力关注对象。
4. 他可能是第二次星潮中改变地球命运的人。

前期不直接揭露，只通过检测异常、莉莉娅惊讶、秘境回响、媒介主动响应等方式埋伏笔。

## 06. 长期伏笔池

### 伏笔一：星潮并非自然灾害

星潮可能是多个世界争夺新生锚点的结果。地球看似获得力量，实则被放上了跨界棋盘。

### 伏笔二：召唤契约源于古老跨界盟约

现代召唤师以为召唤是异能，但真正的召唤契约可能来自远古时代的“星盟”。林烬的羁绊契约是星盟正统，而地下组织的奴役契约是被污染的残次版本。

### 伏笔三：女主们的原世界危机彼此有关

魅魔王庭、血月王朝、青丘影市、机械天界、光辉圣国等看似独立，背后可能都受同一个跨界黑手推动。

### 伏笔四：地球官方并非铁板一块

官方主流保护人类秩序，但内部存在保守派、实验派、利益派。

- 保守派担忧召唤兽风险。
- 实验派想研究林烬契约。
- 利益派可能与财阀勾连。
- 正派官方可作为主角阶段性靠山。

### 伏笔五：地下召唤组织寻找“完美容器”

地下召唤组织试图通过污染契约召唤高位异界存在。他们认为林烬是“天然完美容器”，因此中期会多次盯上主角和女主。

### 伏笔六：女主完全降临需要星门

前中期女主多以契约降临为主。后期若要解决原世界线，需要找到星门核心，让主角团具备主动跨界能力。

# 与作品卖点的关系说明

## 01. 与“都市异能”的关系

世界表层保持现代都市结构：学校、高考、学院、财阀、官方、公会、黑市、网络舆论等。

读者能快速代入，不需要理解复杂修仙宗门体系。

## 02. 与“魅魔吸量”的关系

莉莉娅作为成年魅魔女王候选，通过召唤规则自然登场。

她的能力强绑定：

- 精神控场。
- 幻术。
- 情绪破防。
- 魅惑抗性。
- 战场心理压制。
- 后宫控场与主动暧昧。

世界规则允许她在考试、秘境、日常、官方审查中持续制造看点。

## 03. 与“召唤兽后宫”的关系

召唤契约被设计为平等羁绊而非奴役，解决后宫合规与角色尊严问题。

每位女主有：

- 独立原世界。
- 独立身份。
- 独立能力。
- 独立危机。
- 与林烬的双向选择。
- 战斗价值和日常价值。

## 04. 与“学院/高考前期模块”的关系

异能高考提供开局打脸和资源奖励。

学院提供短期升级和身份平台，但地图结构已预留都市秘境、全球禁区、异界通道，避免长期困在校园。

## 05. 与“中后期自由扩展”的关系

星潮、秘境、禁区、异界坐标、女主原世界线共同构成扩展框架。

后续可以按需要展开：

- 都市秘境事件。
- 财阀资源争夺。
- 全球禁区任务。
- 异界追兵入侵。
- 女主原世界救援。
- 星门战争。
- 跨界秩序建立。

## 06. 与“快节奏男频爽文”的关系

设定尽量直观：

- 九阶等级。
- F 到 SSS 装备。
- 星晶作为通用资源。
- 高考/学院/秘境/禁区递进。
- 每位女主带一个能力标签和一个原世界伏笔。

不在前期堆复杂神话，只让设定为冲突、升级、暧昧、打脸服务。

# 需要千束拍板的世界观问题

1. **米娅能力方向最终选哪一个？**
   - A. 空间系：适合秘境逃生、传送、储物、位移。
   - B. 封印系：适合克制强敌、解锁遗迹、压制污染。
   - C. 炼金系：适合道具、药剂、装备强化、生活日常。
   - D. 秘宝鉴定系：适合寻宝、资源爽点、黑市剧情。
   - 建议：空间 + 秘宝鉴定混合，主能力空间，副能力鉴宝。

2. **主角入学学院是否确定为星海学院？**
   - 星海学院最适配召唤、精神、秘境探索。
   - 若想更强冲突，也可让林烬被战斗派学院看低后转入星海学院。

3. **异能高考篇长度倾向？**
   - A. 10—20章快速完成高考与首个秘境。
   - B. 30—50章完整高考、选拔、城市危机。
   - 建议：首卷 40—60章内完成高考/选拔/首个城市秘境，并拿到学院资格。

4. **官方机构基调要偏友善还是偏审查？**
   - A. 主流友善，少数保守派制造麻烦。
   - B. 整体强势压迫，主角更多靠自己。
   - 建议：A，更适合轻爽文和后续资源推进。

5. **地下召唤组织“黑契会”是否作为第一卷末反派出现？**
   - 可选：第一卷末只露头，第二阶段正式冲突。
   - 建议：第一卷末通过污染召唤事件露头，保持节奏。

6. **莉莉娅原世界“夜冕魔界”的主线危机要多早展开？**
   - A. 前期只通过梦境/追兵轻微伏笔。
   - B. 第一卷末直接出现王庭追兵。
   - 建议：A，避免压垮高考节奏。

7. **装备等级是否采用 F—SSS？**
   - 优点：读者直观。
   - 缺点：略常规。
   - 建议保留，适合快节奏男频。

8. **召唤兽女主在城市中的身份制度是否保留？**
   - 建议保留“智慧召唤伙伴临时身份制度”，能合理支撑学院、都市、同居据点和官方任务。

9. **第二次星潮是否作为全书中后期主线？**
   - 建议确定为中后期大主线，前期只埋秘境波动和异界通道异常。

10. **林烬真实天赋名称是否采用“万界羁绊召唤”？**
   - 备选：
     - 万界羁绊召唤。
     - 星约召唤。
     - 王契召唤。
     - 万界女王契约。
   - 建议：正式能力名用“万界羁绊召唤”，宣传/爽点称呼可用“女王契约”。
