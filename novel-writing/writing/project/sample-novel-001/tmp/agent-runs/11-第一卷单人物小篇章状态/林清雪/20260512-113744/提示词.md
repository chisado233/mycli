# 林清雪

## 1. 基础身份

### 姓名

林清雪

### 人物定位

女主 / 队友 / 战斗搭档 / 现实侧搭档 / 地球防线职业者 / 慢热情感线核心。

林清雪是主角团中最重要的地球现实侧人类女主，不是召唤物，也不通过召唤师契约加入苏念。她承担“苏念仍然属于地球、属于现实生活”的情感锚点功能，同时也是前期职业觉醒、高考小秘境、学院训练、江海城市怪物潮、后续大秘境防线中的长期同行伙伴。

她的定位不是单纯被保护的校园女神，而是从江海一中重点学生逐步成长为华夏镇守体系认可的年轻职业者，在地球侧战斗、防线任务、官方视角与现实身份认同上持续发挥作用。

### 所属势力

初始所属：

- 地球 / 华夏 / 江海市。
- 江海一中异能班。
- 江海市职业高考体系。
- 江海镇守府关注名单中的优秀学生。

后续关联：

- 星海学院或星海学院重点训练营。
- 华夏大秘境镇守体系后备职业者。
- 江海安全区防线与大秘境任务体系。
- 苏念主角团现实侧核心成员。

她与苏念身边的异世界女主不同，没有原世界势力牵引，也没有跨界呼唤流程。她的根基在地球，在学校、学院、镇守府、防线与现实社会秩序之中。

### 初始身份

江海一中异能班高三学生，十八岁以上，已达到地球职业觉醒年龄，是学校重点培养对象之一。

她在职业觉醒前就是江海一中成绩与实战训练都很稳定的优等生，性格认真，自律强，基础战斗素养高。相较苏念的孤儿出身与低资源处境，林清雪更像普通学生眼中的“可望而不可即的优秀模板”：成绩好、天赋高、气质清冷、做事可靠，被老师和同学寄予厚望。

开局阶段，她与苏念是同学兼熟人，有一定互相关注基础。她知道苏念嘴贫、懒散外表下其实很聪明，也知道他不是会轻易放弃的人。因此当苏念被判定为“不稳定召唤系”后，她没有跟随旁人嘲笑，而是第一时间关注他的状态。

建议职业方向：

- 主职业：冰系剑士 / 冰霜战法类职业。
- 初始阶位：一阶觉醒者。
- 战斗风格：冰霜控制、短剑/细剑近战、局部防护、减速牵制。
- 成长后定位：地球侧攻防兼备的防线职业者，可与苏念的召唤联动体系形成现实战术配合。

该职业方向与“清雪”气质匹配，也方便她在学院、防线、城市怪物潮中承担控场、救援、限制怪物潮推进等剧情功能。

### 隐藏身份/秘密

林清雪没有神秘跨界身份，也不是神明血脉、召唤对象或异世界转生者。她的“秘密”更偏现实与心理层：

1. **她早就关注苏念**
   - 表面上只是普通同学关系，实际上她很早就注意到苏念的异常聪明和临场判断力。
   - 她不一定会直白承认，但在觉醒前后，她对苏念的处境比其他同学更上心。

2. **她对自己的优秀有压力**
   - 作为江海一中的重点学生，她习惯被老师、同学、考务组寄予期待。
   - 她不允许自己在关键考核中拖后腿，也不愿成为只能被苏念保护的人。
   - 这种压力会推动她在后续学院与防线阶段主动成长。

3. **她害怕被苏念甩在身后**
   - 苏念觉醒初期被低评时，她还能够以现实侧同学身份靠近他。
   - 但莉莉娅、青璃、白九璃、克露露、诺雅等异世界女主陆续出现后，苏念的世界迅速扩大。
   - 林清雪真正害怕的不是“输给谁”，而是自己再也无法理解苏念正在面对的危险。

4. **她不愿把感情说出口**
   - 她清纯含蓄，自尊心强，吃醋时不会大吵大闹，而是用沉默、回避、训练加倍、话里带刺的方式表现。
   - 她越是在意，越会努力表现得冷静。
   - 白九璃这类擅长看穿人心的角色最容易戳破她的伪装。

5. **她是苏念地球身份的现实锚点**
   - 当苏念不断跨越世界、与异界女主建立契约、成为多界核心时，林清雪始终记得他最初只是江海一中那个被误判后还嘴硬吐槽的少年。
   - 她不代表神秘力量，而代表“苏念从哪里来”。

### 登场阶段

第一阶段：黄金开局与高考小秘境。

具体登场节点：

- 职业觉醒中心：与苏念一同参加十八岁职业觉醒。
- 苏念被判定为“不稳定召唤系”后，她第一个表现出关心。
- 高考小秘境异常中，她与苏念共同遭遇危机，并见证莉莉娅的存在。
- 苏念救下她和其他同学后，她开始察觉苏念的召唤能力并不普通。
- 后续进入学院篇、城市安全区、防线任务，持续作为地球现实侧女主存在。

## 2. 人物魅力

### 外貌与气质

林清雪是清纯含蓄型现实侧女主，外貌不走强烈魅惑路线，而是偏干净、冷白、安静、耐看的校园与职业者气质。

外貌特征：

- 年龄明确为十八岁以上，成年女性。
- 身形修长，训练痕迹明显，动作干净利落。

气质关键词：

- 清冷但不冷漠。
- 干净、认真、自律。
- 面对外人有距离感，面对苏念会露出不自然的柔软。
- 不擅长撒娇，但会用行动照顾人。

她与莉莉娅形成强烈对照：莉莉娅是成熟主动、深渊香气、强暧昧压迫；林清雪是现实清纯、慢热陪伴、欲言又止的拉扯感。

### 性格标签

- 清纯含蓄。
- 慢热认真。
- 外冷内软。
- 自尊心强。
- 行动力强。
- 有责任感。
- 吃醋但嘴硬。
- 不愿拖累别人。
- 现实侧陪伴型。

她不是傻白甜，也不是只会脸红的花瓶。她的含蓄来自性格与成长环境，不代表软弱。面对怪物潮、黑契会、地球防线危机时，她会害怕，但不会退缩。

### 反差点

1. **清冷优等生会悄悄吃醋**
   - 平时说话克制，遇到莉莉娅、青璃等女主与苏念过于亲近时，表面镇定，手里的训练剑却会握得更紧。
   - 她不会直接问“你们什么关系”，而是问“你昨晚又灵能透支了？”“她为什么能随便进你房间？”

2. **看似柔弱，战斗时很狠**
   - 平时温柔克制，战斗中却能精准判断怪物弱点，用冰霜控制为队友创造机会。
   - 她不会为了表现自己逞强乱冲，但一旦必须断后，会毫不犹豫留下。

3. **嘴上说不在意，行动上全是关注**
   - 苏念低评后，她假装只是顺手递资料。
   - 苏念受伤后，她会冷着脸把恢复药剂塞过去。
   - 苏念被官方质疑时，她会站出来作证，却说“我只是陈述事实”。

4. **现实侧女主面对异界女主的危机感**
   - 她不是契约女主，没有契约同步、召唤锚点、跨界命运羁绊。

   - 后续成长重点就是她不靠契约、不靠被召唤，而是凭自己的选择与努力站到苏念身边。

5. **越喜欢越不坦率**
   - 白九璃随便一句茶言茶语就能让她表情失控。
   - 莉莉娅成熟地调侃她时，她会努力保持礼貌，但心里已经乱成一团。

## 3. 与主角关系

### 初始关系

林清雪与苏念初始是江海一中同学，属于熟悉但尚未明确亲密关系的状态。

初始关系基础：

- 两人同在职业觉醒与高考体系中。
- 林清雪知道苏念平时嘴贫、看似不正经，但关键时刻很可靠。
- 苏念知道林清雪是学校重点学生，清冷认真，不太会参与同学间的嘲笑和排挤。
- 两人之间有尚未明说的好感基础，但都没有主动挑破。
- 苏念被低评后，林清雪的关心成为他现实侧少数温暖来源之一。

初期关系关键词：

- 同学。
- 搭档预备。
- 互相关注。
- 未挑明好感。
- 现实侧陪伴。
- 高考危机中的并肩与被救。

### 关系吸引点

林清雪被苏念吸引的点：

1. **低谷中仍嘴硬不认输**
   - 苏念被判定为不稳定召唤系后没有彻底崩溃，而是用嘴贫和吐槽掩盖压力。
   - 林清雪能看出他不是没事，而是不想让别人看见狼狈。

2. **关键时刻可靠**
   - 高考小秘境异常中，苏念即使自身处境危险，也会救同学、救她、反向破解危机。
   - 他不只是嘴上聪明，实战中也能冷静判断。

3. **有底线**
   - 苏念对莉莉娅等异界女主坚持自愿契约，不把她们当工具。
   - 林清雪虽会吃醋，但也会因此确认苏念不是被力量冲昏头脑的人。

4. **仍保留现实感**
   - 哪怕苏念的能力越来越离谱，他在林清雪面前偶尔还是那个会吐槽资源不够、考试麻烦、训练太累的江海学生。
   - 这种现实感让林清雪觉得自己还能靠近他。

苏念被林清雪吸引的点：

1. **她没有在低谷时看轻他**
   - 职业觉醒后，她的关心对苏念很重要。
   - 她没有因为评级而改变态度。

2. **她清醒、认真、不盲从**
   - 面对苏念身边越来越多异常存在，她不是无脑接受，也不是立刻敌视，而是观察、思考、追问。
   - 她能成为苏念现实判断的镜子。

3. **她愿意追上来**
   - 林清雪不是一直等苏念保护，她会加练、参战、承担防线任务。
   - 苏念能从她身上看到地球职业者的韧性。

4. **她代表“家”的方向**
   - 苏念孤儿出身，现实牵挂不多。
   - 林清雪的存在让江海、学校、地球不只是任务地点，而有了情感重量。

### 关系阻碍

1. **苏念身份变化过快**
   - 苏念从低评召唤师迅速变成界域行者、多界契约核心，林清雪很难跟上信息变化。
   - 他有许多秘密不能立刻告诉她，导致误会和距离。

2. **异界女主带来的情感压力**
   - 莉莉娅开篇即与苏念有强烈暧昧契约氛围。
   - 青璃有“剑主”绑定。
   - 克露露会挑衅。
   - 白九璃会拱火。
   - 诺雅有装甲同步与数据化占有欲。
   - 林清雪没有契约关系，容易觉得自己没有“特殊证明”。

3. **现实侧与跨界侧的信息差**
   - 林清雪起初只知道地球游戏入侵、高考、副本、怪物潮。
   - 苏念却已经接触深渊、修仙、神明世界伏笔。
   - 她不愿被蒙在鼓里，但也知道苏念隐瞒有安全原因。

4. **她的自尊心**
   - 林清雪不想靠“被苏念喜欢”获得位置。
   - 她希望自己能在战斗和防线中证明价值。
   - 因此她不会轻易示弱，反而可能在压力下逞强。

5. **官方与防线压力**
   - 苏念身边的异界女主会引发官方警惕。
   - 林清雪作为地球职业者，一方面相信苏念，另一方面也必须面对镇守体系的安全逻辑。
   - 她会成为情感与现实立场之间的缓冲人物。

6. **后宫结构下的长期拉扯**
   - 林清雪不会轻易接受自己只是“众多女主之一”。
   - 她需要时间确认苏念是否认真对待她，而不是因为现实侧熟悉才把她留在身边。
   - 这条线应慢热推进，不能过早完全解决。

### 关系变化路线

第一阶段：同学关心 → 高考危机牵绊

- 职业觉醒后，林清雪关心苏念低评状态。
- 高考小秘境异常中，苏念救下她并展示异常召唤能力。
- 她第一次看见莉莉娅，对苏念产生“他身上有我不知道的秘密”的认知。
- 情绪：关心、震惊、疑惑、微妙吃味。

第二阶段：现实陪伴 → 学院吃醋拉扯

- 进入学院或训练营后，林清雪与苏念仍保持现实侧联系。
- 莉莉娅常驻、青璃加入后，她开始明显感到危机。
- 她嘴上不问，行动上靠近苏念，训练也更加拼命。
- 情绪：不安、吃醋、克制、想追上他。

第三阶段：并肩作战 → 不是只能被保护

- 城市怪物潮和学院任务中，林清雪开始承担战斗职责。
- 她用冰霜控制、救援、牵制证明自己不是苏念的累赘。
- 苏念开始主动将现实侧任务交给她配合，而不是只保护她。
- 情绪：信任提升，关系由被救者转向搭档。

第四阶段：人魔战争 → 情感压力爆发

- 苏念公开维护莉莉娅、克露露等异界女主。
- 林清雪理解他的选择，但也感到自己位置动摇。
- 战后她质问苏念是否真的信任她、是否还把自己当地球人。
- 苏念回应“我信她们，也信你”，关系进入第一次正面确认。
- 情绪：委屈、压抑、爆发、被认真回应后的动摇。

第五阶段：防线成长 → 现实侧核心伙伴

- 林清雪参与地球防线任务，成为华夏镇守体系年轻职业者中的优秀战力。
- 她在苏念不在地球或跨界行动时，承担现实侧联络、防线支援、同伴保护功能。
- 她逐步接受苏念的多界身份，但要求自己也能承担对应责任。
- 情绪：从追赶变成并行。

第六阶段：多界会盟 → 地球代表性的支持者

- 多界会盟与神明世界真相公开后，林清雪不再只是苏念的同学，而是地球侧理解他、支持他的关键人物。
- 她能在地球职业者、官方、主角团之间充当可信桥梁。
- 她与异界女主之间从单纯吃醋，转为互相承认。
- 情绪：成熟、稳定、仍会吃醋但不再自我怀疑。

终局阶段：现实归属确认 → 长期核心女主

- 神庭契约陷阱中，林清雪无法通过召唤师契约回应苏念，但她以地球侧伙伴和自愿选择站在他身边。
- 她证明自己不是契约附属，却同样是苏念不可替代的情感支柱。
- 终局后，她成为苏念往返多界时最重要的现实归属之一。

### 终局关系

林清雪终局为苏念长期核心女主之一、地球现实侧最重要的情感锚点与防线伙伴。

终局关系特点：

- 她不是召唤物，不受召唤师契约归属。
- 她与苏念的关系建立在现实陪伴、共同成长、危机并肩和双方自愿选择上。
- 她接受苏念身边存在多位异界女主，但不会失去自己的位置。
- 她与莉莉娅、青璃、白九璃等女主形成长期轻修罗场关系，日常有吃醋和拉扯，但大局上互相承认。
- 她在地球侧防线、学院后方、多界盟约现实事务中保留独立价值。
- 她代表苏念从江海走出的起点，也代表他终局仍会回到地球。

情感表达边界：

- 双方均成年。
- 所有亲密推进均自愿。
- 只写暧昧氛围、战后陪伴、称呼变化、沉默靠近、旁人调侃、房间误会等合规内容。
- 不写露骨过程。

## 4. 剧情功能

### 主线功能

1. **地球现实侧视角承载者**
   - 林清雪是读者观察苏念现实变化的重要窗口。
   - 她从同学角度见证苏念由低评召唤师成长为多界核心。
   - 她能提出普通地球职业者会关心的问题：苏念的召唤物到底是什么？异界存在是否可信？官方会怎样处理？地球防线怎么办？

2. **职业高考与学院篇核心搭档**
   - 她参与职业觉醒、高考小秘境、学院训练、资源竞争。
   - 她让前期校园与学院内容不只是苏念单人打脸，而有现实同龄人的并肩和情感反馈。
   - 她可在苏念隐藏界域行者身份时承担“知道一点但不全知道”的角色。

3. **江海城市危机参与者**
   - 城市怪物潮、旧港区裂隙、镜湖小秘境异常等事件中，她可以作为江海一中优秀学生参与救援和防守。
   - 她能在苏念、江海镇守府、同学群体之间形成联系。

4. **地球防线成长代表**
   - 中期人魔战争与大秘境防线中，林清雪代表地球年轻职业者的成长。
   - 她的进步证明地球并非只靠苏念和异界女主，地球自身也在适应游戏入侵。

5. **苏念现实身份锚点**
   - 当苏念不断被多界势力争夺、官方观察、异界女主牵引时，林清雪提醒他最初的身份与现实羁绊。
   - 她让苏念不是漂浮在多界之间的工具人，而是有来处、有同学、有城市、有地球归属的人。

6. **神明世界终局主题的非契约证明**
   - 神庭契约陷阱针对苏念与召唤女主的契约，林清雪作为非召唤物、非契约对象，可以从另一个角度证明“自愿选择”不只存在于召唤师契约中。
   - 她的终局站队强调：真正重要的不是印记本身，而是自由意志。

### 情感线功能

1. **慢热清纯线**
   - 与莉莉娅的开篇强暧昧形成节奏对照。
   - 林清雪的情感推进依靠现实陪伴、危机并肩、吃醋拉扯，而非契约同步快速推进。
   - 她适合穿插在战斗后、学院日常、房间误会、防线休整中制造细腻甜点。

2. **吃醋拉扯核心**
   - 她是最适合对莉莉娅、青璃、白九璃、克露露、诺雅产生情绪反应的角色。
   - 她吃醋时不恶意雌竞，而是自我怀疑、克制、追赶和偶尔失控。
   - 这种吃醋能让后宫修罗场更有现实感。

3. **男主情感底线检验者**
   - 林清雪不会因为苏念强大就无条件接受一切。
   - 她会问苏念是否信任异界女主，是否还信任地球同伴，是否把召唤关系当成责任。
   - 她帮助读者确认苏念不是只顾收女和变强的人。

4. **现实陪伴与归属感**
   - 她能在苏念灵能透支、战后疲惫、被官方质疑时提供现实侧陪伴。
   - 她不需要强烈撩拨，只需一句“你还回来就好”就能形成独特甜点。

5. **后宫关系中的清流与压力点**
   - 她不像莉莉娅那样主动，也不像白九璃那样会玩弄话术。
   - 正因如此，她的沉默和委屈更容易让苏念认真对待。
   - 她能推动苏念学习平衡后宫关系，而不是让所有女主都围着他无条件转。

### 势力线功能

1. **江海一中与职业高考线**
   - 林清雪是江海一中异能班重点学生，可连接班主任、同学竞争者、考务组。
   - 她的优秀与苏念低评形成开局对比，也方便后续共同逆袭。

2. **江海镇守府线**
   - 她作为本地优秀学生，可在江海镇守府眼中具备培养价值。
   - 她与苏念一起经历小秘境异常后，可成为官方调查苏念异常表现的重要旁证。

3. **星海学院线**
   - 她可与苏念一起进入星海学院或训练营，延续现实侧互动。
   - 学院资源分配、训练积分、学生竞争中，她能提供同龄人视角与情感拉扯。

4. **华夏镇守体系线**
   - 中期她作为成长起来的地球职业者进入防线任务。
   - 她可以参与大秘境防线、城市安全区支援、学生队伍保护。
   - 她与苏念的关系也能让官方更容易理解苏念并非脱离地球秩序的危险变量。

5. **多界会盟线**
   - 后期林清雪可作为地球年轻一代职业者代表陪同苏念出席部分会盟或防线会议。
   - 她不代表最高权力，但代表“地球人如何看待多界联盟”。

### 可制造冲突

1. **低评后的态度差异冲突**
   - 同学嘲笑苏念，林清雪站在较温和一边，引发同学议论。
   - 她对苏念的关心可能被竞争者误读为偏袒。

2. **莉莉娅身份误会**
   - 林清雪看到莉莉娅与苏念的亲密氛围，却不知道完整契约过程。
   - 她既吃醋又担心苏念是否被深渊存在影响。

3. **信息隐瞒冲突**
   - 苏念不能公开界域行者身份，林清雪察觉他隐瞒。
   - 她理解安全需要，但情感上会受伤。

4. **战斗定位冲突**
   - 苏念习惯保护林清雪，林清雪不愿只当被保护者。
   - 她可能在某次防线任务中过度逞强，导致危机和关系争执。

5. **官方立场冲突**
   - 地球官方质疑异界女主时，林清雪夹在地球职业者身份和对苏念的信任之间。
   - 她必须做出选择：是保持沉默，还是替苏念作证。

6. **后宫修罗场冲突**
   - 青璃称呼“剑主”、莉莉娅调侃、白九璃拱火、诺雅数据占有欲，都能刺激林清雪。
   - 冲突应以轻喜剧、误会、拉扯为主，不写恶性雌竞。

7. **神明世界信任冲突**
   - 神庭可利用圣痕幻象或神罚副本放大林清雪的不安，让她以为自己永远追不上苏念。
   - 苏念需要用行动而非空话证明她的位置不可替代。

### 可提供收益

1. **战斗收益**
   - 冰霜控制、减速、局部防护、限制怪物潮推进。
   - 可与苏念召唤女主形成战术配合：莉莉娅精神控场后，林清雪冰封关键敌人；青璃近战切入时，她负责封路与保护后排；诺雅火力覆盖时，她限制敌人移动。

2. **现实情报收益**
   - 了解江海一中、同学、学院、考务组、学生职业者圈层信息。
   - 可帮助苏念判断地球年轻职业者的真实处境。

3. **官方信任收益**
   - 林清雪作为地球优秀学生和职业者，可以为苏念提供现实侧背书。
   - 她的证词、并肩经历、战场表现能缓解部分官方对苏念的疑虑。

4. **情感稳定收益**
   - 苏念跨界压力极大，林清雪提供现实侧情绪稳定。
   - 她能让苏念记住自己不是神明棋盘上的工具，而是地球人。

5. **团队平衡收益**
   - 主角团中异界成员众多，林清雪能帮助她们适应地球规则、学院制度、防线纪律。
   - 她在日常中可以担当“正常人吐槽役”，让异界女主的文化差异更有喜剧效果。

6. **主题收益**
   - 她证明主角团不是只有召唤契约关系，也有现实自愿同行。
   - 她能让“自愿选择”主题从契约扩展到普通人际关系。

### 可能带来的麻烦

1. **被敌人利用为现实侧软肋**
   - 黑契会、神庭暗线、敌对学生或跨界追兵可能盯上林清雪，以此逼苏念暴露能力。
   - 她必须成长，避免长期成为被绑架工具。

2. **情绪压抑导致误会升级**
   - 林清雪不擅长直接表达，容易把吃醋和委屈压在心里。
   - 如果苏念一直回避，可能导致阶段性冷战或战场配合失误。

3. **过度追赶导致受伤**
   - 她为了证明自己不拖后腿，可能在学院训练或防线任务中过度消耗。
   - 这能制造危机，也推动苏念认真承认她的战斗意志。

4. **官方审查压力**
   - 她与苏念关系密切，可能被镇界司或学院导师询问苏念异常召唤能力。
   - 她需要在保护苏念秘密与遵守地球秩序之间寻找平衡。

5. **修罗场引发团队日常混乱**
   - 林清雪吃醋、白九璃拱火、莉莉娅调侃，会造成大量轻喜剧场面。

## 5. 成长与状态

### 初始状态

年龄与合法边界：

- 十八岁以上，成年女性。
- 地球人类，类人智慧对象。
- 所有情感与亲密氛围均以成年、自愿为前提。
- 不是召唤物，不存在被契约强制绑定的问题。

身份状态：

- 江海一中异能班重点学生。
- 职业觉醒前后被学校和老师看好。
- 初始实力为一阶觉醒者，天赋稳定。
- 家庭背景可保持普通或小康，不必设计强大家族，避免抢走苏念底层逆袭焦点。

心理状态：

- 自律、认真，对未来有明确目标。
- 对苏念有好感但未明说。
- 在苏念低评后担心他，却不擅长表达。
- 对游戏入侵时代的残酷有认知，但还没有真正经历大规模战争。

战斗状态：

- 初始职业建议为冰系剑士或冰霜战法。
- 擅长基础剑术、冰霜减速、短时间控场。
- 防御与爆发都未成熟，容易在高压副本中灵能消耗过快。
- 初期不如莉莉娅等异界女主惊艳，但胜在成长稳定、配合性强。

关系状态：

- 与苏念是同学兼潜在搭档。
- 与莉莉娅初见时带有震惊、警惕和微妙吃醋。
- 与其他异界女主初期会有距离感，后续逐渐互相承认。

成长重点：

## 十三、事件链

### 事件1：第一章契约后氛围强钩子

- 开篇直接呈现结果：
  - 苏念灵能透支。
  - 房间残留深渊香气。
  - 契约印记发烫。
  - 莉莉娅慵懒调侃苏念。
  - 第一永久契约提示出现。
- 正文只写暧昧氛围、战后状态、称呼变化与系统提示，不写露骨过程。
- 随后倒序回到职业觉醒。

### 事件2：职业觉醒低评

- 苏念觉醒为召唤师。
- 检测仪只能识别普通召唤波动，无法识别界域行者。
- 评级结果：不稳定召唤系，潜力低。
- 同学议论，部分人嘲笑。
- 林清雪没有嘲笑，默默关心。

### 事件3：资源困境与隐藏职业触发

- 苏念回家计算普通召唤师资源需求，发现自己几乎走不通。
- 他尝试临时契约，失败。
- 听见莉莉娅来自深渊的危机呼唤。
- 苏念吐槽隐藏能力像“专门听见异世界美人求救的离谱职业”，但正式职业名仍为界域行者。

### 事件4：莉莉娅危机

- 莉莉娅为成年魅魔，夜绯宫贵女。
- 她被敌对派下了魅毒/情毒，并推入真名契约陷阱。
- 她理智濒临失控，却仍强烈祈求不被夺走真名与自由。
- 苏念被界域行者送到她身边，成为外界变量。
- 两人完成高强度契约同步，压下毒性与真名污染。
- 危机解除后，莉莉娅清醒确认、自愿认可苏念，缔结第一永久契约。

### 事件5：返回地球与莉莉娅伪装

- 苏念返回地球。
- 莉莉娅可短时降临，但受灵能与锚点限制。
- 外界以为她是高阶恶魔系召唤物。
- 苏念知道她是真实异界女主，必须保护她的身份。

### 事件6：高考小秘境异常

- 镜湖高考小秘境出现异变，考生陷入异常怪物潮。
- 苏念首次正式召唤莉莉娅联动：
  - 精神控场。
  - 情绪扰乱。
  - 恶魔契约束缚。
- 苏念救下林清雪与部分同学。
- 低评召唤师完成打脸。

### 事件7：秘境核心与圣痕残纹

- 苏念击败异常精英怪，获得梦魇晶与小秘境核心碎片。
- 秘境深处出现不属于深渊、不属于地球的白金圣痕残纹。
- 莉莉娅对其感到不适，提示深渊危机背后也有外来规则污染。

### 事件8：旧港裂隙与黑契会

- 唐绯雨提供旧港裂隙中的召唤媒介线索。
- 苏念调查时发现黑契会江海支部正在非法契约智慧异界对象。
- 秦牧野试图夺取莉莉娅契约。
- 苏念与莉莉娅联手击破黑契会小据点。
- 黑契会禁忌契约卷残页暴露“神眷契约术”雏形。

### 事件9：官方与学院关注

- 江海镇守府复核苏念战绩。
- 星海学院招生组关注他。
- 方明远承认觉醒中心评级可能误判。
- 赵临江上报苏念异常，但暂未掌握界域行者真相。
- 林清雪对莉莉娅身份产生更多疑问。

### 事件10：卷末青璃钩子

- 苏念整理战利品时，小秘境核心碎片中嵌着一枚剑冢残玉。
- 残玉发烫。
- 一道清冷、古老、压抑的声音传来：“你是来夺剑，还是来听剑说话？”
- 苏念意识到下一次界域行走即将出现。
