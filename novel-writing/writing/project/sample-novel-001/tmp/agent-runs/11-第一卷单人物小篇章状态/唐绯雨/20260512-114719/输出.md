## 03｜莉莉娅适应地球与林清雪吃醋（第11—15章）

### 起点状态

- 唐绯雨本人未正式登场。
- 她作为“下城区星晶黑市”中有名的情报商，被周远山或同学口中提及。
- 她掌握黑市召唤媒介、梦魇晶、星晶流通价格，对异常收购动向敏感。
- 立场仍为表面中立，只按价格与规矩办事。

### 终点状态

- 唐绯雨的名字进入苏念视野，但双方尚未见面。
- 她所在渠道传出“有人高价收购能稳定智慧恶魔召唤对象的媒介”的消息。
- 她开始注意到近期媒介收购不寻常，判断背后不像普通召唤师需求。

### 实力/力量体系变化

- 无战斗表现。
- 她的“能力”主要体现为情报嗅觉与黑市资源调度，不涉及正面力量升级。

### 情感/关系变化

- 与苏念：尚未建立直接关系，仅被苏念间接听闻。
- 与莉莉娅：未见面，但她的情报将间接关联莉莉娅的锚点与身份风险。
- 与黑契会：尚无正面冲突，但她对破坏黑市规矩的强制契约买家产生警惕。

### 掌握信息变化

- 已知/疑似：
  - 黑市近期有人收购梦魇晶、恶魔契约媒介。
  - 买家目标可能不是普通召唤物，而是“高智慧恶魔契约对象”。
  - 这类收购可能牵涉强制契约或追踪术。
- 未知：
  - 苏念就是目标召唤师。
  - 莉莉娅的真实身份与第一永久自愿契约细节。
  - 镜湖白金残纹的来源。

### 资源/伤势/污染变化

- 持有/可调动：星晶、召唤媒介、梦魇晶行情情报。
- 伤势：无。
- 污染：未直接接触圣痕残纹或契约污染。

### 下一篇章承接

- 作为黑市情报源被提前埋下。
- 后续高考异常与黑契会线索出现后，她的情报价值会上升。

## 07｜二阶突破与江海重点关注（第33—39章）

### 起点状态

- 唐绯雨已通过黑市渠道听闻镜湖异常、高智慧恶魔契约对象、梦魇晶流动等消息。
- 她判断旧港裂隙内的媒介收购与黑契会有关。
- 她不想黑契会继续破坏黑市规矩，因此愿意把部分线索卖出或递出。

### 终点状态

- 唐绯雨通过黑市情报渠道正式向苏念传递旧港裂隙线索。
- 她未完全站队，但把苏念列入“可观察的长期合作对象”。
- 她确认苏念大概率不是普通低评召唤师，而是有资格接触更深灰色情报的人。

### 实力/力量体系变化

- 无战斗升级。
- 情报能力表现提升：
  - 能锁定梦魇晶与恶魔契约媒介的异常买家。
  - 能把旧港裂隙、黑契会、镜湖异常初步串联。
  - 能在不直接暴露自身的情况下传话给苏念。

### 情感/关系变化

- 与苏念：
  - 从互不相识 → 建立第一次间接情报联系。
  - 唐绯雨对苏念的态度是试探、观察，不白给信任。
  - 她认为苏念若能从旧港活下来，才值得长期合作。
- 与莉莉娅：
  - 未直接见面。
  - 通过黑市传闻知道苏念身边可能有高智慧恶魔契约对象。
  - 对这类契约对象的存在保持谨慎和兴趣。
- 与黑契会：
  - 警惕加深。
  - 不喜欢黑契会用强制契约搅乱灰市，因为这会引来官方清算。

### 掌握信息变化

- 已掌握：
  - 旧港裂隙有人高价买梦魇晶与恶魔契约媒介。
  - 买家与黑契会有关。
  - 对方似乎在寻找镜湖事件中的召唤师。
  - “高智慧恶魔契约对象”已经成为黑契会目标。
- 推测：
  - 苏念可能就是黑契会寻找的目标。
  - 旧港裂隙中存在非法契约仪式或追踪媒介。
- 未掌握：
  - 界域行者真相。
  - 莉莉娅自愿永久契约的具体过程。
  - 圣痕残纹背后的神庭线索。

### 资源/伤势/污染变化

- 资源：
  - 持有旧港裂隙情报。
  - 掌握梦魇晶、星晶、恶魔契约媒介的黑市流向。
  - 可作为后续灰色情报渠道。
- 伤势：无。
- 污染：未直接接触旧港祭坛污染。

### 下一篇章承接

- 她提供的旧港裂隙情报直接推动苏念进入篇章08。
- 唐绯雨暂不亲自下场，以情报商身份保持安全距离。
- 若苏念打掉黑契会旧港据点，她会重新评估与苏念长期合作的价值。

## 08｜旧港裂隙与青璃呼唤（第40—45章）

### 起点状态

- 唐绯雨已把旧港裂隙、梦魇晶收购、恶魔契约媒介、黑契会买家等线索递给苏念。
- 她在黑市侧观察旧港局势，没有直接参与战斗。
- 她判断苏念若能处理旧港事件，说明其价值远高于普通召唤师。

### 终点状态

- 黑契会旧港据点被重创后，唐绯雨确认自己的判断：苏念不是普通低评召唤师。
- 她愿意保留与苏念的合作渠道，但仍不会无偿提供情报。
- 她对苏念、莉莉娅、旧港残页、圣痕残纹的关注度提升。

### 实力/力量体系变化

- 无正面战力变化。
- 情报地位变化：
  - 从普通黑市传闻提供者 → 苏念可用的灰色情报节点。
  - 她对“黑契会媒介收购链”的判断被旧港事件验证，信誉上升。

### 情感/关系变化

- 与苏念：
  - 关系从“第一次传话试探”升级为“可保留合作渠道”。
  - 她对苏念产生现实兴趣：这个年轻人有胆量、有底线，也能带来高价值情报交换。
  - 仍保持商人距离，不主动卷入主角团。
- 与莉莉娅：
  - 未直接建立情感关系。
  - 通过旧港结果进一步确认莉莉娅不是普通恶魔系召唤对象。
  - 对莉莉娅相关情报的价格与风险判断同步提高。
- 与黑契会：
  - 敌意不公开化，但实际更不愿配合黑契会。
  - 黑契会旧港失败后，她会更小心处理相关残党交易。

### 掌握信息变化

- 已确认：
  - 苏念确实与旧港黑契会目标有关。
  - 旧港裂隙存在非法契约仪式。
  - 黑契会在收购梦魇晶、恶魔契约媒介，并寻找高智慧恶魔契约对象。
  - 苏念具备重创黑契会据点的实力与胆量。
- 可能新增判断：
  - 苏念背后的契约方式与普通召唤师不同。
  - 莉莉娅具备完整智慧和强烈自主性。
  - 旧港事件牵涉的不只是黑市交易，还有更深层污染线索。
- 仍未知：
  - 剑冢残玉与青璃呼唤。
  - 神庭暗线真相。
  - 苏念隐藏职业本质。

### 资源/伤势/污染变化

- 资源：
  - 保留黑市渠道、人脉、星晶与召唤媒介行情。
  - 旧港线索已消耗一部分，但换来对苏念价值的判断。
- 伤势：无。
- 污染：
  - 未直接沾染旧港契约污染。
  - 对被污染媒介的警惕提升，后续交易会更谨慎。

### 下一篇章承接
