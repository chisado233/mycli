# 作品企划

## 作品名

暂定：**《全球游戏入侵：我的魅魔召唤物全是异界女王》**

备选短名：**《我的魅魔召唤物全是异界女王》**

标题保留“全球/游戏入侵/魅魔/召唤/女王”吸量方向，突出男主召唤的不是普通怪物，而是来自不同异世界的真实女主、玩家、公主、剑灵、机娘等核心卖点。

## 篇幅规划

- 预计篇幅：180万—250万字，游戏入侵 + 多界召唤 + 后宫爽文。
- 收女/关系推进频率：前期节奏偏快，约十几章推进一位核心成年类人/智慧女主加入或确认明显暧昧关系；每位女主加入都要绑定一种世界体系与一种战斗联动。
- 亲密暗示/事后节点频率：常规约每3章一次，长战斗、连续副本、跨界事件可顺延；只写事后氛围、称呼变化、契约同步率、房间细节、旁人调侃等，不写露骨过程。
- 战斗频率：前30章每3—5章一次副本、怪物潮、职业考核或召唤实战；中期每8—12章一个完整跨界事件。
- 日常频率：约20%—25%，重点写异世界女主适应地球游戏化生活、同居据点、争宠、修罗场和白九璃拱火。
- 情感频率：约25%，每位女主都要有偏爱、独占欲、关系确认或后宫位阶变化。
- 探索/副本/比赛频率：前期以地球新手副本、职业高考、城市安全区争夺为主；中后期扩展到多界副本、深渊裂隙、修仙秘境、机甲战场、跨界版本活动。

## 目标平台/目标读者

适合番茄、七猫、飞卢、起点轻爽男频等平台。

目标读者：

- 喜欢全球游戏入侵、职业觉醒、等级副本、怪物潮、转职升级的男频读者。
- 喜欢召唤师、异世界玩家、真实女主降临、多职业联动战斗的爽文读者。
- 喜欢后宫、多女主、魅魔、吸血鬼公主、剑灵萝莉感、机娘、狐娘等属性的读者。
- 喜欢女主主动撩、修罗场、甜暧昧、轻喜剧后宫日常的读者。

## 核心卖点

1. **多重异世界同步游戏入侵**  
   地球和多个异世界同时被游戏化灾变卷入，所有世界都出现职业、等级、副本、怪物潮、装备、技能、转职和玩家/原住民阵营。地球不是唯一主场，而是多界版本更新中的一环。

2. **男主是特殊召唤师，不只召怪物**  
   普通召唤师只能召唤异界生物、怪物、兽宠；林烬特殊在于能召唤来自不同异世界的真实女主，包括异世界玩家、原住民强者、公主、剑灵、机娘等。她们拥有独立人格、原世界身份和自身主线。

3. **羁绊联动战斗机制**【新增项】  
   林烬与召唤女主缔结“界约羁绊”。羁绊提升后，可触发不同联动形态：职业借用、技能同步、武器化、剑灵附体、恶魔契约技能、血族夜战模式、机甲武装合体等。男主不站后排挂机，而是借助女主体系切换战斗风格，亲自近战、爆发、控场或装甲化作战。

4. **魅魔吸量 + 后宫修罗场常驻**  
   莉莉娅作为第一位深渊魅魔女主，负责开局吸量、主动暧昧、精神控场和后宫基调；白九璃拱火，艾瑟琳雌小鬼挑衅，剑灵萝莉感争宠，机娘冷静但占有欲强，形成持续修罗场。

## 主角团

- 主角：**林烬**  
  18岁以上，地球游戏入侵初期觉醒为召唤师。表面评级为“不稳定召唤系”，实则拥有唯一型天赋“界约召唤”【新增项】，可召唤异世界玩家/真实女主，并通过羁绊同步获得职业、技能、武器、形态或战斗风格。性格嘴贫、聪明、会钻规则，战斗中冷静，有底线，不奴役女主，关系建立在自愿契约与并肩作战上。

- 主要女主/队友：  
  - **莉莉娅**：来自深渊世界的成年魅魔，真实异世界存在。御姐、主动、坏女人感，擅长精神控场、幻术、情绪破防、恶魔契约技能；第一位召唤女主，负责魅魔吸量和后宫基调。  
  - **艾瑟琳**：来自深渊世界的成年吸血鬼公主。雌小鬼感，嘴硬挑衅、表面嚣张，实际容易被林烬反制破防；能力为血刃、夜战、自愈、贵族血统压制。  
  - **青璃**【新增项】：来自修仙世界的仙剑剑灵，成年合法，外表娇小/萝莉感。可让林烬进行剑灵附体、御剑、近战剑修化，补足男主亲自近战爽点。  
  - **诺雅**：来自机甲世界的成年机娘。可变身机甲/装甲，与林烬进行武装合体，提供火力覆盖、飞行机动、数据分析和战场预测。  
  - **白九璃**：来自修仙世界的成年白毛狐娘。拱火绿茶定位，擅长幻术、潜入、情报交易、资源牵线；日常负责茶言茶语、挑拨修罗场和装无辜。  
  - **林清雪**：地球人类女主，18岁以上，清纯含蓄型，同学/搭档定位。前期共同经历职业考核、副本和城市怪物潮，情感推进慢于异世界女主，负责现实侧羁绊与吃醋拉扯。  
  - **希尔薇**：来自精灵世界的成年圣射手【新增项：明确来源世界】。远程狙击、侦查、自然结界、秘境探索，适合副本斥候与远程火力位。  
  - **伊芙琳**：来自圣域世界的成年光辉圣女【新增项：明确来源世界】。治疗、净化、增益、驱散污染，负责团队续航和深渊副本克制位。  
  - **影弥**：来自暗影世界的成年影魔刺客【新增项：明确来源世界】。潜伏、暗杀、反召唤、背刺敌方玩家，偏中后期登场。

## 故事主线

多重异世界同时遭遇游戏化灾变，地球也被卷入版本降临。职业觉醒、副本入口、怪物潮、安全区、等级榜、装备掉落和转职体系迅速改变现实秩序。林烬在职业考核中觉醒召唤师，却因无法稳定召唤普通怪物被判定为低潜力。

危机中，他召唤出来自深渊世界的成年魅魔莉莉娅。众人以为那只是高阶召唤兽，林烬却发现她是拥有记忆、身份和原世界危机的真实异世界存在。两人通过“界约羁绊”完成首次联动，林烬借用恶魔契约技能与精神控场能力，逆转新手副本和职业考核。

前期围绕地球职业高考、新手副本、城市怪物潮和安全区资源争夺展开。林烬逐步召唤莉莉娅、青璃、白九璃等女主，打破“召唤师只能站后排”的偏见，形成多形态战斗路线：魅魔控场、剑灵近战、狐娘幻术、吸血鬼夜战、机娘装甲化。

中期揭示多界游戏入侵并非单一灾难，而是多个世界被同一游戏规则强行并轨。各世界玩家、原住民、BOSS、阵营任务开始互相进入对方副本。林烬既要在地球建立核心队伍，也要帮助女主们解决原世界版本危机。

后期主角团成长为跨界核心势力，林烬依靠界约召唤和羁绊联动，整合深渊、修仙、机甲、圣域、精灵等世界资源，对抗游戏入侵背后的高维系统和跨界灾厄，建立以自愿契约、共同战斗和后宫羁绊为核心的新秩序。

## 文风

- 快节奏游戏入侵爽文，开篇迅速给出职业觉醒、魅魔召唤、首次联动、打脸和暧昧卖点。
- 主角嘴贫但不油腻，聪明、有底线，战斗时敢亲自上场，不做纯后排指挥。
- 战斗写清职业、等级、技能、装备、副本机制，以及男主与女主的羁绊联动形态。
- 后宫日常甜、暧昧、轻修罗场；女主主动撩拨、争宠、吃醋、白九璃拱火常驻。
- 莉莉娅突出魅魔御姐和控场；艾瑟琳突出雌小鬼嘴硬和破防反差；青璃突出成年合法萝莉感与剑灵黏人；诺雅突出机娘冷静、装甲合体和隐藏占有欲。
- 亲密关系只做隐晦表达，不写露骨过程；常规约三章一次事后/暗示节点，战斗副本可顺延。

## 雷点

- 所有亲密关系必须限定成年、自愿、类人/智慧对象。
- 萝莉感女主必须明确成年合法，禁止未成年人亲密关系。
- 女主不是普通召唤兽、宠物、怪物奖励、技能图标，必须有独立人格和原世界身份。
- 禁止强迫、奴役、洗脑式亲密关系。
- 禁止动物化性内容，白九璃只能按成年类人智慧狐女处理。
- 正文不写露骨器官、动作和具体过程。
- 后宫不能长期暧昧不推进，也不能只收不写。
- 学院/高考/职业考核只作为前期模块，不能长期困在校园刷排名。
- 禁止沿用影门事务所、命运债务、官方备案、事务所案件流。
- 禁止把男主写成只站后排喊技能的普通召唤师。

## 宏观调控

- 开篇三章：地球游戏入侵，林烬参加职业觉醒/考核，被判定为低潜力召唤师；新手副本失控，莉莉娅作为真实深渊魅魔被召唤；林烬与她完成首次界约联动，靠精神控场和恶魔契约技能打脸逆袭。
- 十章内：完成一次职业考核或城市新手副本；确立“普通召唤师召怪物，林烬召唤异世界真实女主”的差异；林清雪建立现实侧羁绊；埋下青璃、白九璃、艾瑟琳登场伏笔。
- 首卷：完成高考/安全区/城市副本资源争夺，林烬获得核心训练营或官方重点关注资格；莉莉娅稳定第一女主和后宫基调；青璃或白九璃至少一人正式加入；明确羁绊联动的成长路线。
- 第二阶段：艾瑟琳以深渊吸血鬼公主身份加入，重点写雌小鬼挑衅、夜战血刃、自愈突袭，以及被林烬反制后的破防反差；诺雅登场后开启机甲装甲合体爽点。
- 中长期：约十几章推进一位核心女主加入或关系升级；每卷至少一个大战斗爽点、一个后宫关系节点、一个女主原世界危机伏笔。主线从地球副本扩展到多界版本战场、跨界玩家冲突和游戏入侵真相。
