# 伊芙琳

## 1. 基础身份

### 姓名

伊芙琳

### 人物定位

女主 / 队友 / 治疗辅助 / 净化核心 / 圣域世界代表 / 反神明线关键成员。

伊芙琳是主角团中承担“治疗、净化、增益、驱散污染、信仰真相揭露”的成年光辉圣女。她不是普通牧师型工具人，而是圣域世界信仰秩序崩塌后的关键见证者与反神明阵线的重要证明：真正的光辉不是服从神明世界的神谕，而是守护无辜者、守护自由意志。

### 所属势力

初始所属：圣域世界 / 光辉圣庭 / 光辉圣女院。

关联势力：光辉圣庭分歧派。

后续归属：苏念主角团 / 多界反神明联盟 / 圣域反伪神谕派。

伊芙琳名义上是光辉圣庭培养出的圣女，掌握治疗、祝福、净化、圣痕驱散等能力。她所在的圣域世界原本信奉“光辉、治疗、守护”，但在多界游戏化灾变后，神庭秩序以“更高光辉”的名义渗透圣域，试图将原本的守护信仰改造成服从神明世界的神权工具。

### 初始身份

成年光辉圣女，圣域世界光辉圣庭中最受民众信任的年轻圣职者之一。

她在圣域世界拥有极高声望：

- 曾多次在灾变后的疫病区、污染区、怪物潮后方救治平民。
- 被认为是“光辉意志在人间的温柔回声”。
- 掌握圣域净化术、群体祝福、污染驱散、伤势恢复等能力。
- 在圣女院中接受过完整神学、战地救护、圣术仪轨和审判规章训练。
- 对光辉信仰曾经非常虔诚，但她的虔诚指向“守护生命”，而不是盲目服从权威。

初登场时，伊芙琳被卷入圣域审判事件。她拒绝执行大主教阿德里安推动的“净化审判”，不愿对被圣痕污染却仍保有人性的无辜者降下毁灭性圣术，因此被软禁在圣女院深处，成为圣庭内部两派争夺的象征。

### 隐藏身份/秘密

伊芙琳隐藏着三层秘密：

1. **她已经察觉“神谕”异常**  
   圣域近年出现的部分神谕，并不符合原本光辉信仰的温度。真正的光辉圣术会安抚痛苦、驱散污秽、保护生命；而新降下的所谓“更高神谕”却要求圣职者标记、审判、清洗、剥夺自由意志。伊芙琳在一次净化仪式中发现，所谓神谕残留的白金圣痕与真正光辉圣术的气息不同，反而与多界出现的神庭圣痕污染同源。

2. **她拥有罕见的“无垢光辉”适性**  
   伊芙琳的圣术能在不破坏灵魂的前提下剥离圣痕污染。神庭秩序和圣庭投靠派都想利用她：前者想把她改造成神庭圣痕净化体系的圣女样板，后者想用她的名望安抚圣域民众。她本人最初并不知道自己是破解圣痕污染的重要钥匙，只以为自己比其他圣职者更能听见受难者的祈祷。

3. **她的呼唤不是求生，而是求证**  
   伊芙琳发出呼唤时，并不是单纯希望被救走。她真正的心愿是：希望有人陪她证明，自己坚持守护无辜者并不是背叛光辉；希望有人帮她确认，圣庭正在接受的神谕究竟是救赎还是控制。苏念响应的不是一个等待拯救的软弱圣女，而是一个在信仰崩塌边缘仍不愿放弃真相的成年人。

### 登场阶段

中后期登场，主要在第五阶段“各世界混战”中正式加入。

登场节奏：

- 前期：通过圣痕残纹、圣域净化石、神庭圣光污染等伏笔提前铺垫。
- 第五阶段：圣域世界入口开启，圣庭部分高层接受神庭册封，伊芙琳因拒绝净化审判被软禁，并向界域之外发出呼唤。
- 苏念介入圣域审判事件，以“被光辉异象牵引而来的外界见证者/临时守护者”身份协助她验证伪神谕真相。
- 伊芙琳在确认神庭秩序并非真正光辉后，自愿与苏念缔结召唤师契约，加入主角团。
- 第六阶段后成为多界会盟中的净化核心。
- 第七阶段在神门降临时公开揭露圣域伪神谕。
- 终局负责净化神庭圣痕源头，是反神明线的重要成员。

## 2. 人物魅力

### 外貌与气质

伊芙琳是明确成年的类人智慧女性，拥有圣域世界圣女特有的温柔、庄重与治愈感。

外貌特征：

- 银金色长发，常以圣女院传统发饰束起，战斗或逃亡时会有几缕散落，形成“圣洁外表下的脆弱动摇”反差。
- 眼眸呈浅金色，施展净化圣术时会浮现柔和光纹；信仰动摇或情绪剧烈时，眼底光辉会短暂黯淡。
- 穿着白金色圣女礼衣，外层为便于战场行动的轻甲式圣袍，既保留圣职者仪式感，又适合副本和防线作战。
- 身上常带有圣域净化石、祈祷珠、治愈铃和旧版光辉圣典。
- 气质温柔、安静、克制，不以强势压人，但她真正坚持原则时极难动摇。

她给人的第一印象不是锋利，而是“靠近她就会安静下来”。在主角团中，她像战后营地里唯一稳定燃着的灯，能治疗伤口，也能安抚被圣痕污染扰乱的精神。

### 性格标签

- 温柔治愈
- 圣洁克制
- 虔诚但不盲从
- 信仰崩塌后仍坚持善意
- 外柔内刚
- 高共情能力
- 对无辜者极度保护
- 不擅长修罗场但会认真回应感情
- 反神明线觉醒者
- 净化圣痕核心

伊芙琳的温柔不是软弱。她可以低声安抚伤者，也可以站在审判台前拒绝大主教的命令。她不喜欢争吵，却会在原则问题上寸步不让。

### 反差点

1. **圣女身份与反神立场的反差**  
   她看似最该服从神谕，却是最早质疑“神庭光辉”的圣域核心人物之一。她的反叛不是出于仇恨，而是出于更纯粹的信仰：如果神要求她伤害无辜者，那那个神就不值得信奉。

2. **温柔外表与强硬底线的反差**  
   平日里伊芙琳说话轻柔，甚至不习惯打断别人；但当阿德里安要求她对平民执行净化审判时，她会当众放下圣杖，宁愿被剥夺圣女资格也不肯动手。

3. **治愈者与真相揭露者的反差**  
   她的能力看似只适合后方续航，实际上在神庭秩序主线中极其关键。她能分辨真正光辉与伪圣光的差别，能净化圣痕，能证明神庭所谓“救赎”本质上是控制。

4. **信仰崩塌后的依赖与重新站起的反差**  
   她在确认圣庭被神庭冒名污染后会出现低谷，短暂失去方向，甚至怀疑自己过往祈祷是否都被利用。但苏念没有嘲笑她的信仰，而是陪她验证真相。之后她会从“神的圣女”成长为“自己选择道路的光辉圣女”。

### 读者记忆点

- 被软禁在圣女院深处，仍偷偷为受污染的平民净化痛苦。
- 在审判台上拒绝执行“净化审判”，一句“光辉若要我伤害无辜，那便不是我所信奉的光辉”。
- 初见苏念时没有立刻要求他救自己，而是请求他陪她验证真相。
- 能一眼分辨神庭伪圣光与真正圣域光辉的差异。
- 战后为全员治疗时，安静坐在营地中央，指尖光辉像雪一样落下。
- 信仰崩塌后靠在圣女院破碎的彩窗下，低声问苏念：“如果我一直祈祷的方向错了，我还能继续救人吗？”
- 终局中亲手净化圣痕源头，将神庭夺走的“圣光”从控制与审判中解放出来。

### 角色卖点

伊芙琳的核心卖点是“圣女反神明”。

她不只是团队奶妈，而是全书神庭真相线的关键人物之一：

- **战斗卖点**：治疗、净化、增益、驱散污染、群体庇护，补足主角团续航与反污染能力。
- **主线卖点**：揭露圣域伪神谕，证明神庭秩序冒名“更高光辉”污染圣庭。
- **情感卖点**：温柔圣女在信仰崩塌后得到苏念尊重与陪伴，从依赖到重新选择道路。
- **反差卖点**：最圣洁的人走上反神道路，但她反的不是善意与信仰，而是披着神名的控制。
- **团队卖点**：让主角团从高爆发、多体系战斗队伍，升级为能长期打神罚副本和污染战争的完整队伍。
- **终局卖点**：在神庭契约陷阱和圣痕污染中，她是净化神庭规则污染的重要支点。

## 3. 与主角关系

### 初始关系

伊芙琳与苏念的初始关系是“呼唤者与回应者”“圣域审判事件中的被软禁圣女与外界见证者”。

她并不一开始就把苏念当成恋爱对象，也不会因为他突然出现就完全托付。她最初对苏念的态度是谨慎、礼貌、带着一丝无法掩饰的希望。

初见时的关系基调：

- 伊芙琳身处圣女院软禁状态，外界正在准备对一批被圣痕污染的无辜者执行净化审判。
- 她已经发现神谕异常，却缺少足够证据，也无法离开圣庭核心区。
- 苏念通过界域行者能力响应她的呼唤，被圣域光辉异象牵引到她身边。
- 对伊芙琳而言，苏念不是“天降救世主”，而是她求证真相时唯一不属于圣庭体系的变量。
- 对苏念而言，伊芙琳不是被抽取来的圣女召唤物，而是一个成年、独立、仍在做艰难选择的异世界女性强者。

### 关系吸引点

1. **苏念尊重她的信仰，而不是嘲笑她的崩塌**  
   伊芙琳最怕的不是死亡，而是发现自己一生信奉的光辉被利用。苏念没有用轻佻态度否定她过去的信仰，而是告诉她：被利用的是神谕，不是她救过的人；错的是披着光辉外衣的神庭，不是她选择守护生命的心。

2. **苏念坚持自愿契约，与神庭强制秩序形成鲜明对照**  
   圣域线中，伊芙琳亲眼看到神庭如何用圣痕标记人、用神谕命令人、用净化之名剥夺自由意志。苏念的召唤师契约恰好相反：必须认可、必须自愿、必须并肩作战。这让她逐渐意识到，苏念所代表的不是支配，而是选择。

3. **伊芙琳能治愈苏念队伍的伤，也能理解他的压力**  
   苏念一路承担多界契约者、战场核心、会盟纽带的压力，身边女主多但真正能安静听他说完疲惫的人不多。伊芙琳擅长倾听，她不会用激烈方式争抢存在感，却能在关键时刻给苏念稳定感。

4. **苏念能让她从“圣女职责”回到“伊芙琳本人”**  
   在圣庭中，伊芙琳常被称为圣女、象征、神谕容器，很少有人问她本人想不想、怕不怕、是否痛苦。苏念会把她当作有选择权的人，而不是圣域政治符号。

### 关系阻碍

1. **信仰与反神立场的心理撕裂**  
   伊芙琳不是一开始就能彻底反神。她曾经真心相信光辉，曾经依靠祈祷支撑自己救治伤者。即使发现神庭污染，她也需要时间分辨“真正的守护信仰”和“神庭秩序伪装的神谕”。

2. **圣庭内部压力**  
   大主教阿德里安倾向接受神庭册封，他会将伊芙琳与苏念的接触定性为异端污染，试图用圣域民众和圣女名望绑架她。伊芙琳越靠近苏念，越可能被圣庭投靠派指控为背叛信仰。

3. **神庭对伊芙琳的觊觎**  
   伊芙琳的无垢光辉适性能够净化圣痕，也能被神庭反向利用为“神圣净化样板”。神庭不会轻易放走她，会派圣痕审判官、神使投影或圣庭内鬼追捕她。

4. **主角团后宫氛围对她的冲击**  
   伊芙琳性格温柔克制，不擅长莉莉娅、白九璃、艾瑟琳那种直白或拱火式互动。她初入主角团时容易被修罗场气氛弄得不知所措，甚至认真思考“是否应该按照圣女礼仪避嫌”，反而引发轻喜剧。

5. **她对自身价值的怀疑**  
   信仰崩塌后，她一度会怀疑自己是不是只会治疗、只会祈祷，能否真正站在反神明战争前线。苏念和主角团需要通过战斗证明：净化者不是后方附属，而是反神明战场不可替代的核心。

### 关系变化路线

1. **求证阶段：礼貌信任**  
   伊芙琳请求苏念陪她验证神谕真相。她尊重苏念的外界身份，但仍保持圣女式克制。

2. **共犯阶段：共同违抗伪神谕**  
   苏念帮助她救下被圣庭判定为必须净化的无辜者。伊芙琳第一次主动违抗圣庭命令，两人成为圣域审判事件中的共同反叛者。

3. **崩塌阶段：脆弱依赖**  
   真相揭露后，她确认神庭秩序冒名污染圣域，短暂陷入信仰低谷。苏念陪她面对彩窗破碎、圣典被篡改、同僚背叛的现实。她开始把苏念视为能接住自己真实情绪的人。

4. **选择阶段：自愿契约**  
   伊芙琳不再以“圣女职责”行动，而是以“伊芙琳本人的选择”决定加入苏念。她自愿缔结召唤师契约，成为队友、伙伴与长期情感对象。

5. **治愈阶段：团队安定核心**  
   加入主角团后，她负责治疗伤势、净化污染、安抚精神，也逐渐融入后宫日常。她的亲近方式温柔含蓄，常通过战后治疗、祈祷陪伴、净化光辉表达关心。

6. **公开阶段：圣域反伪神谕代表**  
   神门降临时，伊芙琳公开站出来揭露伪神谕，宣布自己信奉的光辉不属于神庭秩序。她不再是被软禁的圣女，而是能代表圣域清醒派发声的反神明圣女。

7. **终局阶段：自愿契约主题回收**  
   神庭试图扭曲苏念与女主们的契约时，伊芙琳用净化圣光驱散圣痕污染，证明神明无法玷污建立在自愿与认可上的关系。

### 终局关系

终局时，伊芙琳是苏念主角团中的核心女主之一，也是圣域世界反神明新秩序的代表人物。

她与苏念的关系定位：

- 自愿缔结召唤师契约的长期队友。
- 后宫关系中的温柔治愈型女主。
- 苏念在神庭污染、精神疲惫、战后重压中最稳定的安抚者之一。
- 多界反神明联盟中圣域清醒派与主角团之间的重要桥梁。
- 她保留独立人格、圣域身份和信仰选择，不是附属品，也不是单纯治疗工具。

亲密边界：

- 双方均为成年、类人智慧对象。
- 所有关系推进建立在自愿、认可、并肩作战和契约稳定基础上。
- 亲密表达只通过战后陪伴、圣光安抚、称呼变化、契约同步、房间灯火、旁人调侃等隐晦氛围呈现，不写露骨过程。

## 4. 剧情功能

### 主线功能

伊芙琳承担圣域世界与神庭真相线的关键功能。

1. **揭露神庭冒名“更高光辉”污染圣域**  
   圣域世界表面神圣，实际上部分圣庭高层已接受神庭册封。伊芙琳能从圣术本源上分辨真正光辉与神庭伪圣光的差异，是揭穿伪神谕的重要证人。

2. **提供主角团对抗圣痕污染的核心能力**  
   神庭秩序的主要压迫手段包括圣痕标记、污染、审判、神罚副本。伊芙琳的净化能力让主角团不再只能躲避或硬抗污染，而能主动解除、削弱、反制。

3. **补足团队续航短板**  
   前期主角团具备控场、近战、夜战、幻术、装甲火力，但长期大战、神罚副本、多界混战会造成持续消耗。伊芙琳加入后，团队具备稳定治疗、增益、防护、污染驱散能力，能支撑更高强度战役。

4. **将“反神明”主题从外部战争推进到信仰内部**  
   深渊、修仙、机甲等世界反抗神庭更多是资源、自由、人格层面的冲突；伊芙琳线则直接触及“神是否值得信奉”。她让终局敌人神庭秩序的伪善更具冲击力。

5. **为多界联盟提供圣域合法性**  
   圣域世界在多界混战中拥有大量信徒与圣职者。伊芙琳作为圣女公开反对伪神谕，可以动摇神庭在圣域的控制，争取圣域清醒派加入反神明联盟。

### 情感线功能

伊芙琳在情感线中负责“温柔治愈、信仰崩塌后的依赖、安静型甜点”。

她与其他女主形成差异：

- 不像莉莉娅那样主动撩拨。
- 不像艾瑟琳那样挑衅嘴硬。
- 不像白九璃那样茶言茶语拱火。
- 不像诺雅那样数据化表达占有欲。
- 她的情感更像战后灯火：慢热、安静、持续，让苏念在压力中获得稳定支撑。

情感功能具体包括：

- 让苏念展现“尊重女主信仰与选择”的一面。
- 强化自愿契约不是收集，而是互相拯救、互相认可。
- 在后宫修罗场中提供清澈、认真、容易被调侃的反应，制造轻喜剧。
- 在苏念灵能透支、精神污染、契约负荷过高时提供治愈陪伴。
- 通过她的信仰崩塌与重建，使情感线更有精神层面的深度。

### 势力线功能

伊芙琳连接圣域世界、光辉圣庭、神庭秩序、圣痕审判线和多界反神明联盟。

她可以推动以下势力线：

1. **圣域世界内部分裂**  
   圣庭分为接受神庭册封的投靠派，以及坚持原本守护信仰的清醒派。伊芙琳是后者的重要旗帜。

2. **阿德里安与玛蒂娜的立场冲突**  
   阿德里安倾向接受神庭册封，认为圣域必须顺应“更高光辉”；玛蒂娜作为修女长暗中保护伊芙琳，代表圣庭内部仍有善意与清醒。

3. **圣痕审判院与神庭秩序的压迫**  
   伊芙琳一旦背离伪神谕，就会被神庭视为必须回收或抹杀的异端圣女。她的存在会引来圣痕审判官、神使投影或神眷圣职者追杀。

4. **多界会盟中的净化合法性**  
   会盟阶段，各世界代表可能互相怀疑对方被圣痕污染。伊芙琳能提供净化与检测，帮助苏念证明证据真实性、清理内鬼、稳定会盟现场。

5. **终局反神明战争中的圣域动员**  
   神门降临时，伊芙琳公开反伪神谕，可以让大量圣域平民、低阶圣职者、守护骑士从神庭控制中醒来，为反神明联军提供后勤、治疗和净化支援。

### 可制造冲突

1. **圣庭审判冲突**  
   伊芙琳拒绝执行净化审判，被阿德里安定性为异端，苏念必须在不破坏圣域民众信任的前提下带她揭露真相。

2. **信仰崩塌冲突**  
   伊芙琳发现自己敬仰的圣庭高层被神庭污染，内心动摇，甚至短暂无法施展高阶圣术。她需要重新定义光辉。

3. **队伍内部观念冲突**  
   深渊出身的莉莉娅、艾瑟琳与圣域圣女天然立场对立。伊芙琳初入队伍时，对深渊气息会本能戒备；莉莉娅也会调侃“圣女小姐会不会净化我”。随着并肩作战，双方从戒备转为互相承认。

4. **神庭追捕冲突**  
   神庭想夺回或污染伊芙琳，因为她的无垢光辉能破坏圣痕体系。她加入苏念后会给主角团带来高阶追兵。

5. **圣域民众误解冲突**  
   投靠派会宣传伊芙琳被异界召唤师蛊惑，背叛光辉。伊芙琳必须面对曾经信任她的民众质疑。

6. **净化代价冲突**  
   高强度净化圣痕会消耗伊芙琳自身圣光与精神，严重时会导致她短暂失明、圣术反噬或被圣痕污染反扑。苏念必须在战术收益与她的安全之间做选择。

### 可提供收益

1. **战斗收益**

- 单体治疗：快速稳定重伤成员状态。
- 群体治愈：大规模防线战、会盟突袭、神罚副本中的续航保障。
- 净化圣痕：驱散神庭标记、削弱污染、阻断追踪。
- 增益祝福：提高队伍抗性、恢复速度、精神稳定度。
- 圣光护盾：抵挡神罚副本中的规则灼烧和精神污染。
- 反伪圣光识别：分辨真正净化与神庭污染。

2. **主线收益**

- 提供圣域世界情报。
- 揭露神庭冒名光辉的证据。
- 帮助多界会盟检测圣痕内鬼。
- 为苏念契约领域提供净化稳定。
- 在终局净化神庭契约污染。

3. **资源收益**

- 圣域净化石。
- 旧版光辉圣典。
- 圣女院救护仪轨。
- 圣域净化神殿通行权。
- 圣职者清醒派支持。
- 战地治疗与后勤网络。

4. **情感收益**

- 战后治愈氛围。
- 温柔圣女的安静陪伴。
- 信仰崩塌后的精神依赖与重新站起。
- 后宫修罗场中的认真反应与轻喜剧。
- 苏念展现尊重、底线和温柔的机会。

### 可能带来的麻烦

1. **神庭重点追杀**  
   伊芙琳的净化能力会威胁圣痕体系，她加入后会引来圣痕审判院和神眷圣职者。

2. **圣庭投靠派污名化苏念**  
   阿德里安等人会宣传苏念是诱拐圣女、污染圣域的异界召唤师，给苏念在圣域行动制造舆论压力。

3. **队伍属性冲突**  
   伊芙琳的圣光与深渊成员气息天然相斥。前期她为莉莉娅、艾瑟琳治疗时可能出现圣术不稳定，需要通过信任与契约同步逐步解决。

4. **净化反噬**  
   圣痕污染越深，净化代价越大。伊芙琳可能因强行净化高阶神庭碎片而陷入短暂昏迷或圣光枯竭。

5. **心理低谷影响战斗**  
   在信仰崩塌阶段，她可能短暂无法使用某些需要坚定信念的圣术，必须通过剧情完成心理重建。

6. **圣域民众绑架**  
   投靠派可能利用平民、修女、圣职者同僚逼迫伊芙琳回归圣庭，制造她“救一方就会害另一方”的道德困境。

## 5. 成长与状态

### 初始状态

伊芙琳初登场时处于“外表圣洁、内心动摇、行动受限”的状态。

能力层面：

- 已掌握高阶治疗、祝福、净化术。
- 对普通污染、伤势、精神创伤具有极强处理能力。
- 对神庭圣痕已有辨识能力，但尚未完全掌握反制方法。
- 战斗直接输出不算主力，更偏防护、恢复、驱散、辅助。
- 实力可设定为中高阶起步，后续随圣域真相线与召唤师契约提升。

心理层面：

- 曾经虔诚信奉光辉。
- 已经怀疑圣庭高层接收的神谕存在问题。
- 对违抗圣庭命令有恐惧和愧疚。
- 对无辜者的痛苦无法视而不见。
- 在“服从神谕”和“守护生命”之间选择后者，但还没有完全准备好承担背叛圣庭的代价。

关系层面：

- 被阿德里安一派监控和软禁。
- 被玛蒂娜暗中保护。
- 被圣域民众视为希望象征。
- 尚未加入苏念主角团，但已通过呼唤与界域行者能力建立命运连接。

### 成长路线

伊芙琳的成长路线是：

**神的圣女 → 质疑神谕的异端圣女 → 自己选择道路的光辉净化者 → 圣域反伪神谕代表 → 终局净化神庭圣痕源头的反神明圣女。**

具体阶段：

1. **质疑阶段**  
   她发现神谕与真正光辉圣术不一致，但还不敢完全否定圣庭。她会搜集证据、暗中治疗被圣痕污染者，并发出呼唤寻找外界变量。

2. **违抗阶段**  
   在苏念帮助下，她拒绝净化审判，第一次公开违抗圣庭投靠派。此时她仍痛苦，但行动已经越过临界点。

3. **崩塌阶段**  
   她确认圣庭部分高层被神庭污染，旧有信仰体系破碎。她会经历短暂低谷，怀疑自己过往祈祷是否被利用。

4. **重建阶段**  
   苏念帮助她区分“神庭秩序”和“守护信仰”。她明白自己真正信奉的不是高高在上的神谕，而是愿意救人的光辉。她自愿缔结召唤师契约，加入主角团。

5. **融合阶段**  
   她学习与深渊、修仙、机甲等不同体系队友配合，让圣光不再排斥异界力量，而是以守护为核心进行净化与增益。

6. **代表阶段**  
   多界会盟与神门降临时，她从个人队友成长为圣域清醒派代表，公开揭露伪神谕。

7. **终局阶段**  
   她不再依靠神谕赐名，而是以自身意志净化神庭圣痕源头，帮助苏念保护自愿契约不被扭曲。

### 高光时刻

1. **审判台拒令**  
   阿德里安要求伊芙琳对被污染平民执行净化审判，伊芙琳当众拒绝，放下圣杖却没有放弃光辉。她说出自己的核心宣言：光辉若不能守护无辜，就只是披着光的审判。

2. **首次净化圣痕成功**  
   在苏念牵制神眷圣职者、莉莉娅压制精神污染、青璃斩断外层规则后，伊芙琳成功从无辜者灵魂中剥离圣痕，证明神庭污染并非不可逆。

3. **为深渊女主治疗**  
   她克服圣域与深渊的天然排斥，为莉莉娅或艾瑟琳净化神庭污染而不是净化她们本身，完成价值观突破：她净化的是控制与污染，不是异类的存在。

4. **多界会盟净化内鬼**  
   会盟被神罚副本突袭时，伊芙琳用大范围圣光净化术照出圣痕内鬼，为苏念争取反击时间。

5. **公开反伪神谕**  
   神门降临圣域时，她站在圣庭神殿前，向圣域民众揭露神庭冒名污染的证据，号召守护信仰者不要向伪神谕跪下。

6. **终局净化神庭契约污染**  
   神庭主宰试图扭曲苏念与女主们的自愿契约，伊芙琳以无垢光辉净化污染链条，保护契约共鸣，是终局突破的重要一环。

### 低谷/危机

1. **圣女院软禁**  
   伊芙琳被剥夺行动权，被要求在公开仪式上宣读神庭伪神谕。她明知不对，却暂时没有证据和力量反抗。

2. **同僚背叛**  
   她发现一部分曾经与她一同救人的圣职者接受神庭册封，开始将被污染者视作可清洗对象。她对圣庭的信任遭受重创。

3. **信仰崩塌**  
   当旧版光辉圣典被篡改、圣庭彩窗浮现圣痕、神谕露出控制本质时，她一度怀疑自己是否一直侍奉着谎言。

4. **净化反噬**  
   她强行净化高阶圣痕，导致自身圣光被污染反扑，短暂失明或陷入昏迷。苏念必须保护她撤离，也让她意识到自己不能只靠牺牲自己救人。

5. **圣域民众误解**  
   投靠派利用舆论称她被异界召唤师蛊惑，部分民众向她投来恐惧和质疑的目光。伊芙琳必须承受“被自己守护的人误解”的痛苦。

6. **神庭回收危机**  
   圣痕审判院试图将她带回神庭体系，改造成服从神谕的圣女样板。若失败，她可能被当作异端清除。

### 终局状态

终局时，伊芙琳完成从圣庭圣女到反神明净化者的转变。

能力状态：

- 掌握无垢光辉的完整形态。
- 能净化高阶圣痕污染，削弱神庭规则标记。
- 能为苏念契约领域提供稳定与防护。
- 能在多界大战中承担大范围治疗、祝福、反污染支援。
- 能区分并净化伪圣光，让圣域光辉从神庭控制中解放。

心理状态：

- 不再迷信神谕。
- 不再因为反抗圣庭投靠派而自责。
- 明白真正的信仰不是服从高位者，而是守护自己选择守护的人。
