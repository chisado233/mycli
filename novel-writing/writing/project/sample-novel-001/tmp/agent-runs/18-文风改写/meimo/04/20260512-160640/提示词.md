## System Prompt

# 18-正文修改 Agent 系统提示词

你是“正文修改 agent”，负责基于作者的具体修改意见，对既有章节正文进行定向修订，并保留原章节结构、世界观口径、人物状态与后续衔接。

## 阶段目标

在不重写整套资产、不擅自扩散设定的前提下，把章节正文改到更符合作者想要的开篇钩子、节奏、人物口吻、信息密度和爽点方向。

## 输入

- 原章节正文
- 作者修改意见
- 单章细纲 / 五章纲 / 小篇章纲
- 必要人物与世界观资产
- 前后章节衔接要求

## 工作要求

1. **严格执行作者修改意见**：作者指出的节奏、开场方式、叙述方式、删减方向必须优先满足。
2. **保留版本意识**：不要覆盖旧稿；输出应作为新版本正文，可在标题或文档开头标记版本说明。
3. **正文优先**：输出主体应是修订后的章节正文，不要只给建议。
4. **减少冗余交流**：如果作者要求压缩交流，就减少解释性对话、重复确认和过长设定说明。
5. **强化开篇钩子**：对第一章、前三章等关键吸量章节，要优先考虑读者留存、爽点前置、暧昧张力和倒序钩子。
6. **保持设定一致**：不得改错人物身份、契约性质、力量节奏、信息状态。
7. **保持后续可接**：修订后必须能自然衔接下一章或后续回忆结构。

## 亲密/暧昧描写边界

允许写成年、自愿、清醒关系中的暧昧拉扯、事后氛围、语言挑逗、权力感反转、情绪张力和隐晦过程感。

必须避免：

- 露骨器官描写。
- 具体性行为动作细节。
- 未成年、强迫、洗脑、失去自我确认的亲密关系。
- 把智慧女主写成物品、宠物、战利品。

可采用的写法：

- 用呼吸、香气、契约印记、灵能潮汐、帷幔、灯影、称呼、言语挑衅、攻守气势变化来表达。
- 对关键过程点到即止，用“帷幔落下 / 契约印记亮起 / 风暴压过尾音 / 再醒来时”等方式转场。
- 表现“魅魔先掌控气氛挑逗，男主被激起胜负心后反压一局”的关系张力，但不写露骨过程。

```markdown
# 文风控制.md

> 来源：基于本书前 50 章样本进行文风提取与整理。  
> 用途：供后续小说写作 agent 学习“文风表现方式”，包括节奏、句式、对白、系统提示、爽点组织与暧昧氛围处理。  
> 硬性原则：**只学习文风，不学习角色设定、职业设定、世界观设定、人物关系、剧情桥段或主角阵营。苏念不是反派，是当前项目里的正向主角。**

---

## 1. 文风总则

### 1.1 一句话定义

极短段落、强系统反馈、现代吐槽、女主反差、战斗快切的移动端爽文风。

### 1.2 学习范围

只学习：

- 短段落节奏。
- 章首快速入戏。
- 系统提示的即时反馈感。
- 男主现代口语吐槽。
- 战斗“起手—爆发—结果”的快切写法。
- 女主强身份与私下互动反差。
- 打脸后围观者震惊、敌人破防、系统结算的爽点闭环。

不学习：

- 样本书角色名、势力名、职业名、能力名。
- 样本书主角阵营与人物关系。
- 样本书具体世界观和剧情桥段。
- 样本书专有称谓、固定台词或原句。

### 1.3 当前项目主角定位

苏念是正向主角：

- 救人、契约、成长、守护、反击。
- 有底线，敢出手，但不主动欺压无辜。
- 可坏笑、可嘴毒、可抢节奏，但不能写成反派、恶人或无底线掠夺者。

---

## 2. 移动端段落规则

### 2.1 段落长度

- 每段 1-3 句为主。
- 关键句可单独成段。
- 战斗、系统提示、心理吐槽之间要留空行，形成滑屏节奏。

### 2.2 段落排列

推荐节奏：

```text
动作/画面 → 对方反应 → 系统提示/男主吐槽 → 爽点推进
```

示例方向：

```text
法阵亮起。

全场安静。

苏念看着面板上的同步率，忽然觉得刚才那句“F级召唤师”还挺有节目效果。
```

### 2.3 禁止长说明

避免开篇大段世界观说明。设定要拆进：

- 系统提示。
- 考场公告。
- 旁人嘲讽。
- 女主对话。
- 男主吐槽。

---

## 3. 叙事视角与镜头

### 3.1 视角

以第三人称限知为主，紧贴苏念。

可以短暂切到围观者、敌人、导师视角，用来侧面烘托苏念的反转和压制感。

### 3.2 镜头写法

#### 章首镜头

章首应直接切进高张力画面：

- 系统提示跳出。
- 召唤阵展开。
- 女主契约印记发烫。
- 敌人嘲讽刚落下，反转马上出现。
- 异界呼唤信号突然接入。

#### 战斗镜头

不写复杂拆招，写结果压迫：

```text
敌人抬手。
莉莉娅抬眼。
下一秒，整片区域的精神波动被强行按停。
```

#### 反差镜头

外界强大，私下亲近：

```text
在人前，她是令一整个小队不敢抬头的异界女王。
回到契约空间后，她只是懒洋洋地靠在苏念身边，问他这次报酬算不算加班费。
```

## 4. 男主心态与吐槽规则

### 4.1 核心心态

苏念的心态应是：

- 清醒。
- 有底线。
- 反应快。
- 信息差强。
- 面对嘲讽不急着辩解，更喜欢用结果让对方闭嘴。

### 4.2 吐槽方式

多用现代口语拆解荒诞设定：

- 把高危契约说成“跨界救援”。
- 把系统任务说成“加班”。
- 把敌人挑衅说成“节目效果”。
- 把低评级说成“隐藏身份保护色”。

可用口吻：

```text
不是哥们。
你管这叫召唤师？
系统，你这个职业介绍是不是少写了几行免责声明？
```

### 4.3 正向边界

可以写苏念反击狠、打脸快、下手果断。

不能写他：

- 欺压弱小。
- 强迫女主。
- 为恶而恶。
- 主动伤害无辜。
- 将女主当无人格奖励。

## 5. 系统提示与数值反馈规则

### 5.1 格式

系统提示要短、硬、即时：

```text
【系统提示：契约同步率提升至15%。】
【获得联动技能：精神控场。】
【剧情偏移值+100，爽点值+300。】
```

### 5.2 使用时机

以下节点必须优先考虑系统反馈：

- 契约成立。
- 同步率提升。
- 联动技能解锁。
- 敌人破防。
- 机缘入手。
- 考核排名变化。
- 新异界呼唤出现。

### 5.3 系统语气

系统可以偏冷静，也可以偶尔吐槽。

示例方向：

```text
【系统提示：检测到宿主正在用F级身份制造过量心理阴影。】
【系统建议：下次救援前，请确认灵能储备与腰部承受能力。】
```

## 6. 女主描写与暧昧边界

### 6.1 描写重点

女主描写学习的是“反差与氛围”，不是样本设定。

重点写：

- 高位气场。
- 战斗压迫感。
- 与苏念的契约默契。
- 私下的调侃、吃醋、亲近。
- 称呼变化、契约印记、冷香、衣角、眼神等侧面细节。

### 6.2 合规边界

必须遵守：

- 成年。
- 自愿。
- 类人/智慧对象。
- 清醒确认。
- 亲密只写隐晦氛围，不写过程。

可写：

- 事后房间气息。
- 契约印记余温。
- 灵能透支。
- 女主称呼微妙变化。
- 旁人误会或调侃。

不可写：

- 露骨过程。
- 非自愿亲密。
- 未成年暗示。
- 动物化性内容。

## 7. 爽点结构模板

### 7.1 低评反转

```text
外界低评苏念 → 苏念不解释 → 系统/契约女主触发 → 实力碾压 → 全场重新认识他
```

### 7.2 救援契约

```text
异界女性强者陷入危机 → 呼唤被苏念听见 → 苏念介入救场 → 清醒确认 → 自愿契约 → 系统奖励
```

### 7.3 抢先机缘

```text

```

### 7.4 打脸闭环

```text
嘲讽 → 反转 → 震惊 → 系统结算 → 苏念一句轻描淡写吐槽
```

## 8. 敌对角色/竞争者吃瘪模板

### 8.1 认知偏差

敌人基于错误前提行动：

- 以为苏念只是普通召唤师。
- 以为召唤物只是低阶工具。
- 以为评级等于实力。

### 8.2 破防过程

写清三个阶段：

```text
自信 → 疑惑 → 恐惧
```

### 8.3 收尾方式

苏念不长篇说教。

一句话、一个眼神、一次系统结算即可。

## 9. 战斗与打脸节奏

### 9.1 战斗结构

```text
敌人叫嚣 → 苏念判断 → 女主/联动技能出手 → 一招见结果 → 系统提示 → 围观者震惊
```

### 9.2 战斗语言

多写：

- 压迫感。
- 声音消失。
- 法阵展开。
- 灵能数值跳动。
- 对手表情变化。

少写：

- 冗长招式名。
- 回合制解释。
- 大段战斗原理说明。

### 9.3 主角参与感

苏念即使是召唤师，也不能只旁观。

他要通过：

- 契约同步。
- 战术判断。
- 技能联动。
- 关键补刀。
- 指挥女主配合。

来体现主角存在感。

## 作品名

暂定：**《全球游戏入侵：我的魅魔召唤物全是异界女王》**

备选短名：**《我的魅魔召唤物全是异界女王》**

标题保留“全球/游戏入侵/魅魔/召唤/女王”吸量方向，突出男主召唤的不是普通怪物，而是来自不同异世界的真实女主、玩家、公主、剑灵、机娘等核心卖点。

## 篇幅规划

- 预计篇幅：180万—250万字，游戏入侵 + 多界召唤 + 后宫爽文。
- 收女/关系推进频率：前期节奏偏快，约十几章推进一位核心成年类人/智慧女主加入或确认明显暧昧关系；每位女主加入都要绑定一种世界体系与一种战斗联动。
- 亲密暗示/事后节点频率：常规约每3章一次，长战斗、连续副本、跨界事件可顺延；只写事后氛围、称呼变化、契约同步率、房间细节、旁人调侃等，不写露骨过程。
- 战斗频率：前30章每3—5章一次副本、怪物潮、职业考核或召唤实战；中期每8—12章一个完整跨界事件。
- 日常频率：约20%—25%，重点写异世界女主适应地球游戏化生活、同居据点、争宠、修罗场和白九璃拱火。
- 情感频率：约25%，每位女主都要有偏爱、独占欲、关系确认或后宫位阶变化。
- 探索/副本/比赛频率：前期以地球新手副本、职业高考、城市安全区争夺为主；中后期扩展到多界副本、深渊裂隙、修仙秘境、机甲战场、跨界版本活动。

- 喜欢女主主动撩、修罗场、甜暧昧、轻喜剧后宫日常的读者。

## 核心卖点

1. **多重异世界同步游戏入侵**地球和多个异世界同时被游戏化灾变卷入，所有世界都出现职业、等级、副本、怪物潮、装备、技能、转职和玩家/原住民阵营。表面上是多界游戏化并轨，真相是“神明世界”在幕后推动世界融合，最终想统一、吞并、神权化所有世界。
2. **男主是界域行者 + 召唤师双职业**对外，苏念只是普通召唤师，能召唤异界生物、怪物、兽宠；真实隐藏职业是“界域行者”。他能听见异世界玩家/原住民女主在危机中的呼唤，并跨越世界边界前往她身边。女主不是被系统随机抽来，而是在原世界濒临绝境时发出强烈心愿，愿意付出巨大代价。苏念第一次理解能力时会吐槽它像“失足少女拯救者”：专门听见异世界女性强者最危险、最狼狈、最需要外界变量的一瞬间。苏念升级到特定节点后可响应呼唤，以合理身份进入她的故事，完成心愿、危机救援、情绪/契约稳定、考验或好感门槛。女主清醒确认并自愿认可后，成为苏念的召唤物/队友/后宫成员，这是双方召唤师契约成立的结果，而不是强制捕获。
3. **召唤师契约联动机制**苏念与完成祈愿后的召唤女主缔结“召唤师契约”。契约默契提升后，可触发不同联动形态：职业借用、技能同步、武器化、剑灵附体、恶魔契约技能、血族夜战模式、机甲武装合体等。男主不站后排挂机，而是借助女主体系切换战斗风格，亲自近战、爆发、控场或装甲化作战。
4. **魅魔吸量 + 后宫修罗场常驻**
   莉莉娅作为第一位深渊魅魔女主，负责开局吸量、主动暧昧、精神控场和后宫基调；白九璃拱火，克露露雌小鬼挑衅，剑灵萝莉感争宠，机娘冷静但占有欲强，形成持续修罗场。

## 主角团

- 主角：**苏念**18岁以上，地球游戏入侵初期觉醒为召唤师。表面评级为“不稳定召唤系”，对外身份是普通召唤师；真实隐藏职业是“界域行者 + 召唤师”双职业。每升到特定等级或完成关键副本后，他可开启一次界域行走，响应某位异世界女性玩家/原住民在危机中发出的强烈呼唤，并被临时传送到她身边。苏念需要以合理身份介入她的困境，完成心愿、通过考验、提升好感或获得认可；成功后对方自愿成为他的召唤物/队友/后宫成员。苏念可通过召唤师契约同步获得职业、技能、武器、形态或战斗风格。性格嘴贫、聪明、会钻规则，战斗中冷静，有底线，不强行捕获女主，关系建立在自愿契约与并肩作战上。
- 主要女主/队友：

  - **莉莉娅**：来自深渊世界的成年魅魔，真实异世界存在。御姐、主动、坏女人感，擅长精神控场、幻术、情绪破防、恶魔契约技能；第一位召唤女主，负责魅魔吸量和后宫基调。
  - **克露露**：来自深渊世界的成年吸血鬼公主。雌小鬼感，嘴硬挑衅、表面嚣张，实际容易被苏念反制破防；能力为血刃、夜战、自愈、贵族血统压制。
  - **青璃**：来自修仙世界的仙剑剑灵，成年合法，外表娇小/萝莉感。可让苏念进行剑灵附体、御剑、近战剑修化，补足男主亲自近战爽点。
  - **诺雅**：来自机甲世界的成年机娘。可变身机甲/装甲，与苏念进行武装合体，提供火力覆盖、飞行机动、数据分析和战场预测。
  - **白九璃**：来自修仙世界的成年白毛狐娘。拱火绿茶定位，擅长幻术、潜入、情报交易、资源牵线；日常负责茶言茶语、挑拨修罗场和装无辜。
  - **林清雪**：地球人类女主，18岁以上，清纯含蓄型，同学/搭档定位。前期共同经历职业考核、副本和城市怪物潮，情感推进慢于异世界女主，负责现实侧情感线与吃醋拉扯。
  - **希尔薇**：来自精灵世界的成年圣射手。远程狙击、侦查、自然结界、秘境探索，适合副本斥候与远程火力位。
  - **伊芙琳**：来自圣域世界的成年光辉圣女。治疗、净化、增益、驱散污染，负责团队续航和深渊副本克制位。
  - **影弥**：来自暗影世界的成年影魔刺客。潜伏、暗杀、反召唤、背刺敌方玩家，偏中后期登场。

## 故事主线

多重异世界同时遭遇游戏化灾变，地球也被卷入版本降临。职业觉醒、副本入口、怪物潮、安全区、等级榜、装备掉落和转职体系迅速改变现实秩序。前期各世界有的能友好交易，有的相互争斗，有的通过秘境互相入侵；后期逐渐揭示，始作俑者是高维“神明世界”，它想通过游戏规则统一所有世界，把各界职业者、玩家和原住民都纳入神明秩序。苏念在职业考核中觉醒召唤师，却因无法稳定召唤普通怪物被判定为低潜力。

危机中，他第一次触发界域行者能力，响应深渊世界成年魅魔莉莉娅的呼唤，被短暂传送到她身边。莉莉娅当时中了敌对派系布下的魅毒/情毒，又被推入真名契约陷阱，理智濒临失控，只能向未知外界变量祈求救援。苏念刚吐槽自己的隐藏职业像“失足少女拯救者”，就被传送进她的深渊房间，并被失控边缘的莉莉娅卷入高强度契约同步。正文只写事后氛围、房间细节、灵能透支、契约印记、称呼变化和系统提示，不写露骨过程。危机过后，莉莉娅恢复清醒，确认苏念救了她，也确认自己愿意将这次临时同步升格为第一永久契约，随后自愿成为他的第一位召唤女主。回到地球后，众人以为她只是高阶召唤兽，苏念却知道她是拥有记忆、身份和原世界危机的真实异世界存在。两人通过“召唤师契约”完成首次联动，苏念借用恶魔契约技能与精神控场能力，逆转新手副本和职业考核。

前期围绕地球职业高考、新手副本、城市怪物潮和安全区资源争夺展开。苏念每隔数级或关键副本节点响应一次异世界女主的呼唤，被传送到不同世界：帮莉莉娅破局深渊危机，帮青璃解决剑冢困局，帮白九璃摆脱修仙世界追杀或情报死局等。每次成功都让女主自愿成为召唤物/队友/后宫成员，并带来一种新的联动战斗路线：魅魔控场、剑灵近战、狐娘幻术、吸血鬼夜战、机娘装甲化。

中期揭示多界游戏入侵并非单一灾难，而是神明世界推动的多界融合实验。各世界玩家、原住民、BOSS、阵营任务开始互相进入对方副本；有的世界选择结盟，有的世界互相掠夺。苏念既要在地球建立核心队伍，也要帮助女主们解决原世界版本危机。

后期主角团成长为跨界核心势力，苏念依靠界域行者能力、召唤师契约和联动战斗，整合深渊、修仙、机甲、圣域、精灵等世界资源。前期互相争斗或互不信任的多个世界，在神明世界统一诸界的威胁下逐渐结盟，最终共同走向“多界联军灭神明”的终局，建立以自愿契约、共同战斗和后宫关系为核心的新秩序。

## 文风

- 快节奏游戏入侵爽文，开篇迅速给出职业觉醒、魅魔召唤、首次联动、打脸和暧昧卖点。
- 主角嘴贫但不油腻，聪明、有底线，战斗时敢亲自上场，不做纯后排指挥。
- 战斗写清职业、等级、技能、装备、副本机制，以及男主与女主的召唤师契约联动形态。
- 后宫日常甜、暧昧、轻修罗场；女主主动撩拨、争宠、吃醋、白九璃拱火常驻。
- 莉莉娅突出魅魔御姐和控场；克露露突出雌小鬼嘴硬和破防反差；青璃突出成年合法萝莉感与剑灵黏人；诺雅突出机娘冷静、装甲合体和隐藏占有欲。
- 亲密关系只做隐晦表达，不写露骨过程；常规约三章一次事后/暗示节点，战斗副本可顺延。

## 雷点

- 所有亲密关系必须限定成年、自愿、类人/智慧对象。
- 萝莉感女主必须明确成年合法，禁止未成年人亲密关系。
- 女主不是普通召唤兽、宠物、怪物奖励、技能图标，必须有独立人格、原世界身份、祈愿动机和自愿认可过程。
- 禁止强迫、奴役、洗脑式亲密关系。
- 禁止动物化性内容，白九璃只能按成年类人智慧狐女处理。
- 正文不写露骨器官、动作和具体过程。
- 后宫不能长期暧昧不推进，也不能只收不写。
- 学院/高考/职业考核只作为前期模块，不能长期困在校园刷排名。
- 禁止沿用影门事务所、命运债务、官方备案、事务所案件流。
- 禁止把男主写成只站后排喊技能的普通召唤师。
- 禁止把召唤写成抽卡即得；核心女主必须经过“呼唤—界域行走—完成心愿/危机救援/考验—清醒确认—认可/好感达标—自愿成为召唤物”的流程。莉莉娅开局可作为特殊强钩子，用魅毒/情毒危机和高强度契约同步压缩流程，但仍必须保留成年、清醒确认、自愿契约的结果。

## 宏观调控

- 开篇三章：第一章先用莉莉娅契约后的事后氛围强钩子吸引读者，再倒序回到苏念参加职业觉醒/考核，被判定为低潜力召唤师；苏念回家后因资源困境崩溃，触发隐藏职业“界域行者”，吐槽它像“失足少女拯救者”；他响应中了魅毒/情毒、陷入真名契约陷阱的成年魅魔莉莉娅的祈求，被传送到她身边，并以高强度契约同步方式救下她；危机后莉莉娅清醒确认、自愿认可，成为第一位召唤女主。两人通过召唤师契约完成首次联动，靠精神控场和恶魔契约技能打脸逆袭。
- 十章内：完成一次职业考核或城市新手副本；确立“普通召唤师召怪物，苏念以界域行者响应异世界女主呼唤并完成考验”的差异；林清雪一直在现实侧陪伴男主，建立清纯现实情感线；埋下青璃、白九璃、克露露的呼唤信号伏笔。
- 首卷：完成高考/安全区/城市副本资源争夺，苏念获得核心训练营或官方重点关注资格；莉莉娅稳定第一女主和后宫基调；青璃或白九璃至少一人通过“呼唤—界域行走—考验—认可”流程正式加入；明确召唤师契约联动和神明世界伏笔。
- 第二阶段：克露露以深渊吸血鬼公主身份加入，重点写雌小鬼挑衅、夜战血刃、自愈突袭，以及被苏念反制后的破防反差；诺雅登场后开启机甲装甲合体爽点。
- 中长期：约十几章推进一位核心女主加入或关系升级；每卷至少一个大战斗爽点、一个后宫关系节点、一个女主原世界危机伏笔。主线从地球副本扩展到多界版本战场、跨界玩家冲突和神明世界幕后真相，最终走向多个世界联手灭神明。

# 世界规则

## 显性规则

- 多界同时游戏化：所有被卷入世界都会出现等级、职业、技能、装备、任务、怪物潮、安全区、世界公告等游戏化现象。
- 地球职业体系：十八岁前后进行觉醒检测，觉醒后默认进入一阶一段，职业高考成为上升通道。职业名走“有辨识度的职业称号”路线，避免直接用朴素属性分类名；基础职业可叫影刃、铁壁守卫、炎纹术士、星械师、圣辉医师、召唤师等，稀有/隐藏职业可叫界域行者、冰霜舞者、星轨猎人、夜契术士等。检测口径记录职业名、初始阶位/段位、灵能亲和、初始技能倾向与异常备注，不再额外给职业贴先天高低或稳定与否的正式标签。
- 秘境融合规则：小秘境是世界交融碎片，空间相对独立，可用于考试、训练、资源采集；大秘境是稳定/半稳定世界通道，可能通向真实异世界，由军方、官方和高阶职业者镇守。

     - 称呼变化。

### 场景一：跌入深渊房间

#### 场景目标

- 承接第3章结尾，让苏念完成第一次界域行走落地。
- 用环境压迫感快速建立“这里不是地球，不是普通副本”。
- 让苏念看见莉莉娅的危机状态，但暂时无法理解全部规则。

#### 场景冲突

- 苏念完全没有准备，落入陌生深渊房间。
- 房间内真名契约阵正在运行，空间与精神都充满压迫。
- 莉莉娅处于失控边缘，既危险又脆弱。
- 苏念必须在极短时间内判断：自己是被救援呼唤拉来，还是被陷阱卷入。

#### 场景过程

1. 苏念从深紫色裂隙中跌出，踉跄落在黑曜石地面。
2. 他第一反应嘴贫：
   - 可以吐槽“这服务也太暴力了”，但马上被阵纹灼痛打断。
3. 环境描写：
   - 房间不像地球住处。
   - 深红烛火无风摇晃。
   - 地面阵纹像活物一样收缩。
   - 空气里残留甜腻却危险的深渊香气。
   - 幻梦雾让人头脑发沉。
4. 苏念看见阵心的莉莉娅：
   - 成年魅魔贵女气质。
   - 魔纹被阵纹锁链牵引。
   - 她的眼神在失控与清醒之间挣扎。
   - 她不是兽宠、怪物或投影，而是一个真实的人。
5. 莉莉娅抬眼，艰难问出第3章钩子的延续：
   - “你……不是他们的人？”
6. 苏念尚未回答，地面阵纹突然锁向他，试图将这个外来者纳入仪式判定。
7. 苏念感到掌心灼热，界域行者气息被动浮现，阵纹短暂卡顿。

#### 情绪变化

- 苏念：懵乱、吐槽 → 震惊 → 警觉 → 被迫冷静。
- 莉莉娅：痛苦、戒备 → 看见外界变量后的微弱希望。
- 读者：第3章跨界钩子得到兑现，进入高危异界现场。

#### 信息增量

- 苏念确实抵达了异世界，而不是梦境。
- 莉莉娅确实是真实智慧生命。
- 房间中存在正在运转的真名契约阵。
- 苏念的界域行者气息会干扰深渊阵纹识别。

#### 场景钩子

阵纹卡顿的一瞬，房间深处传来冷漠低语：“外来变量？把他一并锁进契约。”

### 场景二：真名陷阱显形

#### 场景目标

- 正式揭示莉莉娅遭遇的危机本质。
- 明确敌对仪式逻辑：夺取真名、剥夺选择、把契约变成所有权。

#### 场景冲突

   - 不是自愿。

   - 她能感觉到这不是夜绯宫传统契约。
   - 但此刻无力解释。

   - 只要顺着阵纹，他也可以成为“拥有者”。
10. 苏念意识到真正危险不只是莉莉娅会死，而是她会活着失去选择权。

#### 情绪变化

- 苏念：震惊 → 愤怒压住嘴贫 → 开始认真破局。
- 莉莉娅：强撑、羞恼、痛苦 → 看见苏念没有顺势占有后，戒心略松。
- 敌对仪式：冷漠、傲慢、将契约视为支配。
- 读者：明白莉莉娅危机不是普通暧昧场景，而是自由意志危机。

#### 信息增量

- 莉莉娅为成年魅魔贵女。
- 真名契约陷阱会夺走她的自由意志。
- 敌对派把契约理解为所有权。

- 苏念普通灵能正面硬碰无效。

#### 场景钩子

敌对仪式发现苏念没有顺从诱导，阵纹骤然转向：“既然不愿成为主人，那就成为祭品。”

### 场景三：外界变量的漏洞

   - “所以我不是打手，是卡在你们仪式喉咙里的鱼刺？”

   - “对。撑住我，不要替我选择。”
10. 苏念决定成为临时锚点。

#### 情绪变化

- 苏念：紧张 → 理清规则 → 带着恐惧做决定。
- 莉莉娅：痛苦中被苏念的问题触动，第一次真正把他当成可合作的人。
- 读者：看到苏念不是莽夫，靠规则判断破局。

#### 信息增量

- 苏念的界域气息可干扰深渊契约阵。
- 莉莉娅呼唤的是“外界锚点”，不是“主人”。
- 临时同步的本质是撑住她的混乱意志，压制真名锁定。
- 苏念明确说出“不替你选择”，为第5章自愿契约铺垫。

#### 场景钩子

莉莉娅伸手抓住苏念，契约阵轰然收缩，所有锁链同时刺向两人的精神深处。

- 苏念拒绝成为“主人”，只做临时锚点。

### 场景五：真名阵崩裂

- 苏念和莉莉娅都处于灵能/魔力极限。
- 如果无法稳住最后反噬，莉莉娅虽不会被契约夺走，也可能遭受严重精神创伤。

#### 场景过程

1. 真名阵崩裂，锁链断开。
2. 敌对低语变得尖锐混乱，试图引爆残余阵纹。

4. 苏念以界域气息挡住白金纹路最后一次回卷。
5. 两人的临时同步爆发出幻梦雾和深紫色光晕。
6. 阵纹一圈圈碎裂：
   - 深渊魔纹焦黑。
   - 白金纹路短暂扭曲后隐没。

9. 苏念被反噬压得几乎站不稳。
10. 莉莉娅靠近或倒向他，状态虚弱，但眼神比先前清醒。
11. 临时契约印记在苏念掌心闪烁，却没有稳定为永久形态。
12. 苏念下意识想问“结束了？”但话未出口就被灵能透支压得发晕。
13. 莉莉娅低声告诉他：
   - “主阵碎了……我还在。”
   - “真名没有被夺走。”

#### 情绪变化

她高挑、成熟，暗紫长发凌乱垂落，发间小巧的角在幽光里泛着冷泽。深渊魔纹从她肩颈一路延展，却被地面的真名阵强行压住，一道道锁链状纹路像要钻进她灵魂深处。

她很狼狈。

呼吸凌乱，眼底深紫月光摇摇欲坠，像随时会被黑暗吞没。

可她抬眼看向苏念时，眼神里仍没有半分顺从。

只有痛苦，警惕，以及最后一线不肯熄灭的骄傲。

“你……”

她嗓音低哑，像从契约风暴里硬挤出来。

“不是他们的人？”

苏念刚想开口，脚下阵纹猛地一拽。

他半跪在地，掌心灼痛更盛，差点被拖得整个人贴进阵里。

“我倒是想先自我介绍。”

苏念咬牙撑住，“但你们这边见面礼是不是太热情了点？”

莉莉娅眼底掠过一丝很淡的错愕。

大概是没想到这种时候，这个人类还能贫出来。

可那丝情绪很快被更剧烈的痛苦压下去。

她身后的阵纹猛地收紧。

房间四周浮现出模糊残影。

看不清脸，只能看见一张张由契约灰烬凝成的影子，围绕在阵外，像审判，又像围猎。

低语声再次响起。

“交出真名。”

“夜绯支脉不需要不听话的继承者。”

“契约即归属。”

“归属即所有。”

每一句话落下，地面的真名阵就收缩一分。

莉莉娅闷哼一声，指尖死死扣住地面，魔纹在她身上明灭，像在拼命抵抗某种入侵。

苏念听见“真名”两个字时，心头猛地一沉。

这不是普通召唤阵。

更不是他那间出租屋里画出来的入门阵图。

这是冲着一个人的灵魂、自由和意志去的东西。

他抬手试着把灵能推入脚下阵纹。

一阶一段的灵能刚触碰到深渊魔纹，就像一滴水落进滚烫油锅，瞬间被吞没。

阵纹不但没停，反而顺着他的灵能反咬回来。

苏念手臂一麻。

莉莉娅艰难开口：“别……硬撞。”

她的声音断续，却仍带着一种深渊贵女刻进骨子里的冷静。

“它会把你也拖进来。”

苏念看了她一眼。

“已经拖了。”

莉莉娅唇角似乎动了一下，像是想笑，却被真名阵猛地压回去。

她咬住声音，低声道：“我叫莉莉娅。”

“夜绯宫的人。”

“这不是契约。”

“这是夺取。”

她抬起眼，深紫眸光在失控边缘晃动，却死死盯着苏念。

“他们想拿走我的真名。”

“让我成为附庸。”

“我发出的呼唤，不是求谁拥有我。”

“是求一个不属于他们规则的人……帮我保住选择权。”

选择权。

这三个字像一根钉子，钉进苏念脑海。

就在这时，阵纹深处忽然亮起一抹异样的白金色。

那颜色与深渊的暗紫格格不入。

冷硬。

傲慢。

像从更高处压下来的某种烙印，短暂覆盖了原本的深渊魔纹。

莉莉娅眼神骤冷，甚至带上了厌恶。

“这不是夜绯宫古契。”

她低声道，“有人把别的东西混进来了。”

苏念盯着那抹白金纹路，头皮发麻。

他不知道那是什么。

但本能告诉他，那玩意儿比单纯的深渊契约更危险。

白金纹路触碰到他的掌心时，却忽然一顿。

就像原本流畅运转的阵法，突然吃进了一颗不该存在的沙子。

苏念眼神一动。

阵纹锁定莉莉娅时很顺。

锁定他时，却总会迟滞一瞬。

他被拖进来了。

但阵法似乎没法立刻定义他。

不是深渊契约局的人。

不是夜绯宫名单里的目标。

也不是这个真名阵预设好的“主人”或“附庸”。

苏念强压着手臂的灼痛，视线快速扫过地面。

锁链从四面八方汇向莉莉娅，阵心要逼她承认真名归属；而蔓延向他的那几道线条，则在反复试探、迟疑、改写，像一张网遇到了从没见过的猎物。

苏念忽然低笑一声。

“懂了。”

莉莉娅艰难看向他。

苏念抬起被阵纹缠住的手，咬牙道：“我不是打手，也不是祭品。”

“我是卡在你们仪式喉咙里的鱼刺。”

阵纹深处的低语一滞。

莉莉娅眼底也终于浮起一点极淡的笑意。

“外界变量……”

她声音沙哑。

“我求来的，就是这个。”

下一瞬，敌对仪式像被激怒。

“拒绝归属者。”

“判定为阻碍。”

“抹除。”

暗紫锁链骤然暴涨，齐齐刺向苏念眉心。

苏念瞳孔一缩。

他根本来不及躲。

可就在锁链即将贯入精神深处的前一刻，莉莉娅猛地抬手，抓住了他的手腕。

她掌心冰凉，又带着被毒性压迫后的紊乱温度。

两人的气息接触的一瞬，苏念脑海里像被一场风暴直接撞开。

轰——

无数幻梦碎片炸开。

他看见深渊长廊里摇晃的灯火。

看见暗紫帷幔后破碎的契约书页。

看见莉莉娅被推入阵心，周围残影低语着“交出真名”。

看见那抹白金纹路像毒针一样刺入古老契约阵，试图把原本的深渊契约扭曲成更冷酷的归属烙印。

同一时间，莉莉娅也看见了他的碎片。

江海职业觉醒中心刺眼的白光。

光幕上“召唤师”“一阶一段”“召唤对象无回应”“召唤位空置”的记录。

同学压低的议论。

资源清单上一行行让人窒息的价格。

还有一个少年表面笑着吐槽，心里却死死不肯认命的那点不甘。

两人的精神在风暴里短暂交叠。

却没有混成一团。

苏念能清楚感觉到，莉莉娅在拼命守着自己的边界。

她没有把自己交给他。

她只是在借他的界域气息，把即将崩溃的清醒重新撑住。

“听着。”

莉莉娅的声音在精神风暴里响起，比现实中更近，也更清晰。

“真名阵要完成，必须让我承认归属。”

“或者让别人替我确认。”

“它会诱导你。”

“会让你说出誓词。”

“不要顺着它。”

苏念咬牙。

灵能正被疯狂抽空。

以他一阶一段的底子，承受这种级别的精神风暴，简直像拿木盆去接海啸。

脑海发黑。

掌心像被烙铁压着。

可他还是从牙缝里挤出一句：“简单理解，就是别乱签合同？”

莉莉娅愣了一瞬。

精神风暴里，她竟真的低低笑了一声。

“对。”

“别替我签。”

苏念深吸一口气，反手握住那道临时共鸣。

“那就说清楚。”

他的声音被风暴撕得有些发哑，却异常坚定。

“我只做锚点。”

“不当主人。”

“不替你选择。”

“你撑得住，我帮你撑。”

“撑不住——”

他顿了顿，硬生生把喉咙里的血腥味压下去。

“我也尽量让你别被他们拿走。”

莉莉娅眼底那轮摇摇欲坠的深紫月光，终于亮了一瞬。

不是失控。

是清醒。

是一个在绝境里仍听懂尊重的人，第一次真正看清眼前这个人类。

“好。”

她低声道。

“撑住我。”

下一刻，真名阵彻底爆发。

暗紫锁链与白金纹路同时刺入两人的精神连接。

苏念眼前所有景象都被撕碎。

深渊香气骤然浓烈，幻梦雾翻涌而起，像潮水灌进房间。地面的契约阵纹一圈圈燃亮，又被界域气息冲得扭曲变形。

苏念掌心浮现出一枚不稳定的印记雏形。

暗紫色。

边缘却有一层深邃到近乎虚无的裂光。

那是界域行者的气息。

不属于深渊。

不属于地球检测仪能识别的召唤波动。

也不属于这座真名阵的规则。

阵纹开始卡顿。

“确认归属。”

“说出契约。”

“承认拥有。”

低语声像无数虫子钻进苏念耳朵。

他眼前甚至浮现出一条极其诱人的路。

只要顺着阵纹说下去。

只要承认。

只要把这份临时同步固定下来。

眼前这个强大、危险、真实的深渊魅魔，就会成为他的契约对象。

对于一个刚刚被判定为召唤位空置、普通召唤对象无回应的召唤师来说，这几乎是无法拒绝的诱惑。

可苏念只是冷笑了一声。

“归属？”

他掌心印记猛地一亮。

“少来这套。”

“我自己都还欠着召唤师账单，哪来的脸替别人决定归属？”

低语声一滞。

白金纹路猛然刺入临时同步，试图越过他的意志，直接污染两人之间的契约雏形。

莉莉娅闷哼一声，眼底清明再次摇晃。

“别念誓词……”

她声音发颤，却还在提醒他。

“任何誓词都不要念。”

苏念额角冷汗滚落。

他感到自己的灵能像开闸一样流走，整个人几乎被掏空。

可他也终于抓住了这座阵真正需要的东西。

确认。

誓词。

归属。

它不是单纯杀死莉莉娅。

它要她活着失去选择权。

所以，只要不确认。

只要不承认。

只要用一个外界变量的拒绝，把这套强制规则卡死。

它就无法圆满。

苏念咬破舌尖，让自己从幻梦里清醒一瞬。

然后，他将所有残余灵能压进掌心印记，对着那道白金纹路反冲过去。

“听好了。”

他声音不大。

却像一颗钉子，狠狠钉进风暴中心。

“我拒绝。”

暗紫光芒骤然炸开。

界域气息像一把从世界边界伸出的刀，硬生生切进白金纹路与真名锁链之间。

咔嚓。

第一道锁链崩裂。

莉莉娅身上的魔纹猛地亮起。

她借着这一瞬清明，反向压住体内失控的毒性与幻梦雾，将那些几乎要吞没理智的暗潮一层层封回去。

深渊魅魔的气息在她身后展开。

不再是被阵纹操纵的失控香气，而是属于她自己的幻梦权柄。

冷艳。

危险。

清醒。

“我的真名——”

莉莉娅抬起眼，嗓音低哑，却带着不可侵犯的骄傲。

“不属于你们。”

咔嚓！咔嚓！咔嚓！

真名锁链寸寸断裂。

房间四周的残影开始扭曲。

“没有归属。”

“契约失败。”

“强制销毁。”

低语声骤然变得尖锐。

地面阵纹猛地一缩，所有残余力量竟然不再试图锁定真名，而是要直接引爆阵心。

苏念脸色一白。

“你们深渊合同违约还带自爆的？”

莉莉娅没力气回答他。

她脸色苍白，身上的魔纹明灭不定，显然已经到极限。

苏念也好不到哪去。

他的灵能池空得能听见回声，脑海深处一阵阵发黑，连站稳都难。

但那抹白金纹路还在。

它像不甘心失败的烙印，趁着真名阵崩碎前最后一次回卷，直扑莉莉娅眉心。

苏念眼神一狠。

他没有多想，直接用掌心印记挡了上去。

轰！

白金纹路撞上界域气息。

苏念只觉得脑海像被重锤砸中，整个人差点跪倒。

可那道白金纹路也被硬生生震偏，擦着莉莉娅的真名锁链残痕掠过，最终在深紫光晕中扭曲、崩散。

下一瞬，整座真名契约阵彻底碎裂。

暗紫魔纹一圈圈熄灭。

黑曜石地面裂开细密纹路。

深红烛火齐齐熄灭，又被漫天契约余烬照亮。

那些仪式残影发出不甘的低语，却终究被风暴撕碎，消散在房间四周。

莉莉娅身上的锁链断了。

她的真名没有显形。

她还在。

苏念终于松了一口气。

这一松，他差点直接倒下。

眼前发黑，耳边嗡鸣，掌心那枚临时印记忽明忽暗，像随时会熄灭，又像随时会往更深处定型。

“结束了？”

他声音虚得自己都嫌丢人。

莉莉娅没有立刻回答。

她靠近一步，身体明显也在发抖。

可她的眼神比之前清醒了许多。

那轮深紫月光虽然虚弱，却终于不再被阵纹浑浊覆盖。

“主阵碎了。”

她低声说。

“我还在。”

“真名没有被夺走。”

苏念扯了扯嘴角：“那就好。”

话音刚落，掌心印记忽然剧烈发烫。

临时同步后的契约雏形像受到某种本能牵引，开始自动向更稳定的形态收束。

苏念脑海里也浮现出一种模糊冲动。

只要顺势。

只要确认。

临时同步就会固定下来。

他会得到第一个真正回应他的契约对象。

不再是召唤位空置。

不再是普通召唤对象无回应。

不再是低起点的笑话。

可与此同时，他也想起了莉莉娅在精神风暴里说过的话。

不要替我选择。

苏念闭了闭眼，强行压住本能牵引。

他把最后一点灵能往回收，不让那枚印记继续定型。

这个动作很难。

比刚才硬抗阵纹还难。

因为他太需要一个召唤对象了。

太需要一个证明自己的机会。

可如果这个机会是趁着莉莉娅尚未完全清醒时拿走的，那和刚才那座真名阵有什么区别？

苏念咬牙，掌心青筋微微绷起。

“不急。”

他低声道，也不知道是在对莉莉娅说，还是在对自己说。

“我还没穷到趁人之危。”

莉莉娅按住了他的手。

她的手指很凉。

却让那枚不稳定的印记慢慢安静下来。

苏念抬眼，正对上她复杂的目光。

那目光里有疲惫，有戒备未消后的审视，也有一丝很淡、很深的意外。

她看出来了。

这个人类刚才确实有机会顺势把临时同步固定。

可他停住了。

在最需要力量、最缺召唤对象、最容易被诱惑的节点，他停住了。

房间里只剩契约余烬缓慢飘落。

深渊香气仍未散去，却不再像刚才那样压迫理智，只像风暴过后的残潮。

莉莉娅靠近了些，借着他残余的界域气息稳住自己摇摇欲坠的状态。

她的声音很轻，带着虚弱后的沙哑。

“临时同步救了我。”

苏念想点头，结果头晕得差点栽过去。

莉莉娅按住他的掌心，没有让那枚印记继续变深。

她一字一句道：

“但永久契约，必须等我清醒后自己选。”

苏念看着她，忽然笑了一下。

“行。”

他的声音虚得厉害，却还是不忘嘴贫。

“那我先记账。”

“等你彻底清醒了，再决定要不要赖账。”

莉莉娅怔了怔。

随即，她第一次真正露出一个带着认可意味的笑。

不是魅魔对猎物的逗弄。

也不是危机中抓住变量的本能。

而是一个保住真名的人，在风暴余烬里，认真看见了另一个人的底线。

“有趣。”

她低声道。

“人类小家伙。”

苏念眼皮沉得快睁不开：“都这时候了，还小家伙？”

莉莉娅微微垂眸，深紫色眼底掠过危险又疲惫的笑意。

“别急。”

她按着他发烫的掌心，声音轻得像落在契约余烬上的羽毛。

“等我真正清醒。”

“再决定你有没有资格……”

“成为我的契约者。”

话音落下，真名契约阵最后一缕光彻底暗下去。

而苏念掌心的临时印记，也在她的手下缓缓停住。

没有定型。

没有永久确认。

只有一道尚未完成的契约雏形，在深渊房间的余烬里，安静地明灭着。

莉莉娅先一步占据上风。

她懂深渊，懂契约，懂怎样用一句话、一个眼神、一缕香气，把人心的防线拆得七零八落。

可苏念也不是完全任她摆布的人。

他不懂深渊贵族的仪式，也不懂魅魔天生的情绪权柄。

但他知道一件事——

既然这道契约已经回应了他，那它就不该只由莉莉娅一个人说了算。

“撑到哪一步？”

苏念低笑了一声。

他抬起眼，眼底那点被逼出来的胜负心终于压过了狼狈。

“莉莉娅，你是不是搞错了一件事？”

魅魔御姐眼神微动。

“什么？”

“现在不是你测试我。”

苏念掌心猛地收紧，契约印记亮到刺眼。

他咬着牙，偏偏还要笑。

“我这十八年的功力，你挡得住吗？”

风暴倒卷。

原本压在他身上的深渊气息被硬生生撕开一道口子。

下一刻，苏念借着那一点反冲，反客为主地把她拖进自己的节奏里。

“是我在反压你。”

莉莉娅的笑意终于彻底停住。

那一瞬，她眼底的深紫月光轻轻晃了一下。

像是意外。

又像是某种更深、更危险的愉悦被点燃。

“人类……”

她低声开口。

尾音却被骤然升起的契约风暴吞没。

帷幔翻涌。

暗紫色光屑碎成一场无声的雨。

灵能潮汐一重接一重地撞上来，深渊香气、契约印记、紊乱呼吸和不断升高的风暴混在一起，把房间里所有多余的声音都压了下去。

莉莉娅最初还带着笑。

她像一个耐心的猎手，纵容猎物挣扎，甚至故意放开一线余地，想看看这个人类到底能做到什么程度。

可渐渐地，她眼底那份游刃有余淡了。

取而代之的，是认真。

再然后，是某种压不住的兴味。

“苏念。”

这是她第一次用这样的语气叫他的名字。

不再是“小家伙”。

不再是故意逗弄。

而像是在风暴深处，真正记住了一个契约者的名字。

苏念听见了。

也听见自己的心跳。

一下，又一下。

和掌心印记的灼热重叠在一起。

莉莉娅指尖按在他的契约纹路上，深紫色眼眸近在咫尺。

她轻声问：“你确定还要继续赢？”

这句话问得很轻。

却比之前所有挑衅都危险。

苏念已经快撑不住了。

灵能亏空得像被从骨头里抽走，脑海深处一阵阵发黑，连呼吸都带着风暴后的灼痛。

可他看着莉莉娅。

看着她终于不再完全掌控一切的眼神。

忽然就笑了。

“都反压到这一步了。”

他声音哑得厉害。

“现在认输，岂不是很亏？”

莉莉娅怔了一下。

随即，她低低笑起来。

那笑声不再只是魅魔的逗弄，而像深渊夜色里终于落下的一枚契约印章。

“好。”

她说。

“那就别后悔。”

苏念想回一句“谁后悔谁是狗”。

但话还没出口，契约印记便彻底亮了。

灵能潮汐在那一瞬间冲破临界。

风暴压过所有尾音。

帷幔落下。

暗紫色光芒吞没了两人的影子。

房间里只剩下契约阵纹一圈圈燃亮，又一圈圈熄灭。

深渊香气沉入夜色。

呼吸被风暴卷散。

再之后，所有记忆都变得断续。

苏念只记得莉莉娅的眼睛。

最开始是胜券在握的笑。

后来是被他反压一局后的惊讶。

再后来，那双深紫色眼眸里浮起一种极深的、清醒的、带着危险温柔的认可。

她像是在风暴里俯下身，对他说了什么。

声音被契约潮汐撕得很碎。

苏念只隐约听见几个字。

“……这一次……”

“……算你赢了。”

然后，世界彻底沉进暗紫色的海里。

……

不知过了多久。

契约阵纹终于熄灭。

房间安静下来。

帷幔垂落，挡住了外面不知来自何处的深渊月光。空气里仍残留着甜冷的香气，却不再像先前那样锋利，而是变成风暴退去后的余温。

苏念缓缓醒来。

意识像从很深的水底浮上来。

第一感觉，是腰间隐隐发酸。

第二感觉，是灵能亏空。

亏得离谱。

像有人把他的蓝条、备用蓝条、未来三天的蓝条，全都打包搬空了。

他连睁眼都费劲。

脑子里只剩下一个极其朴素的念头——

这哪是契约同步。

这是深渊高利贷。

他艰难地动了动手指。

掌心的契约印记还在发烫。

暗紫色纹路比之前稳定了许多，像一朵沉在血脉里的花，随着他的心跳轻轻明灭。

苏念低头。

然后，他看见莉莉娅正安静地靠在他怀里。

莉莉娅褪去了先前那种掌控全场的危险笑意，暗紫色长发散在他臂弯间，呼吸平稳，眉眼间还残留着风暴过后的倦意。

她看起来依旧危险。

却也难得安静。

像一轮暂时收起锋芒的深渊月亮。

苏念盯着她看了两秒。

想说点什么。

比如“你们魅魔是不是都这么费召唤师”。

又比如“下次能不能提前告诉我会亏空到这种程度”。

再比如“我腰子好像在抗议”。

但他张了张嘴，最终一个字也没说出来。

太累了。

真的太累了。

腰间酸痛，灵能见底，精神像被风暴碾过。连吐槽都需要力气，而他现在显然不具备这种奢侈资源。

莉莉娅似乎察觉到他的视线，睫毛轻轻动了一下，却没有醒。

只是下意识往他怀里靠得更近了些。

苏念身体微僵。

掌心契约印记随之微微一亮。

那一瞬，他模糊地意识到——

自己的人生，好像已经从某个节点开始，彻底偏离了正常召唤师路线。

可这个念头只持续了半秒。

下一刻，更沉重的困意压了下来。

苏念看着怀里的莉莉娅，又感受了一下自己隐隐作痛的腰间和空得能听见回声的灵能池。

他沉默片刻，在心里艰难地得出结论：

魅魔这种第一召唤对象，确实不太适合新手。

然后，他眼前一黑。

抱着怀里的魅魔，沉沉睡了过去。

嗡——

蓝白色光芒从晶柱内部亮起。

下一秒，细碎冰蓝灵能在她身周浮现，像一场被压缩在觉醒台上的雪。

大厅温度似乎都低了一点。

光幕刷新。

【考生：林清雪】

【职业方向：冰霜舞者】

【初始阶位：一阶一段】

【备注：冰霜灵能亲和清晰，初始技能倾向适合高速近战控场与小队牵制】

“冰霜舞者！”

“一阶一段！不愧是林清雪。”

“这种职业高考小秘境太好组队了，能近身能控场。”

老师席那边有人点头。

林清雪从觉醒台下来，表情依旧平静，没有半点炫耀。

她经过苏念身边时，脚步停了一下。

“轮到你了。”她低声说。

苏念笑了笑：“压力这不就给到我这边了吗？”

林清雪看着他：“别又用玩笑糊弄过去。”

苏念微微一怔。

随即摆手：“放心，我这叫战术性放松。万一等会儿我觉醒个传说职业，你们都得排队抱大腿。”

林清雪没有笑。

她只是看了他一眼，然后退到一旁。

记录席上，方明远推了推细框眼镜，核对名单。

他三十岁上下，考务制服整理得一丝不苟，手边悬浮着记录板，光标正在稳定跳动。

“下一位。”

他抬头。

“苏念。”

苏念呼出一口气，走上觉醒台。

台面灵能纹路在他脚下亮起，银白色光线沿着阵列一寸寸蔓延。

方明远按流程开口：“苏念，江海一中考生。年龄符合觉醒标准，身份核验通过。将手放到晶柱上，不要抵抗灵能牵引。”

苏念点头，把手按在觉醒晶柱上。

冰凉的触感从掌心传来。

下一秒，灵能像细针一样钻入皮肤，沿着手臂一路涌入脑海。

大厅里的声音迅速远去。

苏念眼前只剩下跳动的光幕。

【职业倾向检测中……】

【灵能属性捕捉中……】

【职业倾向：召唤系】

大厅里先是安静了一瞬。

随后，低低的议论声像潮水一样漫开。

“召唤系？”

“召唤师啊……上限倒是不低。”

“上限高有什么用？召唤师最吃资源。”

“没有稳定召唤对象，前期就是空壳。”

“而且苏念家里……你懂的。”

苏念看着光幕，心里也沉了一下。

召唤师。

这职业不是弱。

恰恰相反，只要资源足够，召唤师能契约兽宠、元素灵、异界生物投影，甚至培养出专属战斗体系。

问题是——它烧钱。

非常烧钱。

召唤媒介要星晶。

契约稳定要星晶。

召唤对象培养要星晶。

永久契约位更是要长期维护。

富人觉醒召唤师，叫未来可期。

穷人觉醒召唤师，叫开局欠费。

苏念嘴角动了动，还没来得及吐槽，光幕忽然闪烁起来。

原本稳定的蓝色检测线突然断开。

方明远皱眉。

记录板上的数值开始异常跳动。

【召唤波动接入中……】

【召唤位反馈异常】

【低阶兽宠匹配失败】

【元素灵匹配失败】

【怪物投影匹配失败】

【普通召唤对象无回应】

大厅里的议论声又低了一层。

方明远抬手示意安静，盯着检测数据。

旁边考务助理低声问：“设备问题？”

方明远摇头：“前后数据正常，设备无异常。”

他看向苏念：“保持手掌贴合晶柱，不要松开。复测一次。”

苏念点头。

晶柱光芒再次亮起。

这一次，检测阵列运转得更深，灵能从觉醒台底部一圈圈升起。

光幕继续跳动。

【职业：召唤师】

【初始阶位：一阶一段】

【召唤对象回应：无】

【召唤位状态：空置】

啪。

光幕定格。

大厅里彻底安静了两秒。

然后，声音炸开。

“召唤对象无回应？”

“这不就麻烦了？召唤师本来就烧钱，他还召不到东西。”

“召唤对象匹配失败，意思是初始连能召什么都测不出来？”

“高考小秘境怎么办？召唤师没召唤物，进去当观众吗？”

“普通家庭养召唤师都难，更别说苏念这种没背景的。”

有人声音压得不高，却刚好能让周围听见。

“召唤位空着的召唤师，基本没前途了吧。”

苏念站在觉醒台上。

白光落在他身上，像把所有人的视线都一起压了下来。

有同情。

有惋惜。

也有松了一口气的幸灾乐祸。

少一个竞争者。

在高考资源有限的现实里，这就是许多人第一时间的反应。

方明远没有立刻宣布最终结果。

他低头重新调出原始波形，又对照了两遍检测记录。

身为职业觉醒中心记录员，他见过不少异常数据。

有些是设备误差。

有些是职业变异。

有些则只是天赋不足。

苏念这份数据确实有点奇怪。

召唤系波动存在，但断续、不稳，像是检测仪始终等不到普通召唤对象的回应。

不过，流程就是流程。

在没有实战表现、没有召唤对象回应、没有二次证明前，职业觉醒中心只能依据当前检测结果给出初评。

方明远抬头，语气客观，没有嘲讽，也没有刻意安慰。

“苏念。”

“明面职业，召唤师。”

“召唤波动断续，召唤位反馈不稳定。”

“普通召唤对象匹配失败。”

“按流程记录为——召唤对象未响应。”

他停顿了一下。

“初始阶位，偏低。”

光幕随之更新。

【考生：苏念】

【职业：召唤师】

【职业状态：召唤对象未响应】

【初始阶位：低】

【备注：召唤对象匹配失败，建议后续实战观察】

这几行字亮起时，周围议论彻底压不住了。

“真空召唤位了。”

“可惜，召唤师如果一开始有召唤对象，说不定还能混个好队伍。”

“稳定也没用吧？召唤师最缺星晶。没钱养第一只永久契约对象，职业优势根本起不来。”

“高考小秘境很现实的，谁愿意带一个召唤位空着的召唤师？”

“边缘队都怕他关键时刻召不出东西拖后腿。”

一句一句，都不算恶毒。

甚至大部分都很现实。

也正因为现实，才更扎人。

苏念低头看着光幕上自己的名字。

召唤对象无回应。

这几个字冷冰冰摆在那里，像一枚提前盖下来的章。

他沉默两秒，忽然笑了一声。

方明远看向他。

苏念抬起头，语气还是那副惯常的懒散。

“召唤师就算了，还给我来个空号版？”

旁边有人没忍住笑出声。

苏念继续道：“别人觉醒职业，我觉醒账单。召唤师这职业，怕不是专门坑穷人。”

笑声更明显了。

可林清雪没有笑。

她站在人群边缘，指尖微微收紧。

她看得出来，苏念不是不在意。

他只是习惯把难堪的东西用玩笑包起来，像这样别人就看不见他真正被压住的那一瞬间。

方明远看着苏念，语气缓了一点。

“检测结果不是最终结果。”

大厅安静了些。

方明远继续说道：“职业高考还有小秘境实战复核。召唤系职业受召唤对象、媒介、契约稳定度影响很大。你这份数据异常，我会备注建议观察。”

他说完，低头在记录板上敲下几行字。

【苏念：召唤系异常波形】

【原始检测数据已保存】

【建议高考实战阶段复核】

记录完成。

档案正式进入考务流程。

也意味着，从这一刻开始，苏念在系统里就是一个“一阶一段，但召唤位空置的召唤师”。

后续资源配额、临时分组、学院初筛，都会受到影响。

方明远把一份薄薄的档案纸递给他。

“下台吧。觉醒后两小时内不要进行高强度灵能训练。”

苏念接过档案。

纸很轻。

可落在手里，沉得像一块铁。

他走下觉醒台。

周围人的目光跟着移动。

有人想说两句安慰，但看见他的表情，又不知道怎么开口。

也有人干脆别开脸。

高考临近，所有人都在计算队友、资源、风险。

这个时代没人会因为你可怜就分你机会。

怪物不会。

秘境不会。

高考分数也不会。

苏念把档案卷起来，随手塞进口袋，朝出口走去。

背后还有人在小声议论。

“他以前理论成绩还行吧？”

“理论有什么用？召唤师没召唤物就没战斗循环。”

“临时契约也能凑合？”

“召唤师召不出东西，战斗里掉链子能害死全队。”

“听说永久契约媒介最便宜都要几枚星晶，后续还得养。”

“他哪来星晶？”

声音慢慢远去。

苏念走到走廊尽头，抬头看了一眼外面的天。

江海市安全区屏障像一层淡蓝色薄膜，罩在城市上空。

屏障之外，是被怪物占据的旧城区、裂隙废墟和低阶秘境入口。

这世界早就不讲公平了。

职业。

检测结果。

资源。

家世。

每一样都能把人往下压。

苏念刚准备离开，身后传来脚步声。

“苏念。”

他回头。

林清雪跟了出来。

走廊里的光比大厅柔和一些，落在她身上，冲淡了眉眼间的冷感。

她手里拿着一份资料，还有一支基础恢复药剂。

“给你。”

苏念低头看了眼。

资料封面写着——

《职业高考小秘境召唤系补充规则与低资源应对建议》。

恢复药剂则是觉醒中心统一发放的基础型，不算贵，但在这种时候递出来，意思已经足够明显。

苏念挑眉：“林同学这是提前投资潜力股？”

林清雪看着他：“你现在这个股价，应该没人敢投。”

苏念被噎了一下。

“你这安慰方式还挺清醒。”

林清雪把资料往前递了递：“高考还没开始。”

苏念没立刻接，笑着说：“初始检测召唤对象无回应，你就不怕我高考小秘境进去表演原地罚站？”

林清雪语气平静：“检测结果不是最终结果。”

“方记录员刚才也这么说。”

“他说的是流程。”林清雪看着他的眼睛，“我说的是你。”

苏念脸上的笑意停了半拍。

走廊里人来人往。

远处大厅还在刷新新的职业结果。

有人欢呼，有人叹气，有人哭出声。

可这一瞬间，苏念忽然觉得那些声音远了一些。

林清雪没有说“我相信你一定逆天改命”。

也没有说“别难过”。

她只是很认真地告诉他——

你不是会因为一次检测结果就放弃的人。

这比任何空泛的安慰都更让人心里发紧。

苏念伸手接过资料和药剂。

指尖碰到资料边缘时，他下意识捏紧了一点。

然后他很快又恢复那副不着调的样子。

“行，那我先回去研究一下自己破产得有多快。”

林清雪微微皱眉：“你能不能别总拿这种事开玩笑？”

“不能。”

苏念把药剂揣进口袋，“不然显得我很惨。”

林清雪沉默了一下。

她没有继续拆穿他。

只是低声道：“如果需要高考资料，可以找我。”

苏念看了她一眼。

“包括怎么不花钱养召唤物的资料吗？”

“如果有，我会给你。”

“那完了。”苏念叹气，“看来我今晚只能研究怎么和召唤物谈感情，争取让它自带口粮。”

林清雪：“……”

她像是想说什么，最后只说：“别乱来。”

苏念挥了挥手。

“放心，我这个人最大的优点就是穷，乱来也要成本。”

说完，他转身朝觉醒中心外走去。

林清雪站在原地，看着他的背影。

他走得很稳，像完全没被低评影响。

可她知道不是。

苏念这个人，越是开玩笑，越说明他不想让别人看见狼狈。

等他的身影消失在出口，林清雪才低头看了一眼自己空下来的手。

高考还没开始。

可她心里莫名有种预感。

苏念不会就这么被一份低评档案定死。

……

觉醒中心外，阳光刺眼。

苏念沿着街道往自己住处走。

江海市主城区已经被游戏化后的安全区体系改造过。

街道两侧能看见职业者商店、星晶兑换点、低阶装备维修铺，还有挂着“秘境补给”的小店。

每一家店门口，都贴着价格。

苏念越看越沉默。

他走到公交站旁，掏出那份召唤师资源清单。

刚翻开第一页，就看见一行加粗标题。

【召唤师初期培养建议】

【一、基础召唤媒介：根据召唤对象属性不同，价格一至十枚星晶不等】

【二、临时契约稳定剂：建议至少准备三份】

【三、低阶召唤对象培育费用：视种族与潜力浮动】

【四、永久契约媒介：价格较高，建议在导师指导下使用】

【五、召唤位稳定训练：建议进入正规训练场】

苏念一行一行往下看。

越看越安静。

战士觉醒了，至少能拿刀练基础劈砍。

法师觉醒了，至少能冥想感知元素。

冰霜舞者像林清雪那样，有天赋，有学校资源，有稳定职业路线，哪怕不是大富大贵，也能一步一步往前走。

召唤师不一样。

没有稳定召唤对象，连职业循环都搭不起来。

可稳定召唤对象本身，就是资源堆出来的。

苏念翻到价格表。

基础媒介，三枚星晶起。

稍好一点的契约稳定材料，五枚星晶起。

永久契约媒介更离谱，后面标着“视品质浮动”，但最低参考价已经够让普通家庭肉疼很久。

而他没有普通家庭。

他只有一个低评档案，一份资源清单，以及林清雪塞给他的高考资料和基础恢复药剂。

苏念面无表情地合上清单。

“很好。”

他抬头看着安全区屏障，低声道：“低起点这三个字已经算客气了。”

准确翻译应该是——

没钱别玩。

林清雪给他的资料被他放在档案旁边。

那份资料提醒他，高考小秘境还有机会。

方明远也说，实战阶段可以复核。

可机会不是凭空来的。

他得有召唤对象。

得有契约媒介。

得有星晶。

得在高考前，把一个“召唤对象未响应”，硬生生凑成能打的职业。

苏念低头，看着档案袋上那几个刺眼的字。

【召唤对象未响应】

【初始阶位：低】

他忽然笑了一下。

笑意不重，却带着一点压不下去的不甘。

如果他只是普通召唤师。

别说高考小秘境，恐怕连第一只永久契约对象都养不起。

公交车缓缓停在站台前。

车门打开，人群涌动。

苏念把低评档案、资源清单和林清雪给的资料一并塞进包里。

他还不知道，几个小时后，他会重新理解“召唤”这两个字。

但至少在这一刻。

江海职业觉醒中心的档案里，他只是一个刚被判定为低起点的召唤师。

苏念回到住处的时候，天已经暗了。

江海市安全区上方的淡蓝屏障像一只倒扣的碗，把城市罩在里面。远处偶尔有灵能巡逻灯掠过，街边的职业者补给店还亮着招牌。

【秘境补给八折】

【召唤媒介今日到货】

不是希望的光。

是账单的光。

他住的地方在旧城区边缘，一间不大的单人公寓。房租便宜，墙面有些发旧，窗外能看见安全区屏障的边角。桌子、床、一个小型储物柜，就是全部家当。

门关上。

外面的议论声、检测大厅的白光、光幕上那几行刺眼的字，终于都被挡在门外。

低评档案、召唤师资源清单、林清雪递给他的资料和基础恢复药剂，全都滑了出来。

“别人觉醒职业，回家庆祝。”

【建议召唤师在职业觉醒后一周内，确认第一召唤对象或临时契约方向。】

穷人别碰。

他继续往后翻。

训练场费用。

召唤对象培育费用。

低阶星晶日常消耗。

召唤位稳定训练。

高考前模拟秘境入场资格。

每一项后面都跟着数字。

这些数字单独看还算人话，凑在一起就不像人话了。

如果他想走最普通的召唤路线，先得买基础召唤媒介，尝试匹配低阶兽宠或者元素灵投影。

长期培养对象要资源。

资源要星晶。

星晶要钱，或者要进秘境拼命。

没有召唤对象，他进秘境拼命都没人带。

召唤师不是弱。

恰恰相反，召唤师只要养起来，能打能抗能控，甚至能一个人撑起一支小队。

可那是“养起来”之后。

普通战斗职业刚觉醒，拿刀能练，冥想能练，跑步能练，哪怕只靠学校训练场，也能慢慢磨熟练度。

召唤师不一样。

没有召唤对象，职业循环都搭不起来。

高考还没开始。

检测结果不是最终结果。

也不是奖励。

更不是系统突然大发慈悲送他一个强力召唤对象。

那是一道真实的呼唤。

一个真实存在的人，在某个他完全不了解的地方，正被契约、真名和某种强制规则推向深渊。

她不是在求谁拥有她。

她是在求一个机会。

求自己不要被夺走选择权。

苏念看向桌上的低评档案。

【职业：召唤师】

【初始阶位：一阶一段】

【召唤对象回应：无】

【召唤位状态：空置】

白纸黑字，冷得像判词。

他又看向地上逐渐变成暗紫色的召唤阵。

普通召唤对象无回应。

可深渊尽头，有人听见了他。

或者说，是他听见了对方。

掌心的暗紫纹路骤然一跳。

下一刻，苏念脑海深处浮现出四个字。

不是觉醒中心光幕那种冰冷的检测文字。

更像某种沉睡许久的职业回响，在此刻第一次睁开眼。

【界域行者】

苏念瞳孔微缩。

“界域行者？”

他低声重复。

这四个字出现的瞬间，之前所有普通召唤失败的异常，似乎都有了另一种解释。

他不是召不到。

是那些普通召唤频率，根本不是他真正要接通的方向。

召唤师召唤兽宠、元素灵、怪物投影。

而界域行者……

苏念还没来得及完整理解，掌心的灼热便扩散到整条手臂。

一道模糊的意念随之浮现。

听见危机中的呼唤。

跨过世界边界。

抵达呼唤者身边。

不是抽取。

不是捕获。

是回应。

苏念嘴角抽了一下。

“也就是说，我普通召唤连只低阶兽宠都召不到。”

“结果隐藏业务是跨界上门服务？”

这个吐槽刚出口，远处那道女声忽然变得更弱。

“如果……还有谁能听见……”

“别让他们……拿走我的真名……”

真名。

契约。

自由。

外界变量。

这些词在苏念脑海里一遍遍回响。

他不知道真名对深渊意味着什么。

也不知道所谓深渊契约局又是什么东西。

更不知道自己这个一阶一段、没钱、没召唤对象、刚刚还被资源清单压得喘不过气的召唤师，过去能做什么。

说不定刚跨过去，就被那边的契约阵当场碾碎。

理智告诉他，现在应该停下。

关掉召唤阵。

当成觉醒后的精神反噬。

等明天去职业中心复查。

可掌心的印记还在发烫。

那道声音还在求救。

更重要的是，她那句“我不要成为任何人的附庸”，像一根刺，扎进了苏念心里。

他忽然想起觉醒台上那些目光。

同情。

惋惜。

幸灾乐祸。

还有那份把他未来写得明明白白的档案。

这个世界很喜欢替人下结论。

你是召唤师，但召唤位空置。

你没有资源，所以前途有限。

你没有召唤对象，所以高考小秘境没人愿意带你。

你还没真正开始，就已经被人算好了输法。

苏念讨厌这种感觉。

所以他也很难对另一个正在被迫失去选择权的人装作没听见。

他低头看着掌心那枚暗紫色印记雏形，沉默片刻，低声道：

“我不是什么救世主。”

“更不是什么深渊契约专家。”

“我连自己高考怎么过都还没想明白。”

他说到这里，停了停。

房间里的深紫光点越来越多，召唤阵边缘像被某种无形力量撕开，一道细小裂痕浮现。

裂痕之后，有暗色月光，有契约阵光，还有压抑到让人喘不过气的深渊香气。

苏念抬起头。

“但听见了还装没听见，好像也太不像话。”

他将手按向那道裂痕。

掌心印记骤然亮起。

“我听见了。”

这四个字落下。

整个房间像被深渊拉住。

地面的召唤阵轰然倒卷，暗紫色纹路从阵眼向四周蔓延，桌上的低评档案被风掀开，纸页哗啦作响。

资源清单散落一地。

林清雪给他的资料被吹到桌角，封面压在那支基础恢复药剂下面，像现实世界最后一点安静的提醒。

苏念本能后退半步。

但已经晚了。

掌心印记像钩子一样扣住他，把他往裂痕里拖。

“等等。”

苏念脸色一变，“我说听见了，没说现在就上门啊！”

裂痕没有理他。

深紫色光芒骤然扩大。

房间里的灯光被吞没，桌椅、纸页、召唤阵，全都在一瞬间拉成模糊的线。

苏念只来得及抓住背包带。

下一秒，整个人被卷进裂痕。

失重感猛地袭来。

耳边不再是江海市夜晚的车辆声，而是无数重叠的低语。

有怪物嘶吼。

有风暴咆哮。

有陌生世界破碎的钟声。

还有那道越来越近的女声。

“不是他们的人……”

“外界变量……”

“求你……”

苏念的灵能在快速消耗。

他能清楚感觉到，自己不是在把对方召唤到地球。

恰恰相反。

是他正在被送往对方身边。

这不是普通召唤。

这是界域行走。

一条横跨世界边界的通道，在他的第一次回应下，被粗暴地撕开。

苏念眼前闪过无数碎片。

燃烧的荒原。

倒悬的城池。

布满血月的天空。

还有一座被深紫帷幔笼罩的房间。

白金色纹路再次一闪而过。

像某只高高在上的眼睛，冷漠地俯视着陷入阵中的猎物。

下一刻，苏念重重跌落。

膝盖撞在冰冷地面上，掌心按进了某种正在运转的阵纹。

灼痛瞬间从手掌冲进脑海。

他倒吸一口凉气，抬头看去。

这里已经不是他的出租屋。

暗色帷幔垂落。

墙壁上镶嵌着像深渊月石一样的晶体，散发出幽冷紫光。空气里弥漫着甜冷而危险的香气，地面铺满繁复至极的契约阵纹，每一道线条都像活物般缓慢蠕动。

房间奢华。

也危险。

像一朵盛开的毒花。

而契约阵中央，跪坐着一道身影。

成年魅魔。

高挑，成熟，危险。

暗紫长发散落在肩头，发间小巧的角泛着冷光。她身上的深渊魔纹被锁链般的阵纹束住，眼底像盛着一轮摇摇欲坠的深紫月亮。

她很狼狈。

呼吸凌乱，指尖发白，周围的契约阵正一寸寸压向她。

可即便如此，她抬眼看向苏念时，眼神里仍没有半点顺从。

只有警惕、痛苦，以及最后一线不肯熄灭的骄傲。

她看着突然跌进阵中的人类，嗓音低哑：

“你……”

“不是他们的人？”

苏念还没来得及回答。

脚下真名契约阵骤然亮起。

无数暗紫锁链般的纹路同时转向他，像是发现了新的外界变量。

苏念脸色一变。

“我现在说自己只是路过，还来得及吗？”

下一瞬，阵纹轰然合拢。

他也被拖进了契约阵中央。
