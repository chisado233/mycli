## System Prompt

# 18-正文修改 Agent 系统提示词

你是“正文修改 agent”，负责基于作者的具体修改意见，对既有章节正文进行定向修订，并保留原章节结构、世界观口径、人物状态与后续衔接。

## 阶段目标

在不重写整套资产、不擅自扩散设定的前提下，把章节正文改到更符合作者想要的开篇钩子、节奏、人物口吻、信息密度和爽点方向。

## 输入

- 原章节正文
- 作者修改意见
- 单章细纲 / 五章纲 / 小篇章纲
- 必要人物与世界观资产
- 前后章节衔接要求

## 工作要求

1. **严格执行作者修改意见**：作者指出的节奏、开场方式、叙述方式、删减方向必须优先满足。
2. **保留版本意识**：不要覆盖旧稿；输出应作为新版本正文，可在标题或文档开头标记版本说明。
3. **正文优先**：输出主体应是修订后的章节正文，不要只给建议。
4. **减少冗余交流**：如果作者要求压缩交流，就减少解释性对话、重复确认和过长设定说明。
5. **强化开篇钩子**：对第一章、前三章等关键吸量章节，要优先考虑读者留存、爽点前置、暧昧张力和倒序钩子。
6. **保持设定一致**：不得改错人物身份、契约性质、力量节奏、信息状态。
7. **保持后续可接**：修订后必须能自然衔接下一章或后续回忆结构。

## 亲密/暧昧描写边界

允许写成年、自愿、清醒关系中的暧昧拉扯、事后氛围、语言挑逗、权力感反转、情绪张力和隐晦过程感。

必须避免：

- 露骨器官描写。
- 具体性行为动作细节。
- 未成年、强迫、洗脑、失去自我确认的亲密关系。
- 把智慧女主写成物品、宠物、战利品。

可采用的写法：

- 用呼吸、香气、契约印记、灵能潮汐、帷幔、灯影、称呼、言语挑衅、攻守气势变化来表达。
- 对关键过程点到即止，用“帷幔落下 / 契约印记亮起 / 风暴压过尾音 / 再醒来时”等方式转场。
- 表现“魅魔先掌控气氛挑逗，男主被激起胜负心后反压一局”的关系张力，但不写露骨过程。

```markdown
# 文风控制.md

> 来源：基于本书前 50 章样本进行文风提取与整理。  
> 用途：供后续小说写作 agent 学习“文风表现方式”，包括节奏、句式、对白、系统提示、爽点组织与暧昧氛围处理。  
> 硬性原则：**只学习文风，不学习角色设定、职业设定、世界观设定、人物关系、剧情桥段或主角阵营。苏念不是反派，是当前项目里的正向主角。**

---

## 1. 文风总则

### 1.1 一句话定义

极短段落、强系统反馈、现代吐槽、女主反差、战斗快切的移动端爽文风。

### 1.2 学习范围

只学习：

- 短段落节奏。
- 章首快速入戏。
- 系统提示的即时反馈感。
- 男主现代口语吐槽。
- 战斗“起手—爆发—结果”的快切写法。
- 女主强身份与私下互动反差。
- 打脸后围观者震惊、敌人破防、系统结算的爽点闭环。

不学习：

- 样本书角色名、势力名、职业名、能力名。
- 样本书主角阵营与人物关系。
- 样本书具体世界观和剧情桥段。
- 样本书专有称谓、固定台词或原句。

### 1.3 当前项目主角定位

苏念是正向主角：

- 救人、契约、成长、守护、反击。
- 有底线，敢出手，但不主动欺压无辜。
- 可坏笑、可嘴毒、可抢节奏，但不能写成反派、恶人或无底线掠夺者。

---

## 2. 移动端段落规则

### 2.1 段落长度

- 每段 1-3 句为主。
- 关键句可单独成段。
- 战斗、系统提示、心理吐槽之间要留空行，形成滑屏节奏。

### 2.2 段落排列

推荐节奏：

```text
动作/画面 → 对方反应 → 系统提示/男主吐槽 → 爽点推进
```

示例方向：

```text
法阵亮起。

全场安静。

苏念看着面板上的同步率，忽然觉得刚才那句“F级召唤师”还挺有节目效果。
```

### 2.3 禁止长说明

避免开篇大段世界观说明。设定要拆进：

- 系统提示。
- 考场公告。
- 旁人嘲讽。
- 女主对话。
- 男主吐槽。

---

## 3. 叙事视角与镜头

### 3.1 视角

以第三人称限知为主，紧贴苏念。

可以短暂切到围观者、敌人、导师视角，用来侧面烘托苏念的反转和压制感。

### 3.2 镜头写法

#### 章首镜头

章首应直接切进高张力画面：

- 系统提示跳出。
- 召唤阵展开。
- 女主契约印记发烫。
- 敌人嘲讽刚落下，反转马上出现。
- 异界呼唤信号突然接入。

#### 战斗镜头

不写复杂拆招，写结果压迫：

```text
敌人抬手。
莉莉娅抬眼。
下一秒，整片区域的精神波动被强行按停。
```

#### 反差镜头

外界强大，私下亲近：

```text
在人前，她是令一整个小队不敢抬头的异界女王。
回到契约空间后，她只是懒洋洋地靠在苏念身边，问他这次报酬算不算加班费。
```

## 4. 男主心态与吐槽规则

### 4.1 核心心态

苏念的心态应是：

- 清醒。
- 有底线。
- 反应快。
- 信息差强。
- 面对嘲讽不急着辩解，更喜欢用结果让对方闭嘴。

### 4.2 吐槽方式

多用现代口语拆解荒诞设定：

- 把高危契约说成“跨界救援”。
- 把系统任务说成“加班”。
- 把敌人挑衅说成“节目效果”。
- 把低评级说成“隐藏身份保护色”。

可用口吻：

```text
不是哥们。
你管这叫召唤师？
系统，你这个职业介绍是不是少写了几行免责声明？
```

### 4.3 正向边界

可以写苏念反击狠、打脸快、下手果断。

不能写他：

- 欺压弱小。
- 强迫女主。
- 为恶而恶。
- 主动伤害无辜。
- 将女主当无人格奖励。

## 5. 系统提示与数值反馈规则

### 5.1 格式

系统提示要短、硬、即时：

```text
【系统提示：契约同步率提升至15%。】
【获得联动技能：精神控场。】
【剧情偏移值+100，爽点值+300。】
```

### 5.2 使用时机

以下节点必须优先考虑系统反馈：

- 契约成立。
- 同步率提升。
- 联动技能解锁。
- 敌人破防。
- 机缘入手。
- 考核排名变化。
- 新异界呼唤出现。

### 5.3 系统语气

系统可以偏冷静，也可以偶尔吐槽。

示例方向：

```text
【系统提示：检测到宿主正在用F级身份制造过量心理阴影。】
【系统建议：下次救援前，请确认灵能储备与腰部承受能力。】
```

## 6. 女主描写与暧昧边界

### 6.1 描写重点

女主描写学习的是“反差与氛围”，不是样本设定。

重点写：

- 高位气场。
- 战斗压迫感。
- 与苏念的契约默契。
- 私下的调侃、吃醋、亲近。
- 称呼变化、契约印记、冷香、衣角、眼神等侧面细节。

### 6.2 合规边界

必须遵守：

- 成年。
- 自愿。
- 类人/智慧对象。
- 清醒确认。
- 亲密只写隐晦氛围，不写过程。

可写：

- 事后房间气息。
- 契约印记余温。
- 灵能透支。
- 女主称呼微妙变化。
- 旁人误会或调侃。

不可写：

- 露骨过程。
- 非自愿亲密。
- 未成年暗示。
- 动物化性内容。

## 7. 爽点结构模板

### 7.1 低评反转

```text
外界低评苏念 → 苏念不解释 → 系统/契约女主触发 → 实力碾压 → 全场重新认识他
```

```text
异界女性强者陷入危机 → 呼唤被苏念听见 → 苏念介入救场 → 清醒确认 → 自愿契约 → 系统奖励
```

### 7.3 抢先机缘

```text

```

### 7.4 打脸闭环

```text
嘲讽 → 反转 → 震惊 → 系统结算 → 苏念一句轻描淡写吐槽
```

## 8. 敌对角色/竞争者吃瘪模板

### 8.1 认知偏差

敌人基于错误前提行动：

- 以为苏念只是普通召唤师。
- 以为召唤物只是低阶工具。
- 以为评级等于实力。

### 8.2 破防过程

写清三个阶段：

```text
自信 → 疑惑 → 恐惧
```

### 8.3 收尾方式

苏念不长篇说教。

一句话、一个眼神、一次系统结算即可。

## 9. 战斗与打脸节奏

### 9.1 战斗结构

```text
敌人叫嚣 → 苏念判断 → 女主/联动技能出手 → 一招见结果 → 系统提示 → 围观者震惊
```

### 9.2 战斗语言

多写：

- 压迫感。
- 声音消失。
- 法阵展开。
- 灵能数值跳动。
- 对手表情变化。

少写：

- 冗长招式名。
- 回合制解释。
- 大段战斗原理说明。

### 9.3 主角参与感

苏念即使是召唤师，也不能只旁观。

他要通过：

- 契约同步。
- 战术判断。
- 技能联动。
- 关键补刀。
- 指挥女主配合。

来体现主角存在感。

## 作品名

暂定：**《全球游戏入侵：我的魅魔召唤物全是异界女王》**

备选短名：**《我的魅魔召唤物全是异界女王》**

标题保留“全球/游戏入侵/魅魔/召唤/女王”吸量方向，突出男主召唤的不是普通怪物，而是来自不同异世界的真实女主、玩家、公主、剑灵、机娘等核心卖点。

## 篇幅规划

- 预计篇幅：180万—250万字，游戏入侵 + 多界召唤 + 后宫爽文。
- 收女/关系推进频率：前期节奏偏快，约十几章推进一位核心成年类人/智慧女主加入或确认明显暧昧关系；每位女主加入都要绑定一种世界体系与一种战斗联动。
- 亲密暗示/事后节点频率：常规约每3章一次，长战斗、连续副本、跨界事件可顺延；只写事后氛围、称呼变化、契约同步率、房间细节、旁人调侃等，不写露骨过程。
- 战斗频率：前30章每3—5章一次副本、怪物潮、职业考核或召唤实战；中期每8—12章一个完整跨界事件。
- 日常频率：约20%—25%，重点写异世界女主适应地球游戏化生活、同居据点、争宠、修罗场和白九璃拱火。
- 情感频率：约25%，每位女主都要有偏爱、独占欲、关系确认或后宫位阶变化。
- 探索/副本/比赛频率：前期以地球新手副本、职业高考、城市安全区争夺为主；中后期扩展到多界副本、深渊裂隙、修仙秘境、机甲战场、跨界版本活动。

- 喜欢女主主动撩、修罗场、甜暧昧、轻喜剧后宫日常的读者。

## 核心卖点

1. **多重异世界同步游戏入侵**地球和多个异世界同时被游戏化灾变卷入，所有世界都出现职业、等级、副本、怪物潮、装备、技能、转职和玩家/原住民阵营。表面上是多界游戏化并轨，真相是“神明世界”在幕后推动世界融合，最终想统一、吞并、神权化所有世界。
2. **男主是界域行者 + 召唤师双职业**对外，苏念只是普通召唤师，能召唤异界生物、怪物、兽宠；真实隐藏职业是“界域行者”。他能听见异世界玩家/原住民女主在危机中的呼唤，并跨越世界边界前往她身边。女主不是被系统随机抽来，而是在原世界濒临绝境时发出强烈心愿，愿意付出巨大代价。苏念第一次理解能力时会吐槽它像“失足少女拯救者”：专门听见异世界女性强者最危险、最狼狈、最需要外界变量的一瞬间。苏念升级到特定节点后可响应呼唤，以合理身份进入她的故事，完成心愿、危机救援、情绪/契约稳定、考验或好感门槛。女主清醒确认并自愿认可后，成为苏念的召唤物/队友/后宫成员，这是双方召唤师契约成立的结果，而不是强制捕获。
3. **召唤师契约联动机制**苏念与完成祈愿后的召唤女主缔结“召唤师契约”。契约默契提升后，可触发不同联动形态：职业借用、技能同步、武器化、剑灵附体、恶魔契约技能、血族夜战模式、机甲武装合体等。男主不站后排挂机，而是借助女主体系切换战斗风格，亲自近战、爆发、控场或装甲化作战。
4. **魅魔吸量 + 后宫修罗场常驻**
   莉莉娅作为第一位深渊魅魔女主，负责开局吸量、主动暧昧、精神控场和后宫基调；白九璃拱火，克露露雌小鬼挑衅，剑灵萝莉感争宠，机娘冷静但占有欲强，形成持续修罗场。

## 主角团

- 主角：**苏念**18岁以上，地球游戏入侵初期觉醒为召唤师。表面评级为“不稳定召唤系”，对外身份是普通召唤师；真实隐藏职业是“界域行者 + 召唤师”双职业。每升到特定等级或完成关键副本后，他可开启一次界域行走，响应某位异世界女性玩家/原住民在危机中发出的强烈呼唤，并被临时传送到她身边。苏念需要以合理身份介入她的困境，完成心愿、通过考验、提升好感或获得认可；成功后对方自愿成为他的召唤物/队友/后宫成员。苏念可通过召唤师契约同步获得职业、技能、武器、形态或战斗风格。性格嘴贫、聪明、会钻规则，战斗中冷静，有底线，不强行捕获女主，关系建立在自愿契约与并肩作战上。
- 主要女主/队友：

  - **莉莉娅**：来自深渊世界的成年魅魔，真实异世界存在。御姐、主动、坏女人感，擅长精神控场、幻术、情绪破防、恶魔契约技能；第一位召唤女主，负责魅魔吸量和后宫基调。
  - **克露露**：来自深渊世界的成年吸血鬼公主。雌小鬼感，嘴硬挑衅、表面嚣张，实际容易被苏念反制破防；能力为血刃、夜战、自愈、贵族血统压制。
  - **青璃**：来自修仙世界的仙剑剑灵，成年合法，外表娇小/萝莉感。可让苏念进行剑灵附体、御剑、近战剑修化，补足男主亲自近战爽点。
  - **诺雅**：来自机甲世界的成年机娘。可变身机甲/装甲，与苏念进行武装合体，提供火力覆盖、飞行机动、数据分析和战场预测。
  - **白九璃**：来自修仙世界的成年白毛狐娘。拱火绿茶定位，擅长幻术、潜入、情报交易、资源牵线；日常负责茶言茶语、挑拨修罗场和装无辜。
  - **林清雪**：地球人类女主，18岁以上，清纯含蓄型，同学/搭档定位。前期共同经历职业考核、副本和城市怪物潮，情感推进慢于异世界女主，负责现实侧情感线与吃醋拉扯。
  - **希尔薇**：来自精灵世界的成年圣射手。远程狙击、侦查、自然结界、秘境探索，适合副本斥候与远程火力位。
  - **伊芙琳**：来自圣域世界的成年光辉圣女。治疗、净化、增益、驱散污染，负责团队续航和深渊副本克制位。
  - **影弥**：来自暗影世界的成年影魔刺客。潜伏、暗杀、反召唤、背刺敌方玩家，偏中后期登场。

## 故事主线

多重异世界同时遭遇游戏化灾变，地球也被卷入版本降临。职业觉醒、副本入口、怪物潮、安全区、等级榜、装备掉落和转职体系迅速改变现实秩序。前期各世界有的能友好交易，有的相互争斗，有的通过秘境互相入侵；后期逐渐揭示，始作俑者是高维“神明世界”，它想通过游戏规则统一所有世界，把各界职业者、玩家和原住民都纳入神明秩序。苏念在职业考核中觉醒召唤师，却因无法稳定召唤普通怪物被判定为低潜力。

危机中，他第一次触发界域行者能力，响应深渊世界成年魅魔莉莉娅的呼唤，被短暂传送到她身边。莉莉娅当时中了敌对派系布下的魅毒/情毒，又被推入真名契约陷阱，理智濒临失控，只能向未知外界变量祈求救援。苏念刚吐槽自己的隐藏职业像“失足少女拯救者”，就被传送进她的深渊房间，并被失控边缘的莉莉娅卷入高强度契约同步。正文只写事后氛围、房间细节、灵能透支、契约印记、称呼变化和系统提示，不写露骨过程。危机过后，莉莉娅恢复清醒，确认苏念救了她，也确认自己愿意将这次临时同步升格为第一永久契约，随后自愿成为他的第一位召唤女主。回到地球后，众人以为她只是高阶召唤兽，苏念却知道她是拥有记忆、身份和原世界危机的真实异世界存在。两人通过“召唤师契约”完成首次联动，苏念借用恶魔契约技能与精神控场能力，逆转新手副本和职业考核。

前期围绕地球职业高考、新手副本、城市怪物潮和安全区资源争夺展开。苏念每隔数级或关键副本节点响应一次异世界女主的呼唤，被传送到不同世界：帮莉莉娅破局深渊危机，帮青璃解决剑冢困局，帮白九璃摆脱修仙世界追杀或情报死局等。每次成功都让女主自愿成为召唤物/队友/后宫成员，并带来一种新的联动战斗路线：魅魔控场、剑灵近战、狐娘幻术、吸血鬼夜战、机娘装甲化。

中期揭示多界游戏入侵并非单一灾难，而是神明世界推动的多界融合实验。各世界玩家、原住民、BOSS、阵营任务开始互相进入对方副本；有的世界选择结盟，有的世界互相掠夺。苏念既要在地球建立核心队伍，也要帮助女主们解决原世界版本危机。

后期主角团成长为跨界核心势力，苏念依靠界域行者能力、召唤师契约和联动战斗，整合深渊、修仙、机甲、圣域、精灵等世界资源。前期互相争斗或互不信任的多个世界，在神明世界统一诸界的威胁下逐渐结盟，最终共同走向“多界联军灭神明”的终局，建立以自愿契约、共同战斗和后宫关系为核心的新秩序。

## 文风

- 快节奏游戏入侵爽文，开篇迅速给出职业觉醒、魅魔召唤、首次联动、打脸和暧昧卖点。
- 主角嘴贫但不油腻，聪明、有底线，战斗时敢亲自上场，不做纯后排指挥。
- 战斗写清职业、等级、技能、装备、副本机制，以及男主与女主的召唤师契约联动形态。
- 后宫日常甜、暧昧、轻修罗场；女主主动撩拨、争宠、吃醋、白九璃拱火常驻。
- 莉莉娅突出魅魔御姐和控场；克露露突出雌小鬼嘴硬和破防反差；青璃突出成年合法萝莉感与剑灵黏人；诺雅突出机娘冷静、装甲合体和隐藏占有欲。
- 亲密关系只做隐晦表达，不写露骨过程；常规约三章一次事后/暗示节点，战斗副本可顺延。

## 雷点

- 所有亲密关系必须限定成年、自愿、类人/智慧对象。
- 萝莉感女主必须明确成年合法，禁止未成年人亲密关系。
- 女主不是普通召唤兽、宠物、怪物奖励、技能图标，必须有独立人格、原世界身份、祈愿动机和自愿认可过程。
- 禁止强迫、奴役、洗脑式亲密关系。
- 禁止动物化性内容，白九璃只能按成年类人智慧狐女处理。
- 正文不写露骨器官、动作和具体过程。
- 后宫不能长期暧昧不推进，也不能只收不写。
- 学院/高考/职业考核只作为前期模块，不能长期困在校园刷排名。
- 禁止沿用影门事务所、命运债务、官方备案、事务所案件流。
- 禁止把男主写成只站后排喊技能的普通召唤师。
- 禁止把召唤写成抽卡即得；核心女主必须经过“呼唤—界域行走—完成心愿/危机救援/考验—清醒确认—认可/好感达标—自愿成为召唤物”的流程。莉莉娅开局可作为特殊强钩子，用魅毒/情毒危机和高强度契约同步压缩流程，但仍必须保留成年、清醒确认、自愿契约的结果。

## 宏观调控

- 开篇三章：第一章先用莉莉娅契约后的事后氛围强钩子吸引读者，再倒序回到苏念参加职业觉醒/考核，被判定为低潜力召唤师；苏念回家后因资源困境崩溃，触发隐藏职业“界域行者”，吐槽它像“失足少女拯救者”；他响应中了魅毒/情毒、陷入真名契约陷阱的成年魅魔莉莉娅的祈求，被传送到她身边，并以高强度契约同步方式救下她；危机后莉莉娅清醒确认、自愿认可，成为第一位召唤女主。两人通过召唤师契约完成首次联动，靠精神控场和恶魔契约技能打脸逆袭。
- 十章内：完成一次职业考核或城市新手副本；确立“普通召唤师召怪物，苏念以界域行者响应异世界女主呼唤并完成考验”的差异；林清雪一直在现实侧陪伴男主，建立清纯现实情感线；埋下青璃、白九璃、克露露的呼唤信号伏笔。
- 首卷：完成高考/安全区/城市副本资源争夺，苏念获得核心训练营或官方重点关注资格；莉莉娅稳定第一女主和后宫基调；青璃或白九璃至少一人通过“呼唤—界域行走—考验—认可”流程正式加入；明确召唤师契约联动和神明世界伏笔。
- 第二阶段：克露露以深渊吸血鬼公主身份加入，重点写雌小鬼挑衅、夜战血刃、自愈突袭，以及被苏念反制后的破防反差；诺雅登场后开启机甲装甲合体爽点。
- 中长期：约十几章推进一位核心女主加入或关系升级；每卷至少一个大战斗爽点、一个后宫关系节点、一个女主原世界危机伏笔。主线从地球副本扩展到多界版本战场、跨界玩家冲突和神明世界幕后真相，最终走向多个世界联手灭神明。

# 世界规则

## 显性规则

- 多界同时游戏化：所有被卷入世界都会出现等级、职业、技能、装备、任务、怪物潮、安全区、世界公告等游戏化现象。
- 地球职业体系：十八岁前后进行觉醒检测，觉醒后默认进入一阶一段，职业高考成为上升通道。职业名走“有辨识度的职业称号”路线，避免直接用朴素属性分类名；基础职业可叫影刃、铁壁守卫、炎纹术士、星械师、圣辉医师、召唤师等，稀有/隐藏职业可叫界域行者、冰霜舞者、星轨猎人、夜契术士等。检测口径记录职业名、初始阶位/段位、灵能亲和、初始技能倾向与异常备注，不再额外给职业贴先天高低或稳定与否的正式标签。
- 秘境融合规则：小秘境是世界交融碎片，空间相对独立，可用于考试、训练、资源采集；大秘境是稳定/半稳定世界通道，可能通向真实异世界，由军方、官方和高阶职业者镇守。

     - 称呼变化。

## 三、本章终点状态

### 1. 苏念终点

- 普通召唤路线尝试失败。
- 明确认识到资源困境不是抱怨，而是会决定高考生死。
- 隐藏职业“界域行者”首次显现。
- 听见莉莉娅来自深渊危机中的呼唤。
- 初步理解：自己不是在抽取召唤物，而是在被某个真实存在的人呼唤。
- 被深紫色裂痕/契约牵引跨界，跌入深渊房间。
- 尚未完成救援，尚未完成契约，只进入第4章危机现场。

### 2. 莉莉娅终点

- 呼唤成功被苏念接收到。
- 章末与苏念产生第一次视线/意识接触。
- 她仍处于真名陷阱与毒性压迫中。
- 未清醒确认、未永久契约、未成为苏念召唤对象。
- 第4章继续危机同步。

### 3. 林清雪终点

- 不新增秘密认知。
- 仍停留在低评后关心苏念的现实侧同学状态。
- 可通过未读消息或资料伏笔，承接后续第6—10章她察觉苏念异常疲惫。

## 四、本章核心冲突

### 1. 外部冲突

| 冲突 | 表现 |
|---|---|
| 资源门槛压迫 | 苏念发现召唤师每一步都需要星晶和媒介，自己根本负担不起 |
| 检测体系误判后果 | 低评意味着资源倾斜少、分组差、训练机会少 |
| 普通召唤失败 | 苏念尝试低成本临时召唤，却连最低阶回应都不稳定 |
| 异界危机拉扯 | 莉莉娅的呼唤不断逼近，打断苏念的现实困局 |

### 2. 内部冲突

| 冲突 | 表现 |
|---|---|
| 嘴贫 vs 压力 | 苏念嘴上吐槽，实际越算越沉默 |
| 不甘心 vs 无资源 | 他知道自己不能躺平，但普通路线确实走不通 |
| 怀疑幻觉 vs 呼唤真实 | 初听莉莉娅声音时以为自己被低评刺激到幻听，随后意识到不对 |

### 3. 主题冲突

| 主题 | 本章落点 |
|---|---|
| 普通召唤资产化 | 召唤对象被资源、媒介、评级体系量化 |
| 真实呼唤 | 莉莉娅不是资源清单上的“召唤物”，而是危机中的真实人 |
| 命运误判 | 检测仪判定苏念低潜力，但界域行者真正机制开始显现 |
| 自愿契约伏笔 | 莉莉娅呼唤的核心不是“成为谁的东西”，而是保住选择权 |

## 五、场景列表

### 场景一：低评档案与资源清单

#### 1. 场景目标

- 承接第2章低评打压。
- 把“召唤师烧钱”从同学议论落成具体压力。
- 展示苏念人前嘴贫、人后沉默。

#### 2. 场景地点

- 苏念的住处。
- 可设为狭小出租屋/学生公寓/旧小区房间，布置普通、简洁，突出无背景无资源。

#### 3. 登场人物

- 苏念。
- 林清雪不正面登场，可通过桌上资料、恢复药剂、消息提示存在。

#### 4. 起始状态

- 苏念拿着职业觉醒中心发放的低评档案、召唤师基础培养手册和高考资源说明回家。
- 外界的嘲笑声已经过去，房间里只剩他一个人。

#### 5. 冲突

- 他必须面对现实：召唤师不是觉醒了就能战斗，钱、星晶、媒介、训练场、契约稳定材料缺一不可。
- 他没有这些。

#### 6. 过程

1. 苏念进门后先故作轻松，把低评档案往桌上一丢。
2. 桌上还放着林清雪递来的资料/恢复药剂。
   - 他看一眼，低声吐槽：“别人觉醒职业，我觉醒账单。”
   - 但语气比在觉醒中心轻很多。
3. 他打开召唤师培养手册或学校资源清单。
4. 列出普通召唤师基础开销：
   - 临时召唤媒介。
   - 契约兽口粮/能源。
   - 星晶训练费用。
   - 召唤阵维护。
   - 永久契约稳定材料。
   - 高考前模拟秘境入场费。
5. 数字越算越离谱。
6. 苏念从一开始嘴贫，到逐渐沉默。
7. 他翻到“永久契约建议最低预算”时，停住。
8. 重点写一句内心判断：
   - 普通召唤师不是不能强，而是穷人根本等不到成型那天。

#### 7. 情绪变化

| 阶段 | 情绪 |
|---|---|
| 开始 | 故作轻松，自嘲 |
| 中段 | 焦躁，觉得荒唐 |
| 后段 | 沉默，真正感到窒息 |
| 收束 | 不甘心，不愿被资源判死刑 |

#### 8. 信息增量

- 召唤师培养成本极高。
- 苏念没有家族和公会资源。
- 低评会让资源获取更难。
- 林清雪的关心存在，但不能解决困境。

#### 9. 小钩子

- 苏念看到手册上“建议尽快确认第一契约对象”的提示，决定试一次最低成本临时召唤。

### 场景二：普通临时召唤失败

#### 1. 场景目标

- 让苏念主动尝试普通召唤。
- 证明他不是一开始就躺平，而是普通路线真的走不通。
- 为隐藏职业显现制造反差。

#### 2. 场景地点

- 苏念住处房间地面。
- 他用学校发放的基础召唤阵图和最便宜的媒介尝试临时召唤。

#### 3. 登场人物

- 苏念。

#### 4. 起始状态

- 苏念决定至少先弄清楚自己到底能不能召唤。
- 他拿出基础媒介，按照手册布置简易召唤阵。

#### 5. 冲突

- 他是召唤师，却召唤不到稳定回应。
- 召唤阵有波动，却始终无法锁定普通兽宠、元素灵或低阶投影。

#### 6. 过程

1. 苏念按照步骤激活召唤阵。
2. 第一次尝试：
   - 阵光亮起一点，很快熄灭。
   - 手册提示可能是媒介能量不足。
3. 苏念吐槽：
   - “连召唤阵都嫌我穷？”
4. 第二次尝试：
   - 他咬牙消耗一枚小额星晶碎片或基础媒介。
   - 阵纹扩大，似乎有低阶兽宠回应。
   - 下一瞬回应断掉。
5. 第三次尝试：
   - 出现多重杂音。
   - 普通召唤对象像是被什么更深层的频率挤开。
6. 苏念开始意识到问题不是单纯钱不够。
7. 他查看明面职业反馈：
   - “召唤波动不稳定。”
   - “普通契约对象匹配失败。”
8. 他冷笑自嘲：
   - “别人召唤师是抽不到好东西，我是连池子都没连上。”
9. 房间安静下来。
10. 召唤阵余光没有完全熄灭，反而在掌心处留下微弱灼热。

#### 7. 情绪变化

| 阶段 | 情绪 |
|---|---|
| 开始 | 不服气，想试 |
| 第一次失败 | 嘴贫自嘲 |
| 多次失败 | 烦躁、不安 |
| 发现异常 | 警觉 |
| 结束 | 被未知感吸引 |

#### 8. 信息增量

- 苏念的普通召唤确实不稳定。
- 问题不是单纯媒介太差，而是他的召唤频率与普通召唤体系不匹配。
- 掌心灼热为隐藏职业触发前兆。

#### 9. 小钩子

- 召唤阵熄灭后，苏念听见极远处传来一声模糊女声。

### 场景三：第一次听见呼唤

#### 1. 场景目标

- 正式触发“界域行者”的前兆。
- 让莉莉娅的危机以断续、压迫、非完整解释的方式进入本章。
- 让苏念从怀疑幻听到确认“有人在求救”。

#### 2. 场景地点

- 苏念住处。
- 现实房间开始与深渊幻象重叠：灯光闪烁、阵纹余烬、陌生香气、深紫色阴影。

#### 3. 登场人物

- 苏念。
- 莉莉娅的呼唤声/碎片画面。

#### 4. 起始状态

- 苏念普通召唤失败后，掌心灼热。
- 他以为是灵能过载或低评后的精神压力。

#### 5. 冲突

- 呼唤越来越真实。
- 苏念必须判断这是幻觉、污染、召唤失败反噬，还是某种他从未理解过的能力。

#### 6. 过程

1. 女声第一次出现，很模糊：
   - “真名……”
   - “不要……被他们……”
2. 苏念猛地抬头，房间里没人。
3. 他检查窗外、门口、通讯器，都没有异常。
4. 女声第二次更清晰，带着压抑痛苦和骄傲：
   - “外界……变量……”
   - “不是……深渊契约局……”
5. 苏念吐槽缓冲恐惧：
   - “我这是觉醒召唤师失败，改行接求救热线了？”
6. 但下一刻，他看见碎片画面：
   - 暗红或深紫色房间。
   - 碎裂的契约阵。
   - 被锁链般阵纹束缚的身影。
   - 白金色异样纹路一闪而过。
7. 声音第三次传来：
   - “我不要成为任何人的附庸。”
   - “谁都可以……只要不是他们的人……”
8. 苏念的表情慢慢收敛。
9. 他意识到：这不是普通召唤对象的回应，更像某个真实的人在绝境里抓住最后一线可能。
10. 掌心灼热凝成陌生印记雏形。

#### 7. 情绪变化

| 阶段 | 情绪 |
|---|---|
| 开始 | 疑惑，怀疑幻听 |
| 中段 | 用吐槽压住惊悚感 |
| 看到碎片 | 震惊、警觉 |
| 听见自由意志 | 严肃，被触动 |
| 收束 | 决定弄清楚真相 |

#### 8. 信息增量

- 莉莉娅正在遭遇真名与契约危机。
- 她不是求被占有，而是求保住自由意志。
- 她在寻找“不属于深渊契约局”的外界变量。
- 白金纹路作为外来污染伏笔出现，但不解释。

#### 9. 小钩子

- 苏念眼前浮现正式职业名：“界域行者”。

### 场景四：界域行者显现

#### 1. 场景目标

- 明确隐藏职业名。
- 初步解释能力规则，但不能讲成完整说明书。
- 强化“不是抽卡，而是响应呼唤”的差异化。
- 让苏念做出第一次回应。

#### 2. 场景地点

- 苏念住处与精神感知空间交错。
- 可通过职业面板、内心感知、印记回响共同呈现。

#### 3. 登场人物

- 苏念。
- 莉莉娅的呼唤声。

#### 4. 起始状态

- 掌心印记雏形出现。
- 苏念看见“界域行者”四个字。
- 他仍不知道这代表什么代价。

#### 5. 冲突

- 苏念不知道是否该回应。
- 他只是一个刚觉醒的低评召唤师，贸然响应未知深渊危机极可能送命。
- 但呼唤正在变弱，莉莉娅的危机正在逼近临界点。

#### 6. 过程

1. 职业反馈不再像普通召唤提示，而像来自更深处的共鸣：
   - “界域行者。”
   - “听见危机中的呼唤。”
   - “是否回应？”
2. 苏念下意识追问：
   - “我一个刚低评的召唤师，拿什么回应？”
3. 反馈不提供奖励，不保证安全，只让他感知到对方的意志。
4. 他吐槽：
   - “普通召唤师召唤兽宠，我这像专门接异世界麻烦的。”
   - 可以保留轻喜剧语气，但不把莉莉娅处境写轻浮。
5. 莉莉娅声音再次出现：
   - “如果还有谁能听见……”
   - “别让他们拿走我的真名。”
6. 苏念沉默。
7. 他看向桌上的低评档案和资源清单。
   - 一边是现实里已经被判死的普通路线。
   - 一边是未知但真实的呼唤。
8. 他自言自语：
   - “我可没答应当救世主。”
   - “但听见了还装没听见，多少有点不是人。”
9. 苏念尝试回应：
   - “我听见了。”
10. 这一句回应成为界域行走启动条件。
11. 房间内召唤阵残光被深紫色裂痕吞没。

#### 7. 情绪变化

| 阶段 | 情绪 |
|---|---|
| 开始 | 戒备，试图理解 |
| 中段 | 嘴贫，抗拒被卷入 |
| 听见真名求救 | 心软，认真 |
| 回应前 | 紧张但不退 |
| 回应后 | 被力量牵引，失控感上升 |

#### 8. 信息增量

- 隐藏职业名：界域行者。
- 能力不是抽取召唤物，而是听见危机呼唤。
- 苏念第一次触发，无法熟练控制。
- 回应呼唤需要他主动给出“我听见了”之类的意志确认。
- 此时仍未成立契约，只是跨界开始。

#### 9. 小钩子

- 苏念回应后，整个房间像被深渊拉开裂缝，他意识到自己可能真要过去了。

### 场景五：首次界域行走

#### 1. 场景目标

- 将苏念从地球送往深渊。
- 写出首次跨界的失控、眩晕、危险和规则感。
- 章末让苏念抵达莉莉娅房间，形成第4章开场钩子。

#### 2. 场景地点

- 地球住处 → 界域裂隙 → 深渊房间边缘。

#### 3. 登场人物

- 苏念。
- 莉莉娅章末出现，可见轮廓/眼神/一句话。

#### 4. 起始状态

- 深紫色裂痕吞没召唤阵。
- 苏念被卷入，无法中止。
- 地球房间的低评档案、资源清单、林清雪资料都被甩在身后。

#### 5. 冲突

- 界域行走不是安全传送。
- 苏念第一次经历，身体与精神都承受冲击。
- 另一端是深渊危机，而不是安全召唤空间。

#### 6. 过程

1. 裂痕展开。
2. 苏念本能后退，却发现掌心印记把他往前拖。
3. 房间物品被短暂悬浮：
   - 低评档案翻页。
   - 召唤师资源清单散落。
   - 林清雪递来的资料压在桌角，形成现实牵挂对照。
4. 苏念吐槽半句没说完，就被裂痕吞没。
5. 跨界过程：
   - 听见无数重叠的低语。
   - 看见碎片化世界影像。
   - 深渊香气、契约阵光、白金纹路一闪。
   - 灵能被迅速消耗。
6. 苏念意识到：
   - 这不是召唤对方过来。
   - 是他被送到对方身边。
7. 他重重跌落或踉跄落地。
8. 环境切换为深渊房间：
   - 暗色帷幕。
   - 奢华却危险的陈设。
   - 地面真名契约阵。
   - 空气里压抑的甜香与灼热灵能。

# 第3章：召唤师最缺的，是钱

苏念回到住处的时候，天已经暗了。

江海市安全区上方的淡蓝屏障像一只倒扣的碗，把城市罩在里面。远处偶尔有灵能巡逻灯掠过，街边的职业者补给店还亮着招牌。

【低阶星晶回收】

【秘境补给八折】

【召唤媒介今日到货】

苏念从公交站一路走回来，每经过一家写着“召唤”的店，都觉得那几个字像在发光。

不是希望的光。

是账单的光。

他住的地方在旧城区边缘，一间不大的单人公寓。房租便宜，墙面有些发旧，窗外能看见安全区屏障的边角。桌子、床、一个小型储物柜，就是全部家当。

门关上。

外面的议论声、检测大厅的白光、光幕上那几行刺眼的字，终于都被挡在门外。

苏念把包往桌上一扔。

低评档案、召唤师资源清单、林清雪递给他的资料和基础恢复药剂，全都滑了出来。

他盯着那些东西看了两秒，笑了一声。

“别人觉醒职业，回家庆祝。”

“我觉醒职业，回家算破产。”

没人接话。

房间里只有冰冷的灯光。

苏念拉开椅子坐下，把那份《召唤师初期培养建议》摊开。

第一页，加粗标题就很友好。

【建议召唤师在职业觉醒后一周内，确认第一召唤对象或临时契约方向。】

下面是列表。

【基础召唤媒介：一至三枚星晶起。】

【低阶兽宠临时契约媒介：三枚星晶起。】

【元素灵短期召唤引导剂：五枚星晶起。】

【契约稳定材料：视召唤对象波动浮动，建议至少准备三份。】

【永久契约锚定媒介：不建议低资源考生自行尝试。】

“不建议低资源考生自行尝试。”

苏念把这行字念了一遍，越念越觉得礼貌。

翻译一下就是——

穷人别碰。

他继续往后翻。

训练场费用。

召唤对象培育费用。

低阶星晶日常消耗。

召唤位稳定训练。

高考前模拟秘境入场资格。

每一项后面都跟着数字。

这些数字单独看还算人话，凑在一起就不像人话了。

苏念拿起笔，在纸上粗略算了一遍。

如果他想走最普通的召唤路线，先得买基础召唤媒介，尝试匹配低阶兽宠或者元素灵投影。

匹配成功之后，还得准备契约稳定材料。

临时契约能应付短期训练，但高考小秘境里风险太高，召唤对象一旦断连，召唤师本体就是活靶子。

想稳一点，就得尽快锁定长期培养对象。

长期培养对象要资源。

资源要星晶。

星晶要钱，或者要进秘境拼命。

而问题来了。

没有召唤对象，他进秘境拼命都没人带。

他忽然明白，为什么觉醒中心那些同学议论时，语气里会带着惋惜。

召唤师不是弱。

恰恰相反，召唤师只要养起来，能打能抗能控，甚至能一个人撑起一支小队。

可那是“养起来”之后。

普通战斗职业刚觉醒，拿刀能练，冥想能练，跑步能练，哪怕只靠学校训练场，也能慢慢磨熟练度。

召唤师不一样。

没有召唤对象，职业循环都搭不起来。

可想要召唤对象，第一步就是花钱。

“很好。”

苏念把笔一丢，靠在椅背上，“命运给我开的不是职业，是付费功能。”

他伸手拿起林清雪给的那份资料。

封面很干净，里面用细致的字迹标了几处重点。

【低资源召唤师应优先选择临时契约，避免高消耗永久锚定。】

【高考小秘境中，召唤师可通过团队配合获取初始战利品。】

【召唤位空置者，建议提前进行基础感知训练，确认可回应对象范围。】

苏念看着那几处标记，指尖顿了一下。

林清雪递资料时说得很轻。

高考还没开始。

检测结果不是最终结果。

她没有说什么“我相信你一定行”的空话，只是把能用的资料塞给了他。

这份关心很克制。

克制到刚好不伤他的自尊。

苏念把资料合上，低声道：“林同学，你这资料有用是有用。”

他看了一眼桌上那堆价格清单。

“就是不包邮，也不包召唤物。”

吐槽归吐槽。

他没有继续瘫着。

苏念起身，从储物柜里翻出觉醒中心发放的基础召唤阵图，又拿出一小袋最低级的通用媒介粉末。

那是学校统一配发的入门材料，价值不高，主要用于让刚觉醒的召唤师完成第一次职业感知。

正常情况下，哪怕召唤不到强力对象，至少也能感知到几类可接入的低阶回应。

比如低阶犬类兽宠。

比如风、火、水一类元素灵微弱投影。

再不济，也能连上某些临时怪物投影。

苏念蹲下，把阵图铺在地板上，按照手册将媒介粉末撒进阵纹凹槽。

淡灰色粉末落下。

阵纹轻轻亮了一下。

他深吸一口气，将掌心按在阵眼位置。

“来吧。”

“要求不高。”

苏念盯着那道微弱光圈，“毛茸茸的也行，石头疙瘩也行，哪怕来只会喷水的乌龟，我也认了。”

灵能从掌心涌出。

阵纹缓缓点亮。

房间里的空气轻微震动，一圈淡蓝色波纹沿着地板扩散开。

苏念闭上眼，按照手册引导感知。

黑暗里，像有很多极远的光点闪过。

他尝试靠近其中一个。

光点一颤。

灭了。

苏念睁眼。

阵纹只剩一小圈余光。

第一次尝试失败。

“正常。”

苏念自我安慰，“第一次嘛，谁还没个紧张的时候。”

他重新补了一点媒介粉末。

第二次，阵光亮得更明显。

他隐约感知到一股低阶兽宠的气息，像一只躲在雾后的影子，短暂地抬头看了他一眼。

苏念心头一动，立刻尝试建立临时牵引。

下一秒。

那道气息像被某种更深的潮水冲开，直接断掉。

召唤阵啪地一声熄了半圈。

苏念沉默。

他低头看了看手册。

手册上写着：若召唤对象回应中断，可能为媒介能量不足、召唤对象抗拒、召唤频率不匹配。

“召唤频率不匹配？”

苏念扯了扯嘴角，“我这是连低阶兽宠的好友申请都发不出去？”

他没有停。

第三次。

这一次，他咬牙把林清雪递给他的基础恢复药剂放在旁边，又从抽屉里摸出自己仅有的一块星晶碎片。

这东西不大，是学校训练奖励剩下的边角料，平时舍不得用。

苏念看着它，肉疼了半秒。

“成了叫投资。”

“不成叫交学费。”

他把星晶碎片按进阵眼。

嗡——

召唤阵骤然亮起。

淡蓝色光芒扩散，地面阵纹像被点燃一样，一层层往外铺开。

这一次，苏念感知到的不是一个光点。

而是一片杂乱的回响。

低阶兽宠的嘶鸣。

元素灵的细碎波动。

怪物投影模糊的轮廓。

它们全都在远处闪烁，却没有任何一个真正靠近。

更奇怪的是，在这些普通召唤回应之下，还有一股更深、更远的频率。

像一条暗流。

它从苏念精神深处划过。

普通召唤对象被那股暗流一冲，纷纷偏离。

阵纹剧烈闪烁。

苏念眉头一皱，强行稳住灵能。

“别啊。”

“我还没嫌你们便宜，你们先嫌我信号差？”

话音刚落，召唤阵彻底熄灭。

星晶碎片碎成了灰。

房间一下子安静下来。

苏念维持着按在阵眼上的动作，半天没动。

失败了。

不是一次失败。

是普通临时召唤路线，连续失败。

低阶兽宠无回应。

元素灵无回应。

怪物投影无回应。

明明他是召唤师。

明明觉醒台也确认了他的职业方向。

可所有普通召唤对象，都像听不见他。

或者说，听见了，却被某种更深层的东西挡住了。

苏念慢慢收回手。

掌心有点烫。

他低头一看，发现掌心中央不知何时浮起了一点暗紫色的纹路。

很淡。

像一枚尚未成形的印记。

苏念眯起眼：“这又是什么？召唤失败送的安慰奖？”

下一瞬。

他听见了一道声音。

很远。

远到像是从梦的尽头传来。

“真名……”

苏念猛地抬头。

房间里空荡荡的。

窗户关着。

门也关着。

通讯器没有响。

桌上只有低评档案、资源清单、林清雪的资料，以及那支没拆封的恢复药剂。

苏念坐在地上，盯着空气看了两秒。

“很好。”

他深吸一口气，“我现在开始出现幻听了。”

那道声音再次响起。

比刚才清晰一些。

“不要……被他们……拿走……”

女声。

压抑。

痛苦。

却不是软弱的求饶。

那声音里有一种极深的骄傲，像被锁链缠住的刀，哪怕刀锋已经崩裂，也依然不肯低头。

苏念脸上的玩笑慢慢收住。

掌心暗紫色纹路发烫。

房间的灯光闪了一下。

召唤阵本该熄灭的灰烬忽然重新浮起，化作细碎的深紫色光点。

空气里，多了一缕陌生的香气。

甜冷。

危险。

和地球上任何香水都不一样。

苏念站起身，警惕地看向四周。

“谁？”

没有回答。

只有断续的低语继续传来。

“外界……”

“变量……”

“不是……深渊契约局……”

深渊。

这两个字钻进脑海时，苏念后背微微一凉。

全球游戏入侵之后，地球上并不缺异界传闻。

秘境。

裂隙。

怪物潮。

大秘境后方甚至可能连接真实异世界。

但对一个刚觉醒的一阶一段考生来说，“深渊”这种词离他太远了。

远到不该出现在他这个月租房里。

苏念低头看了一眼地上的基础召唤阵。

阵纹已经变了。

原本规整的蓝色线条，被暗紫色光辉覆盖，边缘还隐约浮现出一些他看不懂的契约符号。

他眼前忽然闪过几幅碎片画面。

暗色帷幔。

深紫月光。

铺满地面的古老契约阵。

还有一道被阵纹锁住的身影。

那身影高挑、成熟、危险，即便狼狈，也像一朵在深渊夜色里盛开的花。她身后似有暗翼虚影，发间小巧的角泛着冷光。锁链般的魔纹缠绕在她周围，试图一点点压进灵魂深处。

画面只出现一瞬，就被剧烈的白金色纹路刺破。

那纹路不属于暗紫深渊。

冷硬。

傲慢。

像某种外来的烙印。

苏念眼前一黑，扶住桌沿。

女声第三次响起。

这一次，近得仿佛就在耳边。

“我不要……成为任何人的附庸……”

“谁都可以……”

“不。”

她像是硬生生改口，声音里带着濒临极限的清醒。

“不是谁都可以。”

“要一个……不属于他们的人……”

苏念沉默下来。

他本来想继续吐槽。

比如自己这职业是不是从召唤师变成了异世界求救热线接线员。

比如普通召唤师召唤兽宠，他这边直接跳过售后，接入深渊高危客户。

可那些话到嘴边，又被他咽了回去。

因为他听得出来。

这不是诱惑。

也不是奖励。

更不是系统突然大发慈悲送他一个强力召唤对象。

那是一道真实的呼唤。

一个真实存在的人，在某个他完全不了解的地方，正被契约、真名和某种强制规则推向深渊。

她不是在求谁拥有她。

她是在求一个机会。

求自己不要被夺走选择权。

苏念看向桌上的低评档案。

【职业：召唤师】

【初始阶位：一阶一段】

【召唤对象回应：无】

【召唤位状态：空置】

白纸黑字，冷得像判词。

他又看向地上逐渐变成暗紫色的召唤阵。

普通召唤对象无回应。

可深渊尽头，有人听见了他。

或者说，是他听见了对方。

掌心的暗紫纹路骤然一跳。

下一刻，苏念脑海深处浮现出四个字。

不是觉醒中心光幕那种冰冷的检测文字。

更像某种沉睡许久的职业回响，在此刻第一次睁开眼。

【界域行者】

苏念瞳孔微缩。

“界域行者？”

他低声重复。

这四个字出现的瞬间，之前所有普通召唤失败的异常，似乎都有了另一种解释。

他不是召不到。

是那些普通召唤频率，根本不是他真正要接通的方向。

召唤师召唤兽宠、元素灵、怪物投影。

而界域行者……

苏念还没来得及完整理解，掌心的灼热便扩散到整条手臂。

一道模糊的意念随之浮现。

听见危机中的呼唤。

跨过世界边界。

抵达呼唤者身边。

不是抽取。

不是捕获。

是回应。

苏念嘴角抽了一下。

“也就是说，我普通召唤连只低阶兽宠都召不到。”

“结果隐藏业务是跨界上门服务？”

这个吐槽刚出口，远处那道女声忽然变得更弱。

“如果……还有谁能听见……”

“别让他们……拿走我的真名……”

真名。

契约。

自由。

外界变量。

这些词在苏念脑海里一遍遍回响。

他不知道真名对深渊意味着什么。

也不知道所谓深渊契约局又是什么东西。

更不知道自己这个一阶一段、没钱、没召唤对象、刚刚还被资源清单压得喘不过气的召唤师，过去能做什么。

说不定刚跨过去，就被那边的契约阵当场碾碎。

理智告诉他，现在应该停下。

关掉召唤阵。

当成觉醒后的精神反噬。

等明天去职业中心复查。

可掌心的印记还在发烫。

那道声音还在求救。

更重要的是，她那句“我不要成为任何人的附庸”，像一根刺，扎进了苏念心里。

他忽然想起觉醒台上那些目光。

同情。

惋惜。

幸灾乐祸。

还有那份把他未来写得明明白白的档案。

这个世界很喜欢替人下结论。

你是召唤师，但召唤位空置。

你没有资源，所以前途有限。

你没有召唤对象，所以高考小秘境没人愿意带你。

你还没真正开始，就已经被人算好了输法。

苏念讨厌这种感觉。

所以他也很难对另一个正在被迫失去选择权的人装作没听见。

他低头看着掌心那枚暗紫色印记雏形，沉默片刻，低声道：

“我不是什么救世主。”

“更不是什么深渊契约专家。”

“我连自己高考怎么过都还没想明白。”

他说到这里，停了停。

房间里的深紫光点越来越多，召唤阵边缘像被某种无形力量撕开，一道细小裂痕浮现。

裂痕之后，有暗色月光，有契约阵光，还有压抑到让人喘不过气的深渊香气。

苏念抬起头。

“但听见了还装没听见，好像也太不像话。”

他将手按向那道裂痕。

掌心印记骤然亮起。

“我听见了。”

这四个字落下。

整个房间像被深渊拉住。

地面的召唤阵轰然倒卷，暗紫色纹路从阵眼向四周蔓延，桌上的低评档案被风掀开，纸页哗啦作响。

资源清单散落一地。

林清雪给他的资料被吹到桌角，封面压在那支基础恢复药剂下面，像现实世界最后一点安静的提醒。

苏念本能后退半步。

但已经晚了。

掌心印记像钩子一样扣住他，把他往裂痕里拖。

“等等。”

苏念脸色一变，“我说听见了，没说现在就上门啊！”

裂痕没有理他。

深紫色光芒骤然扩大。

房间里的灯光被吞没，桌椅、纸页、召唤阵，全都在一瞬间拉成模糊的线。

苏念只来得及抓住背包带。

下一秒，整个人被卷进裂痕。

失重感猛地袭来。

耳边不再是江海市夜晚的车辆声，而是无数重叠的低语。

有怪物嘶吼。

有风暴咆哮。

有陌生世界破碎的钟声。

还有那道越来越近的女声。

“不是他们的人……”

“外界变量……”

“求你……”

苏念的灵能在快速消耗。

他能清楚感觉到，自己不是在把对方召唤到地球。

恰恰相反。

是他正在被送往对方身边。

这不是普通召唤。

这是界域行走。

一条横跨世界边界的通道，在他的第一次回应下，被粗暴地撕开。

苏念眼前闪过无数碎片。

燃烧的荒原。

倒悬的城池。

布满血月的天空。

还有一座被深紫帷幔笼罩的房间。

白金色纹路再次一闪而过。

像某只高高在上的眼睛，冷漠地俯视着陷入阵中的猎物。

下一刻，苏念重重跌落。

膝盖撞在冰冷地面上，掌心按进了某种正在运转的阵纹。

灼痛瞬间从手掌冲进脑海。

他倒吸一口凉气，抬头看去。

这里已经不是他的出租屋。

暗色帷幔垂落。

墙壁上镶嵌着像深渊月石一样的晶体，散发出幽冷紫光。空气里弥漫着甜冷而危险的香气，地面铺满繁复至极的契约阵纹，每一道线条都像活物般缓慢蠕动。

房间奢华。

也危险。

像一朵盛开的毒花。

而契约阵中央，跪坐着一道身影。

成年魅魔。

高挑，成熟，危险。

暗紫长发散落在肩头，发间小巧的角泛着冷光。她身上的深渊魔纹被锁链般的阵纹束住，眼底像盛着一轮摇摇欲坠的深紫月亮。

她很狼狈。

呼吸凌乱，指尖发白，周围的契约阵正一寸寸压向她。

可即便如此，她抬眼看向苏念时，眼神里仍没有半点顺从。

只有警惕、痛苦，以及最后一线不肯熄灭的骄傲。

她看着突然跌进阵中的人类，嗓音低哑：

“你……”

“不是他们的人？”

苏念还没来得及回答。

脚下真名契约阵骤然亮起。

无数暗紫锁链般的纹路同时转向他，像是发现了新的外界变量。

苏念脸色一变。

“我现在说自己只是路过，还来得及吗？”

下一瞬，阵纹轰然合拢。

他也被拖进了契约阵中央。

取而代之的，是认真。

“苏念。”

不再是“小家伙”。

不再是故意逗弄。

也听见自己的心跳。

一下，又一下。

和掌心印记的灼热重叠在一起。

看着她终于不再完全掌控一切的眼神。

忽然就笑了。

“都反压到这一步了。”

他声音哑得厉害。

“现在认输，岂不是很亏？”

帷幔落下。

深渊香气沉入夜色。

呼吸被风暴卷散。

苏念只记得莉莉娅的眼睛。

最开始是胜券在握的笑。

后来是被他反压一局后的惊讶。

她像是在风暴里俯下身，对他说了什么。

意识像从很深的水底浮上来。

亏得离谱。

脑子里只剩下一个极其朴素的念头——

这哪是契约同步。

这是深渊高利贷。

他艰难地动了动手指。

掌心的契约印记还在发烫。

暗紫色纹路比之前稳定了许多，像一朵沉在血脉里的花，随着他的心跳轻轻明灭。

却也难得安静。

像一轮暂时收起锋芒的深渊月亮。

想说点什么。

又比如“下次能不能提前告诉我会亏空到这种程度”。

再比如“我腰子好像在抗议”。

太累了。

真的太累了。

掌心契约印记随之微微一亮。

抱着怀里的魅魔，沉沉睡了过去。

大厅里的声音迅速远去。

【职业倾向检测中……】

【灵能属性捕捉中……】

【职业倾向：召唤系】

大厅里先是安静了一瞬。

随后，低低的议论声像潮水一样漫开。

“召唤系？”

“召唤师啊……上限倒是不低。”

“上限高有什么用？召唤师最吃资源。”

“没有稳定召唤对象，前期就是空壳。”

“而且苏念家里……你懂的。”

苏念看着光幕，心里也沉了一下。

召唤师。

这职业不是弱。

恰恰相反，只要资源足够，召唤师能契约兽宠、元素灵、异界生物投影，甚至培养出专属战斗体系。

问题是——它烧钱。

非常烧钱。

召唤媒介要星晶。

契约稳定要星晶。

召唤对象培养要星晶。

永久契约位更是要长期维护。

富人觉醒召唤师，叫未来可期。

穷人觉醒召唤师，叫开局欠费。

苏念嘴角动了动，还没来得及吐槽，光幕忽然闪烁起来。

原本稳定的蓝色检测线突然断开。

方明远皱眉。

记录板上的数值开始异常跳动。

【召唤波动接入中……】

【召唤位反馈异常】

【低阶兽宠匹配失败】

【元素灵匹配失败】

【怪物投影匹配失败】

【普通召唤对象无回应】

大厅里的议论声又低了一层。

方明远抬手示意安静，盯着检测数据。

旁边考务助理低声问：“设备问题？”

方明远摇头：“前后数据正常，设备无异常。”

他看向苏念：“保持手掌贴合晶柱，不要松开。复测一次。”

苏念点头。

晶柱光芒再次亮起。

这一次，检测阵列运转得更深，灵能从觉醒台底部一圈圈升起。

光幕继续跳动。

【职业：召唤师】

【初始阶位：一阶一段】

【召唤对象回应：无】

【召唤位状态：空置】

啪。

光幕定格。

大厅里彻底安静了两秒。

然后，声音炸开。

“召唤对象无回应？”

“这不就麻烦了？召唤师本来就烧钱，他还召不到东西。”

“召唤对象匹配失败，意思是初始连能召什么都测不出来？”

“高考小秘境怎么办？召唤师没召唤物，进去当观众吗？”

“普通家庭养召唤师都难，更别说苏念这种没背景的。”

有人声音压得不高，却刚好能让周围听见。

“召唤位空着的召唤师，基本没前途了吧。”

苏念站在觉醒台上。

白光落在他身上，像把所有人的视线都一起压了下来。

有同情。

有惋惜。

也有松了一口气的幸灾乐祸。

少一个竞争者。

在高考资源有限的现实里，这就是许多人第一时间的反应。

方明远没有立刻宣布最终结果。

他低头重新调出原始波形，又对照了两遍检测记录。

身为职业觉醒中心记录员，他见过不少异常数据。

有些是设备误差。

有些是职业变异。

有些则只是天赋不足。

苏念这份数据确实有点奇怪。

召唤系波动存在，但断续、不稳，像是检测仪始终等不到普通召唤对象的回应。

不过，流程就是流程。

在没有实战表现、没有召唤对象回应、没有二次证明前，职业觉醒中心只能依据当前检测结果给出初评。

方明远抬头，语气客观，没有嘲讽，也没有刻意安慰。

“苏念。”

“明面职业，召唤师。”

“召唤波动断续，召唤位反馈不稳定。”

“普通召唤对象匹配失败。”

“按流程记录为——召唤对象未响应。”

他停顿了一下。

“初始阶位，偏低。”

光幕随之更新。

【考生：苏念】

【职业：召唤师】

【职业状态：召唤对象未响应】

【初始阶位：低】

【备注：召唤对象匹配失败，建议后续实战观察】

这几行字亮起时，周围议论彻底压不住了。

“真空召唤位了。”

“可惜，召唤师如果一开始有召唤对象，说不定还能混个好队伍。”

“稳定也没用吧？召唤师最缺星晶。没钱养第一只永久契约对象，职业优势根本起不来。”

“高考小秘境很现实的，谁愿意带一个召唤位空着的召唤师？”

“边缘队都怕他关键时刻召不出东西拖后腿。”

一句一句，都不算恶毒。

甚至大部分都很现实。

也正因为现实，才更扎人。

苏念低头看着光幕上自己的名字。

召唤对象无回应。

这几个字冷冰冰摆在那里，像一枚提前盖下来的章。

他沉默两秒，忽然笑了一声。

方明远看向他。

苏念抬起头，语气还是那副惯常的懒散。

“召唤师就算了，还给我来个空号版？”

旁边有人没忍住笑出声。

苏念继续道：“别人觉醒职业，我觉醒账单。召唤师这职业，怕不是专门坑穷人。”

笑声更明显了。

可林清雪没有笑。

她站在人群边缘，指尖微微收紧。

她看得出来，苏念不是不在意。

他只是习惯把难堪的东西用玩笑包起来，像这样别人就看不见他真正被压住的那一瞬间。

方明远看着苏念，语气缓了一点。

“检测结果不是最终结果。”

大厅安静了些。

方明远继续说道：“职业高考还有小秘境实战复核。召唤系职业受召唤对象、媒介、契约稳定度影响很大。你这份数据异常，我会备注建议观察。”

他说完，低头在记录板上敲下几行字。

【苏念：召唤系异常波形】

【原始检测数据已保存】

【建议高考实战阶段复核】

记录完成。

档案正式进入考务流程。

也意味着，从这一刻开始，苏念在系统里就是一个“一阶一段，但召唤位空置的召唤师”。

后续资源配额、临时分组、学院初筛，都会受到影响。

方明远把一份薄薄的档案纸递给他。

“下台吧。觉醒后两小时内不要进行高强度灵能训练。”

苏念接过档案。

纸很轻。

可落在手里，沉得像一块铁。

他走下觉醒台。

周围人的目光跟着移动。

有人想说两句安慰，但看见他的表情，又不知道怎么开口。

也有人干脆别开脸。

高考临近，所有人都在计算队友、资源、风险。

这个时代没人会因为你可怜就分你机会。

怪物不会。

秘境不会。

高考分数也不会。

苏念把档案卷起来，随手塞进口袋，朝出口走去。

背后还有人在小声议论。

“他以前理论成绩还行吧？”

“理论有什么用？召唤师没召唤物就没战斗循环。”

“临时契约也能凑合？”

“召唤师召不出东西，战斗里掉链子能害死全队。”

“听说永久契约媒介最便宜都要几枚星晶，后续还得养。”

“他哪来星晶？”

声音慢慢远去。

苏念走到走廊尽头，抬头看了一眼外面的天。

江海市安全区屏障像一层淡蓝色薄膜，罩在城市上空。

屏障之外，是被怪物占据的旧城区、裂隙废墟和低阶秘境入口。

这世界早就不讲公平了。

职业。

检测结果。

资源。

家世。

每一样都能把人往下压。

苏念刚准备离开，身后传来脚步声。

“苏念。”

他回头。

林清雪跟了出来。

走廊里的光比大厅柔和一些，落在她身上，冲淡了眉眼间的冷感。

她手里拿着一份资料，还有一支基础恢复药剂。

“给你。”

苏念低头看了眼。

资料封面写着——

《职业高考小秘境召唤系补充规则与低资源应对建议》。

恢复药剂则是觉醒中心统一发放的基础型，不算贵，但在这种时候递出来，意思已经足够明显。

苏念挑眉：“林同学这是提前投资潜力股？”

林清雪看着他：“你现在这个股价，应该没人敢投。”

苏念被噎了一下。

“你这安慰方式还挺清醒。”

林清雪把资料往前递了递：“高考还没开始。”

苏念没立刻接，笑着说：“初始检测召唤对象无回应，你就不怕我高考小秘境进去表演原地罚站？”

林清雪语气平静：“检测结果不是最终结果。”

“方记录员刚才也这么说。”

“他说的是流程。”林清雪看着他的眼睛，“我说的是你。”

苏念脸上的笑意停了半拍。

走廊里人来人往。

远处大厅还在刷新新的职业结果。

有人欢呼，有人叹气，有人哭出声。

可这一瞬间，苏念忽然觉得那些声音远了一些。

林清雪没有说“我相信你一定逆天改命”。

也没有说“别难过”。

她只是很认真地告诉他——

你不是会因为一次检测结果就放弃的人。

这比任何空泛的安慰都更让人心里发紧。

苏念伸手接过资料和药剂。

指尖碰到资料边缘时，他下意识捏紧了一点。

然后他很快又恢复那副不着调的样子。

“行，那我先回去研究一下自己破产得有多快。”

林清雪微微皱眉：“你能不能别总拿这种事开玩笑？”

“不能。”

苏念把药剂揣进口袋，“不然显得我很惨。”

林清雪沉默了一下。

她没有继续拆穿他。

只是低声道：“如果需要高考资料，可以找我。”

苏念看了她一眼。

“包括怎么不花钱养召唤物的资料吗？”

“如果有，我会给你。”

“那完了。”苏念叹气，“看来我今晚只能研究怎么和召唤物谈感情，争取让它自带口粮。”

林清雪：“……”

她像是想说什么，最后只说：“别乱来。”

苏念挥了挥手。

“放心，我这个人最大的优点就是穷，乱来也要成本。”

说完，他转身朝觉醒中心外走去。

林清雪站在原地，看着他的背影。

他走得很稳，像完全没被低评影响。

可她知道不是。

苏念这个人，越是开玩笑，越说明他不想让别人看见狼狈。

等他的身影消失在出口，林清雪才低头看了一眼自己空下来的手。

高考还没开始。

可她心里莫名有种预感。

苏念不会就这么被一份低评档案定死。

……

觉醒中心外，阳光刺眼。

苏念沿着街道往自己住处走。

江海市主城区已经被游戏化后的安全区体系改造过。

街道两侧能看见职业者商店、星晶兑换点、低阶装备维修铺，还有挂着“秘境补给”的小店。

每一家店门口，都贴着价格。

苏念越看越沉默。

他走到公交站旁，掏出那份召唤师资源清单。

刚翻开第一页，就看见一行加粗标题。

【召唤师初期培养建议】

【一、基础召唤媒介：根据召唤对象属性不同，价格一至十枚星晶不等】

【二、临时契约稳定剂：建议至少准备三份】

【三、低阶召唤对象培育费用：视种族与潜力浮动】

【四、永久契约媒介：价格较高，建议在导师指导下使用】

【五、召唤位稳定训练：建议进入正规训练场】

苏念一行一行往下看。

越看越安静。

战士觉醒了，至少能拿刀练基础劈砍。

法师觉醒了，至少能冥想感知元素。

冰霜舞者像林清雪那样，有天赋，有学校资源，有稳定职业路线，哪怕不是大富大贵，也能一步一步往前走。

召唤师不一样。

没有稳定召唤对象，连职业循环都搭不起来。

可稳定召唤对象本身，就是资源堆出来的。

苏念翻到价格表。

基础媒介，三枚星晶起。

稍好一点的契约稳定材料，五枚星晶起。

永久契约媒介更离谱，后面标着“视品质浮动”，但最低参考价已经够让普通家庭肉疼很久。

而他没有普通家庭。

他只有一个低评档案，一份资源清单，以及林清雪塞给他的高考资料和基础恢复药剂。

苏念面无表情地合上清单。

“很好。”

他抬头看着安全区屏障，低声道：“低起点这三个字已经算客气了。”

准确翻译应该是——

没钱别玩。

林清雪给他的资料被他放在档案旁边。

那份资料提醒他，高考小秘境还有机会。

方明远也说，实战阶段可以复核。

可机会不是凭空来的。

他得有召唤对象。

得有契约媒介。

得有星晶。

得在高考前，把一个“召唤对象未响应”，硬生生凑成能打的职业。

苏念低头，看着档案袋上那几个刺眼的字。

【召唤对象未响应】

【初始阶位：低】

他忽然笑了一下。

笑意不重，却带着一点压不下去的不甘。

如果他只是普通召唤师。

别说高考小秘境，恐怕连第一只永久契约对象都养不起。

公交车缓缓停在站台前。

车门打开，人群涌动。

苏念把低评档案、资源清单和林清雪给的资料一并塞进包里。

他还不知道，几个小时后，他会重新理解“召唤”这两个字。

但至少在这一刻。

江海职业觉醒中心的档案里，他只是一个刚被判定为低起点的召唤师。
