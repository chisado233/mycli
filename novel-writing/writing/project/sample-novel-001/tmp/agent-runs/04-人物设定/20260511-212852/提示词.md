# 04 人物设定精简上下文

## 项目核心

- 作品暂名：`全球游戏入侵：我的魅魔召唤物全是异界女王`。
- 类型：全球游戏入侵 + 多界并轨 + 召唤师后宫 + 秘境防线 + 多界反神明。
- 男主：苏念，18岁以上，地球江海市孤儿学生。明面职业是召唤师，被低评为“不稳定召唤系”；真实职业是“界域行者 + 召唤师”。
- 召唤机制：苏念响应异世界女性玩家/原住民在危机中的呼唤，跨界到对方身边，以合理身份完成心愿/考验/好感认可；对方自愿后缔结召唤师契约，成为召唤物/队友/后宫成员。
- 契约机制：临时契约像临时工，永久契约需要强认可；刚觉醒有一个永久契约位，之后约十级或关键转职节点增加。
- 战斗机制：召唤师契约同步后可触发职业借用、技能同步、武器化、剑灵附体、恶魔契约技能、血族夜战、机甲武装合体等。
- 合规边界：所有亲密关系必须成年、自愿、类人/智慧对象；正文只写隐晦氛围、事后状态、契约同步率、称呼变化、旁人调侃等，不写露骨过程。
- 禁止旧名：不得出现“林烬”。男主统一叫“苏念”。
- 禁止技术化措辞：不要用“接口”。

## 黄金前三章

- 第一章以强钩子开场：苏念与成年魅魔莉莉娅完成自愿契约后的暧昧事后氛围。通过房间残留深渊香气、莉莉娅慵懒调侃、苏念灵能透支、契约印记炽热、系统提示“第一永久契约成立”等隐晦方式表现，不写露骨过程。
- 随后倒序：苏念参加18岁职业觉醒，明面觉醒召唤师，被判低潜力；回家后担心召唤师太烧资源，吐槽隐藏职业像“专门救落难异世界美少女的离谱职业”，正式职业名仍为“界域行者”。
- 苏念尝试响应呼唤，传送到莉莉娅所在的深渊房间，帮她破局，双方自愿缔结召唤师契约。
- 前三章完成：职业觉醒、界域行者首次触发、莉莉娅危机、自愿契约、首次联动、返回地球并埋下高考小秘境。

## 主线阶段

1. 开局/江海市：职业觉醒、低评召唤师、莉莉娅契约、高考小秘境、林清雪现实侧陪伴、黑契会露头。
2. 学院篇/前中期收女：进入星海学院或官方训练营；收青璃、白九璃、艾瑟琳、诺雅等；每位女主绑定一个世界体系和战斗联动。
3. 大秘境与人魔战争：魔巢门扩大，深渊诸侯入侵/内斗；苏念通过莉莉娅、艾瑟琳分化深渊势力。
4. 各世界混战：深渊、修仙、机甲、巨龙、圣域、精灵等围绕秘境资源、世界通道、神明规则互相争夺；元气大伤后开始讲和。
5. 神明世界降临：神庭秩序在各世界入口降临，神罚副本、圣痕污染、神使显性化。
6. 终局：苏念与主角团推动多界联军反神明，建立自愿契约、共同战斗、多界互助的新秩序。

- 希尔薇：成年精灵圣射手。清冷守护，自然结界、远程狙击、秘境探索。
- 伊芙琳：成年光辉圣女。治疗、净化、增益、驱散污染，信仰崩塌后依赖与治愈。
