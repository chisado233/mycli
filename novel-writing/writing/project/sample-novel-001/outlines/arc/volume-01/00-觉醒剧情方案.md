# 第一卷觉醒剧情方案

> 用途：重做第二章/觉醒段落前的方案池。最新口径：去掉“失足少女收容系统”职业，将其能力并入金手指系统“失足少女收容系统”；苏念明面觉醒职业仍是召唤师，一阶一段；苏念说话方式改为正常、克制、带少量冷幽默，不再刻意抖包袱。

## 方案A｜富哥嘲讽 + 系统在低谷弹出（推荐）

### 核心结构

1. 职业觉醒中心，林清雪先觉醒“冰霜舞者”，引起全场关注。
2. 富哥同学觉醒一个华而不实但很有噱头的职业“酒吧舞者”或“魅影舞者”，因家里有钱和资源，现场反而被簇拥。
3. 苏念觉醒明面职业“召唤师”，初始阶位一阶一段，但普通召唤对象无回应，召唤位空置。
4. 富哥借机嘲讽：召唤师没召唤物就是空壳，还不如自己的“酒吧舞者”至少能靠资源砸出团队辅助路线。
5. 苏念不讲烂梗，只正常回怼：你职业叫什么不重要，重要的是你别把家里的钱当自己的本事。
6. 就在苏念拿到检测报告、准备离开时，系统弹出：
   - 【检测到高危异界求援信号。】
   - 【失足少女收容系统已激活。】
   - 【首位待收容对象：深渊魅魔莉莉娅。】
7. 章末：苏念听见莉莉娅断续呼唤，进入第三章或直接衔接跨界。

### 优点

- 有外部冲突和打脸预期。
- 富哥职业“酒吧舞者”有反差和嘲讽空间，但不抢主线。
- 系统名很抓人，和魅魔开局强钩子贴合。
- 苏念回怼正常，不靠恶心硬梗。

### 风险

- “酒吧舞者”需要写得像游戏化职业，不要写成纯搞笑职业。
- 富哥不能太蠢，要像现实里靠资源自信的人。

## 方案B｜系统先隐藏，富哥抢风头，苏念最后听见呼唤

### 核心结构

1. 全场按职业稀有度排序，富哥觉醒“魅影舞者”，被老师判定有团队增益潜力。
2. 富哥主动挑衅苏念：没资源的人最怕召唤师，偏偏你还觉醒了最吃资源的职业。
3. 苏念正常回应：你说得没错，召唤师确实吃资源；但职业不是你替我过。
4. 苏念检测结果普通，召唤位空置，外界看低。
5. 没有当场系统面板，只在苏念离开后出现耳鸣、契约印记预热、异界呼救。
6. 最后一行系统弹出“失足少女收容系统绑定中”。

### 优点

- 系统出现更有悬念。
- 第二章现实感更强。

### 风险

- 爽点稍慢，可能不如方案A直接。

## 方案C｜当场系统误导成“废物辅助”，实际收容系统启动

### 核心结构

1. 苏念觉醒召唤师、一阶一段。
2. 检测光幕出现异常，但公开只显示“召唤位空置”。
3. 富哥觉醒“酒吧舞者”，嘲笑苏念召唤不到东西。
4. 苏念脑内私有系统同时弹出，但外界看不到：
   - 【公共检测已屏蔽。】
   - 【失足少女收容系统启动。】
5. 苏念表面沉默，内心第一次感到不对劲。
6. 富哥继续嘲讽，苏念只回一句：你最好一直这么自信。
7. 章末接莉莉娅呼唤。

### 优点

- 有扮猪吃虎味。
- 系统和公共检测形成信息差。

### 风险

- 需要控制苏念心态，不要变成反派腔。

## 方案D｜林清雪视角辅助，看出苏念没有崩

### 核心结构

1. 林清雪觉醒冰霜舞者后，注意到苏念被检测为召唤位空置。
2. 富哥以酒吧舞者职业和资源优势嘲讽苏念。
3. 苏念回怼克制，不激烈争吵。
4. 林清雪发现苏念虽然被看低，但眼神不是绝望，而是在听某种她听不到的声音。
5. 系统只在苏念脑内响起：失足少女收容系统启动。

### 优点

- 可强化林清雪现实侧女主作用。
- 苏念更像正常人。

### 风险

- 第二章焦点会稍微分散。

## 推荐采用

推荐采用 **方案A**：富哥嘲讽 + 系统在低谷弹出。

理由：

- 最符合当前“重做觉醒剧情”的需求。
- 富哥职业“酒吧舞者”提供嘲讽点和职业名反差。
- 苏念正常回怼即可体现性格调整。
- 系统名“失足少女收容系统”在第二章末或第三章初弹出，能直接替代原“失足少女收容系统”概念。
