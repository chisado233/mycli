## 2. 卷0：开篇前状态

### 2.1 公开状态

- 对外身份/社会位置：  
  神庭秩序不以实体势力身份公开出现在地球、深渊、修仙、机甲等世界。诸界只感知到“游戏化灾变”“职业觉醒”“等级任务”“秘境融合”“怪物潮”等现象，并不知道背后存在神明世界统一调度。

- 公开目标：  
  无公开目标。若有神谕残响，也只表现为“赐予职业”“开启试炼”“筛选强者”“奖励勇者”等伪善叙述。

- 公开秩序/规则：  
  职业、等级、技能、任务、奖励、惩罚、秘境通关、阵营选择等表层规则已植入多界，但被包装成自然灾变后的新秩序。

- 普通人/其他势力对它的认知：  
  普通人不知道神庭秩序存在。地球官方只知道全球游戏入侵与秘境融合正在发生。深渊、修仙、机甲等世界也多半将其误解为本世界灾变、外界入侵或未知规则潮汐。

### 2.2 暗线状态

- 隐藏目标：  
  以“诸界归一，众生命定”为终极目标，将多个世界逐步纳入神庭牧场。先通过规则赐予制造依赖，再通过圣痕、神眷、任务链筛选可驯化强者，最终完成世界并轨。

- 暗中行动：  
  - 在地球和各异世界植入职业、等级、任务、奖励体系。
  - 制造小秘境、大秘境与多界融合潮汐。
  - 通过圣痕残纹污染秘境核心、灵魂、装备、传承。
  - 向野心家、叛徒、教团、黑暗组织泄露神眷契约术或神谕碎片。
  - 利用织命书记官记录诸界强者成长轨迹。

- 隐藏盟友：  
  无真正盟友，只有被驯化、诱惑或污染的代理人，包括下界圣徒、神谕祭司、被册封神眷者、投靠神庭的异界强者、被圣痕污染的教团与派系。

- 隐藏敌人：  
  所有不愿被收编的世界本土清醒派均为潜在敌人，包括地球镇守者、深渊反神明派、昆墟清醒派、钢穹自由派、龙族反驯化者、精灵守望者、圣域反伪神谕派、暗影反神庭派。

- 被神庭/圣痕/其他力量影响程度：  
  神庭本体就是影响源。圣痕体系、神谕任务、神罚副本、神眷册封均处于神庭控制下。卷0时它对诸界表层规则影响已高，但对各世界深层本源尚未完全控制。

- 核心资源：  
  神权规则、圣痕、神谕碎片、神眷名额、融界潮汐、世界本源、信仰回响、职业等级规则、织命任务链、神罚副本核心。

- 主要战力：  
  十二神座、圣痕审判院、神罚天使军、织命书记官、神眷诸军、神使投影、下界圣徒、神谕祭司。

- 主力量体系：地球职业体系 / 深渊魔力 / 修仙灵力剑道 / 血族血能 / 机甲科技 / 龙族血脉 / 精灵自然力 / 圣域光辉 / 暗影权柄 / 神庭圣痕 / 混合体系  
  神庭圣痕 / 神权规则 / 织命任务链 / 神罚副本 / 神眷册封 / 混合体系。
