# 血月王庭保守派代表

## 1. 基础身份

- 姓名/称号：血侯赛勒斯（可改）
- 反派类型：家族压迫者 / 阶段反派 / 情感线阻碍者
- 所属势力：深渊世界，血月王庭保守派
- 实力层级：六至七阶
- 登场阶段：中期克露露线
- 退场阶段：血族内战阶段

## 2. 反派动机

- 明面目标：维护血月王庭纯血秩序。
- 隐藏目标：控制克露露的继承权与血月王印。
- 核心欲望：让血族继续由保守贵族掌权。
- 核心恐惧：年轻王族与外界契约者打破旧秩序。
- 为什么必须与苏念冲突：苏念给了克露露自由选择，动摇保守派控制。
- 自认为正确的理由：他认为血族若放弃纯血秩序就会被深渊吞没。

## 3. 威胁设计

- 对苏念的威胁：血族追杀、夜战伏击、贵族舆论压迫。
- 对克露露的威胁：逼迫联姻、剥夺继承权、封锁血月王印。
- 反派优势：血族旧贵族人脉和夜战军团。
- 反派弱点：傲慢，轻视克露露个人意志。

## 4. 剧情功能

- 制造的压迫：克露露不能自由选择命运。
- 引出的事件：血月王庭继承权争斗。
- 承载的爽点：雌小鬼公主被苏念反制后反过来反抗家族。
- 被击败后的收益：血月王印、血族改革派支持。

## 5. 反派轨迹

- 初始状态：血族保守派权臣。
- 升压过程：警告克露露 → 夜战追兵 → 王庭审判。
- 最高威胁时刻：试图剥夺克露露王女资格。
- 失败原因：克露露完成自我选择并获得苏念契约支持。
- 结局：失势或被放逐。
