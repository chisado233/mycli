# 黑契会首领

## 1. 基础身份

- 姓名/称号：顾沉鸦（可改）
- 反派类型：阶段反派 / 非法召唤者 / 价值观对照
- 所属势力：地球，黑契会
- 实力层级：三至五阶，前期对苏念有压迫
- 登场阶段：前期江海市/高考小秘境
- 退场阶段：首卷或学院篇前段

## 2. 反派动机

- 明面目标：通过黑市召唤媒介和非法契约牟利。
- 隐藏目标：研究如何绕过自愿条件，强行奴役高潜力异界女性或召唤生物。
- 核心欲望：用契约支配一切弱者，证明“自愿”只是强者编出来的谎言。
- 核心恐惧：自己的黑契术被证明不如苏念的自愿契约。
- 为什么必须与苏念冲突：苏念第一契约莉莉娅证明召唤对象有独立人格，直接威胁黑契会生意。
- 自认为正确的理由：他认为末世秩序下资源稀缺，召唤物就该被当成资产。

## 3. 威胁设计

- 对苏念的威胁：抢夺召唤媒介、绑架/诱捕契约对象、伪造苏念非法召唤证据。
- 对主角团的威胁：试图研究莉莉娅契约，刺激林清雪现实侧危机。
- 对势力格局的威胁：勾结黑市和灰色研究员。
- 对情感线的威胁：把女主当货物，激怒苏念与莉莉娅。
- 反派优势：黑市资源、人脉、阴招。
- 反派弱点：正面战力一般，过度依赖强制契约。

## 4. 剧情功能

- 制造的压迫：前期价值观反派。
- 引出的事件：旧港区裂隙、星晶黑市、非法召唤实验。
- 承载的爽点：苏念以自愿契约碾压黑契术。
- 被击败后的收益：获得召唤媒介、黑市线索、官方关注。
- 被击败后的后果：暴露更高层实验派或神庭污染痕迹。

## 5. 反派轨迹

- 初始状态：江海黑市幕后人。
- 升压过程：试探苏念 → 盯上莉莉娅 → 旧港区设局。
- 最高威胁时刻：试图强行污染契约印记。
- 失败原因：不理解莉莉娅是自愿契约，不是被苏念控制的货物。
- 结局：被击败，黑契会江海据点覆灭。
