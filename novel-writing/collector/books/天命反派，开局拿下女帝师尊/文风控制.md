# 文风控制.md

> 来源：基于《天命反派，开局拿下女帝师尊》001-050章样本，经“文风提取 agent → 写作模仿 agent → 文风对比 agent → 文风进化 agent”迭代整理。  
> 用途：给小说写作 agent 使用，指导当前项目在不复制原书专有设定、角色定位与原句的前提下，只迁移其高层文风、爽点节奏、句式与移动端阅读体验。  
> 适配方向：全球游戏入侵、苏念、莉莉娅、界域行者/失足少女拯救者、召唤师契约、多女主后宫爽文。

---

## 1. 文风总则

### 1.1 一句话定义

快节奏、高爽度、带现代解构吐槽的男频爽文风：

- 开篇直接给冲突或暧昧现场。
- 男主表面轻佻，内心清醒。
- 叙事重信息差、压制感、抢先感和即时反馈。
- 女主描写强调高身份与反差感。
- 系统提示承担“爽点结算器”和“剧情导航器”作用。

### 1.2 当前项目适配

苏念不是反派，也不能照搬原书的角色身份、阵营关系、世家背景或反派定位；这里只迁移**叙事文风**与**爽点表达方式**。他仍是当前项目里的正向主角：救人、契约、成长、守护与反击。

但在文风上，可以迁移其**高位者心态**：

- 外界可低评苏念。
- 但读者必须很快知道苏念有隐藏职业、系统提示、跨界契约、异界女主等底牌。
- 苏念可以暂时装弱，但内心不能长期自卑。
- 他要把“低评召唤师”当成扮猪吃虎的外壳，而不是精神压迫源。

推荐心态：

```text
外界：F级召唤师，没资源，废了。
苏念：评级挺好，最适合藏牌。等我把异界女王叫出来，你们再重新组织语言。
```

## 2. 移动端段落规则

### 2.1 段落长度

- 每段通常 1-3 行。
- 单段不要堆太多设定。
- 强调“滑屏阅读”爽感：一句动作、一句反应、一句吐槽。

### 2.2 信息排列

优先顺序：

```text
画面/动作 → 角色反应 → 男主吐槽/系统提示 → 爽点推进
```

避免：

- 开篇大段解释世界观。
- 连续 5 行以上纯设定说明。
- 女主、系统、敌人都变成说明书。

### 2.3 常用短句节奏

可多用：

```text
透支了。
彻底干了。
这职业，多少有点不正经。

苏念沉默了两秒。
然后笑了。

不是哥们。
你管这叫F级？
```

## 3. 叙事视角与镜头

### 3.1 视角

第三人称限知，紧贴男主苏念。

叙事既要写外部画面，也要频繁插入苏念的现代式心理吐槽。

### 3.2 镜头类型

#### 强钩子特写

适用于章首：

- 契约印记发烫。
- 莉莉娅靠在黑色丝绒里。
- 房间残留深渊玫瑰香与硫磺余温。
- 苏念灵能透支，腰酸，系统提示跳脸。

#### 压制感全景

适用于召唤、战斗、身份反转：

- 召唤阵覆盖考场。
- 深渊气息压得考生后退。
- 高阶召唤物降临后，全场静音。

#### 心理反差镜头

适用于男主：

```text
外表：苏念脸色苍白，像被抽干。
内心：这波血赚，第一契约女主到手，还白嫖两个联动技能。
```

## 4. 男主心态与吐槽规则

### 4.1 高位者心态

苏念不是传统憋屈升级流主角。

即使开局被嘲讽，他也要表现出：

- 看戏感。
- 信息差优越感。
- 对系统、职业、竞争者/敌对角色的调侃。
- 对敌人“已有取死之道”的轻蔑。

### 4.2 本土化主角心态

苏念不是反派，不为恶而恶；他是“有底线、会吐槽、敢反击、会抢节奏的爽文主角”。

可以写：

- 抢先获得关键机缘。
- 夺回或获取敌对势力的资源。
- 对黑契会、神眷派、强制契约者下狠手。
- 让嘲讽他的考生、老师、财阀、公会当场破防。

不应写：

- 欺压无辜平民。
- 强迫女主。
- 无意义杀戮。
- 为恶而恶。

### 4.3 吐槽风格

常用吐槽方向：

- 用现代词拆解异界套路。
- 把竞争者/敌对角色看作“待处理的麻烦”。
- 把女主危机看作“高风险高回报投资”。
- 把低评当作“扮猪吃虎保护色”。

可用句式：

```text
不是哥们……
系统，你这翻译多少有点灵性。
这届竞争者，业务能力不太行啊。
F级？挺好，最适合藏牌。
```

## 5. 背景虚张声势原则

由于苏念前期没有显赫身份背景，需要用“神秘感 + 系统面板 + 界域行者权柄”制造类似背景压制的爽感。

### 5.1 外界误判

让外界误以为苏念背后有大人物、大势力或未知高阶召唤源：

- 江海考务组怀疑他被某个隐藏公会培养。
- 星海学院导师怀疑莉莉娅来自高阶深渊契约体系。
- 黑契会误以为苏念掌握古老召唤秘术。
- 深渊追兵误判苏念背后站着未知界域势力。

### 5.2 男主借势

苏念可以不解释，让误会自然发酵：

```text
他们愿意脑补，那就让他们脑补。
有时候，未知本身就是最贵的背景。
```

### 5.3 爽点效果

用“误判”复刻背景压制：

```text
敌人：他不过是个F级召唤师。
下一秒：成年魅魔贵女降临，精神控场覆盖全场。
敌人：这他妈是F级？
```

## 6. 系统提示与数值反馈规则

### 6.1 格式

统一使用：

```text
【叮！……】
【目标：……】
【同步率：……】
【爽点值/契约值/剧情偏移值：……】
```

### 6.2 系统功能

系统要承担三种作用：

1. 爽点结算。
2. 剧情导航。
3. 毒舌吐槽。

### 6.3 即时反馈

关键动作后必须有反馈。

适用节点：

- 抢先获得机缘。
- 收服女主。
- 打脸嘲讽者。
- 让敌对角色或竞争者破防。
- 契约同步率提升。
- 获得联动技能。

示例格式：

```text
【叮！检测到竞争者产生强烈嫉妒，爽点值+500。】
【莉莉娅契约同步率提升至18%。】
【获得联动技能：精神控场（初级）。】
```

### 6.4 系统毒舌/人格化

系统可以偏机械，但偶尔损人。

示例方向：

```text
【系统建议：宿主下次救人前，先评估腰部承受能力。】
【检测到目标嘴硬程度过高，建议宿主进行物理降噪。】
【恭喜宿主完成“趁乱抢先”，系统为您的临场判断点赞。】
```

### 6.5 竞争者/剧情对照角色面板

出现竞争者或敌对角色时，面板建议包含：

```text
【姓名：】
【身份：】
【剧情偏移值：】
【当前机缘：】
【一句话人生：】
```

“当前机缘”用于让主角抢先行动；“一句话人生”用于快速建立竞争关系与剧情张力。

## 7. 女主描写与暧昧边界

### 7.1 核心卖点

女主描写重点是：

```text
高身份 + 强反差 + 自愿契约 + 隐晦暧昧
```

### 7.2 莉莉娅开局特化

莉莉娅第一章承担黄金三章吸量。

可写：

- 事后氛围。
- 契约印记发烫。
- 灵能透支。
- 房间凌乱。
- 深渊香气。
- 莉莉娅慵懒调侃。
- 系统吐槽“失足少女拯救者”。

不可写：

- 露骨过程。
- 非自愿持续关系。
- 未成年暗示。
- 动物化性内容。

### 7.3 女主反差细则

女主在外界：

- 高贵。
- 冷艳。
- 危险。
- 不可接近。

女主在苏念面前：

- 会调侃。
- 会吃醋。
- 会主动。
- 会因为契约同步出现失态。

示例结构：

```text
外界眼中，她是深渊夜绯支脉的贵女。
此刻，她却懒洋洋地靠在苏念身侧，用尾尖轻轻勾着他的手腕。
```

## 8. 爽点结构模板

### 8.1 降维打击

```text
敌人轻视苏念 → 苏念不解释 → 莉莉娅/契约技能出手 → 全场沉默 → 系统结算
```

### 8.2 抢先机缘

```text
系统面板显示关键机缘 → 苏念提前入场 → 拿走资源/核心道具/救援对象 → 竞争者迟到破防
```

### 8.3 身份误判反转

```text
外界：F级召唤师。
实际：界域行者唯一职业。
外界：普通魅魔召唤物。
实际：深渊夜绯支脉贵女。
```

### 8.4 女主自愿归心

```text
危机 → 苏念介入 → 契约同步/救援 → 女主清醒确认 → 自愿契约 → 系统反馈 → 女主称呼变化
```

## 9. 敌对角色/竞争者吃瘪模板

### 9.1 无能狂怒

竞争者看着自己以为稳拿的机缘被苏念抢先拿走，只能放狠话。

写法：

```text
他怒。
他恨。
可他只能看着。
因为苏念已经站在了机缘旁边，甚至还嫌弃地拍了拍上面的灰。
```

### 9.2 精神打击

比打败更爽的是让对照者知道：

- 他知道的未来失效了。
- 他以为能控制的局面被苏念打破。
- 他认定的机缘成了苏念的奖励。

### 9.3 众人误判反噬

让嘲讽苏念的人亲眼看到苏念反转：

```text
刚才还在笑的人，此刻一个字都说不出来。
因为他们终于意识到，检测仪不是看低了苏念。
是根本没资格看懂他。
```

## 10. 战斗与打脸节奏

### 10.1 战斗要短

战斗不是详细拆招，而是服务爽点。

推荐结构：

```text
敌人叫嚣 → 苏念吐槽 → 联动技能释放 → 一招压制 → 系统反馈 → 敌人破防
```

### 10.2 苏念不能纯后排

即使是召唤师，也要通过契约联动亲自参与：

- 莉莉娅精神控场，苏念趁窗口进攻。
- 青璃剑灵附体，苏念近战。
- 诺雅装甲合体，苏念火力压制。
- 克露露夜战爆发，苏念切换血刃模式。

### 10.3 打脸必须有余波

不要打完就结束。

要写：

- 围观者沉默。
- 考务组震惊。
- 女主调侃。
- 系统结算。
- 下个机缘提示。

## 11. 章首/章尾钩子规则

### 11.1 章首

章首直接切入：

- 暧昧现场。
- 危机现场。
- 打脸现场。
- 系统提示现场。

第一章建议：

```text
苏念醒来，灵能透支，莉莉娅在身旁调侃，系统提示第一永久契约成立。
```

### 11.2 章尾

章尾严禁固定套话。

可用钩子：

- 新的竞争者/剧情对照角色面板弹出。
- 高考秘境出现圣痕残纹。
- 莉莉娅无法完全回地球，留下深渊追兵钩子。
- 林清雪撞见苏念身边的异常气息。
- 黑契会注意到智慧召唤物。

章尾要让读者想点下一章，而不是机械预告。

## 12. 可用现代梗与禁用项

### 12.1 可用现代梗

少量使用即可：

- 不是哥们。
- 这职业多少有点不正经。
- 风险投资。
- 降维打击。
- NPC。
- 工具人。
- 麻烦。
- 精准扶贫。
- 业务能力不行。

### 12.2 禁用项

严禁复制样本小说专有内容：

- 原书角色名。
- 原书势力名。
- 原书体质/法宝/功法名。
- 原书固定章尾或固定开篇互动。
- 原书长句、对白、标志性桥段。

## 14. 原创文风实例语句

> 以下句子仅供学习句式、节奏、吐槽和氛围，严禁在正文中原样重复使用。

1. 苏念看着检测仪上那条死鱼般的F级曲线，嘴角微挑：“F级？挺好，最适合藏牌。”
2. 【叮！检测到围观者集体轻视，爽点值+37。系统评价：蚊子腿也是肉。】
3. “系统，你管这叫界域行者？我怎么觉得更像异世界失足少女救助热线？”
4. 房间里残留着深渊玫瑰的冷香，契约印记烫得像刚从火里捞出来。
5. 莉莉娅懒洋洋地撑起下巴，暗紫色魔角还泛着余温：“小主人，醒得挺快。”
6. 苏念沉默两秒，扶着腰坐起来：“我现在严重怀疑，这职业不归教育局管。”
7. 【契约对象：莉莉娅。当前同步率：18%。系统建议：宿主下次救人前先热身。】
8. 外界以为苏念召唤失败，只有苏念知道，检测仪不是没测出来，是没资格测。
9. “你们笑得太早了。”苏念看着考场入口，语气轻得像在提醒天气。
10. 下一秒，深渊召唤阵在他脚下展开，笑声戛然而止。
11. 【姓名：叶昊。当前机缘：高考秘境圣痕残片。一句话人生：靠此逆袭，三年后成为苏念的强力竞争者。】
12. 苏念扫了一眼面板，笑了：“挺励志，可惜这一世版权归我。”
13. 莉莉娅的尾尖轻轻勾住他的手腕，声音软得不像深渊贵女：“主人，要我帮你把他们都弄哭吗？”
14. “别急。”苏念摩挲着契约印记，“解决麻烦也要讲究火候。”
15. 考务老师脸上的笑僵住了，因为那只所谓的召唤物，正用看蝼蚁的眼神看着全场。
16. 【叮！竞争者产生强烈嫉妒，剧情偏移值+100，爽点值+500。】
17. “不是哥们，你怎么还急了？你的机缘只是换了个更合适的主人而已。”
18. 苏念最喜欢这种场面：敌人底牌尽出，他连底牌叫什么都还没想好。
19. 莉莉娅在旁边轻笑：“人类真有趣，弱小的时候尤其会嘴硬。”
20. “纠正一下。”苏念抬手，精神控场瞬间铺开，“他们不是嘴硬，是没挨过打。”
21. 圣痕残纹亮起的那一刻，系统提示跳了出来：【检测到神明世界污染样本，建议宿主打包带走。】
22. 苏念挑眉：“污染样本？说得这么难听，这叫未来科研经费。”
23. 林清雪看着他身后的魅魔虚影，半晌才问：“苏念，你到底召唤了什么？”
24. 苏念想了想，认真回答：“一个比较费腰的战略级队友。”
25. 夜色落下时，黑契会的人终于找到了苏念。可惜，他们找来的不是猎物，是灾难本身。

## 15. 不可复制/不可越界清单

### 15.1 不可复制

- 不复制样本小说角色、势力、体质、法宝、功法、原句。
- 不复制样本小说完整桥段。
- 不复制样本小说固定开篇互动或固定章尾。
- 不把示例语句原样放进正文。

### 15.2 不可越界

- 不写露骨过程。
- 不写未成年亲密关系。
- 不写非自愿持续关系。
- 不把女主当宠物、资源容器或无人格奖励。
- 不把男主写成反派、恶人或无差别作恶的杀人狂。

### 15.3 当前项目硬边界

- 莉莉娅是成年魅魔。
- 青璃外表娇小但必须明确成年古剑灵。
- 白九璃是成年类人智慧狐女，不能动物化。
- 所有核心女主加入都必须保留清醒确认、自愿认可与契约成立逻辑。
