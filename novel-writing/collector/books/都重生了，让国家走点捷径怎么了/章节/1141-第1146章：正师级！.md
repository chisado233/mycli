# 第1146章：正师级！

看到周扬的态度如此的坚定，姜老满意的点了点头，而后说道：“你小子倒是个有血性，不过国与国之间的战争必须慎重对待，不能一冲动就开战了！”

　　这时，旁边的杨老将军也附和着说道：“从现在的局势来看，南猴那边明显是想要咱们动手，毕竟南猴军队以及相当一部分民众是亲咱们得，也包括他们党政军的相当一部分领导人。”

　　“如果我们真的动手了，那就给了对方清除异己并煽动百姓情绪的机会，影响还是非常大的！”

　　张老将军也说道：“这就是一个阴谋，逼迫我们动手的阴谋！”

　　然而周扬却淡淡地说道：“是阴谋又如何，不管那帮猴子怎么想的，这都不是他们入侵我国领土，打伤我国边民，扣押我国工作人员的理由！”

　　“而且我也不认为咱们的克制就能让对方回心转意，我们一次次的忍让，最终只会被他们当做是软弱可欺，换来的只可能是变本加厉的挑衅！”

　　周扬当然知道这次流血事件是南猴当局蓄谋已久的阴谋，他们这样做，除了释放亲苏的信号，逼迫其国内部各派系站队外，更重要的就是要通过这种手段逼迫我们动手，主动挑起战争。

　　而南猴那边之所以急着要挑起两国的战争，原因也不仅仅只是想找一个实力更强的主子，更为重要的是南猴这个国家有野心，他们称霸甚至于统一中南半岛。

　　想要实现这个野心，就必须要消除我国在中南半岛的影响力，同时也需要大量的资金支持。

　　而眼下南猴国内最有钱的群体就是华人，因此这几年他们一直在煽动民众情绪，借反/华“”之名剥夺富人阶层财产。

　　但凡被越南当局盯上的企业或个人财产，要么无偿移交给越南当局，要么被扣上“华人资产”的标签，然后被强制充公。

　　为避免日后遭受报复，南猴当局干脆一不做二不休，将以华侨为主体的大批在越长期生活的普通民众驱逐出境。

　　恶魔的屠刀已经高高举起，现在说什么“放下屠刀”已经没有意义了，唯有重拳出击，才能让其“回头是岸”。

　　姜老看了看周扬，而后说道：“你说的有道理，二号首长已经做好了充分的准备，我们或许还会做最后的努力，但这并不妨碍我们着手战争的准备！”

　　“对，先把刀磨快了，到时候要不要剁下去，凭我们的心情！”周扬道。

　　姜老点了点头说道：“我们今天把你叫过来，主要是有这么几件事情和你沟通一下！”

　　“您说！”

　　“鉴于你们所在炮瞄雷达操作员培训方面经验丰富，成绩卓著，军委那边决定再从军中抽调200名精锐，交由你们进行培训，有问题吗？”姜老沉声问道。

　　“报告，没有问题，保证完成任务！”周扬大声说道。

　　“好！”

　　接着姜老继续说道：“再就是你们那边的几个厂子要开足马力搞生产，现在我就正式通知你，你们厂不管生产多少炮瞄雷达和机载雷达，军方都会照单全收，而且部里会保证你们70%的采购费用优先支付！”

　　“是！”

　　随后姜老话音一转，再次说道：“除了上面这些事情外，我们找你过来还有一个非常重要，且非你莫属的任务要交给你！”

　　听到这话，周扬的心里顿时“咯噔”一下，顿时生出一股不祥的预感。

　　毕竟领导给戴的高帽子，那可是不好戴，往往下面全都是坑啊！

　　“首长，您有话就直说，这“非你莫属”四个字听的我心里直哆嗦！”周扬道。

　　姜老瞪了他一眼，然后说道：“出息，还没把任务交给你呢，咱就怂了，这可不是你的做事风格啊！”

　　周扬咧嘴一笑说道：“我倒是也想和咱野战部队的战士们那样，听到任务就嗷嗷叫，但我知道您老这任务肯定不好完成，可不敢胡乱应承！”

　　姜老顿时沉默了，而后看了看他说道：“确实不好完成，所以也就没有直接给你下书面命令，这事儿你尽量做，完成不了我们也不会怪你！”

　　“到底啥事儿？”周扬略带一丝好奇的问道。

　　姜老没有说话，而是从办公桌的抽屉里拿出一份文件，递给了周扬。

　　周扬也不矫情，结果文件立即看了起来。

　　只是当他看到文件的封面后，整个人突然愣住了。

　　这赫然是一份军工企业运营情况的报告，仅凭这个标题他就知道姜老想要他干吗了！

　　他没有看这份报告，而是轻轻放到了面前的桌子上，随后看着姜老说道：“看来是军工产业的日子不好过，您老是想让我给他们找一条出路？”

　　姜老有些赞叹的说道：“你小子是真的聪明，都没看文件里面的内容就知道我要说什么了！”

　　周扬摇了摇头说道：“这还用看，就今年这大环境，军工企业肯定是哀鸿遍野。能勉强维持的恐怕都没几家，欠债度日的恐怕占了绝大多数，日子好过才有鬼了！”

　　姜老叹了口气说道：“你说的没错，这段时间我们对全国包括核工业、航空工业、电子工业、兵器工业、船舶工业、航天工业等部门在内的上万家企业单位进行了调查，发现只有30%的企业勉强予以维持，剩下的70%全都陷入了困境，甚至于还有相当一部分企业连工资都发不出来了！”

　　“我们将这事儿向上面进行了汇报，结果得到的反馈是没钱，要我们部里自己解决这事儿。但你也知道，部里自己变不来钱，只能问问你这个搞钱的专家了！”

　　“您老说笑了，我啥时候成了搞钱的专家了！”周扬有些无奈的说道。

　　“你小子就别谦虚了，你一个小破计划，横竖也就几个人，就能给国家搞来那么多钱，你要还不是这方面的专家，谁敢说自己是专家？”

　　接着姜老话音一转，再次说道：“而且你提出的那个军转民的方案，正在由各个试点单位进行试验，就目前反馈回来的效果而言，还算不错。”

　　“只可惜，这个方案并不适合所有的单位，你就辛苦一点，再给那些纯军工单位找一条出路！”

　　周扬瞬间沉默了，这个任务着实不容易啊。

　　要知道现在全国大大小小的军工单位上万家，从业人员几百万，姜老就是让自己给这几百万人找饭吃，难度可想而知。

　　不过想到前段时间策划的那个军工出口的方案，周扬忍不住心里一动。

　　这时，一旁的杨老开口说道：“周扬同志，你也不要有心理负担，这事儿的难度我们也知道，你只要尽力而为就行了！”

　　张老将军也说道：“能干成啥样就干成啥样，干好了有奖励，干不好没惩罚！”

　　周扬摇了摇头说道：“首长，奖励什么的我倒是不在乎...”

　　话音未落，就听姜老突然说道：“你小子真不在乎奖励？”

　　不等周扬回答，就听姜老再次说道：“我们正讨论怎么嘉奖你小子呢，你要是真不在乎的话，那这事儿就算了！”

　　周扬急忙说道：“别介啊，哪有您老这样式的呢！”

　　姜老笑了笑说道：“这会儿不装清高大气了？”

　　“不装了...不装了！”周扬道。

　　随后，姜老脸上的笑意一收，略带一丝严肃的说道：“经部里研究决定，对山鹰直升机项目的有功人员进行嘉奖，相关嘉奖方案已经定了下来！”

　　“对于你本人，部里打算正式晋升你为正师级干部，继续担任八宝梁研究所所长兼888厂厂长，嘉奖令不日将会下发到你们所。”

　　“是，多谢首长！”

　　姜老看了看周扬，而后说道：“这么年轻的正师级干部，和平时期独一份儿啊！”

　　“谁说不是，我记得也就红军时期咱有过这么年轻的师级干部，其他时候真没出过！”杨老道。

　　这时，姜老略带一丝无奈的说道：“都怪姓云的那个老东西，要不是他悄悄咪咪的把这小子提成了正厅级干部，咱们还能拖一拖！”

　　周扬：@#￥%……&

　　姜老看了看周扬，再次说道：“你小子太年轻了，升的慢一点对你也是一种保护。但既然卫生部那边开了头，我们也就顺势跟进就行了！”

　　“你是真正的天才，我们几个老家伙以及更高层的首长，都希望你能再接再厉，为我们的国防建设做出更大的贡献！”

　　“是！”

　　随后周扬话音一转，再次说道：“首长，您刚才说的那事儿虽然确实有些难度，但也不是完成不了！”

　　姜老急忙问道：“你小子有办法了？”

　　“我点思路，但是我回去好好思考一下，等理顺了再向您汇报！”周扬道。

　　“哈哈哈，我就知道这小子鬼点子多，这事儿就交给你了！”姜老高兴的说道。
