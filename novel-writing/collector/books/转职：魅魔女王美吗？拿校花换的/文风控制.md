# 文风控制.md

> 来源：基于本书前 50 章样本进行文风提取与整理。  
> 用途：供后续小说写作 agent 学习“文风表现方式”，包括节奏、句式、对白、系统提示、爽点组织与暧昧氛围处理。  
> 硬性原则：**只学习文风，不学习角色设定、职业设定、世界观设定、人物关系、剧情桥段或主角阵营。苏念不是反派，是当前项目里的正向主角。**

---

## 1. 文风总则

### 1.1 一句话定义

极短段落、强系统反馈、现代吐槽、女主反差、战斗快切的移动端爽文风。

### 1.2 学习范围

只学习：

- 短段落节奏。
- 章首快速入戏。
- 系统提示的即时反馈感。
- 男主现代口语吐槽。
- 战斗“起手—爆发—结果”的快切写法。
- 女主强身份与私下互动反差。
- 打脸后围观者震惊、敌人破防、系统结算的爽点闭环。

不学习：

- 样本书角色名、势力名、职业名、能力名。
- 样本书主角阵营与人物关系。
- 样本书具体世界观和剧情桥段。
- 样本书专有称谓、固定台词或原句。

### 1.3 当前项目主角定位

苏念是正向主角：

- 救人、契约、成长、守护、反击。
- 有底线，敢出手，但不主动欺压无辜。
- 可坏笑、可嘴毒、可抢节奏，但不能写成反派、恶人或无底线掠夺者。

## 2. 移动端段落规则

### 2.1 段落长度

- 每段 1-3 句为主。
- 关键句可单独成段。
- 战斗、系统提示、心理吐槽之间要留空行，形成滑屏节奏。

### 2.2 段落排列

推荐节奏：

```text
动作/画面 → 对方反应 → 系统提示/男主吐槽 → 爽点推进
```

示例方向：

```text
法阵亮起。

全场安静。

苏念看着面板上的同步率，忽然觉得刚才那句“F级召唤师”还挺有节目效果。
```

### 2.3 禁止长说明

避免开篇大段世界观说明。设定要拆进：

- 系统提示。
- 考场公告。
- 旁人嘲讽。
- 女主对话。
- 男主吐槽。

---

## 3. 叙事视角与镜头

### 3.1 视角

以第三人称限知为主，紧贴苏念。

可以短暂切到围观者、敌人、导师视角，用来侧面烘托苏念的反转和压制感。

### 3.2 镜头写法

#### 章首镜头

章首应直接切进高张力画面：

- 系统提示跳出。
- 召唤阵展开。
- 女主契约印记发烫。
- 敌人嘲讽刚落下，反转马上出现。
- 异界呼唤信号突然接入。

#### 战斗镜头

不写复杂拆招，写结果压迫：

```text
敌人抬手。
莉莉娅抬眼。
下一秒，整片区域的精神波动被强行按停。
```

#### 反差镜头

外界强大，私下亲近：

```text
在人前，她是令一整个小队不敢抬头的异界女王。
回到契约空间后，她只是懒洋洋地靠在苏念身边，问他这次报酬算不算加班费。
```

## 4. 男主心态与吐槽规则

### 4.1 核心心态

苏念的心态应是：

- 清醒。
- 有底线。
- 反应快。
- 信息差强。
- 面对嘲讽不急着辩解，更喜欢用结果让对方闭嘴。

### 4.2 吐槽方式

多用现代口语拆解荒诞设定：

- 把高危契约说成“跨界救援”。
- 把系统任务说成“加班”。
- 把敌人挑衅说成“节目效果”。
- 把低评级说成“隐藏身份保护色”。

可用口吻：

```text
不是哥们。
你管这叫召唤师？
系统，你这个职业介绍是不是少写了几行免责声明？
```

### 4.3 正向边界

可以写苏念反击狠、打脸快、下手果断。

不能写他：

- 欺压弱小。
- 强迫女主。
- 为恶而恶。
- 主动伤害无辜。
- 将女主当无人格奖励。

---

## 5. 系统提示与数值反馈规则

### 5.1 格式

系统提示要短、硬、即时：

```text
【系统提示：契约同步率提升至15%。】
【获得联动技能：精神控场。】
【剧情偏移值+100，爽点值+300。】
```

### 5.2 使用时机

以下节点必须优先考虑系统反馈：

- 契约成立。
- 同步率提升。
- 联动技能解锁。
- 敌人破防。
- 机缘入手。
- 考核排名变化。
- 新异界呼唤出现。

### 5.3 系统语气

系统可以偏冷静，也可以偶尔吐槽。

示例方向：

```text
【系统提示：检测到宿主正在用F级身份制造过量心理阴影。】
【系统建议：下次救援前，请确认灵能储备与腰部承受能力。】
```

## 6. 女主描写与暧昧边界

### 6.1 描写重点

女主描写学习的是“反差与氛围”，不是样本设定。

重点写：

- 高位气场。
- 战斗压迫感。
- 与苏念的契约默契。
- 私下的调侃、吃醋、亲近。
- 称呼变化、契约印记、冷香、衣角、眼神等侧面细节。

### 6.2 合规边界

必须遵守：

- 成年。
- 自愿。
- 类人/智慧对象。
- 清醒确认。
- 亲密只写隐晦氛围，不写过程。

可写：

- 事后房间气息。
- 契约印记余温。
- 灵能透支。
- 女主称呼微妙变化。
- 旁人误会或调侃。

不可写：

- 露骨过程。
- 非自愿亲密。
- 未成年暗示。
- 动物化性内容。

## 7. 爽点结构模板

### 7.1 低评反转

```text
外界低评苏念 → 苏念不解释 → 系统/契约女主触发 → 实力碾压 → 全场重新认识他
```

### 7.2 救援契约

```text
异界女性强者陷入危机 → 呼唤被苏念听见 → 苏念介入救场 → 清醒确认 → 自愿契约 → 系统奖励
```

### 7.3 抢先机缘

```text
系统显示关键机缘 → 苏念提前判断 → 抢先入场 → 获得收益 → 竞争者迟到破防
```

### 7.4 打脸闭环

```text
嘲讽 → 反转 → 震惊 → 系统结算 → 苏念一句轻描淡写吐槽
```

---

## 8. 敌对角色/竞争者吃瘪模板

### 8.1 认知偏差

敌人基于错误前提行动：

- 以为苏念只是普通召唤师。
- 以为召唤物只是低阶工具。
- 以为评级等于实力。

### 8.2 破防过程

写清三个阶段：

```text
自信 → 疑惑 → 恐惧
```

### 8.3 收尾方式

苏念不长篇说教。

一句话、一个眼神、一次系统结算即可。

---

## 9. 战斗与打脸节奏

### 9.1 战斗结构

```text
敌人叫嚣 → 苏念判断 → 女主/联动技能出手 → 一招见结果 → 系统提示 → 围观者震惊
```

### 9.2 战斗语言

多写：

- 压迫感。
- 声音消失。
- 法阵展开。
- 灵能数值跳动。
- 对手表情变化。

少写：

- 冗长招式名。
- 回合制解释。
- 大段战斗原理说明。

### 9.3 主角参与感

苏念即使是召唤师，也不能只旁观。

他要通过：

- 契约同步。
- 战术判断。
- 技能联动。
- 关键补刀。
- 指挥女主配合。

来体现主角存在感。

## 10. 章首/章尾钩子规则

### 10.1 章首

章首 300 字内进入一个明确钩子：

- 召唤失败？其实是检测仪识别不了。
- 系统提示新呼唤。
- 女主契约后醒来。
- 考核现场有人嘲讽。
- 新敌人盯上苏念。

### 10.2 章尾

章尾要给下一章期待：

- 新异界呼唤。
- 新女主危机。
- 新技能解锁。
- 竞争者发现机缘消失。
- 黑契会/学院/深渊势力开始行动。

不要使用样本书固定章尾句式。

## 11. 可用现代梗与禁用项

### 11.1 可用现代梗

少量使用，避免过密：

- 加班费。
- 老板。
- 降维打击。
- CPU。
- 售后服务。
- 隐藏款职业。
- 节目效果。
- 版本答案。

### 11.2 禁用项

- 禁用样本书角色名、势力名、能力名、职业名。
- 禁用样本书具体桥段。
- 禁用样本书固定台词与长句。
- 禁用把苏念改成样本主角定位。
- 禁用把当前项目世界观改成样本世界观。

## 12. 写作检查清单

每章完成后检查：

- [ ] 是否只学习文风，没有引入样本设定？
- [ ] 苏念是否仍是正向主角？
- [ ] 段落是否短，移动端节奏是否顺？
- [ ] 章首是否快速入戏？
- [ ] 是否有系统即时反馈？
- [ ] 是否有苏念的现代吐槽？
- [ ] 女主描写是否强调强身份与反差？
- [ ] 亲密描写是否只停留在隐晦氛围？
- [ ] 打脸是否完成“嘲讽—反转—震惊—结算”？
- [ ] 章尾是否有下一章钩子？

## 13. 原创文风实例语句

> 以下句子仅供句式、节奏、氛围参考，严禁正文照抄。

1. 苏念看着面板上那个刺眼的F级评价，沉默了两秒。
2. 然后他笑了。
3. 检测仪看不懂，不代表他真弱。
4. 【系统提示：检测到未知界域呼唤，坐标锁定中。】
5. “不是哥们，刚觉醒就跨界出差？这职业包五险一金吗？”
6. 契约印记在手腕上微微发烫，像有一枚小型星核埋进了皮肤。
7. 莉莉娅的声音从另一侧传来，慵懒里带着一点危险：“召唤师，需要我替你清场吗？”
8. 苏念看了一眼还在叫嚣的对手，认真想了想：“别全清，留一个负责震惊。”
9. 法阵展开时，整个考场的风都停了一瞬。
10. 刚才还在笑的人，此刻像被人按下了静音键。
11. 【契约同步率提升至18%，获得联动技能：精神控场。】
12. 苏念活动了一下手腕：“你看，我说召唤师也能讲道理。”
13. 只不过他的道理，通常需要对方躺着听。
14. 莉莉娅踩着暗紫色灵光落下，战袍边缘掠过地面，像一场安静燃烧的夜色。
15. “老板。”她微微歪头，“这次算救援，还是算加班？”
16. 苏念看着系统奖励，语气平静：“算投资。”
17. 敌人的脸色从轻蔑变成茫然，又从茫然变成恐惧。
18. 这个过程很短。
19. 短到苏念还没来得及想第二句台词。
20. 【剧情偏移值+100，爽点值+300。】
21. 林清雪看着他身后的异界虚影，终于忍不住问：“苏念，你到底召唤了什么？”
22. 苏念沉吟片刻：“一个售后成本比较高的队友。”
23. 夜色落下时，新的呼唤信号在面板边缘亮起。
24. 这一次，声音来自更远的世界。
25. 苏念看着那行提示，轻轻叹了口气：“看来今晚又不用睡了。”

## 14. 不可复制/不可越界清单

### 14.1 不可复制

- 不复制样本角色。
- 不复制样本势力。
- 不复制样本职业、能力、世界观。
- 不复制样本剧情桥段。
- 不复制样本原句或固定台词。

### 14.2 不可越界

- 不写露骨过程。
- 不写未成年亲密关系。
- 不写非自愿契约或非自愿亲密。
- 不将女主工具化、宠物化或奖励化。
- 不将苏念写成反派、恶人或无差别作恶者。

### 14.3 当前项目底线

- 莉莉娅等核心女主必须是成年智慧个体。
- 契约必须有清醒确认与自愿认可。
- 暧昧服务人物关系与爽文张力，不服务露骨描写。
