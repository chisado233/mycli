# 文学风格收集素材规范

本目录用于收集、拆解和沉淀可复用的文学风格素材，包括叙述语气、句式节奏、意象系统、修辞手法、人物对白风格、心理描写方式、场景描写质感、不同题材的文风模板等。

这里不是简单摘抄“好句子”，而是要说明一种风格为什么有效、适合什么情绪、如何被后续 AI 模仿和迁移。

## 文件命名

```text
序号+名称.md
```

示例：

```text
001+冷峻克制型叙述.md
002+古典华丽型宫廷描写.md
003+诡异潮湿的南方小镇风格.md
004+暧昧拉扯对白风格.md
```

## 记录要求

### 1. 必须说明风格机制

不要只说“文风很美”“很有氛围”。要说明它靠什么产生效果：

- 是短句制造压迫？
- 是长句制造流动感？
- 是冷色意象制造疏离？
- 是反复句式制造仪式感？
- 是口语和停顿制造真实感？
- 是细节堆叠制造沉浸？

### 2. 必须有可模仿示例

每条风格素材至少提供一个原创示例片段，供后续 AI 学习语感。

示例：

```markdown
雨没有变大，只是更密了。檐下的灯被风压得一低，照见他袖口一点暗红。她忽然明白，那不是雨水。
```

### 3. 必须说明适用边界

风格不是越强越好。每条素材应说明：

- 适合什么题材？
- 适合什么人物？
- 适合什么场景？
- 不适合什么场景？
- 用多了会产生什么问题？
