# 世界设定素材规范

本目录用于记录世界类型、地理、历史、势力、种族、国家、宗门、规则、经济、宗教、法律、秘境、禁地等设定。

## 文件命名

```text
序号+名称.md
```

示例：`001+三洲九域世界格局.md`

```yaml
---
name: 三洲九域世界格局
description: 一个由三大洲和九个修炼区域构成的玄幻世界基础格局。

---
```
