# 场景素材规范

本目录用于记录可直接转化为小说桥段的场景素材，包括战斗、日常互动、亲密关系、情绪推进、氛围描写、对话、探索、追逐、灾难等。

## 文件命名

```text
序号+名称.md
```

示例：`001+雨夜刺杀.md`

## YAML 模板

```yaml
---
name: 雨夜刺杀
description: 主角在雨夜遭遇伏击，借环境完成反杀并展示隐藏实力。
source: 原创
tag:
  - 场景
  - 战斗场景
  - 雨夜
  - 反杀
  - 高燃
  - 展示实力
---
```
