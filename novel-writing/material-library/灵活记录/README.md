# 灵活记录素材规范

本目录用于记录暂时无法归类但有价值的信息，如灵感、金句、片段、对话、氛围词、名词库、人名、地名、招式名、组织名、章节名、参考资料等。

## 文件命名

```text
序号+名称.md
```

示例：`001+适合反派登场的金句.md`

## YAML 模板

```yaml
---
name: 适合反派登场的金句
description: 一组适合强大反派初次登场时使用的压迫感台词。
source: 原创

  - 灵活记录
  - 金句
  - 反派
  - 压迫
  - 待整理
---
```
