# 力量体系素材规范

本目录用于记录世界通用力量体系与角色个人力量体系，包括修炼、魔法、异能、科技、血脉、职业、规则、境界、技能树、代价机制等。

## 文件命名

```text
序号+名称.md
```

示例：`001+九境修炼体系.md`

## YAML 模板

```yaml
---
name: 九境修炼体系
description: 从凡体到天命的九个境界，每一境界对应不同资源、寿命和战斗能力。
```
