# 爽点素材规范

本目录用于记录让读者产生满足感、释放感、反转感的爽点素材，如打脸、反杀、逆袭、身份揭露、碾压、突破、大佬撑腰等。

## 文件命名

```text
序号+名称.md
```

示例：`001+退婚当众打脸.md`

## YAML 模板

```yaml
---
name: 退婚当众打脸
description: 主角在被退婚羞辱后揭露实力，在群体见证下完成反转。
source: 原创

  - 爽点
  - 打脸
  - 当众打脸
  - 群体见证
  - 先抑后扬
  - 爽快
---
```
