# 伏笔素材规范

本目录用于记录可埋设、发展、回收的伏笔素材，包括身份、血脉、身世、道具、功法、预言、地图、阴谋、世界真相等。

## 文件命名

```text
序号+名称.md
```

示例：`001+断剑真实来历.md`

## YAML 模板

```yaml
---
name: 断剑真实来历
description: 主角早期得到的断剑实际是上古神器碎片，可在后期回收为主线关键道具。
source: 原创
tag:
  - 伏笔
  - 道具伏笔
  - 已埋设
  - 物品暗示
  - 主线伏笔
---
```
