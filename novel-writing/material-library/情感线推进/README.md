# 情感线推进素材规范

本目录用于记录角色关系与感情线如何一步步推进，包括初遇、试探、心动、误会、吃醋、信任建立、告白、关系确认、亲密升级、危机、分离、重逢、和解、共同成长等。

## 文件命名

```text
序号+名称.md
```

示例：`001+从互相试探到信任建立.md`

```yaml

```
